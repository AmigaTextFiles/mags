Comment écouter ces musiques ?
Deux possibilités s'offrent à vous : si votre navigateur est bien configuré pour interpréter les fichiers aux formats "modules" (.XM, .MOD, etc...), cliquez simplement sur les titres des musiques pour les jouer. L'autre solution est de prendre votre lecteur audio favori et de simplement charger les musiques. Pour une lecture optimale, nous recommandons les lecteurs suivants : EaglePlayer 2.02 (AmigaOS), DeliPlayer (Windows) et XMMS avec son plugin UAEDE (Linux).

Exemple de configuration d'un navigateur pour jouer des modules
L'exemple qui suit s'appui sur le navigateur AWeb et le lecteur audio EaglePlayer.

- Procédez à une installation des deux logiciels.
- Ouvrez Aweb, allez dans son menu préférences "Browser Prefs" ou "Préférences du navigateur" si vous êtes en Français.
- Cliquez sur l'onglet "Viewers" ou "Visualisateurs" ou encore "Types MIME".
- Si le type MIME AUDIO/X-MOD n'existe pas dans votre liste, cliquez sur "Ajouter".
- Ensuite, remplissez les cases comme indiqué :

Type : AUDIO/X-MOD
Extensions : mod med mmd s3m xm dbm
Action : Programme Externe
Nom : (entrez ici le chemin ou ce trouve EaglePlayer) ex: Work:Utilitaires/Musique/EaglePlayer/Eagleplayer
Paramètres : loadmodule %f

Voilà, c'est terminé. Cliquez sur "Sauver" et vous pourrez à présent jouer des modules de musiques par l'intermédiaire de votre navigateur.
