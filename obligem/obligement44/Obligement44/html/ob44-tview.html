Labo : TView par Adelphe Bachelet



Note : retrouvez cet article sur le site bachelet.adelphe.free.fr.

 Demandez le programme... Demandez...

Si comme moi vous n'achetez pas de programmes télé et que vous en avez marre de rater votre série préférée ou votre film, alors TView est fait pour vous.

TView est un petit programme sous MUI (l'archive ne pèse que 79 Ko) qui vous permet de connaître les programmes télé via Internet. Bien sûr, si vous n'avez pas encore Internet, que vous vivez reclus aux fins fonds de l'Himalaya, là, je ne peux rien faire pour vous. L'auteur de ce logiciel est Fredrik Söderberg, il est disponible sur Aminet ou bien sur son site dédié.

Signalons que TView est localisé dans sept langues (Allemand, Français, Polonais, etc.) et qu'il nécéssite MUI 3.8 et la NList.mcc 0.89.

Son installation est simple et rapide, il n'y a pas d'installeur. Il suffit de décompressez l'archive, copier le répertoire TView ou vous voulez, copiez les scripts ARexx dans le répertoire Rexx et voilà, le tour est joué.

Les serveurs

Ensuite, il ne vous reste plus qu'à aller dans les préférences pour régler la langue, les chaines TV que vous désirez (hertzienne, C+, cable, etc.) et le serveur que vous allez utiliser.

Par exemple, pour les Français, le serveur est www.eurotv.com qui est d'ailleurs utilisé pour d'autres pays (Royaume-Uni, Allemagne, Belgique, Suisse, Italie,...). Il existe d'autres serveurs comme www.dagenstv.com pour la Suède et la Finlande, et www.aftenposten.no pour la Norvège et le Danemark.



Demandez le programme


A l'usage, TView est des plus simples : la première fois, il suffit d'aller dans le menu et sélectionner l'action "Récupérer les données". Ensuite, dans la fenêtre principale, un simple clic sur l'emission désirée et TView vous donne des informations la concernant (réalisateur, résumé...). Vous pouvez vous déplacer dans la journée en déplaçant le glisseur des heures. TView possède aussi une fonction recherche (pour les programmes) et alarme (elle vous prévient quelques minutes avant le début de vos émissions favorites).

Pour les télémaniacs, TView deviendra rapidement le Télé 7 Jours des amigaïstes ;-).
