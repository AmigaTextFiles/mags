FTP mit Amiga

Hallo Leute,

Ich habe für alle Leute die keine Ahnung von FTP, Wget... haben, eine gute Lösung gefunden, so dass man es in aller Ruhe erstmal auf seinem Rechner ausprobieren kann!

Dazu wird benötigt zB.:
  • Der FTP-Server -
    • rc-ftpd.lha (Aminet)
    • rc-ftpd_cpu.lha (Aminet)
    • rc-ftpd_loc.lha (Aminet)

  • ein FTP-Client zB. -
    • AmiTradeCenter (Aminet)
    • oder
    • AM-FTP, M-FTP, Ami-FTP...

  • und Wget (Aminet)

Zunächst erstmal die Programme installieren :-)

Danach Miami(DX), Genesis starten, anschließend den FTP-Server starten und konfigurieren (Leider ist die Dok von dem Server nur auf Englisch!) wenn man das geschafft hat, kann man sich um das AmiTradeCenter kümmern und konfigurieren, zB. einen neuen Server Eintrag erstellen wie zB. 'Test' unter Kommentar, bei Login/PW test/test, und bei Adresse 127.0.0.1 Port kann auf 21 bleiben!

Danach kann man auf Verbinden Klicken und man sollte sehen können, dass er sich eingeloggt und danach kann man dann ein bisschen experimentieren mit Daten Downloden bis man es im Schlaf kann. ;-))) Zum Schluss kann man noch auf Trennen klicken und anschließend ATC beenden!

Danach kann man eine Shell öffnen und es mal mit diesen Zeilen probieren,
  • Stack 200000
  • CD RAM:
  • wget -NS -m -A.readme ftp://test:test@127.0.0.1/cd0/Aminet/
und dann kann man es mit allen anderen Optionen von Wget ausprobieren!

Ich wünsche viel Spaß beim Ausprobieren!
