7. CDCat 3.0 mit PowerPC-Unterstützung und neuen Funktionen

»CDCat Release 3:

CDCat, der leistungsfähige und einfach zu bedienende CDROM- Katalogisierer mit MUI-Oberfläche, liegt nach langer Entwicklungszeit nun in einer neuen Version vor: CDCat Release 3 (v3.0).

Bei der Weiterentwicklung wurde viel Zeit investiert, um die Wünsche vieler User nach noch mehr Flexibilität zu berücksichtigen.

So erhalten die User mit CDCat Release 3 eine optimale Kontrolle über die Scankataloge durch den Katalog-Manager. Auch kann CDCat 3 jetzt nach Verzeichnissen suchen. Damit die Rechenpower weiterhin voll ausgenutzt werden kann, bietet CDCat Release 3 nun PowerPC-Suchroutinen, wobei die Routinen für den MC680x0 natürlich weiterhin voll funktionieren.

Auch können alte Kataloge und Configfiles problemlos übernommen werden.

Features von CDCat sind u.a.:

  • geeignet zum Katalogisieren sämtlicher CDs, Festplatten und PD- Disketten
  • superschneller Aminet-Modus, der Aminet-CDs in wenigen Sekunden scannt und dabei Alter, Größe und Beschreibung berücksichtigt.
  • schnelle und flexible Suchfunktion (PowerPC; 68k: 85% Inline- Assembler!!)
  • Direktes Durchsuchen (ohne Scankatalog) von Datenträgern oder Verzeichnissen möglich. So können Sie auch auf einfache Weise verlorene Dateien auf Ihrer Festplatte wiederfinden.
  • Browser-Funktion, über die der Inhalt einzelner CDs verzeichnisweise wie in einem DirUtil angezeigt werden kann, wobei auch die oben genannten Aminet-Erweiterungen angezeigt werden, so dass man sich nun nicht mehr durch die Index-Guides auf den Aminet-CDs quälen muss.

    => Durchstöbern der CDs möglich, ohne dass diese eingelegt sind !!!

  • 15 frei konfigurierbare User-Knöpfe, über die direkt aus CDCat heraus Archive entpackt, Bilder angezeigt, Musikmodule abgespielt etc. werden können.

Neu hinzugekommen in Release 3 u.a.:

  • PowerPC Plugin um die Suche zu beschleunigen (Die alten Routinen funktionieren natürlich immer noch!)
  • Katalog-Manager um Kataloginhalte neu anzuordnen. Einzelne Disks können aus Katalogen entfernt werden, in andere Kataloge kopiert werden etc.
  • Es kann nun auch nach Verzeichnissen gesucht werden.
  • Filterfunktion für die Suchergebnisliste (über Menü erreichbar)
  • Fortschrittsanzeige, wenn über die User-Knöpfe Befehle ausgeführt werden
  • Möglichkeit, Programme im asynchronen Modus zu starten. So muß nicht mehr "run" vorangestellt werden, was Probleme mit dem Ausgabe-Shell- Fenster verursachte.
  • Die Suchergebnisse werden mit der NList MUI-Klasse (von Gilles Masson) dargestellt, so daß die Liste nun z.B. auch einen horizontalen Scroller besitzt.
  • div. Bugfixes

Registrierung:

Die Registrierungsgebühr beträgt 40 DM bzw. 20 Euro. Dann erhalten die User ein Keyfile, um alle Funktionen uneingeschränkt nutzen zu können.

Upgrades:

In diese Version habe ich sehr viel Arbeit investiert, aber da sich die Anzahl der Amiga-User wahrscheinlich nicht unbedingt vergrößert hat, benötige ich auch die Unterstützung der bereits registrierten User von CDCat Release 1 und 2.

Die normale Registrierungsgebühr wurde auf 40 DM erhöht, und registrierter User von CDCat Release 1 und 2 können Ihr Upgrade für 20 DM erhalten.

User, die sich nach dem 1. November 1998 registrieren ließen, erhalten Ihr Update für 15 DM.

Ein Test der Version v1.2 befindet sich in der AmigaPlus 12/97, ein Test der v2.2 in der AmigaPlus 9/98.

Nähere Informationen auf der CDCat-Supportseite http://www.amigaworld.com/support/cdcat Hier gibt es immer die aktuellste Version zum Download, außerdem ist CDCat im Aminet verfügbar, util/misc, CDCat30.lha
