1. Zwei neue WWW-Browser befinden sich in Entwicklung

Zwei neue WWW-Browser sind angekündigt:

  • 'Dry Ice' wird in C++ geschrieben und soll folgende Features bieten:
    • vollständigen HTML 4.0-Support
    • JavaScript 1.3
    • "gewöhnliche AmigaOS-Dinge", wie ARexx, AmigaGuide, etc.
    • Standard for PlugIns

    Möglicher Erscheinungstermin ist der Spätsommer. Weitere Informationen sind in Kürze auf http://www.scifi1.demon.co.uk/wv.html verfügbar.

  • Das Projekt "AmiExplorer" befindet sich noch im Anfangsstadium, so dass nicht garantiert werden kann, dass alle Features implementiert werden. Der Programmierer versucht aber, seine wahrhaft hochgesteckten Ziele zu verwirklichen.

    AmiExplorer soll in einer Light- und einer Profiversion erscheinen, wobei die Lightversion etwa 20 bis 25 Mark kosten und folgende Features beinhalten soll:

  • HTTP
  • FTP
  • E-Mail
  • Javascript
  • Java

Die Profi-Version soll etwa 45 bis 65 Mark kosten und zusätzlich bieten:

  • HTML-Editor
  • Javascript-Editor
  • Java-Editor
  • RealAudio
  • RealVideo
  • ICQ
  • IRC
  • Telnet
  • Gopher

Beide Versionen sollen alle aktuellen HTML-Sprachen /-Dialekte (HTML4.0/MathML/DOM/CSS1/CSS2/SMIL/DSIG/XML) beherrschen.

Ein Erscheinungstermin ist natürlich noch nicht bekannt, aber AMIGA aktuell wird euch exklusiv auf dem Laufenden halten.
