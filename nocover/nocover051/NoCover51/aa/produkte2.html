2. Ankündigung: SecureKey-Datatype v1.0

»TITLE

   SecureKey Datatype

VERSION

   1.0

RELEASE DATE

   4 April 1998

AUTHOR

   Staffan Palmroos (crush@lysator.liu.se)
   (http://www.lysator.liu.se/~crush)

DESCRIPTION

  SecureKey is a system for creating keyfiles protected by the RSA
public key encryption algorithm. The properties of this algorithm
makes it possible for anyone to view the contents of a keyfile without
being able to create fake keys. You could say that the keyfiles are
'write-protected', or 'read-only'.

  This package is a subset of the SecureKey package, containing only
the datatype parts. It is meant for non-programmers who just want to
use the datatype for examining the keyfiles.

REQUIREMENTS

   * All Amigas with AmigaOS 3.0 (V38) or higher.
   * 68000 processor.

AVAILABILITY

   The SecureKey datatype can be downloaded from:
