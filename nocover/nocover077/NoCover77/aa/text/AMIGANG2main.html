2. Neue Informationen von AMIGA-Präsident Bill McEwen

Vor kurzem fand die amerikanische Amiga-Messe AmiWest 2000 statt. Unter 
http://www.ann.lu/detail.cgi?category=news&file=964934374.msg sind 
zahlreiche von AMIGA-Präsident Bill McEwen verbreitete Informationen zu 
finden, darunter folgende bislang nicht bekannte:

* Der Amiga One basiere auf einem Amiga-eigenen Motherboard-Design unter    
  Verwendung von Nicht-Standard-Komponenten. AMIGA errichte eine von Dean   
  Brown (DKB) geleitete Abteilung, um Referenz-Hardwaredesigns für          
  verschiedene Geräte zu entwickeln, die dann Drittanbietern zur Produktion  
  angeboten würden.

* Das Amiga SDK (Software Developer's Kit) verkaufe sich besser als         
  erwartet. AMIGA stehe in Kontakt mit zahlreichen namhaften Consumer-      
  Elektronik- und Software-Unternehmen, inklusive allen großen              
  Spielefirmen. Viele besäßen das SDK bereits.

* Noch in diesem Monat solle eine unter Windows lauffähige Version des SDK  
  erscheinen.

* Das SDK werde ab nächster Woche auch von Redhat vertrieben.

* Sun habe mit der Nutzung der Amiga-Software begonnen, um Java zu          
  demonstrieren.

* Es befänden sich Betaversionen der Amiga-Betriebsumgebung vor der         
  Veröffentlichung.

* Paul Nolan (Photogenics) arbeite auf vertraglicher Grundlage für AMIGA.

* AMIGA werde Usergroups kostengünstige Merchandising-Artikel zur Verfügung  
  stellen.

* Auf der Messe habe AMIGA ein Touchscreen-Gerät, basierend auf der neuen   
  Betriebsumgebung, gezeigt. Das Design stamme von Mick Tinker (BoXeR).

* Abwärtskompatibilität der Betriebsumgeung zum Classic-AmigaOS werde über  
  eine Lösung erreicht, bei der es sich nicht um einen Emulator handele.
