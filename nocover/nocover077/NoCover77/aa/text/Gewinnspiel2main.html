gestellt:

»AFiloFaxPro managet Ihr tägliches Leben. Dazu bietet es eine umfangreiche 
Terminplanerfunktion und darüber hinaus Features wie einen elektronischen 
Notizblock oder Statistikfunktionen, die u.a. die tägliche Computerlaufzeit 
protokollieren. (Unterstützt MUI)«

Eine Demo-Version des Programms ist im Aminet verfügbar.

Henk Jonas hat sein Programm MetaView auf CD-ROM für das Gewinnspiel zur 
Verfügung gestellt:

»MetaView ist ein Anzeige- und Konventierprogramm für Vektorgrafiken, wie 
z.B. Cliparts oder CAD-Zeichnungen.

Neben dem auf Amiga einmaligen Umfang an unterstützten Dateiformaten (WMF, 
CGM, Dr2D, EPS, HPGL, FIG etc.) bietet es eine einfache Katalogisierung von 
Clipartsammlungen.

Durch die ARexxschnittstelle kann MetaView als Import- oder Exportfilter 
für andere Grafikprogramme oder Textverarbeitungen eingesetzt werden. Für 
registrierte Anwender existiert ein Datatype, der den Import von durch 
MetaView unterstützten Dateien in nahezu jede Anwendung auf dem Amiga 
ermöglicht.

Eine ansprechende Oberfläche und die Möglichkeit, Bilder aus der Übersicht 
in einem beliebigen Format an andere Anwendungen zu schicken, zeichnen 
MetaView aus. Eine lokalisierte Version befindet sich bereits im Test und 
wird bald veröffentlicht.

Die aktuelle Version ist im Internet unter 
http://user.cs.tu-berlin.de/~jonash oder etwas sporadischer im Aminet unter 
gfx/conv/MetaView.lha zu finden. Eine EMail-liste auf http://www.egroups.de 
unter dem Stichwort MetaView zu finden.


Der Programmierer ist erreichbar unter:

Henk Jonas
Zionskirchstr. 28
10119 Berlin

E-Mail: jonash@cs.tu-berlin.de«
