4. Spiele-News aus aller Welt
-----------------------------

- ACSYS Erscheinung fraglich

Wie Christian Sobotta vom ACSYS-Entwicklungsteam diesen Monat mitteilte, 
sind erst 50 Bekundungen von Kaufinteresse für ACSYS eingegangen; benötigt 
werden allerdings 500, damit das Spiel erscheinen kann. Deshalb verweist 
Christian nochmals ausführlich darauf, dass es sich nicht um 
Vorbestellungen handelt, sondern nur um Bekundungen eines eventuellen 
Kaufinteresses. ACSYS ist ein sehr guter Turrican-Klon, der in der letzten 
Ausgabe von AMIGA aktuell schon vorgestellt wurde.
 
- Hyperion Software kündigt weitere Portierung an

Fast jeden Monat gibt es eine neue Ankündigung von Hyperion Software zu 
melden; dies ist auch dieses mal nichts anders. Angekündigt wurde eine 
Portierung des PC-Spiels Majesty, welches auf dem Amiga schon im 1. Quartal 
2001 erscheinen soll. Das Spiel soll auf einem PPC-Amiga mit WarpOS, 48MB 
RAM und Grafikkarte laufen. Ob eine Version für den 68060 erscheint, ist 
noch nicht sicher. Außerdem wurde bekannt, dass Hyperion Software auf der 
World of Amiga-Messe in Köln zwar keinen eigenen Stand haben, jedoch einige 
Spiele präsentieren werde.

- Bubble Heroes für Grafikkarten-Benutzer

ArcadiA Products hat auf seiner Internet-Seite einen Patch zur Verfügung 
gestellt, der es Grafikkarten-Benutzern erlaubt, Bubble Heroes auch auf 
ihrer Grafikkarte zu betreiben. Deer Patch soll auf jedem System laufen und 
wird alle Benutzern von Grafikkarten empfohlen.

- Payback-Demoversion

Diesen Monat hat Apex Designs eine spielbare Demoversion von Payback für 
Spielemagazine exklusiv veröffentlicht. Deshalb könnt ihr in dieser Ausgabe 
ein Preview davon lesen.

- Exodus - The Last War

Wie Amiga-News.de berichtete, wurde einem Mitarbeiter die Information vom 
Entwicklungsteam von Exodus mitgeteilt, dass es auf jeden Fall eine PPC-
Version von Exodus - The Last War geben werde.

- SpaceStation 3000 eingestellt

Viele hatten es schon vermutet bzw. befürchtet, diesen Monat hat Digital 
Images bekanntgegeben, dass das Spiel SpaceStation 3000 nicht erscheinen 
wird. Als Grund wurden Fehler bei der Projektkoordinierung sowie 
Startschwierigkeiten des Entwicklungsteams genannt. In Zukunft will man 
sich bei Digital Images jetzt mehr auf den Amiga One konzentrieren.

- Neue Spiele bei AmigaLand

AmigaLand hat die Lizenz bekommen, weitere kommerzielle Amiga-Spiele auf 
ihrer Homepage zu veröffentlichen. Dazu zählen nun Hanse - Die Expedition 
AGA, Carribean Disaster, Whale's Voyage sowie Der Clou! Die Profi-Disk. 
AmigaLand ist unter http://www.vivaline.de/matt/ zu erreichen.
 
- HighAnt Development verlässt Amiga

Marco Frischkorn von HighAnt Development hat diesen Monat bekanntgegeben, 
dass Battle das letzte Spiel für den Amiga sein wird. Als Begründung wird 
angegeben, dass man bei HighAnt Development die Pläne von Amiga Inc. 
einfach nicht als realistisch ansieht.

- Das Blaue Auge II zum freien Download

Timo Kloss von Inutilis Software hat der Amiga Arena die Erlaubnis erteilt, 
das Spiel Das Blaue Auge II - Die Schicksalssäge kostenlos zur Verfügung zu 
stellen. Die Amiga Arena ist unter http://online-club.de/~ARENA/ zu 
erreichen.

- Ermentrud Demoversion

Desweiteren hat Inutilis Software diesen Monat eine erste Demoversion des 
Adventures Ermentrud zum Download freigegeben. Diese Version läuft 
allerdings bisher nur auf CyberGfx-Systemen, weshalb ich davon kein Preview 
präsentieren kann.

- Tales of Tamar

Am 5. November wurde die Version 0.32 von Tales of Tamar veröffentlicht. 
Sie kann zusammen mit dem Client- und Hauptarchiv von http://www.tamar.net/ 
heruntergeladen werden.

- Nightlong Goodies

Auf der Support-Seite für Nightlong auf http://www.clickboom.com wurden 
einige Goodies für das Spiel veröffentlicht. So können Hintergrundbilder 
und Filmsequenzen von dort heruntergeladen werden.
