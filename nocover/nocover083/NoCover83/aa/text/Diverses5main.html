5. Kurzmeldungen (von Oliver Tacke)

---------------------------------------------------------------------------
 Software
---------------------------------------------------------------------------
Für OS3.5 Benutzer ist das zweite BoingBag erschienen. Für OS3.9 Anwender
ist dies jedoch ohne Nutzen.
[http://www.amiga.com/3.5/download/BoingBag2a.lha]

Für registrierte Benutzer von AWeb 3.x gibt es einen neuen Patch.
[http://www.amitrix.com/aweb3x34pch.lha]

Der mp3 Player AmigaAMP ist nun endlich in seiner finalen Version 2.8
erschienen.
[http://www.amigaamp.de/Binaries/AmigaAMP28.lha]

Für den Workbench-Ersatz Scalos wurde ein PlugIn herausgegeben, das die
Kompatibilität zu OS3.9 herstellt. Außerdem verbessert wurden das
Icon-System und VolumeGauge.
Eine Betaversion von V1.2 kann ebenfalls heruntergeladen werden.
[http://scalos.stratum.org/products/scalos/]

EasyPlayer wurde in Version 1.04 herausgegeben.
[http://www.coli.uni-sb.de/~rade/soft/e/EasyPlayer.lha.bin]

Das Programm MainPrefs, das den Umgang mit den Voreinstellern der Workbench
(ab 3.5) vereinfacht, ist in Version 2.02 erschienen.
[http://private.addcom.de/martin.elsner/deutsch/anmai.html]

Der ICQ-Client StrICQ ist nun in Version 0.1734 erhältlich.
[http://www.multimania.com/aubioane/programmes/stricq.lha]

AmigAIM, der (inoffizielle) AOL Instant Messenger für den Amiga, ist in
Version 0.9407 herausgegeben worden.
[http://www.missingpiece.com/AmigAIM/index.html]

EZP@gerNG Classic ist in Version 1.6 verfügbar.
[http://www.sfxsoft.de/de/news.html]

Die endgültige Version des mpeg-Players AMP2 ist erschienen.
[http://ftp.amidog.com/pub/amp/AMP201-lha]

Für die Aldi/Medion/Tevion MD 9310 Grafiktabletts ist eine neue Version des
Treibers FormAldiHyd erschienen, nämlich Version 1.3.
[http://www.platon42.de/]

Die Scanner-Kalibrierungssoftware ICS ist als Version 2.01 verfügbar.
[http://www.ics.coloraid.de]

Der IRC Client AmIRC hat eine höhere Versionsnummer spendiert bekommen:
3.5.6 beta.
[http://ftp.ch.vapor.com/amirc/amirc_356beta.lzx]

Eine neue Version gibt es von Samba und hat die Versionsnummer 2.0.7.
[http://ftp.wustl.edu/~aminet/aminet/comm/tcp/samba_2.0.7.lha]

Für PageStream sind Importfilter für Word8 (und Word97/2000) und für
AmigaWriter erschienen.
[http://www.muenster.de/~tomjoad/]

Dem SMS-Mailer SMSEnginer ist ein Update widerfahren.
[http://blitz-engineers.virtualave.net/download/SMSEngineerMUI.lha]

JST, der Disk-To-HD-Installer für viele ältere Spiele, ist in Version
4.4 erschienen.
[http://jffabre.free.fr/amiga/patches.html]

DirOpus V4.15e beta ist herausgegeben worden.
[http://viper.pl/~opus/DOpus4beta.lzx]

Der Virendetektor Save ist in Version 14.8 veröffentlicht worden und steht
zum Download bereit.
[http://www.vht-dk.dk/]

Der Browser Voyager ist in einer neuen Beta-Version erschienen: V3.3.65.
[http://v3.vapor.com/]

Für EASys ist ein Patch für Version 4.84 erschienen.
[http://www.eternity-computer.de/easys/bin/EASys!_patch_484.lha]

Perfect Paint ist als Version 2.4 erschienen.
[http://gothic.fr.free.fr/amiga/]

Eine Alpha Version des neuen email-Clients Simple Mail ist erschienen.
[http://www.schlawack.net/simplemail/simplemail.lzx]

VHI Studio ist in Version 5.4 erschienen.
[http://www.innovative-web.de]

SoftCinema ist in Version 0.11 herausgegeben worden und unterstützt nun
auch DIVX und 3ivX.
[http://www.amiga.pl/softcinema/]

Der Macintosh-Emulator Shapeshifter ist in Version 3.11 erschienen.
[http://iphcip1.physik.uni-mainz.de/~cbauer/ShapeShifter3_11.lha]

Das Börsenprogramm Bonds ist in einer neuen Version erschienen.
[http://www.programmed-excellence.de/]
