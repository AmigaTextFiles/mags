Leider hat sich meine Prozessorkarte verabschiedet hat, muss der Mac warten...
Stattdessen nehme ich Heute mal einen Klassiker.

FRODO



Technische Daten:
Programmname: Frodo
Getestete Version: 1.5
Typ: C 64 Emulator
Autor: Christian Bauer
Status: Freeware

Systemvorrausetzungen:
- Amiga
- OS 2.1
- 68020 CPU
- 1 MB RAM

Empfohlene Konfig:
- Amiga
- OS 2.1
- 68040 CPU mit 40 Megahertz
- Grafikkarte

Getestet auf folgender Konfig:
- Amiga 1200
- OS 3.9
- 68030/50
- 32 MB RAM + 2 MB Chip


Einleitung:
Da unser erster Computer so ein "Brotkasten" war, und ich ihn auch noch gut in Erinnerung habe, hab ich mir den C 64 für den Amiga geholt. Jetzt ist es mir möglich, meine guten alten C 64 Spiele-Klassiker auf den Amiga zu spielen.

Konfiguration:
Ersteinmal muss man sich die C 64 ROM´s besorgen. Aus Copyrightgründen können sie nicht mitgeliefert werden (Es wird aber beschrieben, wie man daran kommen kann).
Zur Info: Die ROM´s hab ich von einer CD namens "C 64 Sensation".
Nachdem die ROM´s so installiert wurden, wie es ausführlich in der Dokumentation des Programms steht, lässt sich der Emulator starten. Nachdem man einen Doppelklick getan hat, öffnet sich ein Konfigurationsfenster.

In den Fenster können folgende Dinge eingestellt werden:


Wie die Emulation angezeigt werden soll. (Die Zyklen habe ich auf die Standartwerte gelassen!)
Wo der Joystick angeschlossen werden soll (Port 1 oder Port 2), ob eine SID-KArte vorhanden ist (ein Bauplan für eine SID-Karte ist dem Emulator beigelegt). Ob es eine Amerikanische Tastatur verwendet werden soll.
Desweiteren kann man eine Spritekollision ei und auschalten.

Der Emulator kann bis zu 4 Laufwerken verwalten. Sie können in Form von einem Floppydisk-Laufwerk verwendet werden, oder einfach als einer Schublade auf der Festplatte.
Ich habe als erstes Laufwerk meine Ramdisk gewählt und als zweites Laufwerk eine Schublade auf meiner Festplatte.

Zum Schluss sollte man noch die Bildschirmauflösung wählen. Ich empfehle hier LowRes NonInterlaced, das ansonsten das Bild zu klein wird...

Es werden alle Funktionen ausführlich in der zum Programm gehörenden AmigaGuide-Dokumentation beschrieben...

Falls man alle Einstellungen vorgenommen hat, sollte man jetzt speichern. Danch kann man getrost auf OK klicken und den Emulator starten.


Nach etwa 1 Sekunde Verzögerrung öffnet sich der altbekannte Screen des C 64. Mit den Befehl LOAD"$",8 kann man seine Programmschublade im C64 öffnen.

Die C64 Befehle sollten hier bekannt sein, ansonsten kann man diese in der FrodoGuide nachlesen.

Die Spiele laufen zwar etwas langsamer als das Original (C64), aber dafür sehr gut und auch der Sound kommt gut rüber.
