EINLEITUNG:

Globaler Müll einfach sortiert - Das war das Thema des letzten Tests. Aber nun als die letzte Aminet bei mir eintraf, bekam ich die Version 1.4 mitgeliefert und hab sie auch gleich auf die Platte geschmissen. Völlig überweltigt davon, die neue Version zu haben, hab ich mir auch gleich das Guide zur Brust genommen.

NEUES:

  • Deaktivierung von Datenträgern:
    Mit diesem Tooltype kann man Datenträger vom Auffrischen ausschliessen. Man kann damit Netzwerk-Datenträger übergehen und es wird dadurch beim aktiven FTPMount keine neue Netzwerkverbindung aufgebaut.

  • Spezifikation des Dateityp-Kommandos (`Anzeigen´-Gadget):
    Standartmäßig wird Multiview als Anzeiger benutzt, aber durch die folgenden ToolTypes kann man für jedes einen eigenen Anzeiger wählen.

    COM-UNKNOWN=<Shell-Kommando> : Unbekannte Datei
    COM-DOC=<Shell-Kommando> : Dokumentation
    COM-TEXT=<Shell-Kommando> : Text-Datei
    COM-HTML=<Shell-Kommando> : HTML-Datei
    COM-ARCHIVE=<Shell-Kommando> : Komprimierte Datei
    COM-PIC=<Shell-Kommando> : Bild
    COM-SOUND=<Shell-Kommando> : Ton

  • Neue Eigenschaften des GlobalTrashClient Programmes:
    Der Client akzeptiert seit Version 1.4 jeden gültigen AmigaOS-Platzhalter.

FAZIT:

Durch die neuen Funktionen und durch das Einladen neuen Dateityp-Images ist es noch besser und damit auch zum Trash-Manager und Verwalter geworden. Ein dickes .
