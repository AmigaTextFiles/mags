5. Wildfire7\PPC und Wildfire für Movieshop im Preis gesenkt

»For immediate release - zur sofortigen Veröffentlichung!

29. August 1999

Wildfire7\PPC und Wildfire für Movieshop Preissenkung


Ab sofort wird der empfohlene Verkaufspreis des preisgekrönten
Spezialeffekt-Programmes Wildfire in der Version 7\PPC von 279 DM 
auf 249 DM gesenkt.

Weiterhin erscheint Anfang September das `Wildfire für Movieshop`
Beispielvideo. In diesem Video werden alle Spezialeffekte, die
für die Wildfire-Movieshop-Kombination angeboten werden, gezeigt.
Das Video hat ca. 100 Minuten Laufzeit. Der Verkaufspreis wird
hiermit von 40 DM auf 35 DM gesenkt.

Somit ergibt sich folgende Preisstruktur:

249 DM    Wildfire 7\PPC - DAS Spezial-Effekt-Programm
          (bisher 279 DM)

 35 DM    Wildfire-für-Movieshop-Operator Beispielvideo
          (bisher 40 DM) 

 30 DM    Wildfire-für-Movieshop-Operator

 69 DM    Wildfire-für-Movieshop Spezial Effekt Set 1
          über 50 hochqualitative 2D Effekte

 79 DM    Wildfire-für-Movieshop Spezial Effekt Set 2
          über 70 spektakuläre 2D Effekte

 89 DM    Wildfire-für-Movieshop Spezial Effekt Set 3
          über 50 fantastische 3D Effekte

 99 DM    Wildfire-für-Movieshop Spezial Effekt Set 4
          über 50 Profi 3D / 2D Effekte

279 DM    Wildfire-für-Movieshop Bundle 1
          Spezial Effekt Sets 1 - 4 (Sie sparen 57 DM)

539 DM    Wildfire-für-Movieshop Bundle 2
          (bisher 569 DM)
          Wildfire7\PPC, Movieshop Operator,
          Spezial Effekt Sets 1 - 4 (Sie sparen 76 DM)
