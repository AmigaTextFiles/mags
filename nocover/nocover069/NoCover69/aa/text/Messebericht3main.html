»Neuigkeiten von rbm auf der HOME electronics world

Warum immer nur beige Computergehäuse? rbm wird farbige Big-Tower-Gehäuse 
in 16 verschiedenen Farben in Köln vorstellen - auf Wunsch sind die Farben 
auch als Metallic-Lackierung erhältlich.

Sicherheit für Ihren AMIGA - unter diesem Motto stellt rbm digitaltechnik 
eine unterbrechungsfreie Stromversorgung ( USV ) vor. Die USV schützt Ihre 
Daten vor Überspannungsschäden und Stromausfällen. Die beiliegende Software 
ermöglicht dem Anwender von ihm erstellte AReXX-Scripte auszulösen, die 
Daten zu speichern oder Meldungen via Internet versenden zu lassen.

FX-Paint 2.0 wird von Felix Schwarz ( Programmierer von UConv. 4.0 ) auf 
dem Stand von rbm digitaltechnik vorgestellt. Umfangreiche Unterstützungs-
Features zeichnen dieses neue Programm aus. Dazu rbm digitaltechnik: "Diese 
Software verfügt über viele weitere Funktionen, die so einzigartig sind, 
dass sie bisher in dieser Form weder für AMIGA noch für andere Plattformen
verfügbar sind."«
