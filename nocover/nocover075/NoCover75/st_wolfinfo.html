Randa ist wieder da!!!

Nun ist Randa doch wieder da und sogar in neuem Outfit. Sieht sie jetzt nicht direkt hübsch aus? Und zum Lesen der jetzigen Geschichten braucht man auch keine Lupe mehr.

Jetzt auch von Thomas noch verlangen, die Spacys wieder zu beleben, ist wirklich zuviel von einem verlangt. Vielleicht haben wir doch außer Oliver einen anderen Künstler unter uns, der diese Arbeit dem Thomas gerne abnehmen möchte?

Jetzt noch ein paar ganz deutliche Worte eines Clubbers an alle anderen Clubber. Es ist ja ein verdammt trauriges Bild gleich in zwei Feedbacks lesen zu müssen, dass man für diese sehr zeitintensive Pixelarbeiten kein Interesse bekundet kann, weil man keinen Draht dafür hat. Für Thomas muss das ja gewirkt haben, als ob er ein Eimer voll kaltes Wasser über seinen erhitzten Nacken geschüttet bekommen hat. Und neben diesen beiden sehr negativen Äußerungen, gab es aber auch rein gar nicht darüber zu lesen. Kein Statement, keine Erkundigung, kein Vermissen, kein, kein, kein. Nichts, nichts, nichts. Wie sagte weiland Theo Lingen? Traurig, traurig, traurig. Das gleiche könnte man nun ebenso über die Äußerungsbereitschaft der Allgemeinheit zum Ausdruck bringen.

IBrowse 2.2

Auf der Homepage von HiSoft ist seit 05.05. (dieser Termin wurde mir so per E-Mail genannt) IBrowse 2.2 als Demoversion (½ Stunde) oder als Upcreate von IBrowse 2.1 auf 2.2 downloadbar. Upcreate klingt gut, aber das File ist etwas über 1,4 MB groß und ist das Komplettprogramm. Entpackt sind dies stolze 2,75 MB und bevor ich die Installation (eine Neu-Installation gem. Hinweis der Homepage von HiSoft) begann, habe ich erst meine Version 2.1 als .BAK umbenannt. Bei der Installation darf man auch hier den Sprachenkatalog auswählen.

"Català" "Czech" "Dansk" "(Deutsch)" "English (built in)" "Español" "Français" "(Greek)" "Italiano" "Magyar" "(Nederlands)" "(Norsk)" "(Polski)" "Português" "Português-brasil" "Russian" "Slovensko" "Srpski" "Suomi" "Svenska"
aber für die Sprachen, die hier in Klammer gesetzt sind, ist kein Katalog mitgeliefert und beim Versuch der Installation wird eine Fehlernachricht herausgegeben. Die Klammern sind aber bei der Installation nicht sichtbar, man merkt erst hinterher, dass da nichts war. Wann also ein deutscher Katalog verfügbar sein wird und wie man darüber informiert wird, ist mir nicht bekannt. Was sich nun im Einzelnen alles geändert hat, habe ich noch nicht festgestellt. Dafür habe ich das Programm noch nicht lange genug. Zwei Punkte fielen mir sofort positiv auf, die Farbverfälschung, wie bei Tipps&Tricks in NC-74 angeführt, ist verflogen. Auch das Anspringen von Marken auf anderen HTML-Seiten funktioniert jetzt. Der Seitenspeicher lässt sich löschen und das Reloading wird anstandslos ausgeführt. Nur das Verschieben von Framerändern geht noch nicht, oder ich habe in meinen Prefs irgendetwas falsch eingestellt. Übrigends damit man überhaupt mit dem neuen Programm arbeiten kann, ist es vonnöten, den Key von 2.1 nach 2.2 zu kopieren.

iBrowse 2.2 Update

Maintenance update for iBrowse 2 now available at
http://www.hisoft.co.uk/cgi-bin/gp?pg=amigashop(ibrowse)

Improvements / fixes

+ Added ftp/http resume
- Fixed a bug that caused some jpeg images not to be loaded
+ Added a much more flexible download system.
- Fixed a bug that cause files to be left open in the cache
- Fixed crash problems when using multiple browser tabs
- Fixed the editable gui
+ Removed the clicktabs if only 1 browser is open
+ Cycle gadgets with too many entries are now transformed into a listview
  automatically
+ Added support for editable fabmenus
+ Added support for font styles. Now you can define fontstyles that
  matches the contents of <font face=...> tags
- Fixed a bug that could cause problems when changing font preferences
- Made iBrowse recognize OS3.5 as a valid OS version
+ Added a SECURE indicator for secure pages
- Fixed a Y2K compatibility bug with non Y2K aware servers
- Minor changes to the parser to be handle illegal params better
- Rewrite the timer code to avoid a lockup bug in timer.device
- Fixed Network URL MIME type
- Fixed a bug in the HTML reading code that wasted approx 50% of the
  memory needed to load an HTML page
+ Added support for displaying progressive jpegs progressively
Der Vollständigkeithalber habe ich die Mitteilung aus der Mail von HiSoft hier einmal aufgezeigt und jeder kann nachlesen, was er nun bekommt. Wie jeder, der ebenfalls IBrowse 2.2 schon hat, hier deutlich sehen kann, funktioniert die Fontrückumschaltung nach </pre> jetzt einwandfrei.

Wer also nun das neue IBrowse beziehen will und von 1.22 auf 2.2 gehen will, kein Problem. Entweder mit Kreditkarte über Internet oder gem. der Verfahrensweise über die Fa. GTI, wie ich es in NC-73 / Tipps&Tricks beschrieben habe. Dieser muss sich aber das Demoprogramm ziehen und sich neu registrieren lassen. So zu tun, man habe schon 2.1 und es ist nicht der Fall, hat keinen Sinn, es funktioniert nicht.

Ein gewaltiges Manko ist mir heute aufgefallen: IBrowse 2.2 hat verlernt, wie geschachtelte <li>-Anweisungen unter Verwendung von <ul> und <ol> auszuschauen haben. Um jetzt aufzuzeigen, wann eingerückt wird und wann nicht, ist an dieser Stelle unerheblich. Ich will nur sagen, es ist falsch und ich verstehe nicht, dass soetwas beim Beta-Testen übersehen werden kann.

Expo 2000

Nein, ich war nicht in Hannover und berichte nun über meine Eindrücke. Nein, ich möchte keine Reklame über die Expo machen oder ein sonstiges Statement abgeben. Nein, ich bekomme keine Sponsoringbeiträge oder sonstige Zuwendungen. Aber ich habe die Homepage gesehen "www.expo2000.de" und sie hat mir gefallen. Da die Seiten mittels Javascript und CSS aus vielen kleinen Komponenten aufgebaut wird (sogar schnell aufgebaut wird) habe ich einmal einen Sreenshot gemacht und zeige davon einen kleinen Ausschnitt der Startseite.
Die Homepage wird in zwei Sprachen geführt (deutsch und englisch). Die separate Willkommenseite gibt es in zig Sprachen und entsprechenden Schriften. Wer nun einen einfachen Browser eingesetzt hat, kann sich gerade einmal das Grundgerüst anschauen. Denn ein besonderes Erlebnis dieser Website ist die 3-D-Cybertour über das Expogelände. So kann man sich mit der Maus ein virtuelles Bild vom Ausstellungsgelände machen. Es geht aber auch einfacher indem man sich auf der Karte orientiert. Die einzelnen Namen erscheinen dann in Sprechblasen, je nachdem wo der Mauszeiger hinzeigt. Endeckt habe ich eine Seite mit Live-Cams, eine Seite für Kinder mit Spiel und Rätsel, ein Shopingcenter für Fan-Artikel, ein Veranstaltungskalender mit detaillierten Informationen und eine Seite zum Bestellen von Eintrittskarten. Es gibt bestimmt noch eine ganze Reihe mehr, die ich dann leider übersehen habe.
