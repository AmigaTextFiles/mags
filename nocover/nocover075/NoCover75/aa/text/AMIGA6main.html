6. Kurzmeldungen

* In eigener Sache weise ich auf eine kleine, für die Zukunft dieses        
  Magazins aber sehr wichtige, Umfrage hin, die wir in dieser Ausgabe       
  durchführen. Ich bitte euch ALLE, die paar Minuten zu investieren und an  
  ihr teilzunehmen. Danke.

* Am 3. Juni ist eine neue Ausgabe von AMIGAs Online-Magazin Amiga World    
  erschienen, die insbesondere viele Informationen für AmigaNG-Entwickler
  enthält. Sie liegt auf http://www.amiga.com/press/zine/ im HTML-Format
  vor, eine PDF-Version wird in Kürze folgen.

* Der neuesten Ausgabe von Amiga World ist zu entnehmen, dass AMIGA         
  momentan nicht vorhat, Memory Protection (Speicherschutz) in das neue     
  AmigaOS zu integrieren, das bei einem Programmcrash den Absturz des       
  gesamten Betriebssystems verhindern könnte. Dies stieß in der offenen      
  Entwickler-Mailingliste [und auch beim leitenden Redakteur dieses         
  Magazins, der Memory Protection für eine absolute Notwendigkeit hält] auf  
  Unverständnis, so dass AMIGA mittlerweile eine spätere Implementierung    
  nicht mehr ausschließt. Dennoch sollte sich jeder, der ebenfalls für die  
  Integration von Speicherschutz ist, bei Gary Peake (gary@amiga.com) dafür  
  stark machen.

* Zum AmigaOS 3.5 wurde eine Support-Mailingliste eingerichtet. Um sich     
  anzumelden, muss eine Mail mit dem Inhalt "Subscribe" im Body an 
  os35list-request@amiga.com geschickt werden. Nach erfolgter Anmeldung     
  kann man sich über die Adresse os35list@amiga.com zu Wort melden.

* Es befindet sich ein zweites Bugfix-Update zum AmigaOS 3.5 in Arbeit.     
  Einen genauen Erscheinungstermin gibt es jedoch noch nicht.
