Homebank 1.9 erschienen 30.9.2001
10:27
Homebank 1.9 ist erschienen. Diese Betaversion ist allerdinsg nur für registrierte User erhältlich.

Changes:

  • Change : some more internal improvements.
  • Fix : the password window position was popping at the upper left of the screen instead of center when auto openning a wallet.
  • Change : moved the status remind icon to the same position as validated in operation list.
  • Change : automated archives to more flexibility using units (Day/Week/Month/Year) and a associated periodicity can be set from 0 to 100.
  • Change : the display filter to more accurracy, all criteria can now be applied, Date range/Group/Status&Mode/Text/Amount range, all with inclusive/exclusive.
  • Add : a "Reset" button for the display filter, wich lead back to settings default.
  • Remove : the find window which is now obsolete with the new display filter.
  • Change : the date popup, added year increase/decrease, made some optimizations, changed the look, made it sensitive to 1st weekday of country, localized.
  • Add : an option to use the Euro symbol ( € ) with euro compatible fonts (see euro.txt).
  • Add : Euro major functions. If you are in the Euro area, a convert menu will enable to internally convert your wallet to Euro major currency. Your actual currency will only be considered then as the minor currency.
  • Remove : the dual display in «your currency»/Euro, for balance and operation list, each window that contains amounts has now a toggle "Minor" button.
  • Add : support of external images for the string popup buttons.
  • Add : the display sort can now be applied on multiple columns, by default it is set to Status, Date, Info, Amount.
  • Change : the sort criteria is now set by a column clic that popup a window.
  • Fix : automated archives with cheque mode were not dealing with cheque number.
  • Fix : when adding an income amount operation with cheque mode, the cheque number was faultly incremented and the Info gadget filled with the incremented cheque number... Bad...
  • Add : line chart for the overdrawn report window.
  • Add : startup window with progress indications.
  • Change : designed a new HomeBank logo.
  • Change : now you can open several report window at the same time (stat, over, car).
  • Change : rewritten the statistics window with complete filter possibilities, displaying the results as a list, bar chart, pie chart.
  • Add : menu in Operation to manually check the automated archives to be inserted.
  • Add : some options in the filter to force viewing Added & Changed operations.
  • Add : some settings for charts and default statistics options.
  • Change : new images based on the MasonIcons.
   Redakteur: Andreas Magerl

PCI invasion: ProStationAudio on G-Rex 4000 with Terratec audio hardware. 29.9.2001
10:32
ProStationAudio compatible with G-Rex PCI busboards by DCE and Terratec PCI audio boards . Frank Mariak, DCE's software designer of drivers for the G-Rex, confirms that drivers they just made for the 128i and 512i Terratec models have passed preliminary tests for analog I/O with ProStationAudio. Drivers for the 512i are being extended with S/P-DIF support.

G-Rex makers and AudioLabs are actively cooperating to provide the best compatibility between this new PCI busboard and ProStationAudio (a big thank you to Frank for his continued and accurate work).

The G-Rex PCI busboard, available in two models for the A4000 and A1200 is a solution developed specifically for owners of P5/DCE CyberStorm and Blizzard PPC and/or 68K CPU cards. The G-Rex 4000D replaces the stock A4000 backplane, offering slots for PCI and Zorro cards. ProStationAudio takes advantage as well of PCI graphic cards installed on the G-Rex
