Diverse Neuigkeiten
-------------------

1. Opera wird für AMIGA entwickelt

Die Entscheidung ist gefallen: Es wird eine AMIGA-Version des WWW-Browsers 
Opera geben. Im einzelnen informieren die beiden folgenden Bekanntmachungen 
von Opera Software:

»13. Mai 1998

Opera Software kündigt die Entwicklung einer Amiga-Version an -- 
Als Folge der außerordentlichen Reaktion auf seine Initiative "Project
Magic" kündigt Opera Software AS heute die Entwicklung einer Amiga-Version
seines beliebten Browsers an. Die Entwicklung wird die in Reading, UK,
angesiedelte Ramjam Consultants Ltd. unter der Leitung von Tim Corringham
durchführen.

"Wir freuen uns, bei der Übertragung von Opera auf den Amiga dabeizusein.
Opera besitzt ein Amiga-'Feeling' selbst unter Windows 95, ich habe also
volles Vertrauen, daß daraus eine hochwertige Amiga-Anwendung wird und
es einen Grad an Gemeinsamkeiten mit Windows 95 bieten wird, die wenige
Anwendungsprogramme erreichen", sagt Corringham.

"Ich bin selbst seit langer Zeit ein Amiga-Anwender, und es macht mir
große Freude, offiziell die Entwicklung von Opera für die Amiga-Plattform
ankündigen zu können", fügt der Business Director von Opera Software, Jon
von Tetzchner hinzu. "Es war eines unserer Ziele, Opera so vielen Internet-
Anwendern wie möglich zur Verfügung zu stellen, und die Möglichkeit, eine
Amiga-Version anzubieten, erhöht nicht nur die Glaubwürdigkeit für Opera
als ein leistungsfähiges und Anwender-freundliches Programm, sondern es
ist auch ein Anschub für die Amiga-Plattform."

"Dieser Schritt ist nur die logische Folge von Project Magic, wo seit 
seinem Beginn einige Tausende Anwender ihre Unterstützung für Opera auf 
anderen Plattformen als MS Windows anboten. Die Amiga-Gemeinschaft ist in 
voller Stärke angetreten, und wir sind stolz, schon bald ein Produkt 
anbieten zu können, das so fortschrittlich, leistungsfähig und genügsam ist 
wie das Betriebssystem, auf dem es laufen wird", erklärt Helmar Rudolph, 
Initiator und Betreuer von Project Magic für Opera Software.

Beide Gesellschaften streben ein Erscheinungsdatum im Dezember 98 an, 
gerade rechtzeitig für die Zeit der Weihnachtseinkäufe, um den vielen 
loyalen und nicht kleinzukriegenden Amiga-Anhängern eine Quelle der Freude 
und Genugtuung zu bieten.

Opera für den Amiga soll das gleiche Maß an Funktionalität wie sein 
Windows-Gegenspieler bieten und ihn damit in die Spitzengruppe der Browser 
für die Amiga-Plattform bringen.

Der Preis für diese Version ist noch nicht festgelegt, es wird aber 
erwartet, daß er auf gleicher Linie mit der Windows-Version liegen wird.«


»13. Mai 1998

Opera Software kündigt Entwicklung einer Amiga Version an -- Nach langen
Verhandlungen ist es uns eine Freude, die Entwicklung von Opera für den
Amiga anzukuendigen. Die Entwicklung wird in Reading, England, von Ramjam
Consultants Ltd. under der Leitung von Tim Corringham vonstatten gehen, und
wir hoffen, daß wir bis Weihnachten damit fertig sind.

Einige Insider-Informationen von Tim:

- die erste Amiga-Version wird für die bestehenden Amiga-Modelle
  mit Workbench 3.1 und mindestens 4 MByte RAM geeignet sein

- die erste Version soll die Features bestehender Amiga-Browser bieten

- PPC-Version wird folgen - auch sobald wie möglich für neue
  Amigas (was auch immer am Wochenende vom 15.-17. Mai von Amiga, Inc.
  angekündigt wird)
 
- es werden so viele Opera-Features wie möglich integriert, und Amiga-
  spezifische Eigenschaften (z.B. ARexx-Port) kommen hinzu

- Amiga Opera wird eine große Anzahl verschiedener Sprachen unterstützen

- Opera bietet das "Gefühl" einer Amiga-Anwendung selbst auf einem PC,
  wir sind also sicher, daß eine herausragende Amiga-Applikation entstehen
  wird

- Opera ist ständig in Bewegung, aber unser Ziel ist es, sobald wie
  möglich die neuen Features der W95-Version zu übernehmen

- Der Amiga steht vor seiner "Wiedergeburt" mit breiterer Unterstützung
  durch die Industrie, als er sie seit vielen Jahren erfahren hatte. Mit
  neuer Hardware und neuer Software - inklusive Amiga Opera - wird
  er als lebenskräftige Alternative zu Wintel-Maschinen besser dastehen
  als je zuvor!

So, dann freut Euch mal alle schon, und spart bis Weihnachten! :-)«
