4. Kurzmeldungen (von Oliver Tacke)

---------------------------------------------------------------------------
Software (kommerziell)
---------------------------------------------------------------------------

68k/vermischt

Das "Native Developer Kit V3.9", das ein Update der "Amiga Developer CD
2.1" darstellt, ist erschienen und kann heruntergeladen werden.
[http://www.ann.lu/detail.cgi?category=news&file=1010316879.msg]

Eine neue Version der "vapor_toolkit.library", nämlich V15.13, ist
erschienen. Die Library stellt den Programmen aus dem Hause "Vapor" diverse
gemeinsame Funktionen zur Verfügung, so dass die Programme selbst kleiner
gehalten werden können.
[http://vapor.meanmachine.ch/misc/]

Ein neuer Treiber für "VHI-Studio" ermöglicht es in Verbindung mit dem
Mediator-PCI-Busboard, einzelne Bilder oder ganze Sequenzen über eine
TV-Karte zu speichern.
[http://www.elbox.com]

Für Besitzer des "Mediator-PCI-Busboard" und der "Multimedia-CD-ROM" ist
ein Update auf V1.12 erschienen. Es enthält wieder verbesserte Treiber,
außerdem den bereits erwähnten Treiber für VHI.
Auch für diejenigen, die die Multimedia-CD-ROM nicht besitzen, ist Version
2.8 der Software MediatorUP interessant.
[http://www.elbox.com]

Von Individual Computers ist nun die erste finale Version des
"xsurfadsl.device" erschienen, weiterhin ein Update des "appp.device" für
den TCP/IP-Stack AmiTCP (auch Genesis).
[http://www.jschoenfeld.de/jumpnews.htm]

Die Entwicklungssoftware "Inga" für Grafikadventures hat ein Update auf
Version 0.9/5 erhalten. Einzig und allein die Unterstützung für Musik von
CD-ROM fehlt noch.
[http://www.inutilis.de]

Für AmigaOS 3.9 ist nun ein Update erschienen, das den Fonts den Euro
beibringt.
[http://www.haage-partner.com/d.htm]

"PageStream" ist jetzt in Version 4.1 erhältlich und enthält nun einen
integrierten Tabelleneditor, Serienbrieffunktion, etc.
[http://www.pagestream.de]


MorphOS

Eine neue Version der "vapor_toolkit.library", nämlich V15.13, ist
erschienen. Die Library stellt den Programmen aus dem Hause "Vapor" diverse
gemeinsame Funktionen zur Verfügung, so dass die Programme selbst kleiner
gehalten werden können.
[http://vapor.meanmachine.ch/misc/]


---------------------------------------------------------------------------
Software (nicht kommerziell)
---------------------------------------------------------------------------

68k/vermischt

Der ICQ-Messenger-Clone "StrICQ" ist in Version 0.2106 erschienen.
[http://klabba.campus.luth.se/stricq/]

Weiterhin steht eine frühe Betaversion 2.0 von "StrICQ" bereit.
[http://www.stricq.com]

Das Programm "ReqAttack", das dem Verschönern der System-Requester gilt,
wurde in Version 1.96 herausgegeben.
[http://jacadcaps.tripod.com/main.htm]

Der "EasyPlayer", der es ermöglicht, nahezu jedem Shell-basierten
Musikabspielprogramm eine grafische Oberfläche zu spendieren, wurde in
Version 1.12 veröffentlicht.
[http://www.coli.uni-sb.de/~rade/e/EasyPlayer.lha.bin]

Mit dem Programm "WHDLoad" ist es möglich, ältere Diskettenspiele auf
Festplatte zu bannen, auch wenn sie eigentlich nicht dafür gedacht waren.
Es sind zahlreiche neue Plugins erschienen, u.a. für "Lemmings 2",
"Transarctica" und "Ruff'n'Tumble".
[http://www.whdload.de]

"Charon", der Downloadmanager, ist in Version 1.98 erschienen. Mittels des
Programms ist es sehr viel komfortabler, wenn man viele Dateien aus dem
Internet lädt.
[http://www.bigfoot.com/~TheAntony/Charon/]

Für die Programme "Photo Album" und "CyberShow" sind Updates erschienen,
und zwar für ersteres auf Version 6.6, für letztgenanntes auf Version 9.5a.
Es ist nun z.B. möglich, Bilder um 90, 180 oder 270 Grad rotieren zu
lassen.
[http://home.arcor.de/helmuthoffmann/photoalbum/]

Der wohl meistgenutzte Webserver "Apache" liegt in Version 1.3.20 inklusive
PHP-Modul V4.1.1 vor und kann heruntergeladen werden.
[http://sixk.maniasys.com/index_en.html]

Dank der Amiga ARENA steht die Vollversion von "AmiFIG 3.1", einem
Vektorzeichenprogramm, zum Download bereit.
[http://online-club.de/~ARENA/]

Das Audio-Nachbearbeitungsprogramm "Amplify", mit dem man ein- und
ausfaden, Leerstellen am Anfang und Ende entfernen sowie auf 0db aussteuern
kann, ist in Version 1.2 erhältlich.
[http://homepages.compuserve.de/bwoern/]

Martin "Mason" Merz hat Version 7 seiner umfangreichen Sammlung "AppMI" an
GlowIcons für Toolbars herausgegeben.
[http://mason.home.bei.t-online.de/e01.htm]

Das Programm "TaskiSMS", mit dem man vom Rechner aus bequem SMS schreiben
und versenden kann, ist in Version 2.60 erschienen.
[http://taskisms.taski.com.pl]

"EZP@gerNG", ebenfalls ein SMS-Sender, ist in Version 2.0 erschienen und
unterstützt nun u.a. auch das Versenden von Multi-Messages.
[http://www.sfxsoft.de/de/]

Die Programmiersprache "PowerD" steht nun in Version 0.19 alpha 4 zur
Verfügung.
[http://www.volny.cz/kuchinka/]

Der Virensniffer "Watchdog", der in regelmäßigen Abständen den Speicher
nach Viren untersucht, ist in Version 2.5 erhältlich. Benötigt wird
zusätzlich die xvs.library.
[http://home4.inet.tele.dk/vht-dk/amiga/safe/safe.htm]

Wieder sind für den Workbench-Ersatz "Scalos" Updates zu diversen Plugins
erschienen: DefIcons V45.6, Persistent Windows V39.16 und WB3.9 V45.21.
Das Programm "Scalos" selbst ist in Version 40.19 erschienen.
[http://scalos.striatum.org/]

Zum äußerst umfangreichen Erdkunde-Programm "GeoWorld" steht Update Nr. 13
zum Download bereit.
[http://geoworld.amigasoft.de]

Besitzer eines PalmPilots werden sich freuen, denn das Programm "Spitfire²"
zum Verbinden des PDAs mit dem Amiga ist in Version 1.9 erhältlich.
[http://www.amigaspitfire.com]

Die Börsensoftware "Bonds" steht in Version 1.85 zur Verfügung.
[http://www.programmed-excellence.de]

Das Soundtool "WaveTracer DS" ist ab sofort Freeware und darf frei verwendet
werden.
[http://www.VirtualWorlds.de/WT/downloads.html]

Ebenfalls ein Audiotool, nämlich "SampleManager", steht in Version 0.10
zum Download bereit.
[http://ftp.uni-paderborn.de/aminetbin/find?SampleManager]

Das Filesystem "SmartFileSystem" ist in Version 1.194 erschienen. Es
handelt sich allerdings nur um eine Betaversion!
[http://home.wtal.de/js/index.phtml]

Die Datenbanksoftware "XBaze" steht in Version 7.1 zum Download bereit.
Diverse Fehler wurden beseitigt und einige Funktionen hinzugefügt.
[http://www.xbaze.o.k.pl]

Der AOL-Instant-Messenger-Clone "AmigAIM" ist in Version 0.9493 erschienen.
[http://amigaim.missingpiece.com]

Die Programmiersprache "AmiBlitz" ist in Version 2.23 erschienen.
[http://blitz2000.gazchap.com]


PowerPC

Für WarpOS ist Version 0.8.12 von "UAE" erschienen. Mittels des Emulators
kann man auch auf neueren Amigas alte Softwaretitel zum Laufen bringen.
[http://www.amiga-forum.de]

Der Movieplayer "AMP" ist in einer neuen Betaversion V020118 erschienen und
unterstützt u.a. MPEG1/2-Video, AC3-Streams, VCD, CD-i und DVD.
[http://www.amidog.com/amp/]

Das Fraktalberechnungsprogramm "FlashMandelWOS" wurde verbessert und steht
nun in Version 1.6 zur Verfügung.
[http://rz-home.de/~eschwan/cgi-bin/makeframes.pl?fr0=Inhalt_Edgar.html&fr1
=amiga/FlashMandel.shtml]
