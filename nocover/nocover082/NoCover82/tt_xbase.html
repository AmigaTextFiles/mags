XBase ist ein Programm mit dem es relativ einfach möglich ist Dateien - Banken herzustellen.


Die Installation

Die Installation ist einfach, das Verzeichnis mit XBase auf die eigene Festplatte ziehen und die beiden in dem Verzeichnis ENVARC liegenden Icons nach ENVARC auf der eignen Festplatte verschieben oder InstallIcons ausführen.

Es hann noch die popupmenu.library is Copyright © 2000 Henrik Isaksson und die openurl.library written by Troels Walsted Hansen Installiert werden. (Beide brauchen zum einstellen MUI)
Mit der popupmenu.library ist es möglich, abhängieg von der Mausposition ein Popupmenu in XBase darzustellen.
Mit der openurl.library ist es möglich, eine Verbindung zum Internet (Browser) herzustellen.


Der Start von XBase

Der Start von XBase ist einfach, mit der Maus auf das Icon von XBase klicken und es erscheint ein leeres Fenster.
»Dieses leere Fenster erscheint immer wenn XBase so Aufgerufen wird!«
Nun kann unter dem Menüpnkt * (Sternchen) eine Beispiel-Datei sich angeschaut werden. Sie ist auf englisch oder polnisch!
