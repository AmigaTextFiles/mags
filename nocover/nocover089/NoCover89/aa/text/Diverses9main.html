 Software (kommerziell) 
--------------------------------------------------------------------------- 
 
 68k/vermischt 
 
 Für "PageStream" ist ein Update auf V2.1 des "Universalfilters" erschienen,
 das das Laden vieler Grafikformate in PageStream, BME bzw. HHV erlaubt. 
 [http://www.uni.coloraid.de/] 
 
 Für den MAS-Player ist Version 1.4 des MASMPegDevice erschienen. 
 [http://www.platon42.de/platon/files/mus/MASMPegDevice.lha] 
 
 Die Voodoo 3 Warp3D Treiber für das PCI-Busboard "Prometheus" sind
 fertiggestellt worden. Weiterhin aktualisiert wurden alle anderen Treiber
 für das Board. 
 [http://www.matay.pl/] 
 
 Gegen einen entsprechenden Nachweis bietet Haage&Partner den Compiler 
 "StormC" für Studenten und Auszubildende für DM 298,- an. Regulär beträgt 
 der Preis DM 498,-. 
 [http://www.haage-partner.com] 
 
 Für "ArtEffect" und "AmigaWriter" sind Toolbars im GlowIcon Look 
 erschienen. 
 [http://www.haage-partner.com] 
 
 
 PowerPC 
 
 Die Voodoo 3 Warp3D Treiber für das PCI-Busboard "Prometheus" sind
 fertiggestellt worden und liegen auch in einer PPC Version vor. 
 [http://www.matay.pl/]

 Software (nicht kommerziell) 
--------------------------------------------------------------------------- 
 
 68k/vermischt 
 
 Das Filesystem "Smart File System" (SFS) ist in Version 1.190a verfügbar. 
 [http://home.wtal.de/js/index.phtml?SFS.lha] 
 
 Der AOL Instant Messenger Klon für den Amiga, "AmigAIM" wurde in Version 
 0.9428 beta veröffentlicht. 
 [http://www.missingpiece.com/downloads/AmigAIM/AmigAIM_BETA.lha] 
 
 Der Mail- und Newsreader "Microdot II" wurde um diverse Features
 erweitert, wie z.B. das Einrichten mehrerer POP3-Postfächer. Die aktuelle 
 Versionsnummer ist 1.4.15 beta. 
 [http://www.vapor.com] 
 
 Der Compiler "vbcc" für den Amiga ist als Version 0.8 erhältlich. Diverse 
 Fehler wurden behoben und einige neue Features hinzugefügt. Weiterhin soll 
 schnellerer Code generiert werden. 
 [http://www.compilers.de/vbcc/] 
 
 Für den Workbenchersatz "Scalos" ist ein neues Plugin namens "DevFilter" 
 erschienen das es ermöglicht, Laufwerk Icons von der grafischen Oberfläche 
 ausblenden zu lassen. 
 Ein weiteres neues Plugin mit Namen "Titlebar Clock" ermöglicht die
 Erweiterung der Titelleiste des Scalos-Desktops um eine Uhr- und
 Datumsanzeige. 
 [http://www.scalos.com/] 
 
 "Scalos" selbst ist in Version 1.3 beta erschienen. 
 [http://www.scalos.com/] 
 
 Das Malprogramm "DiamondBOX" ist in Version 1.12 veröffentlicht worden. 
 [http://www.DiamondDice.f2s.com/] 
 
 Die Weiterführung des freigegebenen Dateimanagers "Directory Opus 4" trägt 
 Früchte: Version 4.16 ist herausgegeben worden und ist nun für mehr 
 Sprachen lokalisiert. 
 [http://dopus.amiga.pl/files/DOpus416b2.lzx] 
 
 Dem Dateimanager "ClassAction" ist ebenfalls ein Update widerfahren auf 
 Version 4.2. 
 [http://www.martin-elsner.de/] 
 
 Der freigegebene MIDI-Sequencer "Bars&Pipes", der nun weiterentwickelt 
 wird, hört nun auf den Namen "BarsnPipes", um den neuen Status zu
 verdeutlichen. Die aktuelle, bereits erweiterte Version trägt die
 Versionsnummer 1. 
 [http://home.t-online.de/home/alfaj/alfa.html] 
 
 Der SMS-Sender "TaskiSMS" steht in Version 2.58 zum Download bereit. 
 [http://taskisms.taski.com.pl/] 
 
 Für den Compiler gcc ist ein Bugfix (2.95.3-3) erschienen, das Probleme 
 mit den Compileroptionen "-fbaserel" und "-fbaserel32" behebt. 
 [ftp://ftp.geekgadgets.org/pub/geekgadgets/amiga/m68k/alpha/gcc/]

 [http://home.t-online.de/home/sebauer/simplehtml.html]

 Verschiedenes 
--------------------------------------------------------------------------- 
 
 Am 03.11.2001 wird in Brentwood, Essex in England die "World of Amiga" 
 stattfinden. Es ist davon auszugehen, dass die Messe die erste sein wird, 
 auf der der AmigaOne gekauft werden kann. 
 [http://www.worldofamiga.com/] 
 
 Martin Elsner, Programmierer von "ClassAction", hat die Entwicklung des 
 Startmenüs "StopMenu" von Damir Arh übernommen. 
 [http://www.martin-elsner.de/] 
 
 Die lebende Legende "Dave Haynie", der u.a. maßgeblich an der Entwicklung 
 des Amiga 3000 beteiligt war, ist nun Chief Technical Officer bei 
 Merlancia, die mehrere auch AmigaOS unterstützende Computersysteme
 angekündigt haben. 
 [http://www.merlancia.com] 
 
 Vom 20. bis 22. September findet in Köln als Teil der maxpo2001 (Apple 
 Mac-Messe) die Amiga World 2001 statt. Ausgerichtet wird der Amiga Teil 
 der Messe von Falke Media. 
 [http://www.amiga-messe.de/]
