PACKERTEST

Ich glaube jeder kennt ES - das Problem mit dem freien Speicherplatz auf Disketten und auf Festplatten. Im PC-Bereich sind Festplattenverdoppler schon laaange aktuell.

Ich habe nun für Euch 3 Programme bzw. ein Programmpaket getestet womit ihr Byte um Byte "einsparen" könnt. Das erste ist (der etwas betagte) PowerPacker 2.2a, dann der Imploder4.O und das Programmpaket XPK. Alle drei erfreuen sich großer Beliebheit - vorallem beim "Verein für leeeere Brieftaschen ",denn alle Programme sind Freeware.

Der PowerPacker ist wohl der älteste in der Runde. Ab der Version 4.O ist der PowerPacker ein kommerzielles Produkt und KEINE Free- oder Shareware mehr! Als Versuchsprgramm diente mir in allen Fällen DFA-Adress von Dirk Federlein aus Aschaffenburg.

Der erste Kandidat ist der Imploder4.O (PD). Der Imploder präsentiert sich mit einer grafischen Benutzeroberfläche auf der man ein File laden,packen und wieder speichern kann. Der Imploder packt alles so, das es ohne das Programm wieder lesbar ist ,d.h das gepackte File kann man benutzen wie das Orginal. Leider kann man mit dem Imploder NUR Programme packen. Also fällt das Packen von Bildern,Sounds,Befehlen,Fonts usw. aus. Der Imploder liegt mit dem gepackten File nur knapp unterm XPK-Packer und zwar mit genau 536 Bytes. Das Prgramm ist übrigens um ca.4O% ,d.h 24OOO Bytes, kleiner als das Orginal geworden (und das bei allen Packern). Als zweites wollen wir mal den PowerPacker betrachten. Er hing beim Test auf dem 3.Platz, aber wie gesagt es ist eine alte Version. Gepackt wurde mit den Einstellungen : LED-Crunch und efficiency best. So kam der Power Packer mit dem gepackten File auf 262O8 Bytes.

So nun kommen wir zum kompletten Programmpaket XPK. XPK wird schon von vielen Programmen unterstützt (Diavolo Backup ,... ,...). Das Programmpaket befindet sich auf der AmigaMAGAZIN Disk 12/93. Hier sind außer dem eigentlichen Packer noch Zusatzprogramme wie Most(Textanzeiger),ShowIFF (Bildanzeiger) und ein Programm zum Animation abspielen enthalten. Die Daten dürfen natürlich mit XPK gepackt sein. XPK ist eigentlich nicht nur ein Packer - er benutzt nähmlich nicht immer den gleichen "Packer",sondern der Benutzer kann verschiedene Module auswählen um die besten Packraten für das jeweilige File zu erhalten. Es gibt z.B spezielle Packer für Sounds, Programme usw.. Dem Packet ist ein Programm beigefügt was es ermöglicht, das das System so verändert wird, das es gepackte Daten lesen und SCHREIBEN kann ,d.h Daten müssen nicht mehr manuell gepackt werden sondern sie werden gleich beim Schreiben aufs Medium (Pladde,Disk) automatisch gepackt. Das XPK-Paket hatte beim Test die Nase vorn. Es packte das orginal File in 4 Sekunden auf 25348 Bytes.

ZUSATZPROGRAMME:

Beim Imploder erübrigen sich Zusatzprogramme da er ja sowieso nur Programme packt die aus nach dem packen noch lauffähig sind.
Beim XPK gibt es da schon mehr Probleme. Außer den oben angesprochenen Hilfsprogrammen sind Bilder,Sounds,Texte etc. nicht ohne weiteres mehr lesbar. Hierfür gibt es jedoch ein PD-Tool was es ermöglicht das die Daten ebenfalls von allen anderen Programmen lesbar sind. Beim PowerPacker existieren die Programme PPShow (Bildanzeiger) und PPMore(Textanzeiger) die eben diese Datein anzeigen können. Mit dem PD-Tool "PP" können aber auch ALLE Daten (Bilder,...) von normalen Programmen gelesen werden. Neuerdings gibts es ein neues Tool was es ermöglicht automatisch Daten beim Schreibzugriff auf Disk oder Pladde zu paken.


FAZIT:

Alle drei Programme sind zum Zweck die Kapazität von Disk oder Festpladde zu erhöhen bestens geeignet. Man muß sich nur überlegen, ob man oft Bilder,Sounds und Texte oder nur Programme paken will. Wenn man nur Programme pakt ist der Imploder4.O zu empfehlen. Bei allem anderen außer Progamme (und natürlich auch Programme) ist einem XPK und der PowerPacker ans Herz zu legen. Eine grafische Auswertung des Tests gibt aufm nächsten Bild !
