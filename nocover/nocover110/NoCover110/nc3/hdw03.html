Kostenloses Software Update 1.31.27 für die Vortex Golden Gate.

Für die AT Emulatoren Vortex Golden Gate 386SX und Golden Gate 486SLC ist ab sofort das Software Update 1.31.27 verfügbar.


Golden Gate 486SLC2 wird bereits vom Start weg mit dieser Version ausgeliefert.


Von den Verbesserungen dieser Software prfitierten alle Benutzer der Golden Gate.


Speziell bei Golden Gate 486SLC und Golden Gate 486SLC2 kann durch die Einstellung einer FAST OPTION eine Steigerung der Geschwindigkeit bis zu 20 % erreicht werden.


Wer eine 3.5" Leerdiskette und einen an sich selbst adressierten und mit 3,- frankierten Rückumschlag an Vortex schickt, erhält kostenlos das Update.
