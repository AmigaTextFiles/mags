Das Anim-Menü: vJetzt werden alle Menüpunkte aufgelistet und es kommt zu dem jeweiligen Menüpunkt eine Erklärung dazu:

Anim/Pinselanim/Laden: Damit kann man eine Pinselanimation Laden. Eine Pinselanimation ist praktisch ein Pinsel mit vielen Einzelbildern. Hat man die Pinselanimation geladen und man drückt einmal auf die Maustaste, so erscheint ein etwas anderes Bild. Sie können das ausprobieren, indem Sie die linke Maustaste gedrückt halten und einmal über den Bildschirm fahren, dann sehen Sie das jede Pinselseite einmal abgedruckt wird.

Anim/Pinselanim/Sichern: Dient zum Speichern von Pinselanimationen

Anim/Pinselanim/Greifen: Damit kann man eine Animation als Pinselanim aufnehmen. Nehmen wir an Sie hätten 10 Frames erstellt und auf jeder Seite ist ein etwas anderes Bild. Dann wählen Sie den Menüpunkt Anim/Pinselanim/Greifen aus und damit können Sie jetzt aus einer normalen Animation eine Pinselanimation machen.

Anim/Pinselanim/Einstellen: Mit diesem Menüpunkt können Sie Einstellungen für eine Pinselanimation vornehmen z. B.: Dauer der Pinselanimation und Richtung der Pinselanimation...

Anim/Pinselanim/Verwenden: Hier wird der zuletzt aufgenommene Pinsel zum aktuellen Pinsel.

Anim/Pinselanim/Frei: Hier kann ein Pinsel freigegeben werden(z. B. wenn der Speicherplatz knapp ist). Ein freigesetzter Pinsel kann allerdings nicht mehr zurückgeholt werden. Also passen Sie auf das Sie nicht aus versehen einmal einen Pinsel löschen den Sie später vielleicht noch einmal gebrauchen könnten.

Anim/Laden: Damit kann eine Animation in den Speicher geladen werden.

Anim/Sichern: Hier kann eine Animation abgespeichert werden.

Anim/Bewegen: Dieser Menüpunkt dient dazu, den Bewegen Requester einzublenden. Fangen wir oben an:

Es sind 6 Eingabefelder zu sehen, in die man positive oder negative Zahlenwerte eingeben kann. Es sind Felder für die X, Y und Z Achse vorhanden und zwar einmal für die Distanz und einmal für den Winkel. Der Knopf Null bewirkt, daß alle Felder wieder auf Null zurückgesetzt werden. Mit den Feldern Beschleunigung, Verlangsammung und Zähler hat man die Möglichkeit, die Animation zu verlangsamen bzw. zu beschleunigen. Ganz unten sind auch Gadgets für Vorschau, Ende, Laden, Sichern usw... die sich ja fast von selbst erklären.

Anim/Frame/Anzahl: Dort kann man bestimmen, wieviele Frames man in seiner Animation verwenden möchte. Wenn Sie die "Gedehnt" Animationsmethode gewählt haben, und Sie rufen diesen Menüpunkt auf, dann wird automatisch angezeigt wieviele Frames möglich sind.

Anim/Frame/Frame dazu: Bei diesem Menüpunkt kann man bestimmen wieviele Frames in die Animation eingefügt werden sollen.

Anim/Frame/Kopieren: Hier wird ein Requester eingeblendet mit dem man das aktuelle Frame kopieren kann.

Anim/Frame/löschen: Hier erscheint ebenfalls ein Requester indem das aktuelle Frame, eine Serie von Frames oder sämtliche Frames sich löschen lassen.

Anim/Steuerung/Feld Ein/Aus: Damit wird das Animations-Kontrollfeld eingeblendet, das unten am Bildschirmrand erscheint. Hier kann man Frames löschen, hinzufügen, den Leuchttisch einblenden, zu einem beliebigen Frame gehen usw. Alle auf dem Animations-Kontrollfeld aufgezeigten Möglichkeiten lassen sich auch über Tastenkombinationen oder durch die Menüs steuern. Am schnellsten geht das natürlich über die Tastenkombinationen.

Anim/Steuerung/Rate: Jetzt kann man eingeben wieviele Frames pro Sekunde abgespielt werden sollen.

Anim/Steuerung/Bereich: Dort kann bestimmt werden ob die Animation ganz, oder ob nur eine Serie von Frames abgespielt werden soll.

Anim/Steuerung/Vorheriges bis Ping-Pong: Mit diesen Menüpunkten, die ich im 2. Teil schon erklärt habe, lässt sich eine Animation in vielfältiger Weise abspielen. Außerdem kann man auch zu einem bestimmten Frame wechseln. Die Möglichkeiten hier lassen sich auch über Tastenkombinationen oder über das Animationskontrollfeld steuern.

Anim/Methode/Gerafft: Diese Animationsmethode verbraucht weniger Speicher, da mehrere Frames im Speicher untergebracht werden können. Allerdings kann es vorkommen das es im RAM-Speicher schnell zur Fragmentierung kommt. Das animierte Malen und das Laden und Speicher geht hier allerdings langsamer vor sich.

Anim/Methode/Gedehnt: Hier wird jedem Bild der volle erforderliche Speicher zugewiesen. Sie brauchen also wesentlich mehr Speicher. Bei dieser Animationsmethode wird der Speicher nicht fragmentiert und das animierte Malen und das Laden und Speichern geht hier schneller vor sich.

So, daß wäre geschafft. Jetzt wissen Sie einiges über Animationen und Pinselanimationen. Natürlich geht probieren über studieren. Falls Sie etwas noch nicht so genau verstanden haben, so können Sie ja einmal im Handbuch nachschauen oder sich einmal mit mir in Verbindung setzen.
