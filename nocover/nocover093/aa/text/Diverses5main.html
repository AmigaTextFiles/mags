5. Kurzmeldungen (von Oliver Tacke)

---------------------------------------------------------------------------
 Software (kommerziell)
---------------------------------------------------------------------------

 68k/vermischt

 Für Besitzer der Mediator-Multimedia-CD ist ein Update auf Version 1.08
 erhältlich und enthält neue Versionen älterer Treiber.
 [http://www.elbox.com]

 Ebenfalls erschienen ist Version 2.6 der MediatorUP-Software.
 [http://www.elbox.com]

 Besitzer des G-REX-Boards können sich Soundkartentreiber für die Chipsets
 ESS-Solo-ONE und FM-801 herunterladen. Die Chipsets finden u.a. Verwendung
 in einigen Karten von Teratec.
 [http://www.dcecom.de]

 "ProStationAudio" aus dem Hause Audiolabs ist in einer Titanium-Version
 erschienen, die gehörig überarbeitet wurde. Eine Demo ist verfügbar.
 [http://www.audiolabs.it]

 Das DTP-Programm PageStream ist in Version 4.1 erschienen und bietet nun
 u.a. von Haus aus komfortable Unterstützung für Tabellen.
 [http://www.grasshopperllc.com]

 Der Routenplaner "AmiAtlas" von Gerd Frank ist in Version 6 erhältlich.
 Ein erstes Update wurde ebenfalls veröffentlicht.
 [http://go.to/amiatlas/]

 Das Programm "Mediapoint", ein Präsentationsprogramm ähnlich Scala, wird
 weiterentwickelt und um Grafikkarten- sowie AHI-Unterstützung bereichert.
 Erscheinen wird die Software bei GTI/Schatztruhe.
 [http://www.compcity.nl/mediapoint/]

 Cloanto hat Version 5.0 von "Amiga Forever" fertiggestellt und im
 Vergleich zum Vorgänger diverse Änderungen vorgenommen, u.a. ist es
 wesentlich schneller und enthält eine neue Version von Picasso96.
 [http://cloanto.com/amiga/forever/]

 "Warp3D" wird Unterstützung für die Grafikchips Permedia3 sowie ATI-Radeon
 1 und 2 erhalten.
 [http://www.hyperion-entertainment.com/_amiga/index.html]

 Vom Brennprogramm "MakeCD" ist die 8. Betaversion 3.2d erschienen.
 [http://makecd.core.de/]

 Für die Bildverwaltungssoftware "PhotoFolio" ist ein Update auf Version
 2.2 veröffentlicht worden.
 [http://www.scouts.org.au/steeplesoftware/photofolio/en/index.html]

 Für das PCI-Busboard "Prometheus" ist eine Betaversion des
 Grafikkartentreibers für den S3-Chipsatz veröffentlicht worden.
 [http://www.matay.pl/ANGindex.html]


 PowerPC

 "ProStationAudio" aus dem Hause Audiolabs ist in einer Titanium-Version
 erschienen, die gehörig überarbeitet wurde.
 [http://www.audiolabs.it]


 MorphOS

 "CygnusED" ist in Version 4.21b für MorphOS erhältlich.
 [http://www.morphos.de]


---------------------------------------------------------------------------
 Software (nicht kommerziell)
---------------------------------------------------------------------------

 68k/vermischt

 Die memory.library von Thomas Richter ist in Version 43.5 erschienen. Mit
 ihr ist es möglich, das AmigaOS um virtuellen Speicher zu erweitern. Die
 Library ist allerdings kein Patch, sondern sie muss explizit von
 Programmen verwendet werden.
 [http://ftp.wustl.edu/pub/aminet/util/libs/Memlib.lha]

 Für den Disk-to-HD-Installer "WHDLoad" sind wieder eine Vielzahl neuer
 Pakete erhältlich, u.a. für Amazing Spiderman, Operation Wolf und
 Heimdall 2.
 [http://www.whdload.de]

 Der Dateimanager "Diskmaster" steht in Version 2.5RC1 zum Download bereit.
 [http://home.t-online.de/home/rudolph-riedel/archive.html]

 Das Programm "MCP" hat einen neuen Autor bekommen und liegt nun in leicht
 verbesserter Form in Version 1.35 vor.
 [http://www.dgroppe.de/]

 Der Messenger AmigAIM wurde in Version 0.9437beta veröffentlicht.
 [http://www.missingpiece.com/AmigAIM/]

 Das Tool "SRename" wurde in der finalen Version 3.0.0 herausgegeben.
 [http://users.forthnet.gr/ath/cnicol/index.htm]

 Von "YAM", dem wohl beliebtesten Amiga-E-Mail-Clients ist Version 2.3p1
 erschienen, um die MUI-Sicherheitslücke zu schließen.
 [http://www.yam.ch]

 Version 2.4 Beta des beliebten E-Mailers "YAM" ist verfügbar.
 [http://www.boing.net/yam/]

 "EnvHandler", das Env: transparent auf die Festplatte auslagert, ist in
 Version 1.13 erhältlich.
 [http://home.wtal.de/stephanr/downloads/envhandler.lha]

 Der Dateimanager "ClassAction" ist in Version 4.31 erschienen.
 [http://www.martin-elsner.de]

 Der Mailer und Newsreader "Microdot II" ist als Betaversion 1.4.16
 verfügbar.
 [http://www.vapor.com/md2]

 Das "SmartFileSystem" ist in Version 4.1 veröffentlicht worden, jedoch ist
 es nicht kompatibel mit älteren Versionen. Ältere Versionen von SFSCheck
 und SFSSalv funktionieren nicht mehr.
 [http://home.wtal.de/js/index.phtml?SFS4.lha]

 "MagicMenu" ist in der Version 2.33 herausgegeben worden, nachdem Jens
 Langner und Stephan Rupprecht die Entwicklung weiterführen.
 [http://www.magicmenu.de/]

 "SimpleMail" ist als Version 0.12 erhältlich. Behoben wurde u.a. die
 Sicherheitslücke, die durch MUI entstehen kann.
 [http://simplemail.sf.net/]

 Der Workbench-Ersatz "Scalos" ist in Version V40.15 beta herausgegeben
 worden. Einige Plugins erhielten ebenfalls ein Update.
 [http://www.scalos.co.uk]

 Das Programm "MP3TagEditor" ist in Version 2.0 erschienen.
 [http://jacadcaps.tripod.com/]

 Die Scanner-Farbkorrektur-Software "ICS" ist in Version 2.03 erschienen.
 [http://www.ics.coloraid.de/]

 Version 44.130 des "akPNG-Datatype" von Andreas Kleinert steht zum
 Download bereit.
 [http://www.ar-kleinert.de/]

 Die Sequencer-Software "Bars'n'Pipes" wurde in Version 1.07
 veröffentlicht.
 [http://home.t-online.de/home/alfaj/alfa.html]

 Das Astronomie-Programm "Digital Universe" ist in Version 1.03 erschienen.
 [http://www.syz.com/DU/amiga/]

 Version 44.6 des WarpTIFF.datatype steht zum Download bereit.
 [http://www.nanunanu.org/~oliver/warptiff.html]

 Der IRC-Client "AmIRC" ist in Betaversion 3.5.21 für registrierte Benutzer
 erhältlich.
 [http://www.vapor.com/amirc/]

 Hardware

 die Funktion mit dem OnBoard-Ex-Zorro-Board für den A1200 und des Silver-
 Surfers sicherstellt.
 [http://www.jschoenfeld.de/]

 Diverse Treiber für die "VarIO-Karte" stehen zum Download bereit.
 [http://www.jschoenfeld.de/]

 Für die "X-Surf" ist eine Version des ADSL-Treibers erschienen, der
 Probleme der Vorgängerversion beseitigen soll. Außerdem enthält das Archiv
 auch eine neue Version des appp.device, das auch für Nichtbesitzer der
 "X-Surf" sinnvoll ist.
 [http://www.jschoenfeld.de/]
