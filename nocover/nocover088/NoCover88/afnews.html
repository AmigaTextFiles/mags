Voodoo-X Update 1.7.2001
19:55
Voodoo-X ist in der Version 1.5 erschienen.

Hier mal einige Features:

  • Currently supports over 50 archive formats.
  • Retrieve files from within DMS archives (and other disk based archive formats too) without having to extract the entire archive to disk or a virtual disk device like FMS
  • Browse archives inside archives. Quickly gain access to that document inside that LHA inside that LZX inside that DMS that just happens to be inside that ZIP file. ;-)
  • Drag one or more archive icons from Workbench into the main window to browse their contents.
  • Support for multipart archives.
  • Virus checking. If you have Georg Hoermann's xvs.library system installed then you can check archives for file viruses. You can also specify if every file extracted should be checked.
  • Fast decompression, thanks to xadmaster.library. Note that this really depends on the client being used. Some clients are coded in C and might be noticeably slower than the original archive program on older CPUs.
  • Very easy-to-use user interface, thanks to MUI.
  • Quickly copy, move or delete archives using pull down menus. This can be highly convenient when going through download directories.
  • Search window. Search for strings or filename patterns inside archives. Uses assembly optimized string search routines for greater speed.
  • Quickly view pictures, documents or similar files that are inside archives simply by double clicking on a file. Voodoo-X will then extract it and start the viewer of your choice.
