3. Ankündigung: Atomic v1_1

»TITLE

    Atomic1_1

AUTHOR

    John Chufar
    email: chufarj@telepath.com

DESCRIPTION

    - Atomic Clock synchronizer for the Amiga
    - Accurately sets the Amiga Battery Backed-up
      and System Clocks from the Atomic Clock
    - All ARexx code (easy modifiable, and quick to execute)
    - Flexible operation for daylight savings time
    - Sets ANY Amiga clock in ANY timezone in the world
    - Compensates for internal operation
    - Very small and quick
    - Run from CLI or WB (using supplied icon)

NEW FEATURES

    - Year 2000 compatible with updated time servers
    - Fixes incompatbility "bug" which returned "bad args"
      when connected to Year 2000 upgraded UNIX timeserver

SPECIAL REQUIREMENTS

    - OS3.x (Uses Locale.prefs)
    - TCP: device (i.e. AmiTCP, Miami)
    - ARexx

AVAILABILITY

    Any Aminet site, e.g.
     ftp://ftp.wustl.edu/pub/aminet/comm/misc/atomic1_1.lha


PRICE

    Freeware

DISTRIBUTABILITY

    Public domain.
