'Recorder' can record and play back key and mouse input events.

Usage: Recorder -(r|p[q]|a[q]) FILENAME

The mode indicators have the following meaning:

-r Record input events to FILENAME

-p Play input events stored in FILENAME

-a Play events in FILENAME
