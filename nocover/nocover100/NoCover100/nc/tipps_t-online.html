T-Online-Team teilte mit:

Liebe Homepage-Besitzer,

der Webmaster hat auf dem Server für die Privaten Homepages einige neue Dateinamen konfiguriert, die für Index-Dateien verwendet werden können. Es handelt sich um die Namen "default.html" und "DEFAULT.HTML".

Somit sind derzeit folgende Namen für die Index-Datei möglich:

homepage.htm
homepage.html
HOMEPAGE.HTM
HOMEPAGE.HTML
home.htm
home.html
HOME.HTM
HOME.HTML
index.htm
index.html
INDEX.HTM
INDEX.HTML
index.wml
INDEX.WML
default.htm
DEFAULT.HTM
default.html
DEFAULT.HTML

Wenn Sie eine Datei mit einem solchen Namen auf den Server hochladen, wird sie an Stelle das Verzeichnis-Listings angezeigt, falls man eine Adresse in der Form
http://home.t-online.de/home/<eMail-Name>/ oder
http://home.t-online.de/home/<eMail-Name>/<Verzeichnis>/ aufruft.

Falls in einem Verzeichnis mehrere Dateien mit Namen aus dieser Liste existieren, wird diejenige Datei angezeigt, die hier zuerst aufgeführt ist.
