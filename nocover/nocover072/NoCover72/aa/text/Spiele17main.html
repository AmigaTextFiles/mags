11. Emulator-News
-----------------

Mathias "Amidog" Roslund ist auf der Suche nach einem erfahrenen
Amiga 68k Assembler Programmierer, der ihm hilft, seinen AmiGenerator,
eine Sega Mega Drive Emulator, auf 68k zu portieren. Eine PPC
Version ist schon seit fast einem Jahr vorhanden.

Von DarcNES, einem Multi-Konsolen Emulator, ist diesen Monat ebenfalls
eine neue Version erschienen. In der aktuellen Version wurden einige
Fehler behoben und auch einiges verbessert. 

Von der CyberGraphX Version von ANES, ANES CGX, ist auch eine neue
Version erschienen, in der einige Fehler behoben wurden. Die
aktuelle Version lässt sich momentan nur von der ANES Homepage
laden und befindet sich noch nicht im Aminet.

Die WarpUP Version des MAME-Emulators macht gute Fortschritte. Erste
Versionen  sind gerade im Betatesting (basierend auf der aktuellsten
Source-Code-Version von MAME) und eine erste Public Beta wird noch
für den Monat Dezember erwartet, laut dem Autor von WarpMAME, Colin Ward.
