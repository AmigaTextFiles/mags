Zum Teufel mit diesen DMS Archiven!

Produktvorstellung: DMS2HD

Heute möchte ich Euch mal ein kleines und brauchbares Tool vorstellen, welches ich letztens auf dem Aminet Set 7 gefunden habe: DMS2HD!

Wie der Name schon sagt, entpackt das Program DMS Files direkt auf die Festplatte. Bisher war man immer darauf angewiesen, DMS Files per CLI auf die Diskette zu entpacken, was bisweilen etwas nervig war. Das entfällt nun komplett: Man startet einfach DMS2HD und schon erscheint eine Filerequester, in dem das DMS File auswählen kann, sowie das Ziel angeben soll. Den Rest erledigt das Program automatisch!

Der eine oder andere Neueinsteiger wird sich vielleicht nun fragen, was DMS Files überhaupt sind: Nun, DMS ist sowas ähnliches wie ein Packer, mit dem man (früher) häufig komplette Disketten archiviert hat. Da man damit auch bequem kopiergeschütze Disks (NDOS) "archivieren" konnte, war dies ein beliebtes Program für die damalige "Szene". Auch Viren ließen sich so unbemerkt weitergeben. Neben diesem "Zweck" findet man DMS auch auf einigen CD`s (u.a. Aminet Set 1 und Amiga Magazin CD`s) wieder, so daß auch der normale User etwas von diesem Packer hat! Heute hat sich (zum Glück) LhA auf dem Amiga durchgesetzt, so daß man auf DMS kaum noch angewiesen ist. Wer dennoch DMS heute braucht (Demos etc.), der sollte sich dieses praktische Tool mal anschauen!

DMS2HD selber hat den Status "Freeware" und kann somit kostenlos und ohne Einschränkung aus dem Aminet geladen werden. User ohne Internetanschluss finden es auf dem Aminet Set 7 (CD A, Directory util/arc). Wer keine von diesen beiden Möglichkeiten hat, der kann mir auch eine Disk (Rückporto bitte beilegen) schicken und ich kopiere demjenigen das Tool! Der Autor dieses Tools ist übrigens Andreas Falkenhahn, der ja mittlerweile den Spieleteil in der Amiga Aktuell übernommen hat!
