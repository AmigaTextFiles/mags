                        +++----------------------+++
                        +++ HEUTE: Bubble Bobble +++
                        +++----------------------+++


        Ein Bericht von Christian Heitzer, der auf Tatsachen beruht.


 Man benötigt: Das Programm FakeMem.
               Das Programm CPU (auf der WB3.0 im C-Verzeichnis)

 Vorgehensweise: 1. Man kopiert das Programm FakeMem auf die
                    BubbleBobble-Disk (wenn möglich, eine Kopie verwenden!)
                 2. Man kopiert das Programm CPU auf die BubbleBobble-Disk
                    (oder eine Kopie, wenn möglich)
                 3. Man ändert die Startup-Sequence der BubbleBobble-Disk
                    um, indem man vor die eigentliche Startup-Sequence
                    folgende Zeilen hängt:

                    FakeMem
                    CPU NoCache NoBurst NoDataCache NoDataBurst

                    Die normale Startup-Sequence müßte wie folgt lauten:

                    Boot
                    EndCLI

                    Insgesamt müßte die Startup-Sequence dann so aussehen:

                    FakeMem
                    CPU NoCache NoBurst NoDataCache NoDataBurst
                    Boot
                    EndCLI
