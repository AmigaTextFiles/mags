Voraussetzungen:

Hardware:
*  Kickstart 3.0

Installation:
ReqAttack wird mit der Sprachenunterstützung für Deutsch und Englisch von 
einem Installer-Skript an eine beliebige Schublade kopiert. Möchte man 
ReqAttack bei jedem Systemstart aktiviert haben, so muß man einfach das 
Executable "ReqAttack" in die WBStartup-Schublade ziehen.


Überblick:
ReqAttack ist ein Tool, welches das Aussehen alles Standard-Requester des
AmigaOS deutlich aufwertet. Jedes Programm, welches diese Dialogboxen
benutzt, kommt in den Genuß dieser neuen Optik. Den Dialogboxen können
folgende Eigenschaften zugewiesen werden:

+ Logos (auch animiert möglich !)
+ Buttons können eine Grafik erhalten
+ Buttons können individuell eingestellt werden (Größe, Position, etc.)
+ Requesterfensters kan ein Hintergrundmuster zugewiesen werden
+ erweiterte Tastaturbedienung (ähnlich Win95+)
+ Verschiebung des Requesterfensters durch Anklicken an beliebiger Stelle
  innerhalb des Fensters möglich (!)
+ Wahl der Position, an der sich der Requester öffnet


Praxistest:
Nach der Installation und einem Neustart des Systems (oder Doppelklick auf 
das Executable ReqAttack) sollte man eine Aktion durchführen, die einen 
Requester hervorruft. Beispielsweise durch den Aufruf des Menupunktes 
"Version, Copyright" im Menu "Workbench". Sofort fällt das hübsche Logo 
links neben dem Requestertext und der kleine grüne Haken innerhalb des 
Buttons auf. Sehr schön. Erinnert sofort an Win95:

 - grafische Ansprechende Requester

Negativ:
 - keine Punkte
