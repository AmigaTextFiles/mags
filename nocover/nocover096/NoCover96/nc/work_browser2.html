So sieht es aus...

Weil einige wohl nur auf Amiga-OS die Browser kennen, ist es vielleicht interessant zu sehen, was CSS kann.

im MicroSoft Internet Explorer:

MSIE zeigt CSS an, genau wie es im obigen Code geschrieben ist...

und so mit Netscape...

Netscape zeigt CSS an, aber hier sieht man, daß es nicht richtig angezeigt wird. Es fehlt das unterstreichen der Überschrift ...

und der gleiche Code mit AWeb3.

AWeb zeigt CSS absolut nicht an, kein Hintergrund, keine Fontfarbe und kein italic...

und bei IBrowse

ist es genau das gleiche...

wie auch bei Voyager

Voyager zeigt CSS nicht an...

Bei Voyager ist mir noch aufgefallen, daß Javascript:history.back[] nicht funktioniert. ;( Und auch der Öffnen-Requester kann nicht auf Ram Disk: und ähnliche Pfade zugreifen. Vielleicht liegt es an der Demo-Version, aber die Demo von IBrowse hat da keine Probleme.

Fazit des zweiten Teils: Die Amiga-Browser können kein CSS, aber auch bei Windows gibt es Unterschiede in der Darstellung.

Mir persönlich gefällt bei den Amiga-Browsern AWeb am besten und am wenigsten sagt mir Voyager zu.
