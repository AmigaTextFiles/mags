Workshop zum Dateiprogramm XBase 6.4

von Rafaî Kaczmarczyk © 1995-2000


XBase 6.4 ist ein Programm mit dem es relativ einfach möglich ist Dateien - Banken herzustellen.


Die Installation von XBase 6.4

Die Installation ist einfach, das Verzeichnis mit XBase 6.4 auf die eigene Festplatte ziehen und die beiden in dem Verzeichnis ENVARC liegenden Icons nach ENVARC auf der eignen Festplatte verschieben oder InstallIcons ausführen.

Es hann noch die popupmenu.library is Copyright © 2000 Henrik Isaksson und die openurl.library written by Troels Walsted Hansen Installiert werden. (Beide brauchen zum einstellen MUI)
Mit der popupmenu.library ist es möglich, abhängieg von der Mausposition ein Popupmenu in XBase 6.4 darzustellen.
Mit der openurl.library ist es möglich, eine Verbindung zum Internet (Browser) herzustellen.


Der Start von XBase 6.4

Der Start von XBase 6.4 ist einfach, mit der Maus auf das Icon von XBase 6.4 klicken und es erscheint ein leeres Fenster.
»Dieses leere Fenster erscheint immer wenn XBase 6.4 so Aufgerufen wird!«
Nun kann unter dem Menüpnkt * (Sternchen) eine Beispiel-Datei sich angeschaut werden. Sie ist auf englisch oder polnisch!
