4. Kurzmeldungen (von Oliver Tacke)

---------------------------------------------------------------------------
 Software (kommerziell)
---------------------------------------------------------------------------

 Das erste Update (Boing Bag) für OS3.9 ist erschienen und umfasst neben
 Bugfixes auch diverse neue Funktionen. Es ist z.B. nun endlich möglich,
 während eines Kopier- oder Löschvorgangs parallel weiterzuarbeiten.
 [http://www.amiga.com/3.9/download/BoingBag39-1.lha]

 Die Treibersoftware MediatorUP steht in Version 1.8 zur Verfügung.
 [http://www.elbox.com]

 Für GoldED Studio 6 ist ein neuer Systaxparser erschienen. Neben Bugfixes,
 die am alten vorgenommen wurden, unterstützt der neue nun u.a. PHP und
 JavaScript.
 [http://www.aminet.net/pub/aminet/text/edit/html-php.lha]

 Die Treibersoftware für den FastATA Controller ist als V2.4 erhältlich.
 [http://www.elbox.com]

 ProStation Audio ist in der neuen Version 4.05 erschienen und bietet u.a.
 einen nativen Treiber für die Repulse Soundkarte.
 [http://www.audiolabs.it]

 Neue Treiber für TurboPrint sind erschienen für Canon S400, 450, 4500 und
 800, Epson Stylus Color 680, 880 und 980 sowie für Epson Photo 790, 890,
 und 1290.
 [http://www.irseesoft.de]

 Eine öffentliche Betaversion von MakeCD (V3.2 beta 6) ist erschienen und
 bietet u.a. VideoCD-Unterstützung.
 [http://makecd.core.de]

---------------------------------------------------------------------------
 Software (nicht kommerziell)
---------------------------------------------------------------------------

 PlayGUI ist in Version 3.1 erschienen und enthält diverse Bugfixes.
 [http://narr.tsx.org]

 Der Grafiktabletttreiber FromAldiHyd ist in Version 2.2 erschienen. Neue
 Features sind die Unterstützung von Makrotasten und das Abspielen von
 Sounds beim Drücken einer Taste.
 [http://www.platon42.de/]

 Der Bootvorgang-Verschönerer Rainboot ist in Version 3.2 veröffentlicht
 worden und ist u.a. bessere Unterstützung von Grafikkarten bereichert wor-
 den.

 Eine neue Version der asyncio.library für WarpOS ist erschienen.
 [http://members.tripod.de/achimste/download/asyncioppc.lha]

 Der MPEG Player Frogger ist als Version 1.64 veröffentlicht worden und
 soll um 25% schneller sein als die Vorgängerversion.
 [http://frogger.rules.pl]

 Das Grafikprogramm Perfect Paint ist in Version 2.5 erschienen.
 [http://gothic.fr.free.fr/amiga/download/PfPaint_V25.lha]

 Das CDTV Developer Kit ist ab sofort für jedermann zu haben.
 [http://home.t-online.de/home/amiga-freak/cd32developer.html]

 Das JFIF Datatype von Andreas Kleinert, akJFIF, steht in Version 44.96 zum
 Download bereit.
 [http://www.ar-kleinert.de]

 Ebenfalls von Andreas Kleinert stammt die neue Version des MPEG Players
 akMPEG V4.60.
 [http://www.ar-kleinert.de]

 Für Scalos existiert eine neue Version des Persistant Windows PlugIn.
 [http://scalos.striatum.org/products/scalos/modules/Scalos-Plugin_PWindows
 .lha]

 Version 14.0 des Diskettenspiele-auf-Festplatten-Installers WHDLoad ist
 erschienen.
 [http://www.whdload.org]

 Ein Programm der gleichen Art, nämlich JST, ist in Version V4.7 verfügbar
 und ab sofort Freeware. Außerdem steht der Quellcode bereit.
 [http://jffabre.free.fr/amiga/patches.html]

 Von Oliver Roberts stammt ein Datatype für Adobe Photoshop Files.
 [http://www.nanunanu.org/~oliver]

 Das Wav-Datatype von Stephan Rupprecht ist in Version 41.10 erhältlich.
 [http://home.wtal.de/stephanr/]

 Amster, der Client zum umstrittenen mp3-Dienst Napster, ist in Version
 0.8a veröffentlicht worden.
 [http://212.10.48.147/amster/files/0.8a/Amster-main.lha]

 Eine neue Version von EZP@gerNG steht bereit.
 [http://www.sfxsoft.de/de/]

 Hardware
---------------------------------------------------------------------------

 Die G-Rex 1200 PCI ist lieferbar. Das Board bietet BlizzardPPC Besitzern
 fünf PCI 2.1 Steckplätze.
 [http://www.dcecom.de]

 Von Grzegorz Jureszek wurde ein weiteres PCI-Board namens "Prometheus"
 angekündigt. Das Board soll für ZorroIII-Slots ausgelegt sein und vier
 PCI-Slots bieten.
 [http://www.matay.pl]
