9. Kostenloser Upgrade-Patch für Directory Opus Magellan II erschienen

»FOR IMMEDIATE RELEASE

GPSoftware is proud to demonstrate its continuing support for the Amiga 
with the release of a FREE UPDATE patch to its popular File Management and 
Workbench Replacement software, Directory Opus Magellan II.

The update is available free to registered users and upgrades Opus Magellan 
5.8 to 5.81. While some new features have been added such as support for 
the NewMouse standard, improved OpusFTP operations, and improved ease of 
use, this release is mainly a service update providing solutions to a 
number of issues identified by end users.

It is available direct from the GPSoftware web site at

http://www.gpsoft.com.au«
