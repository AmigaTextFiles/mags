Juha-Pekka Jokela
USB 2.0 IDE Adapter on sovitin, jonka avulla sisäinen ATA- tai ATAPI-laite voidaan kytkeä tietokoneen USB-porttiin ja muuttaa se USB-laitteeksi. USB 2.0 IDE Adapter -paketti sisältää USB 2.0 IDE -sovitinkaapelin, ajuri-CD:n, asennusohjeen, lyhyen Ultra DMA 33 -kaapelin ja virtalähteen IDE-laitetta varten.
Sinimusta sovitinkaapeli on siistiä ja tukevaa tekoa. Kaapelin päissä olevista USB- ja IDE-pistokkeista saa mukavan otteen ja kaapelin kytkeminen ja irrottaminen yleensä hieman tiukkaan IDE-liitäntäänkin käy helposti.

Sovitinkaapeli saa pientä miinusta lyhyytensä vuoksi, sillä USB-väylään liitettyä IDE-laitetta ei pysty asettamaan normaalikokoisen tornikotelon päälle tai juurelle niin, ettei laite jäisi roikkumaan sovittimen varaan.

Toisin kuin sovitinkaapelissa, virtalähteen johdoissa on reilusti pituutta. Musta virtalähde on pieni ja kevyt ja se on varustettu virtavalolla, joka osoittautui hyödylliseksi hiljaisia IDE-laitteita käytettäessä. USB 2.0 IDE Adapterin suurin rakenteellinen heikkous on virtalähteestä lähtevä virtapistoke, joka kytketään IDE-laitteeseen. Pistoke on nimittäin samanlainen kuin tietokoneen virtalähteen 5,25" laitteiden pistoke. Pistokkeen saa suhteellisen helposti kiinni laitteeseen, mutta sen irrottaminen käsin, johtoa ja laitetta rikkomatta, on vaikeaa.

USB 2.0 IDE Adapterin mukana tuleva CD sisältää ajurit Windows 98/2000/XP, Mac OS 9.x ja Mac OS X 10.x -käyttöjärjestelmille. Uusimmat Windows- ja Mac-käyttöjärjestelmät sisältävät jo valmiiksi tarvittavat ajurit, joten CD:llä olevia ajureita ei tarvitse asentaa.

Sovittimen toimivuutta testattiin AmigaOne-, Mac Mini, iBook- ja Windows PC -tietokoneilla. AmigaOnessa oli viimeisin ennakkoversio (Pre-release Update #3) AmigaOS 4.0 -käyttöjärjestelmästä, kannettavassa iBookissa Mac OS X 10.0, Mac Minissä Mac OS X 10.3 ja PC:ssä Windows XP.

USB 2.0 IDE Adapter -sovitin soveltuu mainiosti tilanteisiin, joissa siirrettävät tietomäärät ovat suuria (kymmeniä tai satoja gigatavuja), tietoja ei voida syystä tai toisesta siirtää verkkoyhteyden kautta tai IDE-laitetta halutaan käyttää väliaikaisesti toisessa tietokoneessa.

Piiloon tietokoneen kotelon sisälle jäävien laitteiden käyttäminen ulkoisina USB-laitteina sovittimen avulla ei ole erityisen kaunista ja tiedonsiirtokin kahden koneen välillä käy helpommin verkkojohtoa pitkin. Sovitin on kuitenkin monikäyttöinen ja halpa verrattuna ulkoisiin USB-massamuistilaitteisiin.

Vaikka USB 2.0 IDE Adapterin data- ja virtakaapelia ei voi kytkeä väärin eikä käyttäjän tarvitse välittää kytkettävän IDE-laitteen Master/Slave-asetuksesta, ei adapteria kuitenkaan voi suositella aivan aloitteleville käyttäjille. Jos kovalevyn tai CD-ROM-aseman irrottaminen ja asentaminen takaisin pöytäkoneeseen kuulostaa pelottavalta, ei USB 2.0 IDE Adapter ole tarkoitettu sinulle.
