Sakunetin virallinen nodetilanne 14.1.2001

Node - noden Sakunet-osoite, jota käytetään esim. netmailia lähetettäessä
Boss - boxi, jonka kautta kyseinen node hoitaa viestiliikenteensä
Fl - Flageja, jotka vastaavat tiettyjä, verkon sisäisiä asioita
BBS Name - Boxin nimi
Sysop - Boxin operaattorin nimi
Number - Boxin numero
City - Boxin sijaintipaikka
Fl (Flags): N = Uusi tuleva BBS, P = Privaatti, M = Mail-gate (ei BBS) D = BBS on tilapäisesti poissa linjoilta (down)
Fl numero: Seuraavien ehtojen painoarvoista yhteenlaskettu summa, koskee ainoastaan Amiga Zonea pollaavia nodeja:
1 = Uudet Saku-lehdet lähetetään ko. nodelle
2 = MovieDataBasen viikottaiset diffit lähetetään ko. nodelle
3 = Satelliittien taajuustaulukko lähetetään ko. nodelle, aina kun siitä saadaan uusi versio.
Muutoksia:
  • Graveyard (65:51/3) poistettu, ei aktiivista toimintaa aikoihin
