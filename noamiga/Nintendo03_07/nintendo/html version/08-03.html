 Nintendo 64: Quake 2 nutzt Expansions Pak
 
 Die Nintendo 64-Version von Quake 2 wird das 4 MByte-Speicherpaket
 nutzen, somit wird auch Quake 2, wie viele andere Spiele die 1999
 erscheinen werden, in einer hohen Auflösung von 640x480 laufen.

 Nintendo 64: Shadowman auf 256 Mbit Cartridge
 
 Acclaim's Sommerhit nimmt weiter Züge an. Nachdem dieses Spiel auf
 einer 128 Mbit Cartridge geplant war hat man sich bei Acclaim
 umentschieden. So soll Shadowman jetzt auf der großen Cartridge
 erscheinen. Weiterhin gab man bekannt das wie in der PC Version alle
 Cut Szenen und Sprachanteile enthalten sein sollen. Wenn man sich
 überlegt das RE 2 wohl auf einer 512 Mbit Cartridge erscheint. Da
 ist also noch einiges möglich.

 Weihnachten '99 wird Shooterzeit.


 Das könnte wieder ein Weihnachten werden an dem sich die Toptitel um
 die Gunst der Käufer reißen. Nachdem der Termin für Perfect Dark
 immernoch auf Weihnachten'99 steht kommt nun ein vielleicht weiteres
 Highlight dazu. Amorines - so heißt das Spiel das die Forsaken
 Entwickler Probe in der Mache haben. In diesem Spiel könnt Ihr
 wahlweise im Ego Stil oder in der Third Person Perspektive auf
 Ballerjagd gehen. Dabei will Probe eine Mischung aus Turok 2 und
 Forsaken basteln die viel Wert auf Multiplayeraction legt. So soll
 es auch einen Cooperativ Modus geben. Oft gewünscht und noch nie so
 richtig realisiert. Weiterhin wird das Spiel das RAM Pak sowie das
 Rumble Pak unterstützen.

 Game Boy Color: Game & Watch Gallery 3 angekündigt.


 Nachdem hierzulande der 2. Teil zum Start des Game Boy Color auf den
 Markt kam werkelt man in Japan an den letzten Zügen des dritten
 Teils. Neben dem altbekannten Spielprinzip soll vor allem das
 optische besonders zur Geltung kommen. Geplant ist ein Release in
