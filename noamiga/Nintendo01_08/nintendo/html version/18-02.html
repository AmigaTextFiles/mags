Nintendo 64: Neues Bowling Spiel

 Athena gab heute bekannt, das man Ende März ein neues Bowling Spiel
 herausbringen wird. Das Spiel mit dem Namen Super Bowling kann man zu
 4 spielen und jeder Charakter wird über Special Moves verfügen.
 Damit das Spiel nicht schnell langweilig wird, kann man auf
 verschiedenen Bahnen bowlen und sich in kleinen Sub-Games (wie bei
 Tekken 3) warm spielen.

 Nintendo 64: Releasetermin für Command & Conquer
 
 Gute Nachrichten für alle die immernoch auf Command & Conquer für das
 N64 warten. Zumindest für die USA wurde jetzt ein genauer Termin
 genannt. So soll Command & Conquer dort am 28.Juni erscheinen. Da
 das Spiel von Nintendo vertrieben wird sollte dieser Termin auch
 gehalten werden. Zu einem Deutschlandrelease wurde offiziell noch
 nichts bekanntgegeben.

 Nintendo 64: THQ Spiele Line Up für 1999


 Nachdem THQ die Lizens für die WCW an EA abgeben muß besinnen sich
 die Wrestling Macher auf andere Genres und bieten dieses Jahr einige
 Titel an die schon auf anderen Plattformen erschienen sind. Aber
 auch eigene Neuproduktionen sind dabei. Hier die Liste der Titel die
 1999 erscheinen sollen.


 - Nomen Quest (RPG)
 - Rugrats Board Game
 - Nuclear Strike
 - Road Rash 64
 - B.A.S.S. Masters Classic (Angelsimulation)
