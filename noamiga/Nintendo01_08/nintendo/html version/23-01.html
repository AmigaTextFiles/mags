Nintendo 64: Neues 3D-Mortal Kombat für Nintendo 64

 Gegen Ende des Jahres wird ein neuer Mortal Kombat-Teil in 3D für das
 Nintendo 64 in den USA erscheinen. Das Spiel soll dann auch die 4
 MByte-Speichererweiterung ausnutzen.

 Nintendo 64: Command & Conquer kommt!


 Nintendo gab bekannt, daß am 26. Juni Command & Conquer
 für das Nintendo 64 erscheint. Somit hat Nintendo in diesen Jahr mit
 das im Spätsommer erscheinende Starcraft 64 die besten
 PC-Strategiespiele für ihre Konsole.

 Game Boy Color: Mehr Color only-Spiele!


 Möchte Nintendo softwaremässig langsam zum Game Boy Color wechseln?
 Damit die Vorzüge des Game Boy Color richtig genutzt werden können
 (höhere Taktrate, 56 Farben), werden in naher Zukunft immer mehr
 Module erscheinen, welche nur für den Game Boy Color bestimmt sind.
 (Bisher waren die meisten Color Module abwärtskompatibel). Auf alten
 S/W Game Boy werden diese Games dann nicht mehr laufen.
 ---------------------------------------------------------------------
 Game Boy: Sind die Tage des Klassikers gezählt?


 Es scheint fast so, als hätte der ewige Bestseller Game Boy langsam
 aber sicher ausgedient. Das Nachfolgemodell Game Boy Color steht bei
 allen Entwicklern hoch im Kurs. Selbst Nintendo hat jetzt
 angekündigt GBC-only Spiele zu entwickeln. Dies sei sinnvoll, um die
 höhere Taktrate und den Farbmodus des neuen Modells besser zu
 unterstützen. Trotz einer gewissen Kompatibilität zwischen den
 Geräten, dürfte damit der Untergang des
 Schwarz-Weiß-Unterhaltungskünstlers eingeläutet worden sein.
 ---------------------------------------------------------------------
 Game Boy Color: Prince of Persia

 Game Boy Color: Wetrix kommt


 Der beliebte Nintendo 64-Puzzler namens Wetrix wird auch für den Game
 Boy Color im Laufe des Jahres erscheinen. Das Spiel wird von
 Imagineer in Japan entwickelt.
