 Game Boy Color: Neue Verkaufszahlen zum Game Boy Color
 
 Dieses Jahr feiert der Game Boy seinen 10. Geburtstag. Nach wie vor
 ist der Run auf Nintendos Handheld ungebrochen. Dieses Jahr wird die
 70 Millionen Grenze an verkauften Geräten erreicht. Ebenso ist man
 zuversichtlich dieses Jahr 6 Millionen Game Boy Color verkauft
 zu haben.

 Game Boy Color: ZELDA DX in überarbeiteter Version!


 Seit einigen Wochen steht ZELDA DX für den Game Boy Color in
 deutschen Kaufhäusern. Das Spiel ist zwar ein Dual Cardridge (also
 auch kompatibel mit dem alten Game Boy), ist aber nicht nur
 coloriert worden, sondern bietet auch eine Menge neuer Features, wie
 einige Zwischenbilder, oder kleine Spielchen, welche bewältigt
 werden müssen. Das Spiel kostet ca. 69 DM und sollte von jedem ins
 Auge genommen werden, der einen Game Boy Color besitzt und noch kein
 Zelda auf dem kleinen Freund erfahren hat.
