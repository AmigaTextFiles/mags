Nintendo 64: Chris Webber das Zugpferd für NBA Jam 2000

 Nachdem im letzten Jahr Keith van Horn als Rookie auf der Packung von
 NBA Jam 99 prangte sicherte sich Acclaim dieses Jahr den Namen von
 Chris Webber von den Sacramento Kings. Neben dem Aufdruck auf der
 Packung wird Webber auch in der neuen NBA Saison für Werbezwecke
 eingespannt.

 BigN: Donkey Kong 64 in gelbem Modul


 Für Deutschland und Europa plant Nintendo das Modul zu Donkey Kong 64
 in der Farbe Gelb heraus zu bringen. In den USA ist sogar geplant
 das DK64 Pak mit einer grünen Konsole und dem gelben Modul auf den
 Markt zu bringen.

 Game Boy Color: InterAct USA planen Memory Card für den GBC


 Was sich recht futuristisch anhört scheint schon fast fertig zu sein.
 InterAct USA planen eine Memory Cartridge, die man in den Schacht
 des Game Boy Color steckt. Auf dieser Cartridge steckt dann das
 entsprechende Spiel. Danach ist es möglich auf der Memory Card 100
 Spielstände abzuspeichern. Unterstützt werden alle Spiele, die ihre
 Stände auf einer internen Batterie abspeichern. So wie etwa bei
 Zelda DX. Bereits im Oktober soll dieses Teil für 19,95 Dollar in
 den USA zu haben sein.

 Game Boy Color: Driver kommt auch für Game Boy Color


 Die englische Abteilung von GT Interactive hat jetzt Driver für
 Nintendos Game Boy Color angekündigt. Der Titel wird von Crawfish
 Interactive entwickelt und soll noch Ende des Jahres erscheinen. Der
 Entwickler hat bereits genügend Erfahrungen bei Umsetzung für den
 Game Boy Color und war u.a. auch für Game Boy-Versionen von Rainbox
 Six, Street Fighter Alpha, Space Invaders und WWF Attitude
 zuständig.
