 BigN: Nintendo zeigte 64DD auf der E3


 Auf der E3 zeigte Nintendo ein Model des 64DD das in Japan auf
 jeden Fall erscheint. Gezeigt wurde eine Version des Mario Paint
 Nachfolgers Mario Artist. Dieses Tool das aus mehreren 
 Einzelprogrammen besteht wird auch als Modulversion erscheinen.
 Allerdings abgespeckt. Denn bei der 64DD Version wird es eine
 Zusatzcartridge geben an die man alle möglichen exteren Quellen
 anschließen kann. Einen genauen Termin für das 64DD in Japan nennt
 man aber immernoch nicht. Lange dürfte es aber nicht mehr dauern.
 ---------------------------------------------------------------------
 Nintendo 64: Eigenes Foto in Perfekt Dark
 
 Wenn Ende des Jahres Perfekt Dark kommt kann man ein besonderes
 Feature im Spiel nutzen. Wer über ein bis dahin erhältliches Game
 Boy Pak verfügt und ein Game Boy Camera besitzt kann über diese
 Geräte ein Foto in das Spiel integrieren. Mit einem kleinen Tool
 kann man dieses mit Texturen überziehen und im Multiplayermodus
 einsetzen.

 Nintendo 64: Donkey Kong 64 mit Expansion Pak
 
 Donkey Kong 64 wird wohl nicht nur spielerisch einen neuen
 Meilenstein setzen. Es wird auch das erste Spiel sein das nur mit
 einem Expansion Pak läuft. Ohne kann man das Spiel nicht starten.
 Doch keine Angst. Nintendo schlägt wieder einmal zwei klappen mit
 einmal. Wie bei Starfox mit dem Rumble Pak so wird Donkey Kong 64
 inklusive eines Expansion Pak's ausgeliefert. Da sich im Preis
 nichts ändern wird werden sich auch die, die schon ein Expansion Pak
 besitzen nicht weiter ärgern.
