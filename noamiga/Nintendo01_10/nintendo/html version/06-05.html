Nintendo 64: Run Run Run !!!

 Erstmals vorgestellt: NFL Madden 2000. Das Spiel soll sich in Sachen
 Gameplay an die beiden auf dem N64 erschienen Vorgänger halten. Auf
 optischer Seite wird erneut das Expansion Pak unterstützt, um somit
 eine höhere Auflösung zu erreichen; ob es allerdings grafisch mit
 NFL QBC 2000 von Acclaim mithalten kann, ist noch ungewiss.
 Ebenfalls mit 4 Spieler Modus wird dieses Spiel sicherlich auf der E3
 genauer vorgestellt.

 Nintendo 64: Im Zeichen des Sports 


 Weitere Ankündigungen zu den bekannten Sportspielreihen gab EA Sports
 bekannt. Tiger Woods 2000, eine actionlastige Golfsimulation wird
 Euch auf 18+ Golfkursen sicherlich die grösste Freude bereiten.
 Ausserdem könnt Ihr neben einem Solospiel auch mit 3 Freunden
 gleichzeitig einlochen. Ob es sich allerdings als Partyspass eignet
 sei dahingestellt. Der Release ist  von EA Sports auf das dritte
 Quartal festgelegt worden.
 ---------------------------------------------------------------------
 N64 und GBC: Daikatana für das N64 und den GBC


 Der PC Shooter von Quake Macher Romero wird auch auf dem N64 und dem
 GBC erscheinen. Verantwortlich wird für diese Projekte die Firma
 Kemco sein. Obwohl auf der E3 nicht spielbares zu sehen sein wird
 will Kemco beide Produkte Ende des Jahres auf den Markt bringen. Die
 GBC Version soll dabei eher ein Adventure im Zeldastil sein.
 ---------------------------------------------------------------------
 Nintendo 64: Interplay's E3 Line Up
 
 Auch die Firma Interplay wird auf der E3 mit N64 Spielen vertreten
 sein. Das sind zum einen das schon lang angekündigte Earthworm Jim
 3D, Caramageddon 64 sowie Wild Water World Championships. Ein Wave
 Race Konkurrent der aber im Cartoon Look daherkommt. Laut Interplay
 sollen neben 6 Strecken insgesamt 27 Spielcharaktere enthalten sein.
 ---------------------------------------------------------------------
 Nintendo 64: Neues zu Turok: Rage Wars


 Der Ableger von Turok 2, der dieses Jahr um die Weihnachtszeit
 erscheinen soll wird Turok: Rage Wars heißen und sich ganz auf
 Multiplayeraction konzentrieren. Neben einem Single Player Modus
 wird es mehrere Multiplayermodi geben. Aufbauend auf einer
 optimierten T2 Engine sollen die Multiplayermodi denselben Spielspaß
 bieten wie der Single Player Modus. Turok: Rage Wars soll ebenfalls
 auf der E3 vorgestellt werden.
