Homepage eine Seite über das neue Betriebssystem ange- legt, hier ein Bild OS 3.5 Website:

Dieser Website zufolge, sieht das im Einzelenen so aus, um es richtig nutzen zu können, benötigt man ersten einen guten Rechner (CPU) und die andere Hardware soll bzw. müßte auch einiges vorweisen, doch seht mal selbst: Hardeware-Voraussetzungen

Um das Amiga OS 3.5 überhaupt nutzen zu können ist erforderlich:

  • CD-ROM Laufwerk
  • Festplatte
  • 68020 oder ein höherer Prozessor
  • Amiga 3.1 ROMs
  • 4 MB Fast RAM
Für eine schnellere und bessere Arbeit, sollte man haben:
  • 68030 oder ein höherer Prozessor
  • 8 MB Fast RAM
  • Grafikkarte und/oder Scandoubler
  • Modem
Um OS 3.5 voll ausnutzen zu können, benötigt man:
  • PowerPC mit 68060 Prozessor
  • 16-bit Soundkarte
  • 32 MB Fast RAM
  • Grafikkarte
Und das soll in Version 3.5 verändert bzw. neu eingefügt werden.
Features der Version 3.5:
  • (RTG) Grafikkarten Unterstützung
  • Besserer Drucker Support
  • Neues bzw. überarbeitetes Festplatten-Dateisystem
  • Arexx-Update
  • Neue Shell Befehle
  • Dokumentation in HTML
  • (RTA) Soundkarten Support
  • Neue Treiber
  • Verwaltung/Nutzung aller Festplatten über 4 GB
  • CD File System (Unterstützung der "shiny disc", Musik von CD hören beim Arbeiten) (Red.: Ich denke mal, es werden keine Kopfhörer mehr benötigt, sondern alles läuft über einen internen Player)
  • Neues Design der Workbench-Oberfläche
