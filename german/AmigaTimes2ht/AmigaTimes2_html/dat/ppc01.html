Cyberstorm PPC

Die Cyberstorm PPC ist eine High-End Karte für alle A3000/4000 Computer und hat ebenfalls einen RISC Prozessor in verschiedenen Takfrequenzen. Der Sockel für 68040/68060 Prozessoren ist vorhanden kann mit bis zu 128 MB 64 BIT breitem Ultra Fast Ram ausgestattet werden. Ein serienmäßig integrieter Controller erlaubt bzw. bietet Leistungsreserven mit modernem Massenspeicher! Als Anschluß steht ein 68 poliger Ultra Wide SCSI Connector zur Verfügung. Über diesen können weitere Karten angeschlossen werden, z.B. die CyberVisionPPC. Upgrade Möglichkeiten werden auch allen Usern geboten, die einen 68040/68060 Prozessor besitzen oder eine 68k Cyberstorm Karte haben.
