X-DVE - ein kreatives Effektwerkzeug
- Ein Workshop für den Draconer von Gerhard Erler -

X-DVE ist als eigenständiges Programm zur Animationserstellung allgemein bekannt. Auch unabhängig von Movieshop hat es durch seine Effekte schon viele Anwender begeistert. Durch den neu hinzugekommenen XFA-Operator für MovieShop haben wir nun die Möglichkeit, XFA-Animationen direkt in ein Videoprojekt zu integrieren.
Die folgende Anleitung sollte als eine Anregung für die Arbeit mit X-DVE betrachtet werden, wobei die wichtigsten Arbeitsschritte an einem Beispiel nachvollzogen werden. Die vollständige Erläuterung der Funktionen, Möglichkeiten und Animationsfeatures ist nicht Ziel dieses Beitrages. Erläutert wird, wie mit Hilfe von X-DVE eine Überblendung erstellt werden kann, wobei als Beispiel eine statische+dynamische Überblendung mit lediglich einem Effekt gewählt wurde.

1.
Starten Sie das Programm MovieShop. Öffnen Sie Ihr Projekt. Erzeugen Sie eine neue Schnittliste. Legen Sie 2 Szenen ohne Überlappung nebeneinander in eine Spur.
In der Szenenliste fahren Sie das letzte Bild der ersten in der Schnittliste placierten Szene an und kopieren das Bild (Block markieren -> Block kopieren). Exportieren Sie das Bild am besten in die Schublade 'X-DVE:Frames'. (Im Menü "Einstellungen -> Import/Export" sollte Export auf "Vollbilder" gesetzt sein)

2.
Um das Bild weiterbearbeiten zu können, muß folgender Schritt ausgeführt werden: (Nutzer eines System-Datatypes, das Bilder in JPEG oder IFF24BIT laden kann, überspringen diesen Punkt einfach !) Starten Sie das Programm AdPro. Laden Sie das exportierte Bild. Achten Sie darauf, daß unter AdPro im Menü "Settings" -> "Dithering" auf "Floyd-Steinberg" steht, und unter "Palette" "Custom Palette" nicht angeklickt ist. Betätigen Sie "Set Render Screen" und wählen Sie 256 Farben. Betätigen Sie "Execute". Klicken Sie mit der linken Maustaste, sobald das Bild vollständig aufgebaut ist. Wählen Sie unter "Savers" "IFF", und "256 color Image" und bestimmen Sie einen Namen für das Bild, das Sie am besten wieder in der X-DVE:Frames-Schublade abspeichern.

3.
Starten Sie jetzt das Programm X-DVE. Wählen Sie bei Project -> New an. Wählen Sie die Auflösung des MovieShop-Projektes. CCIR ist als solches schon im Auswahlfenster. Andere tragen Sie bitte per Hand ein. Wählen Sie 256 Farben. Wählen Sie bei Object -> Brush aus. Wählen Sie bei Objekt -> Ins (Einfügen) aus. Im nun geöffneten Fenster "Brush Object Requester" wählen Sie -> Select und wählen das vorher mit AdPro konvertierte Bild. Stellen "Fast Remap" ein. Nun steht im ersten Feld "Brush:Bildname". Drücken Sie auf das Feld "Start". Tragen Sie in dem nun geöffneten Fenster "Object setup" folgendes ein: Out:50 (Länge der Überblendung). Wählen Sie bei "DVE Out"-> "Select" und klicken auf "Wind". Im nun geöffneten Fenster wählen Sie z.B. "Carf", bei "Blocks" wählen Sie z.B. "Large" aus. Bestätigen Sie mit OK. Drücken Sie unter "Animation" auf "IFF". Wählen Sie in dem nun geöffneten Fenster bei "Path" die Schublade X-DVE:Frames aus und bestimmen Sie den Namen für die Animationsbildsequenz. Drücken Sie auf "All". Drücken Sie auf "Render". Drücken Sie unter "Animation" auf "Mem". Im nun geöffneten Fenster drücken Sie auf "Path" und wählen das erste Bild der soeben erzeugten Bildsequenz in der Schublade X-DVE:Frames aus. Drücken Sie danach auf "All". Drücken Sie auf "Convert". Drücken Sie auf "Save XFA" und wählen Sie einen Namen für die Animation und die Schublade (z. B.X-DVE:XFA).

4.
Wechseln Sie nun wieder auf MovieShop. Legen Sie den XFA-Operator unter den Anfang der 2. Szene. Die Länge des XFA-Operators sollte in diesem Beispiel 50 Halbbilder betragen (1 Sek.). Wählen Sie die mit X-DVE erstellte Animation aus, drücken Sie dazu "Select XFA" (Die Animation sollte unter "X-DVE:XFA zu finden sein). Stellen Sie bei "Masking" -> Color 0 ein.

Starten Sie die Berechnung !

Um verschiedene Effekte auszuprobieren, ist es nicht notwendig, oben beschriebene Schritte jedesmal durchzuführen. Unter X-DVE befindet sich eine Taste "Preview", mit deren Hilfe sich alle Effekte und Effektkombinationen in einer Vorschau in Echtzeit durchspielen lassen. Eine Auswahl, welcher Effekt geeignet ist, fällt damit besonders leicht. X-DVE enthält aber auch umfangreiche 3D-Betitelungsfunktionen (einfach bei Object->Text auswählen), die zwar Monument nicht ersetzen, aber andere geniale Effekte ermöglichen, Animations Brushes (für dynamisch+dynamisch Überblendung) und einen universellen Sternenfluggenerator. Diese unter X-DVE als Objekte bezeichneten PlugIns lassen sich ebenfalls mit den bereitgestellten Effekten kombinieren, wobei z.B. eine Text-Animation wesentlich leichter zu erstellen ist, da sowohl der Text als auch die Animation in X-DVE generiert werden und dann nur noch mit dem XFA-Operator in MovieShop eingebunden werden.
X-DVE ermöglicht eine unendliche Vielfalt an Effekten und Kombinationen und kann damit dem kreativen Videobearbeiter wärmstens empfohlen werden.
