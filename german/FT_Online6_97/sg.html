Schatztruhe

  Schatztruhe gehört zu den produktivsten Firmen was CD-Roms für 
  Amiga betrifft. Nicht nur die berühmte CD-Rom Serie Aminet
  wird auf der Messe zu bekommen sein.  Auch der berühmteste 
  Editor wird auf der Messe sein Comeback feieren CygnusED.


  Das der Editor ziemlich gut werden wird, dürfte schon daran 
  liegen das für die Programmiereung Bruce Dawson und 
  Olaf Barthel verantwortlich sind. 


  Auch auf eine neue Version des Malprogramms Personal Paint 7.1
  darf man sich freuen. Das Programm besitzt inzwischen 180 Arexx
  Befehle, kann neue Dateiformate lesen wie PhotoCD, GIF, JPEG, 
  PBM uva. Sehr geeignet sind viele Funktionen fürs Internet wie 
  das erstellen von GIF Animationen, auch ein Map Editor ist 
  inzwischen implementiert. Die neue Version bietet auch endlich 
  den ersten PowerPC Code.


  Interessant dürfte auch die neue Wordworth 7 Version sein
  die ebenfalls auf der Messe vorgestellt werden soll. 


  Weiterhin wird sich Schatztruhe natürlich auch auf den 
  Verkauf der verschiedenen CD-Roms konzentrieren.
