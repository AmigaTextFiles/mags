Polygon

Wie zeichnet man ein Polygon das vom Standard abweicht ?
Am besten fangen sie mit einer Linie an und setzen einen Punkt dazu. Um Punkte dazu zu zeichnen drücken sie die "Alt" Taste und die Linke Maus Taste wo sie den Punkt benötigen.Nun rufen sie das Licht - Polygon Menü ( Taste "d" ) auf und verbinden die Punkte mit Kubisch oder anders. Nun verschieben sie die Punkt so bis die gewünscht Form erreicht ist.
Damit das fertige Objekt auch genauso aussieht wie ihr Polygon unterteilen sie die Rundung. Dazu rufen sie das Menü "Polygon Runden" auf. Vorher müssen sie aber noch mindestens 3 Punkte Aktivieren, dazu benutzen sie die Taste "Ctrl" oder "Shift" und die linke Maus Taste.
Nun können sie die zweite Rundung zeichnen und wen sie wollen im Licht - Polygon Menü die Verbindung ändern. Im diesem Beispiel ist es B-Spline. Dies können sie immer weiter ausbauen mit so vielen Rundungen und Linearen Verbindungen wie sie wollen.
Stellen sie nun im Licht - Polygon Menü die Verbindung wieder auf Linear und schließen das Polygon und fertige ist das Polygon.
