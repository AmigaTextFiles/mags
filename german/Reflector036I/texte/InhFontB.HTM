Inhalt FontBox 001 - 025

Die Objektdisketten der FONTBOX enthalten, wie der Name schon sagt, Fonts für REFLECTIONS ab der Version 2.03. Solltet Ihr die Version 2.02 benutzen, so besorgt Euch die REFLECTOR-Ausgabe 004. Dort befindet sich ein Patch, um die Version 2.02 auf die Version 2.03 'abzudaten'. Da mit Sicherheit einige unter Euch noch mit REFLECTIONS 1.6 oder kleiner arbeiten, so hilft Euch dies allerdings nicht. Deshalb könnt Ihr bei uns eine Extra-Version der FONTBOX anfordern, welche die Fonts für diese REFLECTIONS - Version enthält (als Einzelobjekte). REFLECTIONS 3 - User können die Fonts ohne Einschränkungen verwenden.
