AmigaScene - Download

 
Here you can download old issues of 'AmigaScene' and some useful tools. They're packed with 'lha', the most widely used packer for the Amiga. We provide 'lha' for DOS, Windos, MAC and Unix, too.

Unpacking archives

 
After you have saved one of the archives from above to your harddisk you can unpack it into any directory you like. Note that a subdirectory "amigascene" will be created there. We recommend to unpack all archives into the same directory to avoid troubles with local links across different issues. Additionally all files in the most recent archive are updated versions and therefore you'll have to overwrite older files when prompted. We also recommend to unpack older archives first; then new ones. Use this command:

lha x -T archivename.lha targetdirectory
