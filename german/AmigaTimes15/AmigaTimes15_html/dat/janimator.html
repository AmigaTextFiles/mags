JAnimator


Das Hauptziel dieses Programmes ist es, den User mit einer noch ästethischeren Workbench Umgebung beeindrucken zu können. Denn JAnimator kann per Shell Kommando ein Bild oder eine Animation laden und diese auf der WB darstellen. So kann man zum Beispiel eine wunderbar gerenderte, sich drehende Weltkugel auf die WB Oberfläche legen. Jetzt wäre es natürlich noch beeindruckender, könnte man per Klick auf die Animation auch noch seinen Lieblingsbrowser starten. Kein Problem - auch das ist möglich. Allerdings muß man dafür die Befehle wissen, denn JAnimtor hat leider kein anwenderfreundliches GUI für diese Zwecke. Doch keine Panik! So schwer ist das nicht: Einfach ein Shell Fenster öffnen und JAnimator mit dem Befehl 'JAnimator FILE=xyz.iff EXECUTECLI=abc' starten (empfehlenswert ist es hier, JAnimator mit einem 'RUN >NIL:' Befehl aufzurufen)! Das Programm würde nach diesem Beispiel ein Bild oder eine Animation namens 'xyz.iff' laden, sie am WB Screen darstellen und auf Klick das Programm 'abc' starten, welches z.B. ein Browser sein könnte.

Doch JAnimator bietet noch wesentlich mehr Möglichkeiten zum Verschönern der Workbench. So kann man mit dem Befehl 'ALPHA' das Bild auch noch transparent darstellen lassen, wobei hier die Stärke der Transparenz mit einem Wert zwischen 0 und 255 festgelegt werden kann. So ein Effekt kann manche PC User neidisch werden lassen. Natürlich wird einem auch die Möglichkeit gegeben, das Bild immer an einer gewissen Stelle erscheinen zu lassen, indem man die x- und y- Koordinaten bestimmt. Ebenfalls ist es möglich, die Geschwindigkeit der Animation zu wählen, und die Genauigkeit der Darstellung zu definieren. Für OS3.5 Benutzer bietet JAnimator zusätzlich noch eine EXECUTEWB Funktionen an, mit der man direkt Workbench Programme starten kann. Der Autor empfiehlt aber für diejenigen, die mit einer älteren OS Version arbeiten, das Freewaretool WBLoad, welches eine ähnliche Funktion innehat.

Eines der Highlights an JAnimator ist aber, dass man sich problemlos eigene Animationen erstellen lassen kann, indem man eine Serie von Bildern auslesen läßt. Gibt man z.B. pic00 an, sucht JAnimator gleich nach einem pic01, einem pic02, pic03 uns so weiter. Auf diese Weise sucht sich das Programm selbst die Bilder für eine Animation zusammen.




JAnimator ist Freeware und kommt erfreulicherweise mit Installer und optimierten CPU-Versionen. Ein paar nette Demos zum Ausprobieren sind auch enthalten. Für das Darstellen der Animationen und Bilder wird Cbergraphics 3 benötigt, allerdings funktionierte JAnimator bei mir auch unter Picasso96.

Wer schon immer mal ein paar Freunde mit seiner Workbench beeindrucken wollte oder einfach nicht aufhören kann, diese zu 'veredeln', muß JAnimator unbedingt einmal ausprobieren.
