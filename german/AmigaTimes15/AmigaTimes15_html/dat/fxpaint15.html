fxPaint Update 1.5


fxPaint hatten wir ja in der Version 1.0 schon in unserer Ausgabe 4/99 genaustens unter die Lupe genommen und Euch vorgestellt. Auf der Messe in Neuss gab es nun das erste größere Update für dieses Bildbearbeitungsprogramm.

Das Update wird auf CD-Rom ausgeliefert und benötigt die Original fxPaint CD der Version 1.0. Neben den Zusatzs-Kit`s EK 1 und EK 2 beinhaltet das Update natürlich eine ganze Menge an neuen Features. Erstmals gibt es für jeden Prozessor ein optimierte Version, es wurden neue HTM Tutorials eingefügt. Das Laden des Programms dauert auf einem normalen Amiga mit 68030 Prozessor und 80 MB Ram etwas länger als bei der ersten Version, doch das ist nicht weiter schlimm. Ansonsten erwartet den User kaum ein anderes Erscheinungsbild als zuvor. Einige Optionen wurden erweitert, so kann man jetzt durch ein Klick auf das "Öffnen" Icon, gleich auswählen, ob ein Bild geöffnet, ein Fraktal erstellt, Feuer erzeugt, ein Bild gescannt oder ein Bild von einer Digital Kamera eingelesen werden soll. Als Neuerung kann man auch die Überarbeitung des fxAlbums bezeichnen, hier wurde für eine bessere HTML Übersicht gesorgt (u.a. Bildergröße einstellbar/skalierbar), die Diashow verbessert und jetzt kann man auch via Mausdoppelklick gleich ein Bild öffnen bzw. einladen lassen.
    Was ist noch neu? Hier ein kleine Liste von Features:
  • ARexx-Launcher
      - Starten eines ARexx-Skripts direkt aus fxPAINT heraus
  • verbesserte Toolbar
      - nun vertikal verschiebbar
      - optisch "komprimierte" und verbesserte Prozentanzeige
  • verbesserter Plugin-Launcher
      - nur noch ein Klick zum Starten eines Plugins via Popup-Menü
  • verbessertes Bubblehelp-Verhalten
  • neue Effekte
      - JPEG Fraktur entfernen
  • Digitalkamera-Unterstützung
      - benötigt VHI Studio
      - transparente Einbindung von VHI Studio
  • erweitertes stargate.plugin


Fazit:
Alles in allem kann man das Update auf jedenfall empfehlen, es erweitert das Original um nützliche Funktionen und einige kleine Fehler wurden verbessert, das Programm stellenweise optimiert.
