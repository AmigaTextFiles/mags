STFax 3.2

Seit einiger Zeit ist nun die neueste Version des Faxprogramm STFax zu beziehen. Zum Lieferumfang gehören unter anderem eine Diskette auf der sich STFax befindet und auf der anderen MUI (Magic User Interface). Die Installation geht schnell von statten. STFax belegt ca. 1,5 MB Festplatten speicher. Zum begin der Installation mußten wir einen 20 stelligen Code eingeben. Das Programm lag uns in der englischen Version vor und wir besorgten uns den deutsch Katalog von Haage und Partner. Den deutschen Katalog bekommen nur registrierte Benutzer.

Das Programm

Zum Start von STFax wird die Konfiguration automatisch erledigt, so das man sich unötige Konfigurationen erspart. Schon von der Userfreundlichen Bedienung überzeugt das Programm sehr und die optische Gestaltung ist wirklich gelungen. Dank MUI. Das Programm wird von Hilfesprechblasen be- gleitet, womit man dann eigentlich keine Schwierigkeiten haben sollte. Mit STFax lassen sich Faxe verschicken, zu bestimmten Zeiten, empfangen und Faxe abrufen.

Voice-Calling Dabei handelt es sich um ein Voice-Calling System. Das bedeutet, das ihr Amiga zum Anrufbeantworter werden kann. Sie tun per Telefon oder Mikrofon ihre Begrüßung auf die Platte sprechen. Wenn dann der Anrufer bnicht sofort auflegt, wird seine Nachricht als Wave Datei gespeichert. Und Qualität ist wirklich gut. Sie können noch ein automatisches Service System einrichten, das es dem Anrufer erlaubt, eine Fax Übertragung zu starten, was mir den Telefontasten geschieht.

Mini-BBS
Dies klinkt ungewöhnlich, ist aber sehr nützlich. Dadurch können sie ihren Amiga in einen Server verwandeln. Sie benutzen eine Schublade von ihrer Festplatte hierfür, das Daten zum herunterladen enthält. Dies ermöglicht dem Anwender per Modem die Daten herunter zu Laden zu jeder belibiegen Zeit. Doch sie müßen noch einen Secure-Doors einrichten. Dann kann sich jeder User mit dem Passwort und dem Usernamen, die Daten downloaden auf ihre eigene Festplatte. Außerdem erlaubt es STFax, das man über das Modem einen Freund oder Verwandten anruft, wärend man Arbeiten tut.

Das erstellen von Faxe Dies ist eigentlich sehr sinnvoll. Sie benutzen entweder das normale STFax Programm dafür oder erstellen sich per Wordworth ihr Fax und schicken es als Druckerausgabe an ihren Amiga. Dazu benutzt man am besten Turbo Print, und gibt dort die Drucker Device `STFaxPrt` ein und geht auf Workbench Treiber. Dann fragt STFax nach dem Fpad und dem Namen der Datei. Somit ist es auch möglich Grafiken direkt einzubinden.

Fazit:
STFax ist ein muß Amiga User, die häufig das Fax System verwenden. Seine User freundliche Bediener Oberfläche und die optische Gestalltung machen es sehr komfortabel und es läst andere Faxprogramme hinter sich. Alles in allem gab es bei dem Test keine Mängel. Zumindest konnte ich keine Mängel feststellen. Ohne Zweifel ist dies das derzeit beste Fax Programm auf dem Amiga Markt.
