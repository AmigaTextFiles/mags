JPEG-Optimizer (Amiga)

Leider ist dieses Tool nur dem Amiga Anwender vorbehalten. Es hat zwar nichts direkt mit REFLECTIONS zu tun, ist aber dennoch ganz praktisch. Das Tool optimiert JPEG-Files wahlweise mit oder ohne Qualitätsverlust. Ohne Qualitätsverlust (Verluste, wer mag das schon ?) verkleinert es die meisten JPEG-Bilder um ca. 1-5%. Das ist zwar nicht viel, aber jedes gespartes Byte kann wertvoll sein, wenn ein Bild dadurch so gerade noch auf eine Diskette paßt.

Im Archiv befindet sich ein Read-Me File, daß Ihr Euch reinziehen solltet. Das Tool JPEG-Optimizer stammt von Timo Kokkonen.
