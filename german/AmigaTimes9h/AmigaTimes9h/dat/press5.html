Amiga Envoy 3

Amiga Envoy is the only networking software specifically designed for Amiga computers. It allows you to access resources like files and directories on connected Amiga computers as if they were on a local disk. Naturally, this also includes removable media support. Printing over the network is also a feature of Amiga Envoy. A single printer can even be accessed by multiple Amiga computers simultaneously.

Via security and management features, you can also set up access restrictions in a flexible way for multiple users.

Amiga Envoy is easy to use and easy to configure with its font sensitive graphical user interface. Contrary to common types of networking software, there is no need to restart after changing any configuration settings.

The open programmer's interface of Amiga Envoy allows you to add your own or freely redistributable extensions and services to, e.g., update date and time from a remote Amiga computer or even use a remote Amiga computer with your local keyboard and scree n. All the required documentation is included.

Amiga Envoy requires OS 2.04 or higher, at least 1MB of memory, and any SANA-II compliant networking hardware. A harddisk is recommended.

Produktinformation

Amiga Envoy 3

Amiga Envoy ist die einzige Netzwerk-Software, die gezielt für den Amiga entwickelt und auf dessen Fähigkeiten abgestimmt wurde. Mit Amiga Envoy verwenden Sie Dateien und Verzeichnisse anderer Amigas, als wären sie auf einem lokalen Datenträger. Dies schließt selbstverständlich auch Wechselmedien ein. Auch Drucken über das Netzwerk ist kein Problem, sogar für mehrere Amigas, die gleichzeitig einen einzigen Drucker ansprechen.

Die Sicherheits- und Management-Funktionen ermöglichen es Ihnen, Zugriffsbeschränkungen für verschiedene Benutzer flexibel einzurichten.

Amiga Envoy ist einfach zu benutzen und zu konfigurieren und bietet Ihnen natürlich eine fontsensitive graphische Benutzeroberfläche an. Anders als bei herkömmlicher Netzwerk-Software müssen Sie nicht jedesmal neu starten, nachdem Sie irgendeine Einstellung geändert haben.

Über die offene Programmierschnittstelle von Amiga Envoy fügen Sie jeder zeit eigene oder frei verteilbare Erweiterungen und Services hinzu, wie zum Beispiel Services zum Abgleich der Echtzeituhr oder zur Fernsteuerung anderer Amigas. Die gesamte Dokumentation ist mit auf dieser CD-ROM enthalten.

Amiga Envoy benötigt OS 2.04 oder höher, mindestens 1MB Speicher und eine beliebige Netzwerk-Hardware nach SANA-II Standard. Eine Festplatte wird empfohlen.
