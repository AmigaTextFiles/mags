CyberGraphX V4

CyberGraphX was designed to define an independent graphics standard for Amiga graphics boards. The full knowledge of more than 4 years of graphics board software development is included in this outstanding driver package.

Features of previous versions:

  • Universal, hardware independent CyberGraphX-Library
  • Drivers for many popular graphics boards included
  • Modular design, possible usage of multiple graphics boards
  • 15/16 and 24bit support for all Intuition screens incl. Workbench
  • Real Amiga look and feel incl. draggable screens
  • No chipmem required for screen display
  • Extensive support of all features by application software from leading software providers

And now new in v4:

  • Improved memory management and many bug fixes
  • Extended CGXMode program with many new features incl. VESA modes
  • CyberGraphX commodity
  • Direct interface to v43 picture.datatype
  • Extensive documentation and FAQ
  • All neccessary files and drivers on one CD

Requirements:

  • 68020, 68030, 68040 or 68060 CPU
  • AmigaOS 3.x
  • CyberGraphX supported graphics board

Produktinformation

CyberGraphX wurde als unabhängiger Standard für Amiga Grafikkarten entwickelt. Die Erfahrung aus über 4 Jahren Entwicklung von Amiga Grafikkartensoftware sind in diesem herausragenden Treiberpaket enthalten.

Hier sind einige der Features:

  • Universelle, hardwareunabhängige CyberGraphX-Library
  • Treiber für viele populäre Grafikkarten sind enthalten
  • Modulares Design, gleichzeitige Nutzung mehrerer Grafikkarten ist möglich
  • 15/16 und 24bit Unterstützung für alle Intuition Bildschirme inkl. Workbench
  • Das ganze Amiga-Feeling inkl. "ziehbarer" Bildschirme
  • Für die Bildschirmdarstellung wird kein ChipRam benötigt
  • Umfangreiche Unterstützung aller Features durch Anwendungssoftware vieler namhafter Hersteller

Neu in v4:

  • Verbesserte Speicherverwaltung und viele Fehlerkorrekturen
  • Erweitertes CGXMode Programm mit vielen neuen Funktionen inkl. VESA Modi
  • CyberGraphX Commodity
  • Direkte Anbindung an v43 picture.datatype
  • Ausführliche Dokumentation und FAQ
  • Alle erforderlichen Dateien und Treiber auf einer CD

Voraussetzungen:

  • 68020, 68030, 68040 oder 68060 Prozessor
  • AmigaOS 3.x
  • CyberGraphX unterstützte Grafikkarte
