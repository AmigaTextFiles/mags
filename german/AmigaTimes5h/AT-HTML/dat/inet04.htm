Finale Web Cruiser

Die Anzahl der WWW-Browser für den Amiga wird immer größer. Bisher haben sich ja bereits IBrowse, Voyager und A-Web einen recht großen Namen verschafft.

Jetzt soll noch ein weiterer Browser dazu kommen und zwar der Finale Web Cruiser aus der Softwareschmiede Finale-Development. Der neue Browser ist allerdings noch in der BETA-Phase, aber das erste offiziele Relaese wird im Juni/Juli zu erwarten sein. Der Web-Crusier wird den HTML 3.2 Standart sowie viele HTML-Zusatz Features von Netscape und dem Internet-Explorer unterstützen. Außerdem wird er bereits von Anfang an Javascript und Java unterstützen. Die Systemanforderungen für den Browser werden sein mindestens Amiga OS 2.1, 4 MB RAM und eine 68020er CPU. Wer nähre Infos zu dem neuen Browser will, schaut am besten unter www.finale-dev.com/webcruiser/index.html.

Wenn man den letzten Meldungen glauben schenkt, wird sich die Browser- Anzahl für den Amiga aber noch weiter erhöhen. Opera denkt an eine Amiga- Umsetzung wenn sich genügend Amiga-User auf der Opera-Homepage melden und das haben bisher schon jede Menge getan.

Aber noch ein weiterer Browser kündigt sich für den Amiga an, denn durch die Freigabe des Netscape 5.0 Source Code's, wirds vermutlich auch bald vom Netscape eine Amiga-Version geben. Aber wie heisst es doch so schön: Konkurrenz belebt das Geschäft.
