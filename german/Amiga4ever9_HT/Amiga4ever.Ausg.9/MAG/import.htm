IMPORTANT
---------

Für die Soundausgabe der 4ever sind noch einige Grundeinstellungen in Ihrem Browser notwendig.

Falls sie den Player : " AMFM-Play " noch nicht besitzen klicken Sie auf den Button " Download AMFMPlay " am Ende dieser Site und Speichern Sie diesen in Ihrem C: Verzeichnis.

--- IBrowse ---

Starten Sie Ihren Browser und rufen Sie den Menüpunkt " Einstellungen / Allgemein " auf. Wechseln Sie auf " Externe Anzeiger " und klicken sie auf " Neu ". Tragen Sie unter " Typ " im ersten Feld " Audio " ein und im Feld daneben " Modul ". Unter " Endung " tragen Sie bitte " Mods Mod Modul " ein. Unter Attribute tragen Sie bitte " %f QUIET " ein. Klicken Sie auf " OK " und speichern Sie die Einstellungen.


--- Voyager / Voyager NG ---

Starten Sie Ihren Browser und klicken Sie den Menüpunkt " Settings / Settings " an. Klicken Sie " Mime " im rechten Fenster an und dann " Add New ". Den Circle Button " Mime " stellen Sie bitte auf " Audio ". Im Feld daneben geben Sie " Modul " ein. Dann den Action Circle Button auf " Use External Application " stellen. Unter Application geben sie " C:AmFmplay " ein.
( Auch hier sollten Sie AmFmPlay besitzen bzw. den Download in das C: Verzeichnis vornehmen )
Danach OK anklicken und die Einstellungen Speichern.


--- AWeb ---

Starten Sie AWeb und klicken Sie den Menüpunkt " Settings / Browser Settings " an. Dort klicken sie auf " Viewers " und dann auf " ADD ". Geben Sie unter " Type " im ersten Feld " Audio " und im zweiten Feld " Modul " ein.
Unter Extensions geben Sie " Mod Mods Modul " ein.
Unter Command geben Sie " C:AmFmPlay " ein. Unter Arguments geben Sie " %s " ein. Speichern Sie mit dem Button " Save " und Fertig.

Wir vom 4ever Team werden uns aber darum kümmern, das solche Browser-Einstellungen bald der Vergangenheit angehören.
