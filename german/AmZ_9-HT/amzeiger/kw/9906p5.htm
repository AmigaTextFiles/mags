FlashUpdate von p5-PPC-Turbokarten

von
Peter Fischer

Wer stolzer Besitzer einer CyberstormPPC- oder BlizzardPPC-Turbokarte von Phase5 ist und die erste Softwareinstallation durch die der Karte beigefügten CD hinter sich hat, wird früher oder später Fehlermeldungen einiger neuer PowerUp-Programme erhalten, die die Meldung ausgeben, daß eine "PPC-Library" benötigt wird, die die Versionsnummer '46+' enthält.

Nun wird es also Zeit für ein Flashupdate des PPC-Roms!

Da dies einige Schritte beinhaltet, die nicht unbedingt ausführlich in den "ReadMe"-Dateien von Phase5 behandelt werden, hoffe ich, hiermit eine recht ausführliche Anleitung dazu geben zu können.

Die Installation beinhaltet drei Schritte, die ausgeführt werden müssen:

  • Besorgen der neuesten Phase5-Libraries und des aktuellen Update-Files.
  • Installation der Software
  • Das FlashRom-Update

Schritt 1

Kommen wir nun zum ersten Schritt, dem Besorgen der neuesten Phase5- Libraries und des aktuellen Update-Files.

Die aktuellen erhalten Sie, sofern Sie Internetzugang haben, beim FTP- Server von Phase5. Dieser ist unter 'ftp://ftp.phase5.de/pub/ppc/' erreichbar.

Dort downloaden Sie die zum Zeitpunkt dieser Texterstellung aktuellen Files herunter: 'ftp://ftp.phase5.de/pub/ppc/68060-170599.lha' und 'ftp://ftp.phase5.de/pub/ppc/FlashUpdates-300399.lha'.

Die Größe der Dateien beträgt 186 KB und 940 KB. Sichern Sie diese dann auf Ihrer Festplatte.

Schritt 2

Somit kommen wir zum zweiten Schritt, der Installation der Software.

  • Entpacken Sie das Archiv 68060-170599.lha in das RAM.

  • Nun wechseln Sie in das Workbench-Verzeichnis "LIBS" und löschen dort folgende Dateien:

    68060.library
    68040.library
    68040new.library
    68040old.library
    68040dummy.library

    Da jetzt die neuen, im RAM vorhandenen Libraries umbenannt werden und danach in das LIBS-Verzeichnis kopiert werden müssen, sollten Sie, um die einfachste und schnellste Methode zu verwenden, nun das CLI aufrufen und von dort aus weiterarbeiten.

  • Die folgenden Befehle werden nun im CLI eingegeben (natürlich immer mit einem abschließendem `Return`): CD Ram:
    copy RAM:Libs/68040.library libs:68040new.library
    copy RAM:Libs/68040dummy.library libs:68040.library
    copy RAM:Libs/68060.library libs:
    Endcli
  • Damit die neuen Libraries nun auch genutzt werden können, rebooten Sie jetzt Ihren Amiga.
Schritt 3

Nun kommen wir also zum FlashUpdate.

  • Wenn Ihr Amiga wieder gestartet ist, formatieren Sie bitte eine bootbare Diskette im Laufwerk DF0, die Sie zum Beispiel "FlashUpdate" nennen können.

    Via CLI lautet der Aufruf:

    format drive DF0: name Flashupdate
    install DF0:
  • Danach kopieren Sie das für Ihre Turbokarte notwendige FlashUpDate-File auf die Diskette.
    • Haben Sie eine CyberstormPPC-Turbokarte, kopieren Sie das File 'RAM:Flashupdates/PPCUpdate' auf die Diskette. (via CLI: copy RAM:Flashupdates/PPCUpdate DF0:)
    • Haben Sie eine BlizzardPPC68040-Turbokarte, kopieren Sie das File 'RAM:Flashupdates/BlizzPPC040Update' auf die Diskette. (via CLI: copy RAM:Flashupdates/BlizzPPC040Update DF0:)
    • Haben Sie eine BlizzardPPC68060-Turbokarte, kopieren Sie das File 'RAM:Flashupdates/BlizzPPC060Update' auf die Diskette. (via CLI: copy RAM:Flashupdates/BlizzPPC060Update DF0:)

    • Nun schalten Sie den Amiga aus und warten mindestens 2 Minuten!

    • Schalten Sie nun den Amiga wieder an und halten Sie die Taste "s" und beide Maustasten gedrückt, damit Sie in die "EarlyStartupControl" kommen.

    • Dort deaktivieren Sie alle angeschlossen Festplatten-Partitionen. (Disable HD0 u.s.w. im "BootMenu")

    • Klicken sie nun auf "boot with no startup-sequence" und halten Sie erneut die Taste "s" gedrückt. Bitte beachten Sie, daß sich Ihre "Flashupdate"- Diskette auch wirklich im Laufwerk DF0 befindet.

    • Nach dem erneuten Booten erscheint nun auf dem Bildschirm ein Shell- Fenster mit einem Prompt ("1:> ").

      Nun geben Sie den Namen Ihres Updateprogramms ein ("PPCUpdate", "BlizzPPC040Update" oder "BlizzPPC060Update") und drücken die `Return`-Taste.

    • Sollte eine Fehlermeldung direkt nach der Eingabe des gewünschten Updateprogramms erscheinen, ist das Ausschalten der Turbokarte durch das Niederdrücken der Taste "s" fehlgeschlagen. Wiederholen Sie dann das Update durch Drücken der "s"-Taste, wenn die Aufforderung dazu erscheint.

    • Jetzt führt der Rechner das Update durch. Sie erkennen dies nach kurzer Zeit am Erscheinen von Punkten (".") auf dem Bildschirm.

    • Nun warten Sie bis das Update fertiggestellt ist und die Meldung `done` erscheint.

      ACHTUNG: Führen Sie während des Updates auf gar keinen Fall einen Reset durch oder nehmen Sie den Amiga vom Netz!

      Dies würde die Turbokarte zerstören und Sie können dann auch nicht mehr booten!

    • Wenn das Update durchgeführt wurde, können Sie jetzt rebooten und alles sollte einwandfrei funktionieren.

      Wenn Sie einen Blick auf die Versionsnummer der "ppc-library" werfen, müßte jetzt die aktuelle Versionsnummer (aktuell V46.25) erscheinen.

Unter Umständen kann Ihr Workbenchscreen eine andere Auflösung als zuvor haben. In diesem Fall sollten Sie den Amiga erneut booten und dabei gleichzeitig die ESC-Taste drücken.

Dann kommen Sie in das Einstellungsmenü der Turbokarte. Dort stellen Sie im Menüpunkt "System" den Screenmodus von "31khz Default Screen" in "No 31khz Screen" um.

Nach einem erneuten Booten sollte der Screen nun wieder wie gewohnt erscheinen, unter Umständen müssen Sie ihn noch mit den Preferences- Programmen "Overscan" und "ScreenMode" optimal anpassen.

Nun wünsche ich Ihnen ein erfolgreiches Updaten Ihrer PowerPC-Turbokarte! :-) Sollten irgendwelche Fragen offen sein, können Sie sich gerne mit diesen an mich wenden.
