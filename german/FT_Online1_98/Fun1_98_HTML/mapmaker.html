HTML Map Maker V.1.03

 
    Ihr  habt  schon bestimmt Homepages gesehen wo man einen Link
    in  einem  Bild  anklicken muß.  Damit lassen sich die eigene
    Homepages  noch kreativer gestalten.  Stellt Euch einfach mal
    vor,    Ihr    wollt    ein   Adventskalender   im   Internet
    veröffentlichen,  wo sich hinter jeder Tür ein entsprechender
    Link befindet der eine Überraschung parat hat.

    Sowas läßt sich nur dank der Image Maps verwirklichen.  Dabei
    handelt   es  sich  um  einfache  Bilder,  die  Position  des
    versteckten Links wird dabei in der HTML Sprache festgelegt.

    Normalerweise  müßte man ohne solcher Hilfsprogramme wie HTML
    Map  Maker  für  jeden  Link  die  Position ungefähr in Pixel
    raten, dann das ganze in den Browser laden und sehen ob diese
    auch  an  der richtigen Stelle ist.  Dies wäre natürlich sehr
    mühsam  und  nur  bei  vielleicht  3  Links  erträglich, aber
    spätestens dann hättet Ihr gesagt, wer braucht das schon.

    Dank  Map  Maker  läßt  sich  sowas  binnen  weniger  Minuten
    erstellen.   Alles  was  man braucht ist die Grafik, diese mu
    noch  leider  in  dem  IFF  Format  vorliegen.   Erst  in der
    nächsten  Version  sollen DataTypes unterstützt werden.  Dies
    ist aber nicht weiter schlimm weil man ja genug Programme für
    den  Amiga  findet  welche  die  Grafik  in  das  Gif  Format
    umwandeln  können.   Das  einzige was man beachten sollte ist
    die  Auflösung,  so  das die beim Umwandeln ins Gif oder auch
    anders rum nicht verändert wird.  Dies ist sehr wichtig damit
    auch später alle Positionen im Browser stimmen!

    Ist die Grafik geladen so kann man sofort mit der Vergabe von
    Links  beginnen.   Dafür  wählen wir im Programm die Funktion
    Add  Zone.   Die  Grafik  wird angezeigt, jetzt markieren wir
    einfach die gewünschte Stelle, in dem wir die linke Maustaste
    halten.   Auch  hier  wird  von  Map  Maker  noch  nicht  der
    vollständige  Befehl  der  HTML  Sprache unterstützt.  Bisher
    kann  man  nur  rechteckige Zonen Markieren, die HTML Sprache
    läßt  aber  auch runde und bestehende aus Polygonen Zonen zu.
    Das  alles soll aber ebenfalls in die nächste Version kommen.
    Hat  man  alles  so kann der HTML Code gespeichert werden und
    schon ist man fertig.

    Das  wäre schon alles wozu Map Maker fähig ist.  Viele werden
    jetzt  sagen,  das  Programm kann das und das nicht, und auch
    sonst bietet es nicht viel, doch das ist falsch.  Erstens ist
    Map Maker Freeware und zweitens kann man damit sehr wohl sich
    sehr viel Arbeit sparen, und das nicht alles unterstützt wird
    ist  nicht  schlimm,  denn  die  meisten Erweiterungen werden
    sowieso sehr selten benutzt.

    Ich  kann  das  Programm  zumindest  jedem Homepage Ersteller
    empfehlen.  Es ist eine sehr interessante Erweiterung und das
    auch noch kostenlos.
