AMINET SET 5

     Eigentlich ist es ja nicht nötig, einen Testbericht über das
     Aminet  Set  5 zu schreiben, da ja wohl fast jeder Amigianer
     über  diese ausgezeichnete Sammlung von Informationen, Voll-
     Programmen,  Shareware,  PD-Soft, Freeware usw.  des Aminets
     be- scheid weiß.

     Aber  fast  jeder reicht einfach nicht.  Deswegen möchte ich
     gerne  etwas  darüber  schreiben, damit in Zukunft jeder das
     Aminet  benutzt,  ob  per  Modem  oder  per CD.  Was ist das
     "Aminet Set" und warum Nummer "5".

     Das  "Aminet Set" ist eine große Sammlung von allen nur mög-
     lichen  abdeckbaren  Bereichen, daß heißt, es gibt fast kein
     Thema,  zu  dem  man  nichts  im Aminet Set was finden kann.
     Hiermit  sind  wir  dann  auch  gleich  bei der Herkunft des
     Aminet  Set,  nämlich  dem  Internet, wo es das AMINET gibt.
     Somit ist das Aminet Set für alle empfehlenswert, die keinen
     Zugang  zum  Aminet  haben  (oder auch die hohen Kosten beim
     Runterladen scheuen).

     Nummer  "5"  deswegen,  weil es schon 4 weitere Sets aus dem
     scheinbar unerschöpflichen Aminet gab.

     Jedes einzelne Set besteht jeweils aus 4 CDs, die jeweils in
     große Kategorien unterteilt sind.  Die Hauptunterteilung der
     4 CDs beim Aminet Set 5 sind

     CD A: TOOLS mit  84 MB Geschäftssoftware
                      115 MB Entwicklungssoftware
                      19 MB Harddisk Tools
                      88 MB Dokumente
                      9 MB Rund um Hardware
                      59 MB Diverses
                      46 MB Textsoftware
                      75 MB Utilities

     CD B: GFX mit    151 MB Grafiksoftware
                      941 MB Bilder, Animationen

     CD C: FUN mit    813 MB Demos
                      273 MB Spiele

     CD D: MODS mit   100 MB Kommunikationssoftware
                      801 MB Musikmodule
                      43 MB Musiksoftware

     Also gut und gerne 3617 MB an Software auf 7400 Dateien ver-
     teilt,  was  für  den  Preis  von  ca.   DM  58  ein kaum zu
     bewertendes Preis-Leistungsverhältnis ist.

     Desweiteren   enthält   das  Aminet  Set  5  wieder  einige
     Vollversionen,  einiger  Programme,  die mit Updates auf den
     aktuellen  Stand  ge- bracht werden können.  Diesmal gibt es
     eine  uneingeschränkte  Version  von  folgenden  Programmen:

     OctaMED Soundstudio 1.03c
     CanDo 2.5
     AmiAtlas 1.3

     sowie einige ältere Vollversionen von Public Domain Spielen,
     die es vorher nur als anspielbare Version auf den Aminet CDs
     gab, wie z.B.  Pengo, TwinzV3.0, Asteroids, usw..

     Also  kann man sagen, daß es wohl für jeden Amiga-User etwas
     auf dem Aminet Set 5 gibt, auch wenn sie ihren Amiga nur für
     ein           bestimmtes           Gebiet           benutzen
     (Musik,Grafik,Spiele,Büro,usw.).

     Einzelne  Programme  werden auch von mir einzeln bewertet in
     der  Rubrik  "PD-Tests"  in  der  Fun  Time zu finden, da es
     unmöglich wäre alle in einem Test abzuhandeln.


     Fazit: Das Aminet Set 5 (wie auch Teil 1-4) kann man nur 
            jedem Amigianer (User und Userinnen) ans Herz legen.
            Allein das erste Rumstöbern bis man alle Programme,
            die man gerne benutzen möchte findet, ist sehr int-
            eressant und dauert gut einige Wochen.

     Stephan De Bernardi
