SongPlayer - Song and Dance

 
Name: SongPlayer
Version: 1.53
Archive: Aminet:mus/play
Author: Stéphane Tavenard (stephane@wanadoo.fr)
Status: Giftware
Category: Soundplayer
Requires: 68020+, OS3.0+, MUI3.3+, AHI (for 16bit sounds), mpega.library
 
 
SongPlayer is quite a nice player for the various common audio formats. IFF 8SVX, AIFF, Wave, AU, MPEG layer I-III and CDDA can be played. 16 bit sounds as well as CDDA are played via AHI. The CPU really doesn't like doing this, but it does make it possible to listen to CDs without extra hardware, although some fine tuning in the settings is required. With the default settings, groups like Toy Dollz sound about as fresh and lively as Rammstein, the fine boy group from behind the Alps. To those that don't call a fast CPU their own (and let's be honest about this, a 68040 is old hat), you're not going to be blown away with anything. Rather,

 
you'll have to clap on a pair of ear muffs as soon as the first sounds emerge. Only a fast CPU or a sound card with appropriate hardware will help you here. The realm of this tale can only be followed with a 'scope. Poor CPU, squawking speakers, punish me with terrible tones - this sets new levels of S&M, and jolts computer fetishism into a whole new light. If the hardware's right (poor bank account), you will be happy with some of the features and smoother running. In addition there is a simple karaoke function, which displays the text of the song (poor neighbours...).
