SimmLock herausfinden:

Es gibt vier verschieden Simm Locks.

Country-LOCK 
Network-LOCK 
Provider-LOCK 
SIM-Card Lock 

So kannst du sie dir den SimmLock Status ansehen:

#pw+1234567890+1# Provider Lock Status

#pw+1234567890+2# Network Lock Status

#pw+1234567890+3# Provider (???) Lock Status

#pw+1234567890+4# SimmCard Lock Status

Eingabe von Buchstaben und Zeichen:

+ = 2x *  (Zweimal die Sternchen-Taste drücken)

p = 3x *  (Dreimal die Sternchen-Taste drücken)

w = 4x *  (Viermal die Sternchen-Taste drücken)

Software-Version des Handys:

Wenn man wissen will, welche Software auf dem Handy das Handy benutzt, gibt man folgenden Code ein: *#0000#

Beim 6130 gibt man folgende Code ein: *#9999#

Erste Zeile: Software Version *

Zweite Zeile: Datum des Software-Release

Dritte Zeile: Telefontyp

*Hier sind die verschiedenen Versionen:

V 3.14 28/11/97 
V 4.33 11/03/98 
V 4.73 22/04/98 
V 5.00 28/05/98 
V 5.24 14/09/98     (es gibt auch schon neuere Versionen, sind mir aber noch unbekannt)

Simm Clock Modus (Deep Sleep) aktivieren:

Ab der Software-Version 5.24 unterstütz das Handy die Deep Sleep-Funktion, mit der sich, durch Abschalten der Sim-Card, die Stand-By-Zeit verlängern lässt.

Zuerst kann man nachschauen, ob das Handy den Deep Sleep Modus unterstützt, und zwar mit:

*#746025625# oder *#sim0clock#

Wenn die Card diese Funktion nicht beherrscht, kann sie beim Netzbetreiber getauscht werden!

 

 

General Codes für Philips-Handys:

 

Genaral Codes für das Philips DIGA:

Blocking list *#3333*#
Time connected *#2558*#
Flags, something *#3377*#
Security code *#7489*#

Genaral Codes für das Philips FIZZ:

IMEI code *#06#
SW info *#8377*#
Security code *#1234*#

Genaral Codes für das Philips SPARK:

IMEI: *#06#

Genaral Codes für das Philips GENIE:

*#06# IMEI Nummer
*#2254*# Status Register: C, BS, RR, MMI, CREAT.
*#2255*# Aktiviert und deaktiviert den "DEBUG CALL"-Modus; wenn aktiviert, wird das Telefon eine Busy-Line anrufen und es werden einige Hex-Codes auf dem Display erscheinen
*#2558*# Zeitangabe, wie lange du mit dem Netzt verbunden warst (in Tage/H/Min)
*#2562*# nicht 100%ig bekannt; Das Telefon verbindet sich wieder mit dem Netz
*#2565*# nicht 100%ig bekannt; Warmstart
*#3333*# (NO) BLOCKING - Liste (15 Einträge)
*#2377*# "BEER": nicht 100%ig bekannt, Das Telefon macht eine Warteschleife und verbindet sich wieder mit dm Netz, manchmal macht das Telefon einen Neustart
*#3377*# Init, Flags, SIM LCK
*#3353*# Setzt die (NO) BLOCKING - Liste zurück
*#7378*# Name, Length, SIM phase
*#7489*# SECURITY CODE
*#7693*#  Aktiviert und deaktiviert den "SLEEP MODE" für die Sim-Card
*#7787*# nicht 100%ig bekannt: SPURIOUS INTERRUPT
*#7948*# SWITCH OFF: nicht 100%ig berkannt; Ein Timer oder ähnliches
*#8463*#  Gibt Information über den "SLEEP MODE": Wake, Sleep Req., Sleep

Genaral Codes für das Philips GENIE (TCD838):

*#2337*# Aktiviert ein akustisches Signal, wenn das Handy sich wieder mit dem Netz verbindet
