Using ircN

This page will explain how to use some of ircN's key features.

ircNs Channel List:
Once you are in a channel you can 'Right Click' the mouse in the chat window to pop up a menu.
choose Join/Add
You can then choose if you want the channel to be Auto-Joined uppon connection.
Doing this to your favourite channels adds them to the same Join menu.
Then selecting the channels from the Join menu will join that channel.

ircNs Auto-Op bot
ircN can allow you to Auto-op your friends when they enter your channel.
To do this select your friend from the users menu. and 'Right click' them
Then select users/add friend from the popup menu.

Once added select users/ucentral from the same menu.
select the user and press edit.
IT IS VERY IMPORTANT to set the Local/Channel flags and not Global ones.

Type #The-Crypt in the channel gadget and then press 'Add' .
You then select #The-Crypt from the channel gadget and set the following flags for the user...
+a: Auto-Op , +f: Protect and +o: Op
Then remember to press 'Update Flags' to save the changes.

Close the window and then the next time your friend joins the channel you have Ops on you will op them :)
