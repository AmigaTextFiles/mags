Installing ircN

Firstly you need to download the ircN exe from the ircN Download Page

Once downloaded unpack it to a folder , say C:\ircN\

You Must have mIRC installed as you then need to copy the mIRC executable and it's help files into the ircN\System\ directory.
Once you have copied the mIRC files into ircN you may un-install mIRC if you want.

That should be that.
You can now start ircN by using the mIRC.exe in the ircN\system\ dir.
