Setting up mIRC

Firstly:
If you intend to use ircN you should Install it before configuring mIRC.

First step is to execute mIRC.

Once run it should open the configuration window on the 'Connect' page.

Set your 'Full Name' , E-Mail Address and 2 NickNames (1 standard nickname and one to use if the first one is allready in use)

Press the 'Add' button to add servers to the list.
I will show how to add one of the uk ArcNet servers to the list...

Once the 'Add' window pops up add the folowing...

Description: vapor 1
 IRC Server: uk1.arcnet.vapor.com
      Ports: 6667
      Group: ArcNet
No need to add a password so press 'Add'

All Done. Now you can press 'Connect to IRC Server'
