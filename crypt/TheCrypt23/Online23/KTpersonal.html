Amiga Computers


As you can see I am using an Amiga computer. I have been using Amiga's since around 1987 and although both my wife and I each have a PC. I am not a fan of IBM Clones.

I also have a second Amiga A4000 that I keep for emergencies.

My computer still requires some additions like a SCSI Scanner and a couple of 9 Gig Ultra wide SCSI drives for spooling video and audio. I am currently using this computer for the production of Broadcast quality short videos for use in pub Quizzes and the like.

The Amiga is still the only computer that can meet my needs, for a fully pre-emptive multi-tasking environment. I hate the horrible bloat-ware that comes with a Microsoft Operating System. Size alone, doesn't make an operating system good. I want performance and I mean real time performance complete with full multi-tasking. I want to be able to have multiple screens open and to be able to flick between multiple programs at the touch of a mouse button. I need to have several programs running critical code simultaneously. And yes this can be done easily on an Amiga computer.
