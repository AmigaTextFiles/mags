Using mIRC

Once connected to a server press mIRCs 'Channels folder' icon (3rd from the left)

Click the 'Add' button and enter a channel name.
Channel names must start with a hash #
Eg. #The-Crypt , #Amiga , #help

A good way to find channels is to use the 'List Channels' icon (next to the 'Channels folder' one)
But beware . this function works fine on ArcNet as it is so small , but servers like DalNel/IrcNet have thousands of channels and the list can take a VERY long time to complete. This can be helped by entering a 'Match text' string. eg . *amiga* .
Press 'Get List' to find channels on the server.
Or you could use the manual method of using the server command '/join' in the status window
eg. /join #The-Crypt

That's the very basics for you :^)
