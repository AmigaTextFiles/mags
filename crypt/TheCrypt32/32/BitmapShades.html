A software solution that eliminates the problem of flickering screens on A1200 computers by darkening all the bright colors.

BitmapShades is a software solution that eliminates the problem of flickering stripes on DblPal or Multiscan screens on A1200 computers by darkening all the bright colours.

It's an improvement of the program "Color Correct" by Frédéric Calendini.

Bitmapshades starts either from CLI or Workbench.
There are no Parameters needed.
For example you can put it into your WBStartup folder.
Use defined short key (default Ctrl + left Alt + B) to hide/popup window, activate button to start/stop shading and window-close button to quit program.


Prefs

You can keep a prefs file "BitmapShades.prefs" in the program directory or in S:.

The following entries can be done:

Level = 250 ; level of brightness - values from zero to 255
Speed = 2 ; time between checks - values greater or equal 1
XPos = -1 ; x window koordinate - values less zero = centered
YPos = -1 ; y window koordinate - values less zero = centered
Width = 130 ; width of window - values greater or equal 130
Height = 80 ; height of window - values greater or equal 80
Colors = 8 ; used colors - choice of 2, 4 or 8
Active = Yes ; active at launch - choice of "Yes" or "No"
Popup = Yes ; show window - choice of "Yes" or "No"
Popkey = b ; shortkey for popup - can be one from "a" to "z"

The Example represents the default settings of the program.
