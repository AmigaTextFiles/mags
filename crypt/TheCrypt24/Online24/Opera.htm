Installing Opera

The FASTEST Browser on Earth

By Ray C. Hawkins.

Internet Explorer and Netscape Navigator are excellent web browsers but are rather resource hungry applications. So if you are running either of these programs on an old PC, it can slow down your surfing considerably. Even if you have a latest "state of the art" PC you can still access the net even faster. How? Download and Install OPERA, a browser that does not tax your system resources to the hilt.

 

STEP 1

Getting started.

Go on line and open up either Internet Explorer or Netscape Navigator and type www.google.co.uk into the address bar. When Google has loaded type Opera into the search box and click on the search button. When the results appear, Select the top item on the list and you should be taken to the Opera web site. Whilst here you might as well have a read of what Opera has to offer. Now click on the Penguin picture and you will be taken to the Opera Download Page.

 

Opera Download Page.

 

STEP 2

Download the program.

Select a server from a country nearest to you and select the Java version. Now click on Download NOW. A file download dialogue box will appear, make sure you tick the circle next to the option "Save this Program to Disk", then click on OK. A Save dialogue will now appear, select a destination , I chose Desktop from the top of the list and click on save. The 11 megabyte file could take up 20 minutes to download on a 56k modem, so time for a nice tot of whisky.

 

STEP 3

Opera Browser.

When the download is complete, a dialogue box will pop up. Click on the Open button and the Opera installation will begin. As is usual follow the standard requests to install the program. When all the files have loaded click on Finish and the Opera browser will open.

 

Welcome to Opera.

STEP 4

Surfing the Net.

I will now leave to enjoy yourselves now. Try visiting some of your favorite sites and experience how much faster Opera loads them in. There are many other features and settings within Opera, but again I will leave you to have a play with then for yourself. The screengrab below shows Opera in full screen mode.
