INSTALLATION:


Simply you have to run the supplied installer-script (EasyACDDA-Install) or install all by hand following these simple steps:


1) Extract EasyACDDA.lha
2) Copy all files located in EasyACDDA/C to C:
3) Copy the fonts located in EasyACDDA/Fonts to Fonts:
4) Copy EasyACDDA and EasyACDDA.info where you like
5) Rename one of the available images as EasyACDDA.bkg
6) Copy EasyACDDA.bkg into the same place of EasyACDDA 7) Set your favorite tooltypes for EasyACDDA
8) Double click on the EasyACDDA icon
9) Add the task Gui4Cli to PicDT (PictureDatatypePrefs) *
10) Save the PicDT prefs. Quit and restart EasyACDDA again *
11) Save your favorite configuration on your harddisk
12) Enjoy EasyACDDA !
13) There is nothing else to do !

NOTE: Steps 9-10 are only for users with graphic board (*)
