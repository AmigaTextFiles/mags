TimeSwipe

loki@napalmdeath.freeserve.co.uk


The first utility i discovered was Timeswipe. A easy to install little program that runs from your Workbench-Startup drawer. Instalation was a Breeze.

The program and all related material are Copyright to: (C) Satanic Dreams Software.

When you first run Timeswipe, it asks a few questions then saves a log file containing your clock`s date to S:. Then when you boot, it checks to see if the current date is before or too far ahead of the one in the logfile, if it is, it warns you with an OS2+ requester. And asks you if you wish to set the date.




Timeswipe is an excellent utility, When starting your Amiga and it is displaying the incorrect time, Timeswipe not only informs you of this fact, but also allows you to open up the "Time Prefs" screen (without the need to open WB/Prefs/Time. Then it is a simple matter of re-setting your battery clock to the correct time.

Actually Timeswipe is more useful IF you have a working batteryclock installed. Then you would be informed should the clock reset after a crash, Never again will you receive those gloating mails from people who make a habit of ensuring everyones clock is correct to the Second.

Although Timeswipe is a good program, I was looking for more ...So onward and upward.
