Gamers Delight




So you love playing games on your computer. Well regardless if you use a PC/Mac/Amiga eventually you will find a game that, Either wont work on your system, Or used to work, but no longer does, Perhaps because you have upgraded your machine.

Users of the Amiga, In particular are affected, A lot of us buy second hand games (old) or for a touch of nostalgia, dust off an old favourite from our collection.

A lot of games developers have released Patches to allow their software to work properly on various Systems.

Example :- Wasted Dreams (Amiga) would not run using a SCSI CD-ROM

Sadly a lot of these companies are no longer around, So finding these patches can be quite difficult.
