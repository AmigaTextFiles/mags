Introduction

QuickNote is a little notepad, which has only one purpose and that is scrap-notes ;-)

Q : Why did I make this program ?

A : I don't like to have thousands of post-it notes all over my room... This program will replace all those notes, and you don't need to find a pen etc.. just use your keyboard ;-)

This version has a limit to 10kb of data, which I think is way to much for such things (but if you think this is to low, please let me know).



Features


Very little memory usage.
Configureable look (bevel, papercolour and pencolour).
Remember window position and content when exit. (main function) :)
Fast & Easy to use.
Totaly free


Requirements


An Amiga (ohh gosh, you can't mean that) :-)
AmigaDOS 2.04 or higher..
textfield.gadget 3.1 or higher (included in this archive) Place it under Sys:Classes/Gadgets/

Usage


When you start the program for the first time, the window content says "- Welcome to QuickNote 1.3!". This is the because there is no default message available.

The default message is created when the information in the window has been changed and the program closed.

This message will be placed in the same drawer as QuickNote itself, and will be called "QuickNote.msg".

When the program has created the default message and you start the program (again) the message will be loaded and visable as the window content. From here you can edit your note again, and that's the main purpose of this program.

When you exits the program, the window content and position will be remebered until next time you start the program.



Menus :


Clear all (Project)
Clear the current window content.

Load default (Project)
This clears the current window content and replace it with the default message if the message has been changed (A requester will ask you if you really want to replace the current text...)

Save as... (Project)
This is an option to let you save the content to a separate file. By using this you can use QuickNote as a simple texteditor :-)

About... (Project)
Well, I think all of you understand this one ? :)

Quit (Project)
If you have used the option "Save as..." before you choose this you will be asked if you want to save the current text, or just ignore it (if you just used the program as a texteditor and don't want to replace the default message).
This requester will not be showed if you have (after you picked "Save as") choosed "Clear all" or "Load default".

Undo (Edit)
Will undo the current content in the delete-buffer.

Cut (Edit)
Cuts the marked text and places it in the clipboard.

Copy (Edit)
Copies the marked text and places it in the clipboard.

Paste (Edit)
Pastes the content (if any) from the clipboard to the edit window.
