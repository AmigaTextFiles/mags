November's Game Releases

News has just been received that Uropa 2 an 3D Isometric Action Adventure developed by Austex Software, has been released by Vulcan Software and is now available to buy for £29.99 UK Pounds and Uropa 2's release marks Vulcan's second CD-Rom only release.

The Final Odyssey a puzzling action adventure will also be released in the next few days by Vulcan Software along with Genetic Species an 3D Doom Clone following in the next few weeks.

Blade is an RPG strategy game is to be released this week by Alive MediaSoft.. Alive MediaSoft will also be releasing The Haunted a CD-Rom scary adventure game hopefully in time for Christmas.

Street Racer is also set to be released sometime this month by Guildhall Leisure and is expected to be in the shops in the UK.

A number of other games are also planned for release this month, more on them as soon as now. November looks to signal the biggest release of games this year.
