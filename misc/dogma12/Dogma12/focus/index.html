  • A604 - расширение памяти (и не только) для Amiga 600
    Дата публикации: 05/2011 года
    Автор: -= / Vinnny / =-

    Одно из главных отличий - новый дизайн A604 позволяет монтировать фликерфиксер Indivision ECS без удаления внутреннего дисковода (это было проблемным местом для A603). Второе преимущество - появился второй clock-порт, сделанный специально для USB-контроллера Subway. Этот порт может похвастаться увеличением скорости на 60% по сравнению со стандартных портом, что делает USB-шину быстрее, чем когда-либо прежде.

  • Обзор AmigaOneX1000
    Дата публикации: 06/2012 года
    Автор: -= / amifone / =-

    После долгих лет ожидания Amiga-сообщество наконец-то может сказать - мы получили свой лакомый кусочек. В начале 2010 года компания Hyperion подписала контракт с командой разработчиков A-EON на поддержку в AmigaOS4 нового разрабатываемого компьютера AmigaOne X1000. По сути это первый проект HiEnd-Амиги, осуществляемый за последние 17 лет с момента выхода A4000 в 1992 году. Можно ли надеяться, что этот компьютер сможет восстановить интерес к платформе, определить новую планку для будущих Амиг?
