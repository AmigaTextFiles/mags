AMIGA WEB OFFLINE Vol. 1   CD

PHOENIX DP vous annonce la production d'AMIGA WEB OFFLINE Vol.1. Ce CD
contient uniquement des sites Web consultables hors ligne. Il a pour but de
faire connaitre aux nombreux amigaistes ne possédant pas d'accés internet,
la multitude d'informations sur l'Amiga, que l'on peut trouver sur le net.

Ce CD sera comportera de trés nombreux sites commerciaux. Pour les
sociétés désireuses d'y figurer, nous demandons de bien vouloir nous
envoyer une autorisation de diffusion, une archive de leur site sur CD,
disque dur, ou disquettes, ou si cela est impossible, de nous indiquer leur
adresse URL précise et la taille (en Mo) de leur site pour nous faciliter
le téléchargement. Ce CD permettra de faire connaitre votre société aux
personnes non connectés : Alors n'hésitez pas à participer. Plusieurs
sociétés amiga, comme Amitrix, Finale Development etc.. nous ont déjà
donné leur accord.

Une place toute particulière à été reservée aux sites personnels. Si
vous êtes un particulier, un club, une association, nous pouvons inclure
votre site web. Les conditions de participation sont identiques à celles
des sociétés,

Bien sur tous les participants recevrons un exemplaire gratuit d'Amiga Web
Offline !!!.

 AMIGA WEB OFF LINE Vol.1            Prix de vente public: 59 F
