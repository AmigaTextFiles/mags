It is possible to control the computer with the PPC only. We're
working on a linux/m68k port to the PowerUp and the use of seperate
address spaces for each task under linux makes it impractical to have
both CPUs working - thus, the m68k is put to sleep and the PPC runs
the machine.

Later on, we might be using the m68k as a coprocessor to handle
e.g. screen updates. Time will tell.
