Blacks Editor (BED)


Mauro 'M&F' Fontana(mfontana@ing.unico.it)

Program author: Marco Negri (mnegri@iol.it)

This excellent piece of software is one of the best text editor you can find for the Amiga. Written by Marco Negri, the program features some of the most powerful tools you can find only on expensive commercial programs.
The program was written with the idea to help the programmer writing source code for its programs, but has evolved to allow also an easy use for long structured projects.
The program is very advanced and features, among other things:

    About 150 AREXX commands through which it is completely programmable
    Localization
    Completely customizable menus
    Completely configurable keys
    Easy macro learning and execution
    Asyncronous preference requesters
      Options available
      Autosave
      Backup before saving
      Every doc can have its font and local settings
      Auto indenting
      Conversion of line feed during loading and saving to make docs compatible with other platform (PC, MAC)
      Auto detect binary files (so no strange behaviours)
      Setting of the printer device (can print with any device)
    Can be opened on custom screen
    Unlimited number of documents opened at the same time
    Paragraph justification
    Execution of tools without returning to Workbench screen (even through AREXX)
    Configurable SHELL
    Auto configuration after doc extension (i.e. docs ending with .c can have a different configuration than those ending with .txt)
    Amiga guide help

Other features

    Completly and easily configurable
    XPK library supported for import/export
    IBM-Style keyboard supported
    Horizontal scroller and optional vertical scroller
    Optional ARexx macro executed at each file or document opening
    Possibility to open a document without a window
    reqtools.library requesters supported
    Standard Clipboard supported
    Customizable font for gadgets, requesters and text
    Full font-sensitive requesters, with new look
    AmigaDOS Protection Bits handling
    Customizable number of lines to jump with JUMPSCROLL
    Automatic indent supported in paragraph formatting
    OS printer device support
    Saving/loading of entire projects with window and cursor positions.
    And lots more.

The program comes with lots of already predefined AREXX script which can be executed selecting them in the menus or by a completely configurable short-cut. The scripts include lots of automatic operations which can be used to simplify C code writing (searching of function calls in the Autodocs, Compiling, Debugging and lots more) all without leaving BED's custom screen.

The program requires only 400K bytes of RAM to load and it is quite fast, so that is can be used without problems on a plain A1200. (BTW: I've configured the program to work with the E compiler)

Version 1.1 is on Aminet under text/edit.

It is completely free!!.

I have been a tester of the program since it was in alpha stage. I have seen it growing and becoming more and more powerful and better. Living very near to the author I could help him finding bugs and giving suggestions on how to enhance the features and thinking of new ones. The program has now reached its final stage with version 1.1. I can enjoy a personal version 1.1 beta3 with lots of new features added, like word completition, template support and correction of mispellings. I am using BED to write this electronic magazine, as I find it the best text editor one can use.

The previous features, together with new ones like text colouring and foldering should have been included with version 2.0 of the program, but BED development has stopped as its author cannot continue it for lack of time and tools (he has sold his A4000).

Marco is looking for some programmer to take the sources and continue the developing of the program, as there are still some minor bugs to fix and lots of features to be addded. Contact him or me for any suggestion or offer. (see the request in market corner)
