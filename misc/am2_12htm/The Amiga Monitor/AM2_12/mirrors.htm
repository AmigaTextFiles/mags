Anybody interested in hosting an AM Mirror Site should e-mail us at mikewebb@csi.com (please check the AM Link Redirection Page for an up-to-date e-mail address first if possible, but otherwise use this address) with the following information (where applicable):

  • Site Name
  • Server(s)
  • Server location(s)
  • Format(s) you will be carrying
    • HTML Online
    • HTML LhA
    • HTML ZIP
    • AmigaGuide LhA
  • Whether you will carry current issues only, current and some back issues, or all issues in the selected format(s)
  • E-mail address to which you would like archives sent (if different from the one from which you write to us)

Once we reply to you, you will be registered as an official AM Mirror Site, and will receive monthly archives via e-mail.
