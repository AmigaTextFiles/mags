E-Mail Distribution Lists

To join an AM list, send e-mail to mikewebb@csi.com (please check the AM Link Redirection Page for an up-to-date e-mail address first if possible, but otherwise use this address) in which you specify the list(s) you would like to join:


  • HTML LhA
  • HTML ZIP
  • AmigaGuide LhA
    As of this writing, there is a very significant backlog of issues needing to be converted from HTML to AmigaGuide, and new issues have not been available in AmigaGuide for some time; however, this is a high-priority problem and will be addressed as soon as time and resources permit.
  • New issue notification of availability
    To stay up-to-date with AM without having to download entire archives as e-mail attachments.


Notes

  1. You will receive a reply when you have been added to a list.
  2. Addresses that bounce back messages will be removed from the list(s). But because some e-mail problems are temporary, we will attempt to send notification of removal when this occurs.
