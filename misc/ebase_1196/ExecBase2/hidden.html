Hidden Fields

How They Are Used

Most shopping carts uses the <FORM> tag. Iside that you can make different fields which contain data. Most often you will see those fields, and you can change the contents or add to them.

Often, though, the programmer need to pass data around to different pages, and then they often use hidden fields, that is form fields which is set to TYPE="hidden".

Problems

The problem with hidden fields is that if the user leaves the site, the data is lost (well, not if the page still is in the cache of the user).
