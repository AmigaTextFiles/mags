World Foundry to Sign to Vulcan.

I'm pleased to announce that World Foundry will be signing up to Vulcan. As yet final documents have not been signed but it is looking almost certain that things will go through without any problems. I was going to wait till everything was official but since the news has already been released on other news pages I didn't see any point in it.

The deal covers all of World Foundrys current and future games. At the moment this covers PPC and 68k versions of both Explorer 2260 and Maim & Mangle.

Explorer 2260 as I'm sure all of you know is 3D space simulation somewhat along the lines of Frontier. However, don't think that this is just a copy of Frontier, because it is not, it is far more then this and is set to become one of, if not the best space simulation ever. More information about this game can be found at the Collins Encyclopedia Galactia.

There other game Main & Mangle is a realtime strategy game that takes all the best bits from DuneII & Command & Conquer and then builds on them.

More information about both these games can be found at the World Foundry Webpage.
