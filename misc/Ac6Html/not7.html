JAVA para AMIGA
 ***************

'Finale  Development'  esta  desarrollando  "Moca",  una implementacion por
software  de  la 'Java virtual machine' para Amigados/pOS.  

Ademas, usa una novedosa tecnica para conseguir para ser hasta 10 veces mas
rapida  que una maquina virtual java para otras plataformas.

Para los que no sepais lo que es Java, y lo que puede suponer, ahi va eso:

Java  consiste  en  2  partes:   un  lenguaje de programacion de alto nivel
orientado  a  objetos  comparable  a  C/C++  y  la  maquina virtual de bajo
nivel.

Un  compilador de java convierte un programa en lenguaje java en el llamado
"codigo  byte",  el  cual  es muy comparable al lenguaje maquina de toda la
vida,excepto  en  que ha de ser ejecutado en una cpu virtual.  Esto tiene a
la vez una gran ventaja y un gran inconveniente.  

La  ventaja  es que java es un "lenguaje universal",ejecutable en cualquier
plataforma  que  cuente con el software apropiado para emular dicha cpu (de
hecho, Java fue concebido por SUN Microsystems para fastidiar a Microsoft y
su  entorno  windows,  ya  que  podrian  desarrollarse  aplicaciones que no
dependiesen  del  entorno  del  amigo  'billy puertas' para ejecutarse,para
acabar  asi  con  la  casi  total  hegemonia  de  micosoft  en los sistemas
operativos).  (*ED.  ahi keda eso!!!!  :D *)

Java  esta  siendo  todo  un  exito,  y  en  solo  6  meses  ya hay 200.000
desarrolladores  programando  para  java.  Vistas asi las cosas, microsoft,
que  en  un  principio  habia apoyado el lenguaje java, anuncio que sacaria
Visual  J++,una  especie  de visual basic para java pero con la salvedad de
que  los  ejecutables  solo funcionarian bajo entorno 'Windoze', con lo que
todo seguiria como antes...  Veremos en que acaba todo esto.

Bueno, pues eso, su ventaja es la portabilidad.Pero...?Y el problema?.  Muy
sencillo:   (A que emular otro procesador es un tanto lento?(y si no,que se
lo   pregunten  a  los  emuladores  de  pc...).

Para resolver eso, 'Finale Development' lo que va a hacer es integrar en el
paquete  un  potente compilador llamado JIT (siglas de "Just in time"), que
hara  posible  convertir  un  programa  en  Java  a  codigo maquina 68x00 o
PowerPC, con lo que su ejecucion seria mucho mas rapida.

Segun FD, las primeras demos de 'Moca' estaran disponibles en breve, aunque
de  momento  solo se podra utilizar en un Browser que estan diseñando ellos
mismos,  llamado "Finale  web  cruiser".

Sin  embargo,  aseguran estar en conversaciones con distribuidores de otros
browsers de internet para llevar moca a ellos.
