Latest On Foundation Demo

Foundation last was reported in an article in last issue's News In Brief which revealed more details on this up coming game from author Paul Burkey.

A demo has now been released so now you have the chance to see some of these features in action and get a feel of this mixture between Civilization and Settlers.

But some have experienced problems and to help you download the latest version of this demo and help in getting it run, a support page has been set up by Sadeness Software.

I would strongly recommend that you visit this site if you want to try out the Foundation demo. An AGA version and CyberGraphX version are available.
