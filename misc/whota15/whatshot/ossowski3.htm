Amiga Forever 1.0 Preview Edition
The Amiga Emulator CD

The final version 2.0 of "Amiga Forever" is not yet finished, so Claonto decided to release, ahead of the planned schedule, a special "Preview Edition" of Amiga Forever, with a version number of 1.0. This version allows the more technically-experienced users to immediately explore many of the features of the final release, without paying an additional price (the list price of the upgrade from the preview version to the final version will not be higher than the difference between the two).

Amiga Forever 1.0 "Preview Edition" includes:

  • Amiga Explorer 1.0 extension to the Windows desktop: connect an Amiga to the PC via a serial cable, and then access all Amiga files directly from the Windows desktop.
  • Exclusive interview with Jay Miner on topics ranging from the birth of the Amiga to computers and society.
  • Amiga ROM and Workbench files from version 1.0 to version 3.0.
  • UAE for Windows and Fellow for DOS Amiga emulators.
  • "MenuBox" launcher to easily access the emulation software, the documentation and the installation software.
  • Personal Paint 7.1 and DirDiff 5.1 from Cloanto, preinstalled in the Work partition of the emulation environment.
  • Picasso 96 drivers for the Windows version of UAE, supporting palette modes up to 256 colors, as well as true color.
  • MSH filing system to allow Amiga computers to read MS-DOS floppy disks
  • DirectX version 3a for Windows 95 and Service Pack 3 for Windows NT 4 (required by the Windows versions of UAE, unless these or higher versions are already installed)
  • Amiga Interactive Guide, by Gareth Knight, which is an encyclopedia of Amiga technology, history and opinions.

Amiga Forever 1.0 "Preview Edition" does not include the following features, which will be released in the final package (Amiga Forever 2.0):

  • Amiga Explorer support for parallel port and TCP/IP (Ethernet).
  • Audio tracks with Jay Miner's interview.
  • Extended user documentation.
  • Floppy disk with a copy of the Amiga-side data transfer software (also stored on CD-ROM).
  • Additional Amiga Workbench extensions.
  • VESA drivers (for the DOS versions of UAE and Fellow).
  • 3D Amiga Boing sticker: the final touch to transform any PC into an Amiga!
