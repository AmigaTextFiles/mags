DarkChild

The following article is from the Amiga Flame web site.

Developer:MWI

Requirements:GFX Board Only

Darkchild is in development by MWI and is a Fantasy RPG in the tradition of Eye of the Beholder and Amberstar.

A Non-Playable demo of Darkchild has been released and Graphics card users will be pleased to know that it only runs on there boards at present. Darkchild does not feature Texture Mapping, but it does feature 640x480 resolution, while other RPG games on the Amiga usually only had 320x256.

The final version of Darkchild will run on GFX Boards and AGA, using there rtgmaster.library software. There is also the possibly of a PPC version but a decision has not yet been made.

Currently Darkchild has no Publisher but they hope to get one perhaps in the future.
