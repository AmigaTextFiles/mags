Gloom 3

In a previous article, I mentioned that Gloom 3 would be coming out on CD and was going to be distributed by Guildhall, well Alpha Software, Gloom 3's developers have left the game engine much the way it was for Gloom Deluxe, but have included 50 new levels spanning over 7 stages and around 10 new nasties with all new GFX and sound. The game will also improve the fun for multiplayer game, as there will be 30 new 2-player combat levels included, which will also contain zombies to make things that bit harder and also modem support will be much improved.

To run the game, you'll be able to get away with using a base A1200, but it'll be slow in all but the smallest of screenmodes. To get a decent game out of it, it's recommended that you have at least an 030 with 4Mb Fast RAM. But to play the game in 1x1 screenmode an 040 or 060 is recommended.

But graphics card owners will not left out as the game supports GFX cards, which will allow them to play the game in a Workbench window.

By the way, a demo of Gloom 3 is available here and the latest information on Gloom 3 can be found at The Offical Gloom Pages.

Pinball Brain Damage News

Pinball Brain Damage was orignally going to be published by Vulcan Software, but has now been picked up by APC&TCP, and have just released some new details on the game.

The game will include Hi-Res table graphics (640x256) with 256 colours in normal mode and Super-Hi-Res (1280x512) in multiball mode, which is probably the highest resolution yet to be used in any Amiga game.

For more information on Pinball Brain Damage, goto The Amiga Nutta.

Total Destruction 3D

A group called Deepcore Entertainment Software are writing the first true 3D, polyon based, texture mapped beat-em-up for the Amiga. If you've seen Virtual Fighter and Tekken, you'll know what I mean.

This game will no doubt require a pretty powerful machine to play it and from what I've heard it will require at least an AGA/CyberGFX machine with an 030 for 320x200, 2x2 resolution and an 040 for the 320x200, 1x1 resolution and also 2Mb Chip RAM and 8Mb Fast RAM.

The game is being scheduled for release in Summer 98 and will be on CD-ROM only.

Trauma Zero

Trauma Zero is a new Shoot-em-up that is being developed by a new Italian team, with a demo currently available on the Aminet comprising of 4 separate archives. But, the demo isn't playable and is a rolling one, and I've seen it and it looks a lot like R-Type. The game will contain 256+ colour graphics, will scroll at a super smooth 50Hz and will feature a simultaneous 2-player option. The game is AGA and will be hard disk installable.

Voyager NG 3.0 Announced

Oliver Wagner recently announced that Voyager NG 3.0 will be expected to be shown at the Colonge Amiga show in November.

The main features of Voyager NG 3.0 include HTTP/1.1 transactions, Javascript 1.1, Integration of the Mail and News applications (similar to Netscape Communicator), FastMem Blitting of images on AGA systems, and a versatile plugin interface capable of handling streamed network data.

As Voyager NG 3.0 is likely to be more expensive than V2, upgrades from V2 to V3 will priced at only the difference from V2 to V3.

But there will be a final release of V2 before V3.0 is released.
