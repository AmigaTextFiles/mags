A week or so ago, I received an e-mail from Chuck Patterson, who maintains the AmigaRing, informing me of his Amiga Ring, after noticing one of my web pages on ClickBOOM's Amiga Ring and also voiced concerns about the ClickBOOM Amiga Ring's graphics, which didn't mention anything about the Amiga and that there it was more of ClickBOOM commerical ring than an Amiga Ring. On surfing through Amiga web pages for a couple of years, I've run in to the AmigaRing many times over and even though I've been maintaining at least one web page for year, it was until I heard of the ClickBOOM's Amiga Ring, that I became a member of that ring and never thought of becoming a member of the AmigaRing has there has been a lot of fuss made about ClickBOOM's Amiga Ring, since it's appearence, few months ago. But, from now, I will pledging my support to the AmigaRing as well as ClickBOOM's Amiga Ring for as long as both around.

Well, after receiving this e-mail, I checked out the AmigaRing web page at Joining the AmigaRing!

NOTE: Many people have been attempting to join the ring without following all of the steps below. You cannot just add yourself to the ring without having the appropriate code & images. PLEASE follow the directions CLOSELY. You must have the code & images on your page BEFORE joining the ring (and preferably before applying).

    Here are the steps that need to be followed to join the AmigaRing. Please follow them closely to preserve the integrity of the ring! If you have any questions, feel free to email me at midian@azstarnet.com.

  • Save the images that is within the HTML fragment, you should be able to RIGHT-CLICK the image and select SAVE IMAGE to download it, then upload it to the server where your web page is. Please do not reference it from another server without permission.
  • Copy the following HTML code fragment to your web page where you want the links to go. After you submit your site, this code will be sent to you in Email so you can COPY and PASTE it if you prefer.
BEGIN code fragment


[ AmigaRing Home | Prev5 | Prev | Random | Next | Next 5 | List ]

END code fragment
AmigaRing image and layout designed by Keith Blakemore-Noble
  • Click here to fill out the form, submitting your page to the AmigaRing queue. Please note this does not insert you into the ring! It just registers you with the program and assigns you a site ID number.
  • After filling out the form, you will receive your site ID number. Write it down.
  • You will see six sections in the HTML that have underscored areas ("_____"). Replace ALL of these with your SITE ID number. For example, if your site ID is 10, the href should be something like: http://www.webring.org/cgi- bin/webring?ring=amiga;id=10;next.
  • Now, to be added to AmigaRing, you need to e-mail the owner of any existing AmigaRing site and have him/her add you to the ring. Make sure to include your site ID number. To keep the time that your page is up without links to a minimum, you may want to contact the owner of a site if you know where they are.
  • The person that you requested to add you to the ring should do so...(he/she'll most likely visit your web first), then they should send you an E-mail confirming it. Then... you're DONE! :)
Special Note: If you have the desire, feel free to design your own layout that is more appropriate to your web page. Not all image themes will match everyone's page. The only limitation is that the links relate to what they are for and are easy to find. If your web page is text based, feel free to use a text layout. Before using the images and layout designed by someone else on the ring, be sure to send them email and ask their permission.

Adding someone to the AmigaRing!

Important Notes!

  • Remember that the only pages that are applicable to the ring are pages whose content has to do with the "Amiga". Never link a page that does not contain said content.
  • If you get mail from someone asking to be added to the ring, check their page first!! Make sure that the content is appropriate and that they have added the HTML fragment, the appropriate images, and everything else necessary. Also make sure that all of their AmigaRing related links are created properly.
  • If you add someone to the ring, please E-Mail the person to tell them that their site has been added.
