Following from an article in the last issue, here is information on Stratego, a game by Deimos Design.

Stratego - Features

Here are the game features currently developed / planned for Stratego. If Stratego's features change, you will be informed of those changes.


Game Features

  • One or two player games [same Amiga and TCP/IP networks].
  • Authentic Stratego rules [i.e. no additions to, or omissions from, the standard Stratego rules].
  • Save games in progress [even remote games!].
  • Save favorite team setups.
  • Save player prefs.
  • Chat mode for remote games.
  • Completely localised via language catalogs.
  • Configurable sound effects for actions and pieces.

    • System Features

  • Completely Mulitasking.
  • Almost entirely event driven - requires very little CPU time.
  • Compatible with ECS, AGA, and 3rd Party Graphics Cards.
  • Stratego's default Screen is mode-promotible.
  • NEW!: Stratego will have both 16 and 256 colour graphics sets.
  • NEW!: Stratego will run on it's own screen, or on WorkBench.
  • NEW!: Stratego transparently takes advantage of chunky BitMaps, if available.

    • Stratego - Requirements

    As you can see, even the most neglected of Amigas [:-(] will have more than enough juice to run Stratego.


    Minimum Requirements

  • DISPLAY: ECS Chipset
  • CPU: 68000
  • CHIP RAM: Approximately 1 Meg
  • OS: 3.0

    • Recommended System

  • DISPLAY: ECS with Flicker Fixer / AGA Chipset with Multisync Monitor
  • CPU: 020+
  • CHIP RAM: 2 Megs
  • OS: 3.1+
  • EXTRA: Hard disk, with approximately 1 Meg free
