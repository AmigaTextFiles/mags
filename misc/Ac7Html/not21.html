GPSoftware anunciando la nueva versión del OPUS.
------------------------------------------------

GPSoftware   demuestra   la  continuidad  de  seguir  soportando  el  Amiga
anunciando la nueva versión del OPUS en el World Amiga Show en Londres.  El
nuevo  producto  viene a demostrar que GPSoftware/Wizard Developments harán
una nueva versión a la ya existente 5.5.

En los primeros inicios del 95, cualquier usuario del Amiga podía comprarse
el  OPUS 5 con el único concepto de integrar el Workbench en un programa de
manejo  de  ficheros,  totalmente  transparente  al sistema.  Este concepto
define lo que es hoy en día el OPUS 5.5 realizado en Mayo del 96.

Durante este tiempo, una gran incertidumbre en la plataforma Amiga a estado
presente para todos los usuarios que utilizaban el Opus.

Gracias  a  los  usuarios,  por los ánimos, apoyo, registros, iniciativas y
todo  eso  que hace que los usuarios del Amiga sean especiales, se hará una
nueva versión.

Pero  lo  bueno  viene  ahora.   Una  nueva versión del OPUS, conocida como
Magellan,  esta siendo desarrollada desde hace unos 12 meses, o sea, que es
un  nuevo producto.  Como si el OPUS hubiera sido creado desde código cero.
Esta  versión,  lo que quiere definitivamente es ser superior especialmente
en  el  modo de reemplazar al WorkBench con un mejor control de los iconos,
acciones, mas nuevos menús, extras popup y nuevas formas de background.

Después  de  muchos  testeos  betas  con el WorkBench y otros programas del
Amiga,  se  da  soporte  a los NewIcons, un mejor soporte y optimización al
sistema  de  las tarjetas CyberGraphX, iconos de fondo mas rápidos, rutinas
de  dibujo  y de mostrar pantallas mucho mas rápido que la 5.5.  También se
ha  vuelto a reescribir desde cero el OpusFTP.  Es un modulo complementario
con  cambios  y  que es unas cinco veces mas rápido en el acceso a Internet
con   un   completo   sistema   GUI  para  un  Address  Bock  y  parámetros
operacionales.

Aquí tenéis en ingles las nuevas opciones del nuevo Opus:

* Enhanced Lister Functionality including

- Command functions can now act on icon mode listers
- New popup menus
- Optional Space Gauge showing free space on drivers
- Inline Editing for Name mode listers

* Enhanced Opus FTP including

- New GUI Address Book
- New GUI for configuration system
- Up to 5 times faster access with less memory usage
- Individual directory cache exclusive to each FTP Lister
- Optional idle timer
- Auto Index download for AmiNet index files

* A new integrated system of Star Menus

*  New  Desktop  Folder mechanism - Drag and drop items to the desktop with
support of optional popup menus

* New Icon Positioning system - Configure areas where icons will appear and
their priority

* Global control of icon labels and icon borders

* Direct byte-for-byte copy of icons

* Icon label splitting for long labels

* New Icon Command function for special "command" files

* New Icon Information requester with extensive popup options

* Faster icon dragging routines

*  Use  traditional  WorkBench  icon  positions for icons or exclusive Opus
positioning

* New Popup menus with shadow look

* New Enviroment GUI's for greater control over Opus configurations

* CyberGraphiX support for dragging icons and general display speed with 24
bits backdrops

*  Integrated NewIcons support not just for icons but for all system images
including graphic button banks

*  Improved  compatibility  with  MUI, Datatypes, MCP and a number of other
programs

*  New  enviroment  variables  to allow you to customise Opus functionality
even more

* Rewritten Text Viewer for faster access and better scrolling specially on
CyberGraphx screens

* New Filetypes including ability to match on disks

* New Scripts system of disk inserte/disk notification gives control

* over new disks or newly mounted remote file systems

* New Command Functions plus enhanced Argument Variables

*  Buttons/Images:   Full NewIcon image support plus more support for image
and animation file including DPaintV/PPaint animbrushes

*  Groups:   Can  now contain icon-less items plus you may now add left-out
commands to groups

* New ARexx commands

* More than 40 new callback hooks for easier user programming of Opus

Atención:

La  versión  Directory  Opus 5 Magallan, en su upgrade, estará disponible a
partir de la feria de Londres, los días 17 y 18 de mayo.

Eso  es  todo  lo  referente  a la nueva versión del DOPus tan famoso en el
Amiga.   Espero  que  os  registréis, porque estoy seguro de que casi todos
estáis usando este programa.

Un  Saludo de DarkG.
