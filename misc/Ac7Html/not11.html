¡¡Usa Tarjetas ETHERNET PCMCIA de pc para A1200 y A600!!.
---------------------------------------------------------


Si amigos, lo que parecía mas que factible, por fin de ha conseguido:

Usar tarjetas PCMCIA de Ethernet para un A1200 o A600.

Todos nos hemos preguntado alguna vez si seria posible conectar una tarjeta
PCMCIA  de pc ETHERNET usada en portátiles en nuestro amigas, las respuesta
era sencilla:  SI, sí disponemos del driver...

Hace  tiempo  en  una  AUI  vi  un anuncio de una empresa de Inglaterra que
vendía  'drivers'  de  Ethernet para A1200 o A600 por PCMCIA, el precio era
desorbitado y además le tenias que comprar la tarjeta a ellos...

Pero  ahora  como por arte de magia un amiguero ha creado un driver y lo ha
puesto en AMINET como FreeWare!!  ósea..  ¡¡¡¡GRATIS!!!

Ya  no  tenemos escusa, una tarjeta de red de portátiles suele ser bastante
asequible  para  todos  los bolsillo, y encima el driver funciona con todos
los   'stacks'  de  TCP  para  amiga  como  son  el  MIAMI  o  el  AMITCP!!
¡¡¡Increíble!!.

¿Para que necesito una tarjeta de red??

Bueno,  la  verdad  es  campo  algo reducido, por lo menos en AMIGA, puedes
usarla  para  conectar  varios (en teoría varios cientos..  :) )de AMIGAS y
compartir recursos como Disco Duro, CDs, módem, etc..

Otra  posibilidad  es  la  de  integrar el AMIGA en una red pecera para por
ejemplo  conectarte  a  Internet  mediante  una  RDSI  de PC!!  o hacer una
Render-Farm  basada  de  PC  con  AMIGAS para la Edición de vídeo, algo muy
interesante.

El  uso  mas  común que se le puede dar es el de conectar tu PC y tu AMIGA,
consiguiendo unas transferencias mucho mayores que por Serie/paralelo.

También  otro  campo que se podía explotar como ocurre en otras plataformas
es  el  del  juego  en  red,  esperemos  que  alguna  compañía soporte este
innovador driver.

Una  tarjeta  de  red  para  pc en ISA puede costarte como mucho unas 5.000
ptas,  mas  las  de  PCMIA  que  difiere  mucho  según  el modelo, pero con
seguridad  puedes  tener  el  AMIGA  en red por muy poco dinero, siendo una
buena alternativa al 'Siamese'.

El autor de este invento es Bruce Abbott

Los requerientes son:


 - Una CNet CN40BC o Accton EN2216 PCMCIA tarjeta Ethernet.
   (Algunas  otras podían funcionare sean 'Compatibles con NE2000')

  - Amiga 1200 o Amiga 600 (tienen adaptador PCMCIA.. :) )

  - Software de Red. (AmiTCP 3.0b, Miami 2.0g o Envoy 2.0).

  - Una red :^)
