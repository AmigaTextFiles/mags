DEMOSTRACION EN VIVO DEL WEB CRUISER.
--------------------------------------

¡  En la reunión del "Michigan Computer Users' Group", el 19 de Junio a las
7  de  la tarde, se realizará una demostración de "Web Cruiser" !.  A todos
los  asistentes  se  le entregará un copia demo de New York.  ¡ No faltes !
Para más detalles, contacta con Matt Ignash.

Web Cruiser soportará Java vía MOca.

"Web  Cruiser"  de  Finale  Development,  disponible  para  los compradores
norteamericanos  pronto  a  través  de GamaSoft, da a tu Amiga una serie de
tremendas  herramientas  de  productividad para Internet, incluyendo correo
electrónico  multimedia,  gestión de Usenet news, y un navegador compatible
con  soporte  de  frames  que  está  preparado  para  Java  con los plugins
opcionales "MOca Java" y "JavaScript" disponibles este verano.

¡  Web Cruiser es el primer navegador que está preparado para soportar Java
!.   Pero incluso con esa ventaja, Web Cruiser supera a toda la competencia
dándote las siguientes características profesionales:

* Soporte total HTML, incluyendo frames.
* Satisface las normas MIME para capacidades "plug-in".
* Soporta directamente CyberGFX 2+.
* Puede usar datatypes V40, V43, y V45.
*  Interfaz  ClassAct  super  rápido  ClassAct, deja a Ibrowse mordiendo el
polvo.
*  Lector  de  noticias  de  Usenet  "New  York" con gestión de suscripción
jerárquica.
*  Programa  de  correo  electrónico "Voodoo Lite" multimedia visualiza los
ficheros de anexos en sus formatos nativos!

Por  tanto  te  deshaces  de  ese  lento  navegador  y  pide  el  Suite  de
productividad para Internet más completo, Web Cruiser.
