NOVEDADES POWER-UP.
-------------------

* Módulo PowerUP StormC 3.0
---------------------------
El  módulo "StormC 3.0" para PowerUP está ya listo para los propietarios de
las  tarjetas  para  desarrolladores  comerciales.   Con  este  sistema  de
desarrollos serás capaz de portar muy rápidamente tus actuales fuentes en C
y  C++.   La posibilidad de generar binarios "mixed" y "fat" hace muy fácil
manejar  las  partes  importantes de tu software en código nativo para PPC.
"StormC"  mezcla  los  binarios  y  te  da  pleno  control de las variables
globales y estructuras.  El paquete incluye:

*  StormENVIRONMENT  3.0 para crear facilmente projectos para AmigaOS 68K y
AmigaOS con werUP.

* StormC PPC, un compilador completo ANSI-C/C++ para 68K y PowerPC AmigaOS.

*  StormLINK, una versión PPC de nuestro rapidisimo linkador.  StormLINK es
capaz  de  generar binarios estándar, mezclados y "fat" para aplicaciones y
librerías compartidas.

* Nativo ANSI-C/C++ 68k y PPC y librerías matemáticas.  ArtEffect:  Efectos
POWERUP
 
Este  paquete incluye los módulos estándar ArtEffect en versión nativa para
PowerPC.    La   también   nativa  PPC  aeplugin.library  dará  a  terceros
desarrolladores  la  posibilidad  de  realizar  sus  propios mólulos pulgin
nativos PPC para ArtEffect 2.0.

* Lanzamiento de los módulos powerUP para SViewNG
-------------------------------------------------
Los  usuarios  registrados  de SuperView/SViewNG han sido informados de que
los  módulos plugin opcionales para powerUP (TM) para la librería SuperView
están  ya disponibles.  La fecha oficial de lanzamiento es el 1 de Junio de
1997.   El  paquete  de módulos plugin para PPC incluye al menos 26 módulos
plugin  para  la  librería  SuperView,  SVObjects  y  SVOperators, así como
funciones   compartidas   comunmente   soportadas.   Nuevos  módulos  están
actualmente   bajo  desarrollo  y  se  añadirán  al  paquete  cuando  estén
disponibles,  de  modo  que  el número de módulos puede estar en permanente
crecimiento.   Como  ejemplo  de como las operaciones pueden ser aceleradas
por  los  módulos plugin para powerUP elegimos un gráfico de tamaño 716x580
de  24  Bit y se giró 125.3 grados con el operador de la librería SuperView
"RotateFree",  el cual hace uso extensivo de operaciones con enteros y coma
flotante así como de varias funciones matemáticas transcendentales.

Aquí están los tiempos:
CyberStorm 060/50		21 segundos
CyberStorm 060/50 + 603e/150	 2 segundos

Esto  es  obviamente más de 10 veces más rápido.  De hecho es mayor, puesto
que:

a)  El  tiempo  de  ha  redondeado  hacia  arriba  (debería ser en realidad
alrededor de 1 segundo).

b) No había mucha memoria de 64 bit en la tarjeta CyberStorm.  La operación
habría  tenido lugar posiblemente en la lenta memoria de la placa madre del
A4000.

Los  módulos  plugin  PPC  opcionales están ahora disponibles para usuarios
registrados  por  un  precio, que es igual al precio de registro del normal
SViewNG.    Algunos   paquetes  comerciales,  los  cuales  ofrecen  soporte
licenciado de la librería SuperView, probablemente se venderán próximamente
también  con  estos  módulos.   Se seguirá además desarrollando paquetes de
módulos con añadidos exclusivos para programas comerciales específicos.
