Futura aparición de MetalWEB, un Editor de HTML.
------------------------------------------------

Amigos de AmigaChannel!

Queríamos  aprovechar  este mensaje para anunciarles la futura aparición de
un  nuevo  producto para el Mercado de Shareware de Amiga!.  Se trata de un
editor   HTML  WYSIWYG  (What  You  See  I  What  You  Get).   Con  grandes
posibilidades,  su  nombre  es  MetalWEB  y  viene  a  intentar  suplir las
carencias actuales en este tipo de programas para este fantástico Ordenador
que tenemos...

La  versión  demo  definitiva  aparecerá  dentro  de unas semanas o días en
AMINET.   Para  presentarlo a todo el mercado de Amiga, pero hemos decidido
enviarles  antes unos Screenshots y gif animados para que vean la capacidad
de este software!

Las características principales son:

* WYSIWYG
* No se necesitan conocimientos de HTML
* Visualización de las imágenes mediante por DATATYPES
* Fonts de diferentes tamaños (H1, H2, H3...)
* Uso fácil mediante Drag'n'drop
* Versión especial para 000, 020, 030, 040, 060.
* Completamente traducido al Castellano y al Catalán, además del Ingles

Requiere:

* MUI 3.7 o superior
* 4 Megas de RAM mínimas
* Datatypes para poder visualizar las imágenes
* Disco duro recomendable
