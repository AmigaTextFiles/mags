transAM


The following article contains the latest information on Pios's transAM, a next generation Amiga.

The transAM is aimed to be the next-generation Amiga, the high-performance homecomputer of the future, which the Amiga community has been waiting for.

The transAM runs true multitasking, multithreading and multimedial OS environments and features latest modern technologies like multiprocessing and object orientation.

The transAM is based on the unique modular hardware architecture from PIOS, see separate hardware page, thus providing optimal performance and upgradability... and all this for a much lower price than you might expect!

Final prices are yet to be announced.


Technical Specifications

Basics ATX-type motherboard
CPU Module
Expansion 5 PCI bus slots, 1 occupied with graphics card
2 ISA bus slots
Audio 16-Bit audio, 44.1 kHz, stereo in and out
internal sound inputs for CD and AUX
MIDI in and out
Input/Output 2 serial interfaces RS- 232
2 serial interfaces RS- 422
1 PC-style game port
1 digital audio port
MIDI In and MIDI Out ports
2 USB (Universal Serial Bus) ports
provision for serial IrDA interface (infrared remote control)
bidirectional parallel interface (Centronix compatible)
SCSI-2 for harddisks and CD- ROM, internal 50-pin and optional external connectors
EIDE for harddisks and CD- ROM, 2 internal connectors for 4 drives
floppy disk controller

Apart from the standard configuration, the mainboard and CPU Module together with the case are also available as "bare-bone" versions. Alternate system configurations with 200 MHz or up to 300 MHz versions of the CPU Module can be obtained from our distributors.

Standard Configuration

The fully equipped transAM-200 ships with:

CPU Module and Memory CPU 603p, 200 MHz
RAM: 16 MB
2L Cache: 512 KB
Graphics PCI bus SVGA card 2 MB
Drives EIDE Harddisk, 2.5 GB
CD-ROM drive, ATAPI, 12x speed
HD floppy disk drive

The transAM comes in an ATX miditower case with a PC-standard serial mouse and keyboard. Additional hardware components (Ethernet, ISDN, Modems, or ZIP-Drives, etc.) are available from our distributors.

For tight integration with existing Amiga hardware and smooth migration to our new platform, we are currently negotiating to include the Siamese System from HiQ and Eagle GmbH.

Actual capacities (e.g. harddisk size) may vary.

Software

By default, the transAM shipment includes the following software packages preinstalled on harddrive and on CD:
  • pOS
  • BeOS
  • Linux

  • MetroWerks Codewarrior Lite for BeOS
  • An Office software package

More details can be found on the separate software page.
Additional software is available from our distributors.
