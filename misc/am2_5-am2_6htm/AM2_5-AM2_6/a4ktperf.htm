The Amiga 4000 Tower: Performance Numbers

Benchmark information from several programs on several Amigas

By Michael Webb, Editor-in-Chief, MikeWebb@CompuServe.COM

The following performance information was obtained using several Amiga systems, as indicated in the first table. In each case, I attempted to optimize each system for performance, by remapping ROM into fast RAM where possible, activating CPU caches, etc. Most, if not all, of the tests performed probably took place with multitasking and interrupts disabled.

For a discussion of these tests and their results, see this page of the review.

First, I must make a few notes about the software used in this performance evaluation:

  • SysInfo v3.11 is by Nic Wilson.
  • SysSpeed v2.4 is by Torsten Bach and ALiENDESiGN.
  • XOpa v1.87 is by Axel Dörfler and Alexander Bartz.

  • Scenery Animator v4.06 68000 is by Brett Casebolt and Natural Graphics.
  • Scenery Animator v4.06 68020/68881 is by Brett Casebolt and Natural Graphics.
  • CygnusEd Professional v3.5 is by CygnusSoft Software.

Also, the software used in this evaluation (as much the benchmarkers as the actual applications) is not to be considered absolute and/or empirical in its results. Like anything that can be used in the same general manner (for performance analysis), by definition, its accuracy is limited by the methods it uses for examination. Probably the most striking example of this is the large differences between SysInfo's and SysSpeed's MIPS measurements. Despite using the same units, they produce very different results. Therefore, take all of these numbers with a grain of salt. But for relative comparisons of the same software on different Amigas, the results are probably fairly trustworthy.
