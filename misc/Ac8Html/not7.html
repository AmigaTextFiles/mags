MR. HARDWARE COMPUTERS SACA LA VERSION DEMO DE RETAIL ESCORT V4.1
-----------------------------------------------------------------

FECHA: 26/06/1997

Central Islip, NY - Martes, 17 de Junio de 1997:  "Mr.  Hardware Computers"
anuncia  hoy  que ha lanzado la largamente esperada versión de demostración
de  su Sistema de Punto de Venta para Amiga.  Estamos satisfechos por poder
lanzar una demo de nuestra más excitante versión de "Retail Escort".  Hemos
trabajado  mucho  y  duro  para  ofertar  un software de Gestión de Negocio
Profesional para Amiga.

Nuestra Visión de lo que el Software para la Gestión de un Negocio debe ser
es  simple.   Las  herramientas  de  gestión  que  uses  deben  ser  lo más
intuitivas y fáciles como sea posible.  El software debe hacer tu vida como
dueño/gestor  más  fácil.   Tú  tomas las decisiones, pero el software debe
hacer  el  trabajo.   Conociendo  que  es  lo  que necesitas, sólo debe ser
cuestión de un informe o dos más allá.  Con estos pensamientos en mente nos
presentamos a ti con la versión v4.1 de "Retail Escort".

La  versión  de  demostración  de "Retail Escort v4.1" está disponible para
bajar  de  Aminet  bajo  el  nombre  "RE41DEMO.lha".   Busca  "Retail" para
localizarlo.

"Retail Escort" te ayuda a manejar tu negocio con:

Cuentas  por  Cobrar,  Cuentas  a  Pagar,  Montones de Informes de Negocio,
Chequear   Cuentas,   Tarjetas   de  Crédito,  Notas  de  Comprador,  Fácil
Facturación,  Control  de Inventario, Seguimiento de Pedidos, Estimación de
Precios,  Ordenes  de  Compra,  Listas de Correo, Impresión de Etiquetas, y
mucho más.

"Mr.    Hardware   Computers"  es  una  empresa  100%  Amiga  y  está  100%
comprometida  al  posterior  desarrollo  de "SBase4", "SBase4Pro", la serie
"Escort",  y otros programas excelentes para Amiga.  Te invitamos a que nos
ayudes  a probar que el software de Amiga es el mejor para la gestión de un
negocio,  y  por  consiguiente, el Amiga es mejor para gestionar un negocio
que cualquier otro ordenador.
