CRONUS ANUNCIA EL "AMINET SET 5"
--------------------------------

FECHA: 14/07/1997

Arizona,  USA  -  "Cronus"  se  complace en anunciar que el "Aminet Set 5",
producido por Stefan Ossowski de "Schatztruhe" estará disponible en Estados
Unidos  a  partir  del  25  de  Julio de 1997.  "Aminet Set 5" ofrece a los
usuarios  de  Amiga  todo  desde  el  lanzamiento de "Aminet Set 4", más la
última  versión completa sin restricciones de ningún tipo de "Octamed Sound
Studio".   "Aminet  Set  5"  también  presenta casi un 1 GB de datos que no
aparecerán  en  la  serie  regular de Aminet en CD-ROM.  Fechado a Junio de
1997,  el  set  contiene  aproximadamente  4  gigabytes de software en 7400
archives.  Montones de cosas por sólo 35.95$ (más gastos de envio)

Este CD está disponible a través de Cronus en
