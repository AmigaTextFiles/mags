LA UTILIDAD "WUSAGE" PARA ESTADISTICAS DE WEB ESTA YA DISPONIBLE
----------------------------------------------------------------

FECHA: 14/07/1997

Me  gustaría anunciar que se encuentra disponible una versión para Amiga de
nuestro programa de estadística de servidor de WEB "Wusage".  Este programa
produce  atractivos informes HTML, con gráficos GIF en línea del tráfico de
tu sitio WEB.  La versión Amiga significa que cualquiera con un Amiga puede
producir estadísticas de sitios WEB tan bien como aquellos disponibles bajo
Unix  o  NT.   Esto  es útil, por ejemplo, si tienes un Amiga en casa y una
cuenta  virtual  de  servidor  WEB  en una máquina Unix corriendo Apache en
cualquier otra parte.
