Сайт tPA снова в строю!
Levitator сообщает "После полуторагодичного перерыва сайт team PowerAmiga снова в строю! Перерыв был связан с временной потерей контакта с администратором сервера на котором находился сайт. Также в планах работа над совершенствованием сайта."

ezTCP 0.10
Каждые несколько дней Павел Федин aka Sonic выкладывает новую версию разрабатываемого TCP/IP-стека ezTCP для MorphOS. Сегодня стала доступна очередная версия. За прошедшее время реализовано полноценное API bsdsocket.library v4, исправлена куча ошибок (за основу было взято ядро AmiTCP), в том числе в реализации событий. Hа данный момент идёт работа над сборкой DHCP-пакета, а также чистка оставшихся проблем ядра (зависание AmiSSL и проблемы с некоторыми DNS-серверами). В ближайших версиях появится DHCP-клиент и сервер, графический сислоггер, панель управления совместимая с MiamiDX. Для работы портированных Unix-приложений с ezTCP нужна библиотека usergroup.library.

Программное обеспечение

AmiNetRadio 4.0 & SDK
Вышла новая версия многоформатного музыкального плеера AmiNetRadio для AmigaOS3 и MorphOS. Сделано множество исправлений в коде программы, переписаны почти все плагины, добавлены новые ARexx-команды, исправлена работа через AHI, и прочее. Также на сайте появилось SDK для разработки своих плагинов.

AWeb 3.5.07, WookieChat 1.9
Готова новая бета-версия web-браузера AWeb для AmigaOS 3.x и 4.0. Версия для MorphOS появился чуть позже. Также вышла новая версия IRC-клиента WookieChat с различными усовершенствованиями.

CD-диверсия
Для AmigaOS4 и MorphOS появился набор утилит для конвертирования файл-образов компакт-дисков в ISO-9660 из форматов CDRWin, Alcohol 120%, CloneCD, BlindWrite, DiscJuggler, Instant Copy, NeroCD.

Ext2FS 0.16 и SGIXFS 0.7
Marek Szyprowski обновил драйверы файловых систем Ext2FS и SGIXFS. Исправлены ошибки.

AROS набирает обороты
Hаконец-то открылся архив программного обеспечения для AROS. Теперь всё наработанное программное обеспечение, ранее разбросанное по всему интернету, должно слиться в одном месте. Марк Картер опубликовал статьи об инсталляции дистрибутива ArosMin на чистый винчестер, а также об инсталляции AROS поверх предустановленного Linux (Slackware).

PolyOrganiser 0.84
Frederic Rignault обновил электронный органайзер PolyOrganiser (записная книжка, планирование встреч, контакты, дела) для MorphOS.

Cubic IDE 1.0
Dietmar Eilert выпустил Cubic IDE (бывшее название GoldED Studio) - интегрированную среду разработки для AmigaOS/MorphOS. Пакет включает в себя редактор GoldED8, бесплатные C/C++ компиляторы (типа gcc), HTML-плагин Webworld, пакет Joyce для проверки орфографии, и прочее. Cubic IDE хорошо интегрируется с SAS/C, vbcc, gcc, StormC.

ECX 1.4.7
Hа Аминет было выложено обновление для AmigaE-компилятора ECX (AmigaOS/MorphOS). Исправлено почти два десятка ошибок, улучшена генерация 68k-кода.

FreePascal Compiler 2.0.1 (04.10.2005) MOS
Обновился компилятор FreePascal для MorphOS. Из-за нестабильности был сделан откат на компоненты из предыдущего релиза, исправлена проблема в инициализации FPU.

Ambient 1.42 rc2 CVS
Разработка Ambient, графической оболочки для MorphOS, идёт полным ходом. С домашней странички можно скачать очередную бета-версию и оценить количество нововведений.

Thilo Koehler выпустил новую бета-версию MIDI/Audio-секвенсера HD-Rec. Изменения: запись в .wav и .aiff создавала повреждённые файлы, исправлена работа с AHI, улучшен GUI, добавлен MIDI-роутер, исправлены прочие ошибки.

APC&TCP будет распространять MED SS 2
Распространением новой версии трекерного редактора MED SoundStudio 2 (бывшее название OctaMED SS) будет заниматься APC&TCP. В начале 2006 года ожидается OS4-версия редактора. Разработку возглавляет Gerd Frank, автор AmiAtlas. Есть вероятность появления 68k- и MOS-версий.

NTFS 0.4, Ext2FS 0.20
Вышли новые версии 68k/MOS-драйверов для файловых систем NTFS (добавлена поддержка съёмных устройств, юникода, исправлены ошибки) и Ext2FS (добавлены асинхронные кэши буферов, что значительно ускорило скорость записи, драйвер теперь может форматировать диск и создавать полную структуру EXT2, исправлены ошибки).

Bars&Pipes 1.25
Alfred Faust обновил MIDI-секвенсер Bars&Pipes для AmigaOS. Изменения в основном коснулись GUI.

Обновление на Amiga.sf

Изменение спецификации на Amy'05
Guru Meditation и Troika на форуме AmigaWorld.net объявили об изменениях в спецификации материнской платы Amy'05. С появлением AGP-слота, который работает в PCI-режиме, изменился и форм-фактор материнской платы. Шина данных для некоторых компонентов теперь вместо 33MHz будет работать на частоте 66MHz. В первую очередь это сделано для AGP и Ethernet-чипа. С платы было убрано несколько чипов, количество слоёв уменьшено с 8 до 6, что также удешевило стоимость.

P.A. Semi - новый производитель процессоров PowerPC

Dan Dobberpuhl, соучредитель, директор и CEO компании говорит: <Мы начинаем с нуля, переобдумывая каждый шаг на пути к прорыву в дизайне процессоров, достигающих небывалый уровень в показателе производительности на ватт потребляемой энергии. В результате наши заказчики, которые смотрят в будущее, получат новую волну эффективных, высокопроизводительных систем на базе POWER.> Первым ожидается двуядерный процессор на 2GHz с потреблением всего 5 Ватт. Очевидно, что архитектура PowerPC набирает популярность вопреки, например, последним выходкам Apple. ;)

Pegasos III без северного моста
Просочились слухи, что планы по использованию процессора MPC7448 и северного моста Tsi108 в Pegasos III отменяются. Было принято решение об использовании двуядерного процессора MPC8641D, таким образом северный мост в новой материнской плате не потребуется. Дата выхода - второй квартал 2006 года. Ждём официальных пресс-релизов.
