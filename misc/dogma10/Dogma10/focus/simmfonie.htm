SIMMFonie - Переходник памяти ZIP -> SIMM
  Автор -= / Vinnny / =- © 02/2005
Название: SIMMFonie
Разработчик: Kato Development Group
Тип изделия: Переходник памяти ZIP -> SIMM
Период разработки: 1997

SIMMfonie это простой механический переходник памяти ZIP -> SIMM, сделанный прежде всего для использования в системах A3000 и A3000T, однако он может применяться и с другими устройствами. Переходник вставляется в разъёмы, которые обычно были заняты дорогой и медленной ZIP-памятью, позволяя теперь использовать два обычных 72pin SIMM-модуля памяти (поддержаны SIMM 4MB и 8MB, 80ns или быстрее, до 16MB суммарно). Есть две версии SIMMfonie, одна спроектирована для A3000, другая для A3000T. Причиной служат разные расстояния между контактами разъёмов ZIP-памяти, которые на каждой из этих материнских плат немного отличаются.
