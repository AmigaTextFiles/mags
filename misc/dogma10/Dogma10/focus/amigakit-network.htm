Локальная сеть на Амиге также нужна пользователям, как и интернет. Существуют разные решения для обеих целей. Наиболее простые - использование сетевой карты. Сетевых карт, работоспособных с Амигой существует уже великое множество. Хочу познакомить вас с новинками, появившимися в магазине AmigaKit.com. Это две карты: Amiga PCMCIA Network Card MKIII [NETPCM003] и Amiga PCMCIA Wireless Card [NETPCM010]. Первая является обычной сетевой картой, соединяемой проводом типа 'витая пара', вторая использует беспроводное соединение.

Amiga PCMCIA Network Card MKIII [NETPCM003]

Если вы хотите подключить свою Амигу в локальную сеть или просто использовать быстрый широкополосный интернет через маршрутизатор, то эта сетевая плата станет основой для данной цели.

Эта 10MBit PCMCIA-сетевая карта совместима с Amiga1200 и Amiga600. Карта комплектуется XJack-переходником, оборудованным разъёмом RJ45.

Стоимость карты Ј30.

Amiga PCMCIA Wireless Card [NETPCM010]

Соедините Амигу с беспроводной сетью! Теперь ваш компьютер сможет обмениваться информацией по беспроводной сети с другими компьютерами, типа ноутбуков, PC- и Apple-компьютеров.

Одна из главных особенностей (если вы имеете беспроводный широкополосный маршрутизатор) состоит в беспроводном соединении Амиги с интернетом. Теперь ваша Amiga не будет привязана к конкретной комнате и телефонной розетке.

Эта беспроводная сетевая PCMCIA-карта выполнена по стандарту IEEE 802.11b (Wi-Fi) и совместима с Amiga1200 и Amiga600. Драйвер поддерживает 128bit беспроводное шифрование для безопасности.

Стоимость карты Ј35.

Обе сетевых карты комплектуются:

  • 1x дискета, содержащая драйвер и утилиту, предотвращающую ошибку сброса PCMCIA на A1200.
  • Компакт-диск, содержащий последние версии дистрибутивов сетевых пакетов EasyNet, AmiTCP 3, Amiga Samba, SMBFS и Samba Handler.
  • Описание со скриншотами по установке драйвера с TCP/IP-стеком Genesis.
  • Описание со скриншотами по установке драйвера с TCP/IP-стеком Miami Deluxe.
