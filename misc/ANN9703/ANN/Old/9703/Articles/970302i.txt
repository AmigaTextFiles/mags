               What is KOFFIE?

               KOFFIE is a Java-compatible Virtual Machine for the Amiga.
               KOFFIE is build using BEAST. In fact the KOFFIE Virtual
               Machine is just another BEAST class (BST_KoffieVMClass).
               KOFFIE is at this moment a software interpreter to run the
               Java bytecode, the Java classes are BEAST classes. E.g. The
               BEAST KOF_PrintStreamClass emulates the Java
               'java/io/PrintStream' class. In the future KOFFIE will also
               have the possibility to turn the Java bytecode into
               assembler or C.
               AWT will be supported by the use of the BEAST BeaVis
               classes.

               What is the current status of KOFFIE

               Only a small number of bytecode is implemented, only enough
               to get the first demos to work.
             + The three supplied java classes should work, _very_ basic.
             + _Don't_ expect anything else to work.
             + KOFFIE is still quite slow because of the disjunct
               implementation towards BEAST. after the bytecode is fully
               implemented KOFFIE will be optimized.
