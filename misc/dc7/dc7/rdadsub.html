Submitting Reader Ads

If you want to place an ad, then you can do so for free. Either send it in just ASCII format, or both ASCII and as a picture. The reason we don't want just a picture is that we might not have enough room on the disk for everybody to have picture ads. But we probably will...

Important! If you want to submit an ad, you must send at least one article. This rule will be enforced! The only exception is if it is specifically specified elsewhere in this mag (eg, sending programs for review, or subscribing). You're probably thinking that you've got no idea what to write about. Well it can be on anything; not just programming. It doesn't have to be that good at all, really, just as long as it's something! It doesn't even have to be related to Amigas, if you want...
