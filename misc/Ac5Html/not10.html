HAAGE & PARTNER NEWS SEPTEMBER, 1996

PROFESSIONAL C(++) DEVELOPMENT TOOLS FOR THE FUTURE OF THE AMIGA

StormC 1.1 STARTER

16  Sept.   96 - HAAGE & PARTNER Computer anuncia una nueva version STARTER
de StormC 1.1.  Esta versión es especialmente para estudiantes y novatos en
ANSI  C  y  C++ y está al precio de US$ 169.  Este paquete ofrece todas las
ventajas  que  hacen  de  StormC  la mejor elección para programación en el
Amiga:   sistema  de  desarrollo  integrado,  rápido  editor  de fuente con
colorado de sintaxis, muy rápido compilador en ANSI C y C++, optimizando el
linker y el sistema run-time.

StormC  STARTER hace la programación del Amiga más fácil y afectiva.  Todos
sus  componentes  han  sido diseñados para ser fácil de usar.  StormC es un
compilador  dos-en-uno que ofrece los dos C estándar:  el conocido ANSI C y
el moderno C++ estándar (de acuerdo con AT&T 3.0) para objetos orientados a
programación (OOP).

Nueva versión 1.1 de StormC

También  ha  sido  realizada  la  versión  1.1  de nuestro nuevo sistema de
desarrollo  para  StormC  en Agosto.  Tiene muchas nuevas características y
actualizaciones  y  es  la primera versión internacional.  El programa está
localizado  en  Inglés,  Italiano,  Francés  y  Alemán  y  el  manual  está
disponible  en  Inglés,  Alemán  y  pronto  también  en  Francés.   La demo
internacional  de  la versión 1.1 está disponible en nuestra página en y en
Aminet (biz/demo).

Pronto: Versión Internacional de StormWIZARD

 Una  de las parte que más tiempo supone a la hora de programar es el crear
un  Interfaz  Grafico  para  el  programa.   Para  ello ofrecemos un editor
GUIfácil  de  usar, llamado StormWIZARD.  El uso de este editor te ahorrará
mucho tiempo programando y obtendrás un fantástico GUI.  StormWIZARD estará
disponible  en  Inglés  a  finales  de Septiembre a US$ 69.  La última demo
disponible está en nuestra página y en AmiNet.

Planes de Futuro

 Estamos  trabajando también en una versión especial de StormC para el casi
comercializado  PowerPC Amiga de Phase 5.  Su desarrollo se está retardando
y  esperamos  poder mostraros una pre-versión en COMPUTER 96 en Cologne (15
de   Noviembre  al  17).   Allí  mostraremos  un  nuevo  sistema  operativo
desarrollado  por  una  casa Alemana:  proDAD, llamado p-OS y presentaremos
una versión especial del StormC para p-OS también.

 Por  todas  esas razones StormC es el sistema de desarrollo para el futuro
del  Amiga.   StormC  soporta desarrollo para el sistema operativo de Amiga
actual y la próxima generación de PowerPC Amiga y sistemas p-OS.
