Realizado el Kit para desarrolladores de Directory Opus

 Tras  muchas  peticiones  de  los  usuarios de Opus y tras el gran interés
generado por el reciente lanzamiento de la versión 5.5 de DOpus, GPSoftware
se   complace   en  anunciar  la  realización  del  Kit  de  Software  para
Desarrolladores de DOpus.

 Este  archivo  contiene  información para los desarrolladores de Amiga que
desean  escribir  sus  propios modulos para Opus 5.5.  El archivos contiene
AmigaGuide,  autodocs  y  incluye archivos para DOpus5.5, instrucciones del
library  plus,  y  ejemplos de como crear modulos en 'C' para usar con Opus
5.5.  También se incluye el código fuente del programa de Opus ViewFont más
el fuente para un nuevo e innovador módulo IconClock y más.

 Estará  disponible  en  Aminet pronto, pero está disponible ya y gratis en
las    páginas   de   GPSoftware   de   Directory   Opus   localizadas   en
http://www.livewire.com.au/gpsoft/dopus55.html.
