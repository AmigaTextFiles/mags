Amiga Models

Here is a list of the current Amiga range, modified from last issue's, and looking more optimistic. Possible future Models are listed here also.

This is quite different from the list last issue, and a better range too - surely a sign that things are gatting better for the Amiga!

Name: A1200
Manufacturer: Gateway 2000 (currently)
Released: 1992,1995,1997
Current Price: £250 (can be as low as £210)
CPU: 68020 @ 14MHz
Ram: 2MB Chip, No Fast, No SIMM sockets
Gfx, etc: AGA
Other Specs: OS3.1; PCMCIA, CPU slot, IDE
£250 is better value for money, and several retailers are selling bundles with accelerators, extra RAM, hard drives, etc; most notably good value one has to be Power's with Viper 50MHz 68030, SCSI II, 6MB RAM for £270.

Name: A4000T
Manufacturer: Gateway 2000
Released: 1994,1995,1997
Current Price: £1049 without CPU / £1179 with '040
CPU: Bare / 68040 @ 25MHz / 68060 @ 50MHz
Ram: 2MB Chip, 4MB Fast, 4 SIMM sockets
Gfx, etc: AGA
Other Specs: Tower case with A4000 CPU slot, IDE, SCSI 2, Zorro3 slots, ISA slots, Video slot
A lot cheaper than under Escom - but if you really need an A4000 CPU slot, the Infinitiv 1500 is cheaper still.

Name: Infinitiv 1300
Manufacturer: Micronik
Released: 1997
Current Price: £349.95
CPU: 68020 @ 14MHz
Ram: 2MB Chip, No Fast, No SIMM sockets
Gfx, etc: AGA
Other Specs: Tower case, A1200 CPU slot, IDE
Basically an A1200T, intended to be upgraded by the user - no hardware bundles have been announced, but are expected.

Name: Infinitiv 1400
Manufacturer: Micronik

An A1200T just waiting for a graphics card to be added.

Name: Infinitiv 1500
Manufacturer: Micronik
Released: 1997
Current Price: £599.95
CPU: 68020 @ 14MHz
Ram: 2MB Chip, No Fast, No SIMM sockets

An A1200T that manages to be an A4000 clone at the same time (and a lot cheaper too). Perfect for a Cyberstorm and Cybervision PPC...

Manufacturer: Macrosystems

Despite what some may think, this is an Amiga, and is a viable alternative to the A4000T or A1200T clones for video work.
