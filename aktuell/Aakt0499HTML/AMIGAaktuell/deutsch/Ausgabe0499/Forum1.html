IRCNET


deutsch: irc.gmd.de -> port 6667 #amigager (usual amiga users) 
deutsch: irc.germany.net -> port 6667 #amiga (usual amiga users) 
englisch: irc.gmd.de -> port 6667 #amigascne (amiga scene) 
englisch: irc.gmd.de -> port 6667 #thescene (amiga&pc scene) 
englisch: irc.gmd.de -> port 6667 #amiga (ww amiga users) 
englisch: irc.vapor.com -> port 6667 #amiga (vapor server amirc) 
    

wobei in den englischen kanälen durchaus deutsch geschrieben wird, was daran liegt, dass es genug deutsche user und ops dort gibt ;)

passwort ist bei den oben genannten kanälen nicht erforderlich.

bei dem von mir benannten server "irc.gmd.de" handelt es sich um einen der schnellsten innerdeutschen irc server - falls es doch einmal zu problemen kommt, so weise ich auf weitere server hin, als da wären: "irc.uni-stuttgart.de", "irc.uni-paderborn.de", "irc.funet.fi", "irc.netsurf.de" - jeweils mit dem port 6667

bei dem recht kleinen germany.net kann man sein nick/handle registrieren und schützen lassen. näheres erfährt man, indem "/msg nickserv help" (ohne der anführungszeichen...hehe) eingetippt wird - wenn das nicht hilft, einfach einen der user im kanal #amiga fragen.

cHAVEZ aká Mark Buonfrate <chavez@pironet.de>«
