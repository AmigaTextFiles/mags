1. Kostenloses Java2-Paket angekündigt

Nordic Global, der Hersteller des TCP/IP-Stacks Miami, hat für den Sommer dieses Jahres ein komplettes und in der Standardversion kostenloses Java- Paket für den Amiga angekündigt. Hier die Presseerklärung:

»DaytonaTM ist ein komplettes JavaTM 2-Laufzeitsystem für AmigaOS, basierend auf der "Java 2 Platform" von Sun Microsystems, Inc. Es besteht aus den folgenden Teilen:

DaytonaTM-Hauptpacket, welches ohne Gebühren frei erhältlich ist. Neben anderen Dingen unterstützt Daytona im einzelnen:

  • Die komplette Java 2 API und deren Klassen, direkt aus Sun's Implementation abgeleitet und deshalb zu 100% kompatibel zu den offiziellen Java-Spezifikationen und Versionen (im Gegensatz zu einigen anderen Java-Interpretern für das AmigaOS).

  • Multithreading, unter Verwendung einer native Amiga Implementation. Keine Unix-Code-Emulation oder sonstige Bibliotheken erforderlich.

  • Datei und socket I/O (multithreaded) inklusive multicasting, URL Abwicklung und Unterstützung diverser Internetprotokolle

  • AWT(Abstract Windows Toolkit): Das Grundpacket Daytona enthält AWT Laufzeitklassen, allerdings ohne direkte Unterstützung von amigaOS- Screens. AWT-Verbindungen für amigaOS-Screens sind im DaytonaPLUS-Paket enthalten.

  • JFC (Java Foundation Class): Swing,Java2D.

  • COBRA

Das DaytonaDEVTM-Paket, ebenfalls ohne Gebühr erhältlich. DaytonaDEV ist für Entwickler gedacht, die ihre eigenen Javascripts schreiben und diese in Java-bytecode kompilieren wollen. DaytonaDEV enthält folgende Einzelheiten:

  • Java-bytecode-compiler: Startup-Routinen und alle benötigten Klassen.

  • Verschiedene Tools (Bytecode-Disassembler, Dokumentationsgenerator, Archiv-Management-Tools usw.)

Das DaytonaPLUS-Paket wird als Shareware veröffentlicht. Der genaue Preis wird zu einem späteren Zeitpunkt bekannt gegeben. DaytonaPLUS ist ein unabhängig von Daytona entwickeltes Add-on, welches zusätzliche amigaspezifische Leistungen bietet:

  • AWT bindings für direkte AmigaOS-Screens inkl. Unterstützung für CyberGraphX und Picasso96 high/true-color Screens.

  • BOOPSI "Look-and-feel" für Swing

  • Applet API und Laufzeitsystem mit eigenem Applet Viewer, und eine dokumentierte Applet API Bibliothek, welche es erlaubt, mittels Daytona AmigasOS Web-Browsern mit Applet API Unterstützung auszustatten.

  • PowerPC-Unterstützung, für PowerUP und WarpUP (noch nicht völlig entschieden)

  • JIT (Just-In-Time) Compiler für 68K und PPC

  • Unterstützung für vorcompilierte native (68k und PPC) Versionen aller Standard Java Klassen. Daytona und DaytonaDEV enthalten eine komplette Bibliothek mit 68K/PPC-vorkompilierten Klassen für die Benutzung unter DaytonaPLUS.

Alle Leistungsmerkmale sind noch eventuellen Änderungen unterworfen, nicht alle Features müssen in der ersten Version realisiert sein.

Daytona, DaytonaDEV und DaytonaPLUS sind Warenzeichen von Nordic Global Inc. Java und alle Java-basierenden Zeichen sind Warenzeichen oder eingetragene Warenzeichen von Sun Microsystems, Inc. in den USA und anderen Ländern. Nordic Global Inc., Daytona, DaytonaDEV und DaytonaPLUS sind von Sun Microsystems, Inc. unabhängige Produkte.«
