5. Softwaretest: gir06 v0.4 (von Axel Teichmann)

»Getestetes Programm:  gir06 (comm/tcp) Gsm internet realtime audio player 
Version:  0.4 
Art:  Shareware (5 US$) 
Autor:  Sinisa Kesic <sinisa.kesic@po.tel.hr> 
Tester:  A.Teichmann@gmx.net 
Testzeit:  1 Tag 
Verwendete Hardware:  Amiga4000/040 OS3.0 25Mhz 
 16 MB Fast Ram / 2 MB Chip Ram 
  

Installation:

Zur Installation kann man die komplette Directory dorthin kopieren, wo man es aufrufenn möchte.

Aufmachung:

Gir06 ist ein Echtzeit Internet Audio Abspieler für GSM 6.10 sprach Kompression.

Es liegen leider nur Anleitungen im Guide format in Englisch, Dänisch, Kroatisch,Italienisch und Spanisch vor. Wer auf Deutsche Hilfestellung hofft, wird leider enttäuscht. Im Archiv befinden sich Versionen für 020er und 040er Prozessoren, und auch einige kleine WAV -> GSM und RAW -> GSM Konverter.

Praxistest:

Nach Klick auf das Icon, bekommt man eine kleine GUI Oberfläche geboten, in der man aus einer Liste einige Beispiel Web Sites anwählen kann. Geht man dan auf Connect , wird versucht über einen TCP Stack eine Verbinung aufzunehmen.

Sowie eine verbindung besteht, hört man in nicht all zu berauschender Qualität Musik oder Sprache (Dafür ist es Echtzeit).

Desweiteren kann man über die Oberfläche die GSM Samples Kopieren oder Speichern oder wenn man eine neue Site gefunden hat, in die Interne Liste dazunehmen.

Durch einbau in einen Web Browser (als Defenition MIME TYPE "sound/x-gsm") lässt sich so konfortablerweise fast jeder Webbrowser durch GIR um ein echtzeit Audio Player erweitern.

Vollversion:

Durch erwerb der Vollversion bekommt man ein Keyfile, welches die kleinen Nervrequester entfernt. Ansonsten gibt es keinerlei einschränkungen.

Positiv:

Funktioniert einwandfrei. lässt sich als Mime type auch in Browser intregrieren.

Negativ:

Sehr Cryptisch gehalten (nichts für Anfänger). Englisch Kenntnisse erforderlich.

Fazit:

GIR06 ist eigentlich nur für Leute interessant, die wirklich ALLES aus Ihrem Amiga rausholen wollen, um anderen zu zeigen : "Siehst du ,es geht doch !".

Aber eine sinnvoller Anwendungsbreich für einen Echtzeit Audio Abspieler gibt es wohl kaum, da es auch nicht sehr viele WEB sites mit GSM Files gibt.«
