9. Emulator-News

Diesen Monat ist mit GBE ein neuer Gameboy und sogar Gameboy Color Emulator erschienen. Portiert wurde der Emulator von keinem geringeren als Mathias "AmiDog" Roslund. Der Emulator läuft momentan nur auf PPC, dafür aber entsprechend gut. Auch AGA wird mittels spezieller C2P Routinen voll unterstützt.

Im Aminet wurde eine neue Version von MAME (Multi Arcade Machines Emulator) veröffentlicht. Aktuelle Version ist jetzt 0.36 Beta 8.

Weil noch nicht alle Vorbestellungen bezahlt wurden, verzögert sich die Veröffentlichung von Fusion PPC. Spätestens Weihnachten wird sie aber veröffentlicht werden.

Juan Antonio Gomez Galvez hat mal wieder zugeschlagen: Auf seiner Homepage wurden neue Version von AmiMasterGear, AmiMSX und AmiGameBoy veröffentlicht. AmiMasterGear ist jetzt in der Version 0.7 vorhanden, AmiMSX in der Version 2.52 und AmiGameBoy in der Version 0.91. Juan Antonio scheint nun doch wieder am Amiga zu entwickeln; nachdem er letzten Monat schon eine neue Version von AmiMasterGear veröffentlicht hatte und nun diesen Monat sagenhafte 3 neue Versionen rausbrachte, dürfte die Sache wohl klar sein. Vor einem Jahr hatte sich Juan Antonio vom Amiga offiziell verabschiedet.

Die CyberGraphX Version von ANES ist mittlerweile in der Version 1.15 aktuell. In diesem Monat wurden einige Updates zu ANES veröffentlicht.

Von DarcNES wurde ebenfalls eine neue Version veröffentlicht. Zu finden wie immer auf AmiDog`s Webseite.
