11. Emulator-News

Ein Entwicklerteam arbeitet momentan an einem neuen SNES Emulator für den Amiga, der komplett in Assembler geschrieben wird, was eine hohe Emulationsperformance erzeugt. Dabei soll es auch einen PPC-only Modus geben, der auch in Assembler geschrieben ist. Ab einem 68040 soll der Emulator benutzbar sein.

Matthias "AmiDog" Roslund hat eine neue Version des N64 Emulators TrueReality veröffentlicht. Sie kann von http://www.amidog.com/ geladen werden. Dort wurde auch eine neue Version von DarcNES veröffentlicht. DarcNES ist ein Multikonsolen Emulator. Da alle guten Dinge bekanntlich drei sind, hat AmiDog auch noch eine neue Version des Sega Mega Drive Emulators AmiGenerator veröffentlicht. So, jetzt aber nix wie auf http://www.amidog.com/ und die neuen Emulatoren runterladen.

Juan Antonio Gomez Galvez hat auf seiner Homepage eine neue Version seines AmiMSX Emulators veröffentlicht. Es handelt sich um Version 3.0, die allerdings noch im Beta Status ist. Sozusagen ist diese Version ein Preview.

Eine neue Multi GUI für fast alle Emulatoren wurde diesen Monat im Aminet veröffentlicht. Die GUI namens Nostalgia wurde grafisch sehr anspruchsvoll (Achtung: Hohe Auflösung!) gestaltet und unterstützt auch PPC. Wer alle seine Emulatoren mittels einem Programm bedienen will, sollte sich Nostalgia unbedingt mal anschauen.
