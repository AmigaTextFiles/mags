Software (kommerziell)
---------------------------------------------------------------------------

68k/vermischt

Der HD-Recording-Software 'Audio Evolution' (ehemals PlayHD) ist ein Update auf Version 3.20 widerfahren.
[http://www.audio-evolution.com]

'Digital Almanac' ist in Version 3.5 erschienen und wurde u.a. um Buttons im GlowIcon Stil von Martin Mason Merz erweitert.
[http://members.tripod.de/achimste]

'MakeCD' ist in Version 3.2 beta 7 erschienen und enthält einige Bugfixes, z.B. bei den SCS3_Atapi-Treibern.
[http://makecd.core.de/ftp/MakeCD32d_public_beta_7.lha]

Für 'CyberGraphX 4' ist ein Update erschienen, das für Besitzer eines G-Rex PCI-Bus-Boards gedacht ist.
[http://www.schatztruhe.de]

Für das Grafiktool 'UltraConv' ist ein Update auf Version 4.1 erschienen.
[http://www.innovative-web.de/reg.php?rlang=d]

Das Programm 'Pro Station Audio' liegt in der neuen Version V4.10 vor und bietet Unterstützung für diverse neue Formate.
[http://www.audiolabs.it]

Das 'isdnsurfer.device' ist in Version 1.944 erhältlich und unterstützt Sync-PPP, wodurch nun die Einwahl in alle Netze per Call-By-Call möglich ist.
[http://www.jschoenfeld.de/jumpnews.htm]

PowerPC

Für den iMac Emulator 'iFusion' ist das Update auf V1.2 erschienen, das kleinere Fehler beseitigen soll.
[http://www.blittersoft.com/cgi-bin/ams/shopzone30.cgi/iFusion.html]

'Digital Almanac' ist in Version 3.5 erschienen und wurde u.a. um Buttons im GlowIcon Stil von Martin Mason Merz erweitert. Weiterhin ist IFF-ANIM Kompression nun auch im PPC-Modus möglich.
[http://members.tripod.de/achimste]
