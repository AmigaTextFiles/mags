»FOR IMMEDIATE RELEASE

We are proud to announce the release of AmiFTPd v1.0, a new, native Amiga FTP service daemon.

AmiFTPd boasts ease of setup, a GUI configuration editor, multiple account support, anonymous support and a secure Dynamic Virtual File System (DVFS)

A demo has been uploaded, to Aminet, so that it may be evaluated before 
purchase. 
 

See our web pages at http://www.cnetbbs.net/ for more information about AmiFTPd.
