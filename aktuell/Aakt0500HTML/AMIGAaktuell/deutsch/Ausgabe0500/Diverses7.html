23.3.2000 VHI Studio 5.0 verfügbar

Der Nachfolger des bekannten VLRec NG, VHI Studio 5, ist ab sofort verfügbar.

Diese Software erlaubt die Aufnahme, das Editieren und "Grabben" von Bildern mit Hilfe eines Digitizers (durch VHI werden verschiedene unterstützt, z.B. VLab, Frame-Machine, Masoboshi MVD 819, GVP IV24) oder einer Digitalkamera. Weitere Extras, wie etwa die Manipulation von Bildern, animierten Effekten, WebCam- und Zeitraffer-Möglichkeiten sind ebenfalls im Programm integriert. Mit VHI Studio ist es darüberhinaus nun sogar möglich, alle Bilddaten über eine virtuelle Disk von jedem Programm aus laden zu können, auch wenn sie noch nirgends gespeichert wurden.

Neuerungen gegenüber der Version 4 von VLRec NG sind:

  • Digitalkamera-Unterstützung hinzugekommen
    • Unterstützung von Seriennummern und damit die Möglichkeit kommerzieller VHI-Treiber für Digitalkameras

    • Einlesen des Digitalkamera-Inhaltes direkt in das Snapshot-Center

    • Informationen über das aufgenommene Bild werden im Snapshot-Center angezeigt

    • Informationen über den Status der Kamera können angezeigt werden

    • Bilder können via Buttondruck aufgenommen werden

  • direkte Bildbearbeitungsmöglichkeiten im Snapshot Center
    • Spiegelungen

    • Rotationen

    • Optimierungen

  • Neuer Treiber CamEx.vhi
    • drei virtuelle Eingänge mit unterschiedlichen Animationen (die übrigens als JPEG an VHI Studio übergeben und in Echtzeit dekodiert werden)

    • Simulation einer Digitalkamera (einzigste Einschränkung: Die Simulation merkt sich nicht, ob ein Bild im Speicher der Kamera gelöscht wurde oder nicht)

  • VHI Studio VD (Virtual Disk)
    • alle Bilddaten im Speicher von VHI Studio sind auch als Dateien verfügbar

    • MPEG-Encoding benötigt KEINE temporären Dateien mehr
      • Speicherplatzersparniss auf HD

      • schnelleres Encoding, da die Bilddaten nicht mehr von Platte sondern vom wesentlich schnelleren Hauptspeicher geladen werden und weder vorher gespeichert noch danach gelöscht werden müssen

    • Zugriff auf Digitalkameras wie auf eine Festplatte von jedem Programm aus

  • Multikanalvorschau (bis zu neun Eingänge gleichzeitig im Blickfeld)

  • Neuer Treiber für die GVP IV24

  • Treiber für Olympus-Digitalkameras optional erhältlich

  • Treiber für Kodak-Digitalkameras optional erhältlich

  • Direkte Unterstützung von Picasso 96 (nicht wie bisher über die cybergraphics.library-Emulation von P96)

  • Erneutes Einladen von Bildern in das Snapshot Center ist jetzt möglich

  • IFF-ILBM-Files können nun geladen und gespeichert werden

25.4.2000 fxPAINT EK #2 fertiggestellt

Das zweite Update zu fxPAINT wurde fertiggestellt. Die neue Version bietet eine Fülle neuer Features und dank vielen Optimierungen eine wesentlich gesteigerte Geschwindigkeit. Die wichtigsten Neuerungen im Überblick:

  • fxVIDEO-System: Tutorial-Videos direkt innnerhalb von fxPAINT!

  • neues Zeichenwerkzeug: Bevelcircle

  • 6 neue Effekte (Edge-Detect, Sharpen, Texture, "Deep inside", DisplaceMap und ein neuer Buttonrand-Effekt)

  • echtes Antialiasing für Text

  • weiter verbesserte Zeichentablettunterstützung

  • stark verbessertes fxALBUM (insbesondere verbesserte HTML-Album- Erstellung)

  • wap.plugin: fxPAINT ist das erste und einzigste Programm zum Lesen und Schreiben von WBMP-Bildern!

  • AXIS-Farbkalibrationssystem für optimierte Bildschirmdarstellung

  • Mousewheel-Unterstützung für Mäuse nach dem NewMouse-Standard (z.B. VMC 4D+ Maus)

  • beschleunigte Lade- und Speichermodule dank Portierung auf PPC

Desweiteren wurden viele von Usern angeregte Verbesserungen im Detail vorgenommen.«
