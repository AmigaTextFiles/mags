7. Amiga-Message-Board-Software erschienen

»NEWS 27-01-99: FOR IMMEDIATE RELEASE!

Announcing news Amiga WWW Message Board software! Ray A. Akey of ZenMetal Software is pleased to announce the release of Base0 Web Service (B0WS). B0WS is the premiere message board software for the Amiga and, can be used as a "bbs-style" discussion forum or as a support message board. B0WS supports both flat and hierarchical message areas and now has theme support.

Those interested in ordering B0WS should access the Official B0WS Homepage at http://www.cnetbbs.net/bows/. There are links to current systems running B0WS in various themes and looks, recent version changes and ordering information.«
