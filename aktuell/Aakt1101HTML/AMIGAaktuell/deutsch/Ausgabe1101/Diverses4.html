4. Kurzmeldungen (von Oliver Tacke)


Software (kommerziell)


68k/vermischt

Das Amiga-Emulator-Paket für x86-Rechner, AmigaOS XL, ist ab sofort erhältlich.
[http://www.haage-partner.com]

Vom Browser "Voyager" ist eine neue Version erschienen (V3.3.117beta).
[http://www.vapor.com]

Der Treibersoftware des PCI-Bus-Boards "Mediator" ist ein Update widerfahren auf Version 2.5. Außerdem ist der lang erwarteten Warp3D- Treiber für die Voodoo3-Grafikkarten erhältlich.
[http://www.elbox.com]

Weiterhin gab es ein Update der "Mediator Multimedia CD" auf Version 1.06. Zusammen mit diesem Update erhält man u.a. ein Programm, das es in Verbindung mit PCI-TV-Karten und dem Mediator ermöglicht, Fernsehbilder im Hintergrund der Workbench abzuspielen.
[http://www.elbox.com]

Die "Vapor-Toolkit.library", die von diversen Vapor-Programmen genutzt wird, ist in Version 15.11 erhältlich.
[ftp://ftp.vapor.com/pub/misc/]

Ein Update von CyberGFX auf Version 4.3beta ist erschienen.
[http://www.vgr.com/grex/]

Das Programm "ImageFX" wurde aktualisiert zur Version 4.5 und ist ab sofort erhältlich.
[http://www.novadesign.com/]

Ein Update für die virtuelle Sternwarte "Digital Almanac III" ist erschienen.
[http://soft-ware.de/dalmanac/]

PowerPC

Ein Update für die virtuelle Sternwarte "Digital Almanac III" ist erschienen. Der Fehler beim Auswerten der Mausposition wurde behoben.
[http://soft-ware.de/dalmanac/]

MorphOS

Vom Browser "Voyager" ist eine neue Version erschienen (V3.3.117beta).
[http://www.vapor.com]


Software (nicht kommerziell)


68k/vermischt

Der MP3-Player "Amplifier" ist in Version 2.33 herausgegeben worden und bietet nun auch für AmigaOS 3.0/3.1 den vollen Funktionsumfang, der vorher Nutzern von AmigaOS3.5+ vorbehalten blieb.
[http://www.katodev.de/deutsch/amplifier.html]

Das beliebte Programm "MagicMenu" ist in Version 2.31beta zum Download freigegeben worden.
[http://www.magicmenu.de/]

Der Virenscanner "Safe" ist in Version 17.3 erschienen.
[http://home4.inet.tele.dk/vht-dk/amiga/safe/safe.htm]

Ebenfalls ein Update erhielt "Virus Executor" auf Version 2.22.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/ve.htm]

Die "xvs.library", die zum Aufspüren und Entfernen von Viren verwendet wird, ist in Version 33.36 herausgegeben worden.
[http://home4.inet.tele.dk/vht-dk/amiga/news.htm]

Der Instant Messenger "CIM" ist in Version 1.05 erschienen und ermöglicht nun die Möglichkeit verschlüsselter Chats.
[http://www.geocities.com/SiliconValley/Bridge/5737/cim.html]

Auch der Instant Messenger "AmigAIM" erhielt ein Update, nämlich auf Version 0.9433 beta.
[http://www.missingpiece.com/AmigAIM/]

Neue und verbesserte Pakete für "WHDLoad" sind erschienen, u.a. für Jumping Jack Son und Zool. Mit "WHDLoad" ist es möglich, Diskettenspiele auf Festplatte zu installieren, obwohl sie nicht dafür vorgesehen waren.
[http://www.whdload.org]

Die Warp-Datatypes von Oliver Roberts sind aktualisiert worden: WarpJPEG (V44.25), WarpPNG (V44.21), WarpTIFF (V44.5), WarpBMP (V44.6) und WarpPSD (V44.3).
[http://www.oliver-roberts.co.uk/]

Die Amiga-Portierung von "mySQL" ist in Version 4.0.0 (Alpha) erschienen.
[http://sixk.maniasys.com/index_en.html]

Das Programm "ReqAttack", das die Workbench-Requester verschönert, ist in Version 1.93 herausgegeben worden.
[http://jacadcaps.tripod.com/amiga.htm]

Der Dateimanager "ClassAction" ist in Version 4.3 herausgegeben worden.
[http://www.martin-elsner.de/]

Das Grafikprogramm "Perfect Paint" steht in Version 2.75b zum Download bereit.
[http://gothic.fr.free.fr/amiga/index.html]
