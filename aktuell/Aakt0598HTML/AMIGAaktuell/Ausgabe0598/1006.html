6. Ankündigung: BattleDuel v1.7.85

»TITLE
BattleDuel V1.7.85

AUTHOR
Jochen Terstiege <j.terstiege@cww.de> Michael David <m.david@cww.de> Marco Seine <m.seine@cww.de>

DESCRIPTION
BattleDuel is a game like 'Artillery Duel' on the C-64: Two players fight against each other with cannons placed in a windy mountainous landscape. To hit the other they change the power and angle of the cannons. A duel is finished if one of the cannons is totally damaged.

Some features:

  • up to 8 players
  • computer opponents
  • normal duel mode, 2 tournament modes, practice mode
  • team mode
  • ECS/AGA/Graphicboard support (only one program)
  • network support: nullmodem, modem, parnet and TCP/IP
  • nice graphics and sound effects
  • full multitasking
  • OS friendly
  • support for public screens

NEW FEATURES
  • CyberGFX and Picasso96 support
  • new graphics with 128 colours for graphic boards
  • complete localization
  • team mode with four cannons
  • tournaments with 8 players
  • program restarts automatically after changes in the preferences
  • information requester ("i"-key)
  • synchronisation test (network mode)
  • fade effects when program is started/quit
  • sound-channels can be swapped
  • volume is adjustable
  • initial cannon angle for each player
  • opponents settings can be hidden (network mode)

SPECIAL REQUIREMENTS
  • Kickstart 2.0
  • Flicker-fixer or graphic board recommended
  • 1 MB chip memory
