Amiga PowerPC

1. Kurzmeldungen (von Oliver Tacke)

Updates gibt es zu den WarpOS Datatypes von Oliver Roberts: WarpJPG, WarpPNG, WarpBMP. Außerdem gibt es eine neuer Version des Voreinstellers.
[http://www.nanunanu.org/~oliver]

Ebenfalls von Oliver Roberts stammt ein Datatype für Adobe Photoshop Files.
[http://www.nanunanu.org/~oliver]

Der MPEG Player Frogger ist als Version 1.64 veröffentlicht worden und soll um 25% schneller sein als die Vorgängerversion.
[http://frogger.rules.pl]

Das JFIF Datatype von Andreas Kleinert, akJFIF, steht in Version 44.96 zum Download bereit. Es werden sowohl PowerUp als auch WarpOS unterstützt.
[http://www.ar-kleinert.de]

ProStation Audio ist in der neuen Version 4.05 erschienen und bietet u.a. einen nativen Treiber für die Repulse Soundkarte. Weiterhin wurde die Aus- nutzung des PowerPC Prozessors verbessert.
[http://www.audiolabs.it]

Für MorphOS wurde es eine neue Version von lha veröffentlicht.
[http://www.morphos.de/meanmachine/files/Fr3dY/lha-mos114i.lha]

Von Stephan Rupprecht ist Version 45.2 des ILBM Datatypes fertiggestellt worden.
[http://home.wtal.de/stephanr/]

Eine Portierung von SoX (V12.17.1) für MorphOS ist erschienen.
[http://www.morphos.de]

Die MorphOS-Version von MUI wurde weiterentwickelt und trägt nun die Versionsnummer 3.9 Release 2.
[http://www.morphos.de/meanmachine/files/MUI_Release-2.lha]

VoyagerPPC V3.3.92beta ist für MorphOS erschienen. Nutzer ohne Keyfile können das Programm im Demo-Modus nutzen.
[http://www.morphos.de]

Oliver Tacke <oliver@aakt.de> - schickt mir eure News!
