»Produktinformation

Die Aminet CD 28 - Dezember 1998 - beinhaltet über ein Gigabyte (ungepackt) an Software in Tausenden von Archiven. Seit der Aminet 27 CD sind mehr als 500 MB an Neuheiten hinzugekommen. Alle Programme sind Deutsch beschrieben; viele verfügen über eine deutsche Dokumentation. Als besonderes Highlight enthält die Aminet 28 eine Vollversion von CygnusEd Professional Release 3.5 inkl. einer günstigen Update-Möglichkeit auf die Version 4.0!
