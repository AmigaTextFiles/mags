»AMIGA ist dabei!

Zahlreiche Highlights begleiten den diesjährigen Messeauftritt von AMIGA International. Das AMIGA-Museum präsentiert vom ersten Modell alles bis hin zur neuesten Generation. Das geniale Betriebssystem AMIGA OS 3.5 ist nun verfügbar und kann auf der Messe eifrig getestet werden. Die Prämierung des AMIGA Animationswettbewerbes0m findet am 14.11.99 um 14.00 Uhr auf dem AMIGA-Stand statt. Und in Zusammenarbeit mit dem Veranstalter - jede Menge surfen und chatten im Cybercafe bei AMIGA. Selbstverständlich stehen der Fachpresse zahlreiche Ansprechpartner aus dem Hause AMIGA Int. Inc. zur Beantwortung von Fragen zur Verfügung.«

»Neuigkeiten von rbm auf der HOME electronics world

Warum immer nur beige Computergehäuse? rbm wird farbige Big-Tower-Gehäuse in 16 verschiedenen Farben in Köln vorstellen - auf Wunsch sind die Farben auch als Metallic-Lackierung erhältlich.

Sicherheit für Ihren AMIGA - unter diesem Motto stellt rbm digitaltechnik eine unterbrechungsfreie Stromversorgung ( USV ) vor. Die USV schützt Ihre Daten vor Überspannungsschäden und Stromausfällen. Die beiliegende Software ermöglicht dem Anwender von ihm erstellte AReXX-Scripte auszulösen, die Daten zu speichern oder Meldungen via Internet versenden zu lassen.

FX-Paint 2.0 wird von Felix Schwarz ( Programmierer von UConv. 4.0 ) auf dem Stand von rbm digitaltechnik vorgestellt. Umfangreiche Unterstützungs- Features zeichnen dieses neue Programm aus. Dazu rbm digitaltechnik: "Diese Software verfügt über viele weitere Funktionen, die so einzigartig sind, dass sie bisher in dieser Form weder für AMIGA noch für andere Plattformen verfügbar sind."«

»ANNEX sorgt für DANCE- und MUSIC-Power

Auf der Bühne der HOME electronics world gibt es Dancefloor in Perfektion. ANNEX wird den Besuchern an den drei Messetagen in Köln Gutes für Ohren und Augen anbieten. Lassen Sie sich überraschen und verpassen Sie ANNEX in keinem Fall Gesponsert von AMIGA Int.«
