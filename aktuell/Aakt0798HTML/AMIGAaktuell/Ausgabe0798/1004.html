DESCRIPTION

ConDios is an ANSI text, graphics and animation viewer. The special thing about ConDios is that it emulates console.device and due to that it can support stuff console.device doesn't. ConDios uses an engine in the condios.library which is written in assembler for optimal speed.

It support 16 color ANSI (EGA), 8 color ANSI (Amiga), monochrome and a special mode to allow stripping ANSI sequences. ConDios can process any file that contains ANSI 3.41, VT100 (VT54), VT220 ESCape control codes, CSI sequences, SGR, ISO, control codes and EMCA94 Latin 1.

The ConDios viewer can be started from Workbench or from CLI. From CLI (or Shell) you have a whole set of options which should enable ConDios to view any ANSI file correctly. It can view a file of any size as it uses buffered reading. You can even use the viewer as a terminal display with the use of a fifo (ie. fifo-handler). ConDios can open on any public screen you specify.

In benchmark testing ConDios is up to 17% faster than Amiga's console.device with plain text and 55 times faster with ANSI animations (A4000/030/25).

SPECIAL REQUIREMENTS

Required:

condios.library (included)
