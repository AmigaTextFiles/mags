4. Inhaltsübersicht: Aminet Set 8 (von Sofiane Ben Hassine)

»Aminet Set 8

Die Schatztruhe hat es mal wieder geschafft, uns ein neues Aminet Set auf vier randvollen CDs zu pressen. Auf diesen vier CDs findet man den Inhalt der Aminet 28 - 31.

Auf der ersten CD findet man die Vollversion von Directory Opus 5.5. Dort sind auch alle Tools letzten vier CDs zusammengefasst. Die zweite enthält alle Grafik-Daten sowie eine Vollversion von ArtEffect 1.5, so dass man schon Grafikbearbeitung durchführen kann. Die dritte CD enthält eine Vollversion von Gloom 3. Der Schwerpunkt dieser CD ist Fun. Das heißt, alles was ein Zocker braucht, Spiele, Demos, Patches. Die vierte und letzte CD enthält alle Mods sowie spezielle Software zum Abspielen. Die Vollversion ist CygnusEd 3.
