3. Village Tronic sucht Namen für 3Dfx-Erweiterung

»Presse Erklärung:

3Dfx add on für die PicassoIV - GIB DEM KIND EINEN NAMEN!

Angekündigt für das 3. bis 4. Quartal 1999 ist unser neues add on board für die modular erweiterbare PicassoIV Grafik-Karte:

Die Karte wird 3Dfx Technologie auf ZorroIII Amigas wie z.B. A3k, A4K, A1200 (Towersysteme mit ZorroIII Busplatine) im Zusammenhang mit einer PicassoIV zur Verfügung stellen. Sie werden mit dieser Kombination eine 3D- Funktionalität für Amiga-Classic Systeme erhalten, die es bis dato noch nicht gegeben hat.

Mit dem zusätzlich erhältlichen PabloIV Modul können sie ihre 3D- Anwendungen auf ihrem 25" Fernseher darstellen und ihre Bildschirmausgabe auf einem Videorekorder aufzeichnen.

Von der technischen Seite wird die Karte eine Auflösung von max. 960x720 Pixel ausgeben können, in Zusammenarbeit mit einem Speicher von 8MB texture/pixel RAM. Programmierer werden eine native GLIDE® API und später einen zusätzlichen Warp3D Treiber für das Haage 3D API-System zur Verfügung gestellt bekommen.


CHIPSATZ: Voodoo, 3Dfx 
BUSSYSTEM: interner PicassoIV PCI connector, PIV im Z3-Modus 
RAM: 8 MB 
AUFLÖSUNG: 960x720 in 16 bit 
ZUSÄTZL. MODUL: PabloIV 
DRIVER SYSTEM: Glide®, Warp3D 
MIN. ANFORDERUNGEN: ZorroIII, (A3k, A4k, A1200+ZIII board) 
 8 MB usual fast-ram, 1 MB chip, CPU 040/25 minimum
