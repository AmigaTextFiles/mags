3. Neuigkeiten von OnyxSoft (von Dieter Beck)

Das Programm TheMPegEncGUI von Daniel Westerberg (Schweden) hat einen Sprung von Version 2.12 auf Version 2.2 gemacht, welche einige neue Funktionen mit sich bringt. Da wäre zu erwähnen, dass man jetzt mehrere Konfigurationen speichern kann und dadurch auch Freunden, die nur geringe Erfahrung mit den Optionen (die ja bei LAME 3.91 sehr zahlreich sind) haben, eigene Konfigurationen schicken kann, und sie somit sofort mit dem Encoden loslegen können. Die weiteren Neuerungen sind viele Änderungen bei den Optionen von LAME und natürlich viele Fehlerbereinigung gegenüber Version 2.12.

Zur Zeit ist das Archiv nur von der Support-Site zu beziehen. Darin enthalten ist auch eine LITE-Version, die ausschließlich LAME unterstützt. Eine weitere Version unterstützt zusätzlich auch MPEGA.

Konfigurationen der Vorgängerversion können importiert werden, so dass man sich die langwierige Einstellerei sparen kann.

Ferner gibt es zwei weitere Updates zu Programmen desselben Autors: DRemind in der Version 1.6 - diese Version läuft jetzt auch auf Rechnern, die keine Akku-gepufferte Uhr haben, denn die Uhrzeit kann nun z.B. auch via Internet gestellt werden - und das Programm MultiRen in der Version 1.52.

Alle Archive können auf der OnyxSoft-Homepage (http://www.onyxsoft.nu) heruntergeladen werden.
