9. Neu eingetragene AmigaNG-Software (von Thorsten Schölzel)

Hier die jüngsten AmigaNG-Neuerscheinungen bzw. -Nachtragungen:

  • Kill: Prozessabbruch durch Signal

  • Filedialog: Filedialog-Tool

  • Phat Edit: Grafischer Texteditor

  • Rebol: Programmiersprache

  • Easier-Emacs: Bildet bearbeitende emacs-Operationen

  • AmiFlag: AmigaVP-Version vom "Flag and Bombs"-Spiel

  • Clip: Clipboard-Tools

  • Ewinease_b035: Erstellt GUI-apps in einfachem C

  • Exev 1.0: AmigaVP exec-Befehl

  • Pix2PNG: Konvertiert pixmaps nach PNG

  • Povelate_b: POV-Ray 3.1g (binary only)

  • Povelate_s: POV-Ray 3.1g (elate source only)

  • Python 1.6: Python 1.6

  • Regina: GNU-Rexx-Interpreter
