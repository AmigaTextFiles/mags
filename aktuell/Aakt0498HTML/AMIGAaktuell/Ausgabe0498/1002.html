Asimware is proud to announce today's release of AsimCDFS v3.9.

This is a maintenance update for the AsimCDFS v3 package.

The update highlights include:

Added support for the following drives:

  1. Yamaha CDR-400 CD-R drive
  2. Yamaha CRW-4260 CD-RW drive
  3. Yamaha CRW-2260 CD-RW drive
  4. Sony CDU-926S CD-R drive
  5. Sony CDU-415 CD-ROM drive
  6. Plextor UltraPlex CD-ROM drive
  7. Plextor PX-R412C CD-R drive
  8. Teac CD-R55S CD-R drive
  9. Ricoh MP6200 CD-RW drive
  10. Ricoh MP6201 CD-RW drive
  11. Pioneer DR-503s (24x) CD-ROM drive
  12. Pioneer DR-506s (32x) CD-ROM drive
  13. Toshiba XM-6201B (32x) CD-ROM drive
  14. ATAPI CD-R/RW drives

AsimCDFS v3.9 also added the following:

  1. 4-way IDE splitter support for the Amiga 1200
  2. CDDA to 8svx 8-bit mono conversion in the virtual CD-Audio volume.
  3. updated the Inhibit dos packet to prevent any commands from being sent to the CD-ROM drive by asimcdfs.device
  4. asim_atapi.device now supports the complete trackdisk command suite
  5. asim_atapi.device now supports the NewStyleCommands protocol

Registered users can obtain this release via our WWW or ftp site.

ftp: ftp://ftp.asimware.com WWW: http://www.asimware.com/

You will need your AsimCDFS serial number to obtain access.

The AsimCDFS software package is a CD-ROM control system which allows the user to read most CD-ROM discs with an Amiga computer and a suitable SCSI or ATAPI CD-ROM drive.

AsimCDFS is equipped with the following advanced features:

  1. access to ISO 9660, High Sierra, Rock Ridge and Macintosh HFS CD-ROM formats;
  2. FishMarket, a CD-ROM disc containing the public domain Fred Fish collection;
  3. AsimTunes, an audio librarian/controller providing advanced librarian and playback features;
  4. support for direct digitizing from standard audio CDs and playing audio via the Amiga hardware;
  5. CDTV and CD32 emulation modules, complete with autobooting from CD-ROM discs;
  6. integrated support for Kodak and Corel PhotoCD discs. Colour WorkBench icons are created for easy identification. Support for resolutions from 192x128 up to 3072x2048;
  7. support for both SCSI and ATAPI CD-ROM drives;
  8. preferences editor for all AsimCDFS settings; and,
  9. full ARexx command set available for all programs.

Asimware Innovations Inc. is a Canadian software development house providing innovative software solutions since 1992. We are specialists in DVD, CD-ROM, CD-R and CD-RW technologies.«
