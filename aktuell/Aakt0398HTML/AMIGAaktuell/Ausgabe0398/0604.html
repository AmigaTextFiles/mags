DESCRIPTION

CheckHTML checks whether an HTML document is conforming with the HTML (4.0) DTD. This will be done by using the sgmls parser from James Clark, the HTML DTD and a small Shell script. CheckHTML can verify HTML 4.0, HTML 3.2 and HTML 2.0 documents. Newer HTML versions can be easily supported by placing the new DTD into the 'sgml:' directory and adding their public identifier to the 'CATALOG' file. The upcoming XML standard could be easily supported the same way!
