6. Fehler im zweiten Wildfire-Update

»Wildfire Update 2 Fehlerkorrektur:

Wichtiger Hinweis für alle Anwender, die das 2. Update installieren:

Wenn Sie einen PPC-Processor benutzen und das Update installieren, funktioneren der Preview und das Storyboard nicht mehr. Das File `wf_image2.elf` aus der `system` directory hat einen Fehler. Es gibt zwei Möglichkeiten, den Fehler zu beheben:

1. delete wf:system/ppc/wf_image2.elf (löschen Sie das File einfach)

oder

2. copy Wildfire5:English/ppc/system/wf_image2.elf to wf:system/ppc/wf_image2.elf (oder besser: ersetzen Sie das defekte File durch das Original-File von der Wildfire-CD!)«
