DESCRIPTION
    Replacement AmigaGuide (2.0) and Multiview (3.0) viewer. Reads and displays XPK compressed files. Uses special format for Amigaguide files that lets them load *FASTER* than uncompressed files normaly load. Uses very little memory -- less than normal Amigaguide and *much* less than other XPK based viewers.

    Alpha release of work in progress.
FEATURES
  • Keyboard shortcuts in 3.1 now work again.
  • Custom (IFF based) high-performance Amigaguide file format:
  • Conversion/Compression utility provided.
  • XPK compression.
  • Index chunk allows for FAST loading.
SPECIAL REQUIREMENTS
    Requires 2.04 and above of Workbench. 3.1 recomended.
