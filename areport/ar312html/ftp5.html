DESCRIPTION
    The DFA datatype can be used to view DFA created database files using tools like Multiview. Additionally, it is possible to print customized address lists using this datatype and Multiview (for example)

    The display format can be customized, i.e. you can tell the datatype which fields of the address database you want to get displayed.

    This datatype can read all DFA address database formats from version 1.0 to version 2.25 of DFA!

    DFA is the definite address database tool. It supplies a very nice GUI, a powerful Arexx port, a flexible client server mechanism and much more. It is simply the ideal mean to manage several or many addresses for the beginner as well as for the professional Amiga user.

    This is version 39.1. Freeware, C-source included.
SPECIAL REQUIREMENTS
    Requires Amiga-OS 3.0 (V39). Runs on all Amiga computers (500+, 600, 1200, 3000, 4000) with this OS (or a newer one) installed.
