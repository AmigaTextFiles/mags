VDisk v2.7

TITLE

VDisk v2.7 (Update from v2.6)

AUTHOR

Etienne Vogt <Etienne.Vogt@obspm.fr>

DESCRIPTION

vdisk.device is a driver for a recoverable RAM disk. Main features are:
- Up to 16 units with virtually unlimited size (512 MB)
- Dynamic memory allocation and (optionally) de-allocation
- Full support of all AmigaDOS filesystems
- Will survive the heaviest system crashes as long as its own data is not corrupted
- In case of a recovery failure, the cause can be reported by a support command

SPECIAL REQUIREMENTS

VDisk requires AmigaDOS 2.04 or higher (v37).

AVAILABILITY
