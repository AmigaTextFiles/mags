Picture Manager Professional V4 (PMPro) is an excellent commercial image catalogization program with a large number of features for creation and handling of thumbnail tables.

It is a powerful database and allows loading, processing and displaying images via SuperView-Library and includes an automatic conversion function for catalogs of images, which allows to convert between all file formats SuperView-Library does support ... AND MORE ...

It even can directly scan LHA and LZX archives for images.
