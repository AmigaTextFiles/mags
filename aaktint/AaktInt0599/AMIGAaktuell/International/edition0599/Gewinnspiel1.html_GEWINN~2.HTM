1. Results of the last competition

This time 61 readers attended at the competition and 10 at the FD-charts. Especially the number of attendents of the FD-charts can rise, does't it?

The solutions and results:


Standard question:  Nordic Global announced a Java package for free. What is its name? 
  

Answer: Correct was 'Daytona'.


Master question:  Name another Amiga program, which is planned or at least considered to have integrated Java support. 
  
Answer: Due to the fact, that any WWW-browser for the Amiga can made capable of Java using Daytona, any browser that already exists or is under devolpment has been accepted as well as the JavaVirtualMachine 'Merapi', even if it is uncertain, if this program will be released. 
  
Question of opinion:  Do you think it is realistic to sell a computer for 500 US-Dollars that offers the power of a PC for 3,000 US-Dollars as AMIGA wants to?
  

The results:


Answer Number of Votes 
1 (Yes, absolutely.) 6 
2 (Yes, maybe.) 20 
3 (No, maybe not.) 34 
4 (No, absolutely not.) 1 
  

Conclusion:

In this edition the Amiga user users have different opinions. Most users are sceptical, if such a machine can be put on the market, even if noone removes the doubt, that it might be done. But a strong minority thiks, that it can be done.
