2. NetBSD (gateway! Vol.3)

With the appearance of NetBSD revision 1.3.2 a milestone in the history of the free UNIX like operating system was set.

This version of Gateway! Volume 3 contains NetBSD 1.3.2 completely:

  • Installation files for all of the 16 supported platforms

  • All the sources in packed form

This CD is a must for any UNIX expert and for anybody, who has more than one platform at home, but is also for the less experienced user.

More informations regarding this CD can be found in the Internet under: http://core.de/
