AMIGA news

  • Ramjam Consultants published a free Java compiler called Jikes, which can be downloaded from http://www.ramjam.u-net.com/software/jikes/index.html. The company plans to develop more Java software for the Amiga.

  • The programs FTS and AGM by Felix Schwarz are freeware now and can be downloaded from http://innovative.in-tec.de. »April 9, 1999 FTS and AGM are freeware, now. Because there weren't any requests for AGM it became freeware and can be downloaded. FTS, a system for contolling another Amiga via direct connection with modem or nullmodem cable was planned as shareware, but can be downloaded as freeware, because the development was much too slow. It is most likely, that there are no future revisions of AGM and FTS.« (Source: Innovative)
