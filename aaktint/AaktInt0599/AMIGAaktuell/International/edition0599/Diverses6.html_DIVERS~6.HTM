6. Photogenics is back!

Presumed ones live longer, we Amiga users know that. This is also true for the well-known picture editing software Photogenics, we didn't hear of for a long time. But Photogenics wants a comeback:

»Photogenics 4

After 4 years of hard work, Paul Nolan finally published the new revision of the well-known and loved picture editing software Photogenics.

Photogenics is a graphics package, which allows you to create phantastic pictures from nowhere or the edit existing pictures. It can be used for various functions from simple file converting to advanced photo manipulation and picture touch ups. It offers extremely realistic painting tools like e.g. chalk and water colour to create phantastic master pieces in minutes.

Numerous new functions allow never expected experiments with the graphic design.

The list of the new features:

  • Changing of the colours, transparency of the picture editing mode and the position of the drawn in real time.

  • Realistic tools like airbrush, chalk, pen water colours and also smear functions.

  • Various symmetry and text tools.

  • Best work results also when using a standard mouse.

  • Working without time stealing masking with any of the painting tools. You can choose between 3 dozens of work modes.

  • Easy erasing of errors with the right mouse button without altering the good parts.

  • Unlimited number of layers allow an efficient picture composition.

  • Phantastic pyrotechnic paint techniques like lens flares, fire and explosions. Combined with the smear tool you can produce Supernovae, waves of fire and even sun sets and waterfalls.

  • Innovative user interface with freely positionable elements.

  • Full multithreading: No waiting, many effects in real time.

  • Advanced, but easy useage of the alpha channel. Use any picture as a mask or just draw it on a picture. Changes on the mask are displayed in real time in the main picture.

  • 24-bit-alpha channels. Only Photogenics offers the possibility to us colour alpha channels, which allow a masking of red, green and blue. Drawing with gray creates common masks. Create unusual effects with the useage of 24-bit-alpha channels for masking picture editing effects.

  • Symmetry tool - allows incredible effects in combination with the paint tools in seconds.

  • Text tool - Just click anywhere into the picture and write into a paint layer. The text is fully editable until it has been fixed on the lower picture.

  • Multiple display with various magnification factors.

  • Impressive colour manipulation with 5 colour selectors, RGB and HSV slides. You can use colours from pictures, with the changes becoming visible in real time.

  • Extended file format support. Current formats to be loaded and saved are IFF-ILBM, IFF-DEEP, JPEG, Targa, TIFF, BMP and PNG, but more formats will follow. Photogenics also supports loading using 24-bit datatypes.

  • 24-Bit print support using Studioprint.

  • Open plug-in architecture for easy extendability of third parties. Photogenics is basicly a collection of plug-ins, which form a program without transition. Third party developers can access the same API as Photogenics, what allows developers to write any extension of the program core.

Minimum requirements: 68020, 8MB RAM, Amiga OS 3.0, CD-ROM drive.

Recommendes system requirements: 68060, 16 or 24-bit display, 16MB RAM, Amiga OS 3.0, CD-ROM drive.
