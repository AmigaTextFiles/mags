4. Informations about the GlowIcon collection (by co-editor Oliver Tacke)

»On the GlowIcon collection CD-ROM you find about 3500 icons, which are adaptions of the OS3.5 GlowIcons, which are saved in the NewIcon format, so Non-OS3.5-users can use them.

You find icons for many different programs from nearly any application type. Icons for games and many drawers with their symbols. Icons for drives and def_icons for various data formats. It has to be mentioned, that all the icons have the size of the OS3.5 icons, so the real optic "clean-up" of a drawer is no problem.

In addition there are icons for docks as you can create them e.g. with DirOpus and alternative toolbars for various programs like e.g. DirOpus, STFax, Voyager and YAM. Their are all adaptions of the GlowIcons.

As a bonus you find various backdrops in different resolutions, so you can re-design your Workbench, if you like to.

A small archive with examples can be found on http://private.freepage.de/amigafan/glowicons_demo.lha downloaden.
