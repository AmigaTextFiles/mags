5. Preview: Ali Baba

The newcomers from Medici Software released a demo version of their frist game named Ali Baba. Ali Baba is a classic jump 'n' run game and requires AGA and some extra memory. Unfortunately the graphics looks like in an A500 game. It is quite drab and is stuttering when scrolling (on an 68040 Amiga!). So one can only hope for improvement. Very smooth is the animation of the main character and of the enemies. It seems, that many unique pictures were drawn for this. Unfortunately this positive aspect cannot convince due to the drab design. In addition the controls have been quite uninteresting. Basic jump 'n' run actions are not supported, yet. So you can only hope, that Medici Software changes some things before releasing the final version of Ali Baba. The demo version can be downloaded on http://www.medici.org.uk/.
