Amiga PowerPC

1. In a Nutshell (by Oliver Tacke)

The playstation emulator FPSE has been extended and now has the version number V0.08 rev 5.

For MorphOS now the ttf.library is available, which permits display of TrueType fonts.
[http://www.morphos.de/files/ttflibmos_20001030.lha]

The MPEG player Frogger was released in version 1.62 beta 3
[http://frogger.rules.pl]

The RC5 client has been published in version V2.8011 build 464b. On a PPC603e the program now is faster up to 39%, on a PPC604e up to 70%.
[http://distributed.amiga.org/]

For the S.u.S.E. Linux PPC 7.0 distribution an installation instruction for the Amiga is available.
[http://home.germany.net/101-109037]

YAM 2.2 has been released for MorphOS.
[http://www.morphos.de/fileslinks.php3]
