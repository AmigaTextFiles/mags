Software
-------------------------------------------------------------------

Version 3.9 of the CD32-install kits appeared. Now it supports Turrican 1 and 2 and others.
[http://www.airsoftsoftwair.de]

The program "Image Editor", A picture processing program using a NetPBM base appeared in version 1.7.
[http://www.amclust-net.de/software/archive/ImageEditor.lha]

The Virge graphic-device driver for the Mediator busboard runs better in a new version 3 release 3.
[http://www.meanmachine.ch/~vgr/mediator/CGXV3_ElboxR3.lzh]

The stormamiga.lib in V45.00 is now available for SAS/.
[http://www.CyberdyneSystems.de/stormamiga_lib/index_d.html]

Amster, the client for the Napster-service is now available in V0.8.
[http://www.amiga.dk/amster/]

Again, the free version of Directory Opus has been improved and appeared as V4.15d beta.
[http://viper.pl/~opus/DOpus4beta.lzx]

Stephan Rupprecht released version 44.23 of his ILBM datatype. HAM- and EHB graphics will now be shown correctly on PicassoIV or CGX-screens.
[http://stephan-rupprecht.de]

TurboPrint got an update to Version 7.16. It now contains better and new drivers for Canon-, Epson- and HP-printers.
[http://www.irseesoft.de/tp716.lha]

The virtual observatory "Digital Almanac" is available as an update to version 1.3.
[http://www.soft-ware.de/dalmanac]

The InstallerNG appeared in a new version.
[http://installerng.light-speed.de]

An import filter for amiga-writer-files appeared for Pagestream.
[http://www.muenster.de/~tomjoad/]

EZP@gerNG is available in version 1.2.
[http://www.sfxsoft.de/de/]

The widely used email-client for the Amiga, YAM, appeared in version2.2.
[http://www.yam.ch]

Perfect Paint is now available in version 2.3. It has now more features and less bugs.
[http://gothic.fr.free.fr/amiga/]

The program "NetInfo", containing different network and internet tools, is now version 2.8.
[http://www.vapor.com/netinfo/]

A new version of the cnet.device has been released : V1.5
[http://www.jyu.fi/~sintonen/sw.html]

The MPEG-decoder AMP is now available in a beta V2 pre 7.
[http://www.amidog.com/amp/]

The MPEG-player Frogger appeared in version 1.62 beta 4.
[http://frogger.rules.pl]

Version 3.0 of the text-displayer "Next" appeared. The program supports the formats *ix-"man pages", WordPerfect, IFF, FinalCopy, FinalWriter, WordWorth and others.
[http://www.andreasneumann.de/software/next/]

The Palm-desktop Spitfire² is now available in version 1.3.
[http://www.amigaspitfire.com/]

The MUI-class NListtree appeared in version 18.6 beta 3.
[http://www.aphaso.de/www/html/german/aphaso.html]

The RC5-client is available in version 2.8011 build 464b.
[http://distributed.amiga.org/]

GoldED V7 is to be released in December.
[http://golded.tripod.com]

The Silversurfer.device is available online in version 2.104.
[http://www.jschoenfeld.de]

The newest beta-release of the program DataM-II appeared.
[http://www.amclust-net.de/software/archive/DataM.lha]

ProStation Audio is now available in version 4.01.
[http://www.audiolabs.it/]

The address organizer ADM is now available in V3.65.
[http://www.digix.de]

The latest version of VirusExecutor, V2.08 can be downloaded from the internet.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/VirusExecutor.lha]

From 27.11.2000 fxScan V3.0 is available and now contains a text-recognition.
[http://www.innovative-web.de]

The IRC-client AmIRC is now available in version 3.5.5 beta.
[http://www.vapor.com/amirc/]

AmigaAIM in version 0.9398 is now ready for download.
[http://www.missingpiece.com/AmigAIM/]

The latest version of the Filer of Rudolph Riedel carries the version number 2.5 beta 9.
[http://home.t-online.de/home/rudolph-riedel]

NetBSD is available in version 1.4.3.
[http://www.netbsd.org/Ports/amiga/]
