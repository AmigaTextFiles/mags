NewMasterMind

At last something new. Not a Tetris- nor a PacMan- nor an Arkanoid clone but a Mastermind clone. The principle of the game is well known, isn´t it? If not, I´ll try to explain: You have to try to crack a color code four places long. This code is made of up to eight colors. You´re allowed to try ten times to crack the code. After each trial you can see how many colors are set right and how many of the used colors are in the code.

In the mode for children you get a report on the code, that means you can see which colors are on the right place etc. Originally this game was for two people but you can misuse your Amiga as a playmate.

The game runs on WB. The graphics are, for a WB game, quite nice, but there might be problems on Amigas with fewer colors. There is no sound.

The only advantage of this game, in comparison to the original board game, are the computer opponent and the possibility of using more colors. Nevertheless a fun game.

Arkanos

Should MegaBall fear this competitor? Surely not, because although this Arkanoid/Breakout clone has nice graphics, sound and nearly the same extras it is not as fantastic as the good old MegaBall.

Besides the standards of clone like good level design and good adherence to the laws of physics, Arkanos has extra blocks which fall down after some blocks are destroyed. These blocks give special effects: e.g. shooting, the ball destroying all blocks without rebounding, your paddle growing and more.

But there are also some extras that make you lose a life or begin the level again. These extras are not funny.

By the way, this game is freeware, while a leveleditor is available for 20 DM (approx. 10 Euro)


A nice game but a bit boring, although it is, to be fair, a good game.

Mattonite

And yet another Arkanoid clone

This time neither the conversion nor the game fun is done well.

The game flickers everywhere (I am not a pro but I think the programmer forgot the doublebuffering...), otherwise the graphics are o.k., the sound, too.

The leveldesign of the first levels is dull, there is just one big block with no space on the sides. Very clever.

Like Arkanos there are extras like bigger paddles, shooting and a ball which cuts through all blocks. This extra makes the game so easy that you can play the level without any problems.

Due to the flickering sometimes the ball is invisible and the game sometimes defies the laws of physics. It should be impossible to not guess the direction of the ball because rebounds always conform to a basic law: angle of incidence = angle of reflection.

Every other Arkanoid game is better than this one, I recommend downloading MegaBall, the game that will never get a real competitor.
