Various Amiga news

7. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

68k/Miscellaneous

For "TurboPrint" new printer drivers have been released in beta state for Canon S 400, 450, 4500, 800, Epson StylusColor 680, 880, 980 and Epson StylusPhoto 790, 890, 1290
[http://www.irseesoft.de]

Jens Schönfeld stated in a commentary at [http://www.amiga-news.de], he works on a new driver for X-Surf, which can use DSL with the normal version of Miami resp. with Genesis or AmiTCP.
[http://www.amiga-news.de/de/news/comments/8253.html]

For the "Prometheus" PCI bus board of Matay new driver versions for the Voodoo3 2000 and Voodoo3 3000 appeared.
[http://www.matay.pl/ANGindex.html]

The driver software for the "Mediator" PCI bus board became improved and now has the version number 2.3.
[http://www.elbox.com/]

PowerPC

The browser "Voyager³" has been released in a new beta version (3.3.105) which runs without a keyfile.
[http://v3.vapor.com/]

---------------------------------------------------------------------------
Software (non commercial)
---------------------------------------------------------------------------

68k/Miscellaneous

The version V15.9 of the virus scanner "Safe" has come out.
[http://www.vht-dk.dk/]

The Mac emulator "Basilisk II" was published in version 0.9-1.
[http://www.uni-mainz.de/~bauec002/B2Main.html]

For "PageStream" appeared the AmigaWriter import filter in version 1.1.
[http://www.muenster.de/~tomjoad/awfilter.html]

V0.76 of the sample editor "Audiomaster 2000" has been released and since V0.72 it contains a completely rewritten AHI part.
[http://www.nucleus.com/~fenn/]

The HD-installer "WHD-Load" now has the version number 14.3 and it holds several bugfixes.
[http://www.whdload.org/]

The Palm desktop "Spitfire²" appeared in version 1.6 and contains several new features.
[http://www.amigaspitfire.com/]

The progression of the once commercial file manager Directory Opus has led to a new version (4.15m).
[http://dopus.free.fr/files/DOpus415mbeta.lzx]

There appeared an update to V2.02 for the scanner calibration software "ICS".
[http://www.ics.coloraid.de]

The e-mailer "YAM" was developed on, after the sourcecode had been released for free by author Marcel Beck. The new version has the number 2.3.
[http://www.yam.ch]

The Workbench replacement "Scalos" has appeared in the new beta version 1.3.
[http://www.scalos.com]

The stock-market software "Bonds" has been published in a new version.
[http://www.programmed-excellence.de]

The "MySQL" portation for the Amiga now has the version number V3.23.39.
[http://sixk.maniasys.com/index_en.html]

From the assembler "ASM-One" the version 1.47 has been released.
[http://www.euronet.nl/users/jdm/homepage.html]

The "xvs.library" appeared in version 33.26.
[http://home4.inet.tele.dk/vht-dk/amiga/xvs/xvs.htm]

The virus killer "VirusExecutor" again has been updated, this time to V2.19.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/VirusExecutor.lha]

The webserver "Apache" has been placed into the Internet in version 1.3.20 with "PHP-Module" V4.0.6.
[http://sixk.maniasys.com/index_en.html]

The e-mail program "SimpleMail" has been released in a new alpha version (0.7).
[http://simplemail.sf.net/]

V1.190a of the "SmartFileSystem" has been published.
[http://home.wtal.de/js/]

The "Asyncio.library" came out in version V40.5 for WarpOS.
[http://ftp.uni-paderborn.de/pub/aminet/util/libs/asyncioppc.lha]

A new version of the movie player "AMP2" has appeared.
[ftp://ftp.amidog.com/pub/amp/AMP2-BETA-010618.lha]

For the iMac emulator "iFusion" an update to V1.3.0 appeared, which now offers audio support, too.
[http://www.blittersoft.com/]

MorphOS

The compiler gcc received an update to version 2.95.3.
[http://www.morphos.de/fileslinks.php3]
