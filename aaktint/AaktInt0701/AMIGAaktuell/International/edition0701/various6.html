6. News from Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»CD32 Games Install Kit v4.4NG released

Here's a new update for owners of the "The Best of Airsoft Softwair Gold Edition" CD-ROM. Version 4.4 NG of the Install Kit supports much more games now (182 in total). I also made some minor optimizations and bug fixes. To download this update please use the program "Updater" which is contained on the CD-ROM, or click here. This version of the CD32 Games Install Kit is only available for owners of the CD-ROM. The last public, shareware version is 3.9.«

»CD32 Games Install Kit v4.41NG released

A heavy bug was fixed in this release. Please download and install it if you're using version 4.4!«

»Install Kit for Windows

As some emulation sites already posted this news, here's the "official" :-) annoucement for the Windows version of my CD32 Games Install Kit: It will work in connection with WinUAE and it will be available quite soon. The compatibility is very high because every game uses custom patches (just like in the Amiga version). Additionally you won't have to install the game, you can run it right from the CD although in some cases some files have to be copied (which need to be patched). CD audio support is not supported yet but will be added for sure later. The first version of the CD32 Games Install Kit 4.5 (Win32 1.0) can be expected in one week. But currently it will be only available for owners of the "The Best Of Airsoft Softwair Gold Edition" CD-ROM, so order it now :-). Some Screenshoots are available on Airsoft-Software-Homepage.«
