Henk Jonas gave his program MetaView on CD-ROM for the competition to us:

»MetaView is a displayer and converter program for vector graphics like e.g. cliparts or CAD-drawings.

In addition to its support of file formats (WMF, CGM, Dr2D, EPS, HPGL, FIG etc), which is unique on the Amiga, it offers an easy catalogizer function for clipart collections.

Due to its ARexx port MetaView can be used as import or export filter for other graphic programs or text processing software. For registered users there exists also a datatype, which allows the import of any file format supported by MetaView to nearly any application.

A nicely designed GUI and the possibility to send pictures in any format from the overview to other application, are the best features of MetaView. A localized version is in a test phase and will soon be released.

The actual version can be downloaded from http://user.cs.tu-berlin.de/~jonash or can be found in the Aminet gfx/conv/MetaView.lha. An eMail list can be found on http://www.egroups.de on the subject "MetaView".

The programmer can be contacted on:

eMail: jonash@cs.tu-berlin.de«
