Various Amiga news

4. News from Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»On the Airsoft Softwair homepage recently the first update of the Da Cool Installer for two years has been released. Actually, thiis is an all new program, because I rewrote the old Da Cool Installer (for stability reasons). Now, version 1.7 should be bugfree and I cite from the Voyager Readme "stable as a rock".

Da Cool Installer is a general installer for games/demos etc. It supports ECS emulation, trackdisk.device emulation and various boot modes. Theretically it is possible to install any AmigaDOS game easily on the hard disk.

Da Cool Installer v1.7 can be downloaded from the Airsoft Softwair homepage as you read this.«

»On the Airsoft Softwair homepage 6 new boot configurations for Rainboot have been released. So there is e.g. another Britney Spears bootup from Pixel Art, a Shadow of the Beast bootup or the cool Workbench 85 and AmigaOS 3.5 bootups from Teknique. Everything is with screenshots and can be downloaded.

If you use Rainboot regulary, you should consider to register (just 10 DEM (approx. 5.10 Euros)).«
