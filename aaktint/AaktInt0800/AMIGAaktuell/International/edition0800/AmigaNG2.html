News about the AmigaNG

2. New informations from AMIGA president Bill McEwen

Recently, there American Amiga show AmiWest 2000 was held. On http://www.ann.lu/detail.cgi?category=news&file=964934374.msg there are various informations spread by Bill McEwen, also some previously unknown:

  • The AmigaOne will be based on Amiga's own customized motherboard design, using off-the-shelf parts. AMIGA are putting up a hardware division, lead by Dean Brown, to create custom hardware designs for various devices that will be then offered to 3rd parties to manufacture.

  • The SDK is selling better than forecasted. AMIGA have been in contact with numerous major consumer electronics and software companies, including (straight quote) "every major gaming company out there". Many of these already possess the SDK.

  • A version of the SDK for Windows will be made available this month.

  • Red Hat will start selling the SDK next week.

  • Sun have started using Amiga software to demo Java stuff.

  • There are going to be public beta releases of the OE prior to release.

  • Paul Nolan (Photogenics) is now doing contract work for Amiga.

  • AMIGA will start offering low-cost promotional merchandise to user groups.

  • At the show, Amiga were showing off an ARM-based touchscreen unit, running the new OE. The design is by none other than Mick Tinker (BoXeR), and it's not being sold commercially yet.

  • Backwards compatibility to the Classic Amiga from the new OE will be dealt with by something that "is not an emulator".
