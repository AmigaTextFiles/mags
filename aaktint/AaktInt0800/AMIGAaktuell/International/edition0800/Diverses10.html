Various Amiga-News

10. News in Brief (by Oliver Tacke)

Software

The second update for OS3.5 (BoingBag 2) is to replace the demo version of the TCP/IP stack Miami with the full version. This is what the president of Haage&Partner said in public.

Voyager V3 will get a new layout engine, which will be a multiple times faster in HTML display than the current.
[http://v3.vapor.com/index.php?finger=owagner]

Now, MakeCD is available with the versions 3.2c and 3.2d. The last one is a beta version and contains support for DAO with ATAPI drives as well as Sony drivers
[http://makecd.core.de]

An update of Easys! to version 4.8 has been released. Also the problems of the installer script of the CD V4.7.2 have been fixed.

For AmigaWriter a patch to V2.2 has been released. Now, it is possible to load files from FinalWriter and Wordworth.
[http://www.haage-partner.com/d.htm]

The latest version of the web server software Apache is 1.3.12 with mod_php, PHP Version 4.0.1 and Patch Level 2.
[http://www.stud.uni-karlsruhe.de/~uck4/AmigaApache/]

The DTP program PageStream is now available in version 4.7.
[http://www.haage-partner.com/]

For the CD writer software BurnIt, there is now a master driver version V2.72, which also supports the CD writers Teac54i and Ricoh8040.
[http://www.titan-computer.de]

There are news about the browser "Dry Ice", which was announced some time ago. You find e.g. an alpha version for download.
[http://web.ukonline.co.uk/andrewmarkwell/dryice/]

Photogenics Version 5 Beta Release 95 has been released.
[http://www.paulnolan.com/Amiga/download.html]

A demo version of the program ArtEffect version 4 can be downloaded from Haage&Partner's FTP server.
[ftp://ftp.haage-partner.com/ArtEffect/ArtEffect4-Demo.lha]

Service Pack 18 for GoldEd was made available in the Internet.
[http://members.tripod.com/golded/studio08.htm]

In addition a special, freely available version of GoldEd 5 can be downloaded.
[http://golded.tripod.com/studio20.htm]

Andreas Kleinert also released updates for his programs: akPNG.datatype V44.80, akMPEG2 V3.10 and akMPEG V4.46
[http://home.t-online.de/home/Andreas_Kleinert/]

IrseeSoft released a beta version of Turboprint 7.12, which offers improved HP DeskJet 9xx drivers and some improved Canon and Epson drivers. Turboprint 7.10 or 7.11 has to be installed previously.
[http://www.irseesoft.de/tp712beta.lha]

Jens Schoenfeld announced, that x-surf.device version 1.2 is available, now. Now, multiple can be operated in an Amiga. The duplex mode is being activated when the driver is being opened. In addition to that a bug in the Promisc mode was removed.
[http://www.jschoenfeld.com/xsurf.lha]

Hardware

The PCI board "Mediator" for the Amiga 1200 and for Amigas with ZorroIV board can be pre-ordered at Power-Computing. The board offers 4x PCI 2.1 compatible slots and will cost 129.95 GBP (approx. 224.05 Euros).
[http://www.powerc.com/cgi-bin/psearch.cgi?page=/mediatorx.html&rn=244&aid=&uid=BEqPbgSpYdMC6&sid=&cid=4&act=ppage]

Power-Computing announced various add-ons for the "Mediator"-PCI board, which include e.g. graphic boards based onf S3 Virge 4MB or 3dfx Voodoo 3 16MB, an MPEG-2 hardware decoder and a USB controller. [http://www.powerc.com/cgi-bin/psearch.cgi?page=/mediatorx.html&rn=244&aid=&uid=BEqPbgSpYdMC6&sid=&cid=4&act=ppage]

Elbox Computer and Vision Factory Development announced on July 24, 2000 in the Amiga news, that they are working on a CyberGraphX solution for PCI graphic boards, which could be operated with the PCI-board "Mediator".
[http://www.vgr.com/mediator/pressrelease1.txt]

Miscellaneous

In autumn, there could be another German Amiga show hosted by the Falke Verlag. The decision would be made within the next two weeks after consulting AMIGA, said CEO Ali Goukassian.

Rumours

It has been said, that the release of a handheld (portable game console like the Gameboy), which would be based on an ARM processor and the new AmigaOS, was planned for late November.

Various big companies had interest in building new Amigas, especially the handheld type. The companies were Dell, IBM and - quite funny - Gateway.
