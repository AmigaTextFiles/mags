2. Correction of a mistake in the test of ATC in AMIGA aktuell 06/00 (by Ch. Aichinger)

»There I wrote:

[Begin quote]...The page "ADT" contains several options for downloading from Aminet. These options can also be found in the ATC preferences, but are here again for quick changes... [End quote]

This is wrong! In the preferences, you define default values which are used when creating a new ADT server. They have nothing to do with the preferences you set in the server manager on the page "ADT". There you can set individual options for each "ADT-Server" you hav defined.
