An this is a new chance for all of you to win a prize:

Frank Weber gave his program CDCat for the competition to us.

»CDCat Release 3 (MUI) is the PPC/68k CD-ROM catalogizer for the Amiga. In addition you can use it also for archiving all hard drives, floppy disks and directories (e.g. with many graphic files).

It is also quite valuable for Aminet-CD's because age, size and archive descriptions are considered. CDCat offers a super fast search mode (PPC; 68k 85% inline assembler!), in addition you archived CD's can be viewed by the browser without the CD put into the drive.

CDCat is also able to unarchive, view pictures, etc. But you can still configure any program.

More features:

  • XPK support
  • notepad for management of found files
  • DiskIDs for quick identification of the CDs, etc.

A demo version of the program is available in Aminet. The current revision is 3.0. More informations about CDCat can be found on http://www.amigaworld.com/support/cdcat/.

New in revision 3 are e.g. PowerPC support, the catalogue manager for reorganizing the catalogues, the filter function, etc.
