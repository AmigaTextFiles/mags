News about the AmigaNG

2. Amiga SDK without Linux distribution / Download from internet

Contrary to earlier news the Amiga Software Developer's KIT (SDK), which is the base for writing software for the planned AmigaNG, will not have a Linux distribution, though it is required for operating this product. The Linux version recommended by AMIGA is Redhat 6.1 (revision 6.2 does not work according to AMIGA) and can be freely downloaded from the Internet (a serverlist can be found on http://www.redhat.com/download/mirror.html). By the way, developers were able to run the SDK on other Linux distributions like e.g. Suse and Corel Linux. This is why AMIGA considers to offer support for those, too.

It is also not impossible, that there might be a cheaper student version of the SDK. The common price is about 99 US-Dollars (approx. 107.61 Euros). The product, which requires a x86-PC, can be purchased in your local Amiga store.
