Various Amiga news

5. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

For PageStream an update to version 4.0.9 appeared. Improvements among others involved the printer output and the PDF compression.
[http://www.haage-partner.com/]

The isdnsurfer.device for the ISDN board with the same name is in version 1.943 ready for download.
[http://www.jschoenfeld.com]

A new beta version (V3.3.95) of the browser Voyager has appeared.
[http://v3.vapor.com/]

In addition to that there is a new beta version of the Image Decoder for V3.3, which is NOT compatible with V3.2.
[http://v3.vapor.com/]

The printer driver system Turboprint is available in version 7.17 beta.
[http://www.irseesoft.de/tp717beta.lha]

For the program Digital Almanac III now the update to version 3.1 can be downloaded.
[http://www.soft-ware.de/dalmanac]

Elbox has finished the S3-Virge drivers for the Mediator PCI busboard. They are said to be 100% Picasso96 compatible.
[http://www.elbox.com]

For StormC 4 an update has appeared, which repairs several bugs and improves the support for Mixed Binaries.
[http://www.haage-partner.com/]

The iMac emulator iFusion after a long wait is ready and will be delivered.
[http://www.microcode-solutions.com/home.htm]

A new version (V1.3) of the MASMPegDevice has been released
[http://www.platon42.de/platon/files/mus/MASMPegDevice.lha]

---------------------------------------------------------------------------
Software (non-commercial)
---------------------------------------------------------------------------

For Apache, the probably worldwide mostly used webserver, a new patchlevel appeared (1.3.19 - PHP 4.0.4. pl 1).
[http://www.stud.uni-karlsruhe.de/~uck4/AmigaApache/]

GhostScript, a well-known viewer for pdf and PostScript files, is available in version 6.50.
[http://www.muenster.de/~tomjoad/]

The program VirusExecutor has been published in version 2.16.
[http://home.inet.tele.dk/vht-dk/amiga/ve/VirusExecutor.lha]

Beside of that VEPatch.Brain V1.31 for VirusExecutor has appeared.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/VEPatchBrain.lha]

The movie-player Softcinema has been released in version 0.12 beta.
[http://www.amiga.pl/softcinema/bin/SoftCinema_012.lha]

The AOL Instant Messenger clone AmigAIM was published in beta version 0.9414
[http://www.missingpiece.com/AmigAIM/]

The virus finder Safe has appeared in version 15.3.
[http://home4.inet.tele.dk/vht-dk/amiga/safe/Safe.lha]

The SMS transmitter TaskiSMS was published in Version 2.55 with several bug-fixes.
[http://taskisms.taski.com.pl]

Version 4 of the 3D hardware driver system Warp3D has appeared.
[http://ftp.haage-partner.com/3DWorld/Warp3D-4.0.lha]

According to Thomas Frieden from Hyperion the Voodoo 3 driver for Warp3D runs stable, but is not yet ready.
[http://www.vgr.com/cybergfx/]

For Scalos a new DefIcons PlugIn appeared. Minimum requirement is AmigaOS 3.5.
[http://scalos.striatum.org/products/scalos/modules/Scalos-Plugin_DefIcons.lha]

As well for Scalos there is a new version of the Persistent Windows PlugIn.
[http://scalos.striatum.org/products/scalos/modules/Scalos-Plugin_Persist.lha]

A port of the databank system MySQL (V3.23.36) is ready.
[http://sixk.maniasys.com]

The Disk-to-HD installer program JST has appeared in version 4.7b.
[http://jffabre.free.fr/amiga/bintools/LATEST/jst.lha]

The file manager ClassAction was published in version 4.1 and now supports among others LHA and ZIP archives, too.
[http://www.martin-elsner.de/]

To the driver for graphics tablets, FormAldiHyd, an update to V2.3 happened.
[http://www.platon42.de/platon/index2.html]

SnoopDOS appeared in version 3.7.
[http://ftp.wustl.edu/~aminet/aminet/util/moni/SnoopDOS.lha]

---------------------------------------------------------------------------
Hardware
---------------------------------------------------------------------------

The PCI busboard Predator-SE from Eyetech is available. It must be connected to the expansion slot of the Amiga 1200 and offers 5 PCI ports.
[http://www.eyetech.co.uk/NEWS/PREDA001.HTM]

Miscellaneous
---------------------------------------------------------------------------

The mailing list for AmigaAmp of Thomas Wenzels has moved. Now it can be used directly via amigaamp@amigaamp.de
[http://www.amigaamp.de]

Met@box now officially announced, what everybody had expected: The development of the PPC board "Joecard" has been stopped, as well as all other activities in the Amiga sector.
[http://joecard.com]

Crystal Software from now is called ClearWater Interactive.
[http://www.cw-i.com]
