4. News from Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»In the near future at Airsoft Softwair a big update of the NewInstaller, the NewInstaller 2, will appear. This version will be usable as standard installer (Sys:C/Installer) and have all the advantages as there are background pictures, sounds, internet integration etc. An exclusive preview version of this revolutionary tool will be ready for the users of The Best Of Airsoft Softwair Gold Edition CD-ROM in a few days. Also the final version will be available as full version for the owners of the CD-ROM. NewInstaller 2 will set new standards, because using it for the first time it will also be possible to provide complete software packets as exe-files (as it is common use on other systems for yonks). More details will be released soon. For those, who do not own The Best Of Airsoft Softwair Gold Edition CD-ROM and want to get some impressions of NewInstaller version 2 there will be some screenshots on the Airsoft Softwair homepage.«
