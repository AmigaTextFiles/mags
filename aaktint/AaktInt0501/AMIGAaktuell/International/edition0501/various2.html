»April 06, 2001:
Boing Bag #1 for OS 3.9


In time for the Gateway Computer Show Amiga 2001 AMIGA. Inc. and H&P presented the Boing Bag #1 for OS 3.9. The most interesting function certainly is AsyncWB, which allows the asynchronous copying and deleting of files on the WorkBench. Additionally there are: new GenesisPrefs, reworked HDToolBox, Amplifier, ViNCEd, WBRun, UnArc, RAWBInfo, BenchTrash and several more improvements.

Addendum: In the meantime there is a new version (5.254.174 Bytes) of the Boing Bag #1, which contains several small changes: new versions of Blanker, CrossDOS, Exchange and FKey, reworked German locale for IconEdit.

Read the complete text or download the update directly.

April 6, 2001:
H&P at the Gateway Computer Show 2001

Also in this year H&P attended the Gateway Computer Show Amiga 2001 in St. Louis, USA (March 30 to April 1). Mainly we presented the new StormC 4 Compiler System and held workshops about StormC and AmigaOS 3.9. A short report, created for amiga-news.de, highlights the impressions.

April 6, 2001:
PageStream 4.0.9 for Amiga


A few days ago Grasshopper LLC finalized version 4.0.9 of PageStream for Amiga and Macintosh, the Windows version was available sooner. The changes involve corrections to the printer output, PDF compression, faster functions for corrections and hyphenation and some more.

Customers that have registered for the online service have received an e-mail with the data for download. All other customers may still have themselves registered.

April 10, 2001:
Update Patches for StormC 4


For StormC 4 there is a first update available, which repairs several problems and improves and completes the support for mixed binaries and shared libraries

April 17, 2001:
STL for StormC 4


Raymond Zarling has ported the Standard Template Library (STL) for StormC 4.

This library is an extensive collection of algorithms and data structures, which can very much facilitate the creation of programs. It is also defined as part of the recent C++ standard. The Amiga sources can be found on the Aminet, the original sources are on stlport.org and the documentation you can find on sgi.com.
