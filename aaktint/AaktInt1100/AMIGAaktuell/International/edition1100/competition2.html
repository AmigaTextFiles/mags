Frank Weber donated his program CDCat for the competition.

»CDCat Release 3 (MUI) is the PPC/68k CD-ROM cataloguer for the Amiga. In addition you can also use it for archiving hard drives, floppy disks and directories (e.g. with many graphic files).

It is also quite valuable for Aminet-CDs because age, size and archive descriptions are considered. CDCat offers a super fast search mode (PPC; 68k 85% inline assembler!), in addition your archived CDs can be viewed by the "browser" by directories without the CD being in the drive.

CDCat is also able to unarchive, view pictures, etc, or you can configure any other program to do these jobs.

More features:

  • XPK support

  • Notepad for management of found files

  • DiskIDs for quick identification of CDs

etc.


A demo version of the program is available on Aminet. The current revision is 3.0. More information about CDCat can be found on the support-site 
http://www.amigaworld.com/support/cdcat/ 
 

New features in revision 3 include PowerPC support, the catalogue manager for reorganizing the catalogues, the filter function, etc.
