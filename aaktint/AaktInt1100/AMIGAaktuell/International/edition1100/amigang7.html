News about the AmigaNG

7. In a nutshell (by Oliver Tacke)

Taifun, the AmigaDE-successor of the excellent special-effect program Wildfire, is now available in an alpha stage.
[http://www.wk-artworks.de/taifun/taifun.tar.bz2]

The AmigaSDK is now also available for Windows. Requirements are a 200 MHz CPU, 64 MB RAM, 100 MB HDD-Space and an ethernet device.
[http://www.amiga.com/products/SDK.shtml]

Gary Peake from AMIGA said in a developer-mailinglist that it would be possible to give some developers a free cross-upgrade from the Linux version to the Windows version.

AMIGA's internet-website now contains an information-section about the Amiga ONE (http://www.amiga.com/products/one/) also there's a discussion forum.

The URL http://206.170.238.141 is the first online internet-website hosted on an AmigaSDK - AmigaDE computer. It uses the mini-HTML-server zAMiLo in version 0.96.
