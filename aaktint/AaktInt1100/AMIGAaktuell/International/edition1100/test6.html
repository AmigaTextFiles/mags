6. Software report: SMS client »TaskiSMS« (by Klaus Moddemann)

I received lots of emails after my software test report was published in the last issue of AMIGA aktuell. Most dealt with the program's preferences. 
Well, at "user" you enter your personal data such as name, email address and a signature which can be added to a message automatically. 

Some users had questions concerning the server connection, too. You can either send the messages via a mailserver at GMX, hotmail or other mail services. In this case you have to enter the mail server's name in the field named "SMTP server". For the GMX server it's mail.gmx.de. Alternatively, you can use your own provider if it offers a proxy server. In that case you have to enter the proxy server's name in the field named "proxy server" and in the field named "port" the corresponding number. Finally you have to tick the field "use". Finished. Your provider's service center should be able to tell you the exact method for accessing the proxy server.

Some problems may occur when sending messages into the e-plus network and unfortunately the author hasn't fixed this bug yet. I'll take care of this problem when I find some spare time. You'll find the author's answer in the next issue of AMIGA aktuell.
