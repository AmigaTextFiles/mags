2. Manual: Document preview in Amiga programs (by Thomas Nawroth)

Many of you probably know the preview function of Word Perfect (PC). It gives the user a preview of the chosen document before loading it.

With a little trick that's also possible with Wordworth or any other program.

Follow these steps:

1) Load the desired document

Special for Wordworth: In the menu "view" chose "tidy up" and "zoom", "adapt page".

2) make a screenshot

for example with PPaint 6.4: chose "load screen" function.

3) cut brush and save

Now you can cut the desired area as a brush. For Wordworth this is probably a DIN-A4 page in most cases. Before saving the brush, please ensure that you activated "yes (image)" in "prefs" - "icons".

Now you can save the brush temporarily. You can put it into the Ram Disk as it's not really needed, because PPaint creates a preview of the brush as an .info file.

4) assign icon

With an icon installer program you can now replace the original icon with the preview icon.

In this way you can create a document preview for any type of application...
