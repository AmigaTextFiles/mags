2. First BoingBag-Update for AmigaOS 3.9 released

Amiga Inc. and Haage&Partner simultaneous with the fair in St. Louis announced the availability of BoingBag 1 for AmigaOS 3.9. The update may be downloaded at http://www.amiga.com/3.9/download/BoingBag39-1.lha. Here are the the new features and bugfixes:

»BoingBag 1 for AmigaOS3.9

Contents of Update

Internet Software

GenesisPrefs now has a ReAction interface! The AWeb SE documentation has been supplemented with information about plugins.

Harddisk use

HDToolBox now avoids the listing of several drivers, which are not harddisk drivers. There are several bugfixes. In IoTools as well several bugs are repaired, and the compatibility is improved.

Multimedia

There are new and improved ARexx commands for AMPlifier.

Improved shell and new and extended shell commands.

ViNCEd now permits selection of a single word with a double click. For experts: It supports the logging by a CSI Sequenz. Additionally several bug-fixes.

WBRun makes it possible to start programs in Workbench mode out of the shell or from a script. SetEnv has a "SAVE" option. When selected, the environment variable is saved not only to "ENV:" but as well to "ENVARC:". MakeDir has an "ALL" option to create complete directory paths. A bug in LoadResource is fixed and MakeLink has been written anew.

Workbench and Workbench Accessories

AsyncWB adds asynchronous copying and deleting of files and directories to the Workbench, which has long been waited for. The windows "New Drawer" and "Rename", too, no longer block the Workbench.

UnArc now permits the selection of files which are to be dearchived. Find supports the saving of its own window position and size and now works with a directory with an underscore in its name. Furthermore search paths can be added simply by drag&drop or pull-down menus.

RAWBInfo was improved, too: A mouse-click on an icon tooltype with shift-key pressed enables or disables this tooltype. When you drag icon tooltypes from one information window to another, you can add them by holding down the shift key instead of replacing them.

BenchTrash now is able to work asynchronously and to replace the delete function of the Workbench. AnimatedIcon again works as it is supposed to.

ReAction Classes

The ReAction classes responsible for the display of ReAction surfaces are completely reworked, resulting in an optimized view, many bug-fixes and several new features.

Several Bug-Fixes

The FontCache will be automatically activated after the BoingBag installation. xadmaster during depacking no longer will replace spaces with underscores. Now rexxmathlib.library is contained. And much more ...
