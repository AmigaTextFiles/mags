Amiga PowerPC

1. In a Nutshell (by Oliver Tacke)

There are updates for the WarpOS datatypes by Oliver Roberts: WarpJPG, WarpPNG, WarpBMP. Furthermore there is a new version of the preferences.
[http://www.nanunanu.org/~oliver]

As well from Oliver Roberts comes a datatype for Adobe Photoshop files.
[http://www.nanunanu.org/~oliver]

The MPEG player Frogger has been released as version 1.64 and is said to be faster by 25% than the predecessor version.
[http://frogger.rules.pl]

The JFIF datatype by Andreas Kleinert, akJFIF, is in version 44.96 ready for download. As well as PowerUp, WarpOS is supported.
[http://www.ar-kleinert.de]

ProStation Audio has been released in the new version 4.05 and offers a native driver for the Repulse sound board. Furthermore the utilization of the PowerPC processor is improved.
[http://www.audiolabs.it]

For MorphOS a new version of lha is published.
[http://www.morphos.de/meanmachine/files/Fr3dY/lha-mos114i.lha]

Stephan Rupprecht completed version 45.2 of the ILBM datatype.
[http://home.wtal.de/stephanr/]

A port of SoX (V12.17.1) for MorphOS has appeared.
[http://www.morphos.de]

The MorphOS version of MUI was developed on and now has the version number 3.9 Release 2.
[http://www.morphos.de/meanmachine/files/MUI_Release-2.lha]

VoyagerPPC V3.3.92beta has been released for MorphOS. Users without a keyfile can run the program in demo mode.
[http://www.morphos.de]
