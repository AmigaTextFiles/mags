Various Amiga news

3. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

The first update (BoingBag) for OS3.9 has appeared and contains beside bugfixes several new functions. For instance now at last it is possible, during a copying or deleting action, to go on working.
[http://www.amiga.com/3.9/download/BoingBag39-1.lha]

The driver software MediatorUP is available in version 1.8.
[http://www.elbox.com]

For GoldED Studio 6 a new syntax parser has been released. Beside bugfixes in the older version the new one now supports among others PHP and JavaScript.
[http://www.aminet.net/pub/aminet/text/edit/html-php.lha]

The driver software for the FastATA Controller is available as V2.4.
[http://www.elbox.com]

ProStation Audio has been released in the new version 4.05 and offers among others a native driver for the Repulse sound board.
[http://www.audiolabs.it]

New drivers for TurboPrint have been published for Canon S400, 450, 4500 and 800, Epson Stylus Color 680, 880 and 980 as well as for Epson Photo 790, 890 and 1290.
[http://www.irseesoft.de]

A public beta version of MakeCD (V3.2 beta 6) has appeared and offers among others VideoCD support.
[http://makecd.core.de]

---------------------------------------------------------------------------
Software (non-commercial)
---------------------------------------------------------------------------

PlayGUI has appeared in version 3.1 and contains several bugfixes.
[http://narr.tsx.org]

The graphics tablet driver FromAldiHyd has been released in version 2.2.

New features are the support of macro-keys and the playing of sounds when a key is pressed.
[http://www.platon42.de/]

The boot-process beautifier Rainboot has been published in version 3.2 and is among others enriched by better support of graphics boards

A new version of the asyncio.library for WarpOS has been released.
[http://members.tripod.de/achimste/download/asyncioppc.lha]

The MPEG player Frogger has been published as version 1.64 and is said to be faster by 25% than the predecessor version.
[http://frogger.rules.pl]

The graphics program Perfect Paint has been released in version 2.5.
[http://gothic.fr.free.fr/amiga/download/PfPaint_V25.lha]

The CDTV Developer Kit is from now on available for everybody.
[http://home.t-online.de/home/amiga-freak/cd32developer.html]

The JFIF datatype by Andreas Kleinert, akJFIF, is in version 44.96 ready for download.
[http://www.ar-kleinert.de]

From Andreas Kleinert as well comes the new version of the MPEG player akMPEG V4.60.
[http://www.ar-kleinert.de]

For Scalos there is a new version of the Persistant Windows PlugIn.
[http://scalos.striatum.org/products/scalos/modules/Scalos-Plugin_PWindows.lha]

Version 14.0 of the diskgames-to-harddisk installer WHDLoad has been published.
[http://www.whdload.org]

A program with similar purpose, that is JST, is available in version V4.7 and has become freeware. Additionally the source code is available.
[http://jffabre.free.fr/amiga/patches.html]

From Oliver Roberts comes a datatype for Adobe Photoshop files.
[http://www.nanunanu.org/~oliver]

The Wav datatype by Stephan Rupprecht is available in version 41.10.
[http://home.wtal.de/stephanr/]

Amster, the client to the disputed mp3 service Napster, has been released in version 0.8a.
[http://212.10.48.147/amster/files/0.8a/Amster-main.lha]

A new version of EZP@gerNG is available.
[http://www.sfxsoft.de/de/]

---------------------------------------------------------------------------
Hardware
---------------------------------------------------------------------------

The G-Rex 1200 PCI is available. The board offers users of BlizzardPPC five PCI 2.1 slots.
[http://www.dcecom.de]

Grzegorz Jureszek announced another PCI board with name "Prometheus". The board is said to be constructed for ZorroIII slots and offers four PCI slots.
[http://www.matay.pl]

---------------------------------------------------------------------------
Miscellaneous
---------------------------------------------------------------------------

The Amiga market again has been enriched with several merchandizing articles. Beside CD containers there are cups and tumblers made of high-grade steel.
[http://www.amiga.com/amigamall/index.php?cart_id=01259371408&type=accessories]

The German part of ESCENA's website received a new layout, but there still are several blank pages. There is not yet any new information about AmigaOne.
[http://www.escena.de]

The documentation of the once planned PIOS One now is accessible to everybody.
[http://www.amigadev.net/index.php?&subpage=home&function=docs]

Thomas Raukamp passed on his position as editor-in-chief of AmigaPlus (formerly amigaOS) to Nico Barbat and will devote his time solely to the Atari periodical st-computer and the mac life.

On the mailinglist of AmigaOne Fleecy Moss answers questions about AMIGA's correction of course, about the AmigaDE and AmigaOS 4+.
[http://groups.yahoo.com/group/amigaone]
