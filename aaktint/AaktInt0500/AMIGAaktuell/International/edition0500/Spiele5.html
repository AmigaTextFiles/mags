5. Preview: Heretic II

This month you find an unusual preview. Because I do not own a PowerPC board, I could not introduce Heretic II, of course. But the release of this demo is an important event and so I got informed in the IRC and asked for experiences with the demo. The results of this summons is this inofficial preview. Apart from one exception everybody got Heretic II running and it was found out, that the game runs without stuttering in a resolution of 320x256. But I was unable to find out, how this might be with a 3D accelerator. The main informations were, that it runs fine in 320x256. One of the three people I asked didn't get Heretic II running. But this problem could be solved, because Hyperion Software already released a small update. Further comments/progress reports about the demo version of Heretic II can be sent to me.
