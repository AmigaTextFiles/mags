9. Debian-Linux m68k available at Klaatu

»Press release

Debian-Linux m68k for Amiga is now available for a price of 19.00 DEM (approx. 9.69 Euros) for the 2-CD-set at Klaatu Soft&Hardware.

It can be run on any amiga with MMU, which are the A3000, A4000 and any other Amiga, if there is an extension board with MMU present. 8 MB RAM and min. 250 MB hard disk space are further requirements.
