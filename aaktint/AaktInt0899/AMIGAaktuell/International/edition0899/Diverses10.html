10. Product information: 'DragonTools' v2.2

»Status: Freeware 
Short : Setting up ISDN boxes "Speed Dragon" and "Power Dragon" 
  

DragonTools suports you with the setting of the ISDN boxes "Speed Dragon" and "Power Dragon" by Hagenuk.

Functions

  • Changing the data transceiving speed between Amiga and databox

  • Setting up the analogue ports

  • Activation of telephone number suppress mode

  • Programming of call redirections using Databox and Vst.

  • Reading and erasing of any of the 5 telephone charge counters

  • erase caller list

  • Reset to standard settings

  • Load new firmware into the DataBox

Features

  • Graphical user interface

  • Doesn't require MUI

  • Comfortable selection of the serial interface

  • Supports transceiving rates from 300 to 115200 baud

  • Uses locale catalogues

  • Supports decimal charge factors from firmware 3.10 or better

  • Dial help using the optional available "DragonDaemon"

  • Fast transmission of the basic settings, if there are little changes

  • Supports the new DataBox "Power Dragon"

Requirements

  • Amiga using Amiga-OS 3.0+

  • DataBox "Speed Dragon" with firmware 2.14+ or "Power Dragon"

  • ard disk
