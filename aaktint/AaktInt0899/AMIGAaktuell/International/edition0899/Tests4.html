4. Contents overview: Aminet Set 8 (by Sofiane Ben Hassine)

»Aminet Set 8

The Schatztruhe did it again. They pressed a new Aminet Set on four completely filled CDs. On these CDs you find the contents of Aminet 28 - 31.

On the first CD you will find the full version of Directory Opus 5.5. There are all the tools of the last four CDs. The second includes all the graphic files and a full version of ArtEffect 1.5, so you can do graphic editing. The third CD contains a full version of Gloom 3. The main part of this CD is fun. That means everything a player needs, games, demos, patches. The fourth and last CD contains all the mods and special software for playing them. The full version is CygnusEd 3.
