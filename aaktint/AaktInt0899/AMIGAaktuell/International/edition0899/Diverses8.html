8. Schatztruhe announces Aminet 32

Aminet CD 32 - 8/99

Aminet CD 32, dated August 1999, contains nearly 1 gigabyte of (uncompressed) software in thousands of archives. Since the release of Aminet Set 8 over 530 MB new software has appeared plus 190 MB of software, which didn't fit on Aminet 31. The current edition includes the full version of the well-known text editor GoldED 4.8.0.
