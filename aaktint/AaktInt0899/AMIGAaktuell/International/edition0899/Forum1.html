1. Letter to the editor regarding the use of the Linux kernel with the AmigaNG (by Harald Knoop)

»I think, it is wrong to make Linux the Kernel of the Amiga operating system. Why does Amiga, Inc. abandon QNX, which is nearly ready? In addition with a sudden system swap the development of many programmers, which needed several years, has been lost.

As an ALTERNATIVE Linux is fine, anyway. But if it should become the operating system of the Amiga, what I don't hope, there must be a port of OS 3.5 for the NG Amiga!

Harald Knoop «
