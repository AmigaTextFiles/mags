description:

"Jewels of the Oracle" is one of dozens of classic graphics adventure on the Mac. Step by step, as usual, the player moves via mouse but there's a special feature which uses animations in order to lead to another point of view instead of normal transitions.

The game takes place in an ancient building where the player has to solve logical riddles such as small puzzles or Sokoban sequences in order to collect a lot of jewels. He is supported by an oracle's mouth which can give more or less useful mystic prophecies via mouse click, but these rather intensify the atmoshpere.

game speed:

Benefiting from deactivatable intermediate animations which can be replaced by pretty fast screen changes the game speed is quite alright even without a graphics board. Unfortunately, the music is often interrupted but this should not happen on faster systems.

Result:

If you like graphics adventures and also solving tricky puzzles then you will probably like "Jewels of the Oracle", too. Again you can say: "For the price offered the game might be worth testing."
