6. ALD is the successor of the Amiga Web Directories an

After the Amiga Web Directory was closed there is an (inofficial) successor of this service:

»3.2.2000 amiga-news.de from now on with Amiga Link Directory (ALD)

On January 23, 2000 amiga-news.de introduced the new Amiga Link Directory (ALD) to the public, which wants to become THE directory for AMIGA pages. The technology to do this is from the software house Innovative (http://www.innovative-web.de) in Herzogenaurach, Germany.

Entries are sorted by categories and alphabet or can be found using the search function. To get always the latest informations the persons/companies listed have the possibility to change their entry using a previously defined password. There can be shown additional informations, if the listed persons/companies wish. So the address and telephone number can be found as a logo or the eMail or homepage address.

The ALD will be present in parts on the Aminet CDs (but only with the eintries, it is wished for). For the first time it is possible to build a bridge between the AMIGA online and offline world and to improve the communication of the Amiga community. Companies can can get in touch users without internet access.

Please help us to build the larges online directory of AMIGA pages and list your website today!

The ALD can be found on: http://www.amiga-news.de/cgi-bin/anwd-db.pl

  • Your amiga-news.de team«
