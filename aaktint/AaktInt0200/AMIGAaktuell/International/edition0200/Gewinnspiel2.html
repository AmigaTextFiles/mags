Andreas Falkenhahn gave his program NewInstaller for the competition to us:

»There are many software packages in the Aminet without installation program, even if Commodore developped the Installer program, which offers an easy to use user interface and has also many powerful functions.

But many authors do not have the time or are to lazy to write an Installer script. Many of them also found it too difficult.

But this time is over. NewInstaller makes it possible for any author to include an Installer script with his software package. But stop - you do not have to write the script yourself - NewInstaller does this for you fully automatically.

The only thing you have to do is to add an icon with the default tool NewInstaller to your software package. In addition you can set further options using the tool types.

NewInstaller is completely integrated into the Internet. You just have to add two tool types and the user can register your program online, if he wants to. Or you add a tool type and he can directly browse to your Internet homepage!

But there is more: NewInstaller can make the installation to a first class multimedia event. It can show various pictures and play samples and modules. And it does not matter, if the installation runs on AGA/ECS/OCS/graphic boards or whatsoever... The pictures are always shown in the best possible quality and adjusted to the Workbench colours as well to the Workbench's size!

Many things can be changed with the Tool types.

I decided to release NewInstaller as Shareware.

If you register and want to get your key file per eMail, this costs 10 DEM (approx. 5.10 Euros). If you want to receive it per mail on floppy disk the price raises to 15 DEM (approx. 2.65 Euros)!

A demo version (apart from the nerv requester fully operational) can be downloaded from the Airsoft Softwair Homepage (http://airsoft.home.pages.de/) or from the Aminet laden (util/wb/NewInstaller1_5.lha).«
