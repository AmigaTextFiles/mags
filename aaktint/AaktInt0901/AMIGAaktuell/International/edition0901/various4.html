Various Amiga news

4. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

68k/miscellaneous

For the "X-Surf" board a first beta version of the PPPoE driver is available. Only experienced users should try the installation.
[http://www.jschoenfeld.de/jumpnews.htm]

The manual of "Pagestream 4" now is available as PDF file in English language.
[http://www.pagestream.de]

The program "Digital Almanac III" has been released as version 4.1.
[http://soft-ware.de/dalmanac/]

The Warp3D drivers for the "G-Rex-PCI-Busboard" have been released. and are available for download.
[http://www.dcecom.de/]

For the "Prometheus-PCI-Busboard" several new drivers have been published, among others updates for the Picasso system.
[http://www.matay.pl/]

As well for the "Prometheus-PCI-Busboard" the version 1.5 of the developers software was released.
[http://www.matay.pl/]

For the "Mediator-PCI-Busboard" version 2.4 of the drivers software appeared.
[http://www.elbox.com/]

Oliver Wagner announced, that in one of the next days the publishing of the new version of his WWW browser Voyager may be expected. Furthermore the author stated, that he will not exclusively develop the program for 68k Amigas and MorphOS, but that he considers also a support of the Amiga emulator Amithlon - using x86 native code - as well as the AmigaOS 4.0.
[http://v3.vapor.com/index.php?finger=owagner]

PowerPC

The program "Digital Almanac III" has been released as version 4.1.
[http://soft-ware.de/dalmanac/]

---------------------------------------------------------------------------
Software (non-commercial)
---------------------------------------------------------------------------

68k/miscellaneous

The AOL Instant Messenger clone "AmigAIM" has been released in version 0.9432 beta. Several bugs were fixed, so for instance a problem when closing the program, while a service is logging in.
[http://amigaim.missingpiece.com/downloads/AmigAIM_BETA.lha]

A new specimen of the Warp datatypes of Oliver Roberts has been published. It is the "WarpTIFF-Datatype", which also supports the specifications 5.0 and 6.0 of the TIFF format. It has the version number 44.3.
[http://www.nanunanu.org/~oliver/warptiff.html]

There are updates of the other Warp datatypes, too: "WarpJPEG" V44.23, "WarpPNG" V44.19 and of the preference "WarpDTPrefs" V44.6.
[http://www.nanunanu.org/~oliver/]

For the games-to-harddisk installer "WHDLoad" new packages have appeared, among others for Operation Stealth, Cruise For A Corpse and Maniac Mansion.
[http://www.whdload.org/]

New versions 44.120 of the datatypes "akPNG", "akTIFF" and "akJFIF" have been released.
[http://www.ar-kleinert.de/]

The e-mail client "SimpleMail" is in version 0.8 alpha ready for download. Innovations since last version are for instance SSL support for POP3 server via AmiSSL and a menu bar in the main window.
[http://www.simplemail.de.vu]

The Workbench substitution "Scalos 1.3" received an update to version 40.10 beta and it contains many bugfixes compared to the previous version.
[http://www.scalos.com/]

Version 4.1 of the programming language "Emperor" is available in Aminet. With help of this language it is possible to create a GUI for programs just by clicking.
[http://de.aminet.net/pub/aminet/dev/c/Emperor_v4.1.lha]

The program "RB2Exe", which permits to transform Rainboot scripts to stand-alone demos, has appeared in version 1.5. Beside other improvements the memory consumption of the created programs is decreased by nearly 60%.
[http://www.airsoftwair.de/]

The "xvs.library", used for the detection of viruses, has been released in version 33.30.
[http://home4.inet.tele.dk/vht-dk/amiga/xvs/xvs.htm]

The screengrabber "SGrab" has been published in version 1.18.
[http://home.wtal.de/stephanr/downloads/sgrab.lha]

For "Scalos" a plugin with name "StackWindow" has appeared, which permits arranging windows in a space-saving manner.
[http://scalos.striatum.org]

For the virus killer "Virus Executor" a new version of "VEPatch.Brain", the version 1.36, has been released.
[http://www.vht-dk-dk/]

The text-viewer "EvenMore" has been published in version 0.58 and now even permits display of MS-Word files of the versions 95 and 2000.
[http://www.evenmore.co.uk/]

The Amiga port of "mySQL" goes ahead. Version 3.23.41 has appeared and requires at least a 68040 processor.
[http://sixk.maiasys.com/index_en.html]

Of the file manager Directory Opus 4, carried on as Open Source, version 4.16 beta3 has been released.
[http://dopus.amiga.pl/]

Version 2.20 of the virus killer "Virus Executor" has been published.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/ve.htm]

There is a new version, V0.81, of the SinclairZX-Spectrum emulator "ASp".
[http://www.greew.freeserve.co.uk/ASp.html]

The virus-scanner "Safe", which recognises viruses, but cannot remove them, has been released in version 16.4.
[http://www.vht-dk.dk/]

Miscellaneous
---------------------------------------------------------------------------

Haage & Partner via the periodical "AMIGAplus" announced an Amiga emulator based upon the operating system QNX. An official press release still is expected.
[http://www.haage-partner.com]

Merlancia acquired the rights at the design of the Walker and the AmigaMCC. Possibly in the future we can expect a computer in a nifty AudiTT [German sports limousine] design.
[http://www.amiga.com/corporate/080901-merlancia.shtml]

The Macintosh show maxpo, planned in September in Cologne, Germany, where Falke Media wanted to organize an Amiga area, is cancelled. The »Amiga World« is said to be arranged as independent show at a date near the end of the year.
[http://www.amiga-messe.de]

The Internet site "amiga-news.de" celebrates its third anniversary. From here "Cordial Congratulations!" by the AMIGA aktuell International team.
[http://www.amiga-news.de]
