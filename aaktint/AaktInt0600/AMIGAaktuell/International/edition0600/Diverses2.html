Various Amiga news

2. News from Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»A new Shareware program has landed! It's the first new Shareware program from Airsoft Softwair since 2 years. RB2Exe is THE add-on you always wanted for Rainboot2. RB2Exe converts Rainboot configurations into fully stand-alone executables, that run on every Amiga system with OS2.04 and up. So you can do your own multimedia demonstrations and shows now with Rainboot. «

»One week after the release of the CD32 Games Install Kit v3.5 the Cheat and Info Kit add-ons have been released. The CD32 Games Info Kit contains information and pictures about many games and the CD32 Games Cheat Kit offers you cheats for many CD32 games. Both add-ons are now available for download in the Downloads section.«

»As I promised, here are the additional icons for the CD32 Games Install Kit. There are a lot of icons in the archive right now, supporting all Amiga icon systems (MagicWB, NewIcons and even standard ones). Be sure to download this icon package, if you are using the Install Kit. The other Install Kit bonus programs will be made available the next days.«
