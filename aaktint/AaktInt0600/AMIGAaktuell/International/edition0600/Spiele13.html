13. Hints for players without a graphics board (Matthias Puch)

Especially for Lucasfilm adventures such as "The Dig", "Day of the Tentacle" or "Sam & Max" the next hint might be interesting for speeding up the game.


  1. First of all you install a screenpromoter such as ModePro from the Aminet on the Amiga side.
  2. This screenpromter has to be started before Fusion and has to configured so that Fusion is opened on a 320x240 screen with 640x400 pixels. This can be done by adding "Fusion Video Task 0" in the list "programme name" and at "select screenmode" you choose Lowres and a resolution of 640x480. You needn't use autoscrolling.
  3. If you start Fusion with these options then the first upper quarter is displayed. If you drag the screen a little bit (with the right Amiga key) you may start the game.
  4. Finally choose a small screen and drag the screen part to the center of the screen. Now the game not only runs in a pretty big window but also is much faster because the AGA processor is burdened less.

Have fun trying!
