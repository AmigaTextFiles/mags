1. Answer to the question of Gordon Muth regarding IRC in edition 3/99 (by cHAVEZ)

»Gordon Muth <Gordon.Muth@rz.ruhr-uni-bochum.de> hat in der asked for IRC servers and channels in the last edition of AMIGA aktuell international:

IRCNET


deutsch: irc.gmd.de -> port 6667 #amigager (usual amiga users) 
deutsch: irc.germany.net -> port 6667 #amiga (usual amiga users) 
englisch: irc.gmd.de -> port 6667 #amigascne (amiga scene) 
englisch: irc.gmd.de -> port 6667 #thescene (amiga&pc scene) 
englisch: irc.gmd.de -> port 6667 #amiga (ww amiga users) 
englisch: irc.vapor.com -> port 6667 #amiga (vapor server amirc) 
    

but in English channels it is also written in German, which is due to the fact that there are enough German users and ops there ;)

Passwords aren't necessary with the channels listed above.

The server "irc.gmd.de" I named is the fasted server within Germany - if ther are some problems, there are various other servers as: "irc.uni-stuttgart.de", "irc.uni-paderborn.de", "irc.funet.fi", "irc.netsurf.de" - each with the port 6667.

The quite small germany.net allows to register and to protect someones nick/handle. You get more information, if you type "/msg nickserv help" (witout quotation marks..haha) - if this doesn't help just ask one of the users in the channel #amiga.
