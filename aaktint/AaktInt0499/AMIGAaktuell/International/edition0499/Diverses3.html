Usage:

The operator is quite easy to use. First you have to start Wildfire - there is no further knowledge of the user regarding Wildfire 7 required. In Wildfire there aren't any settings to be made, everything works completely automatically!

You can draw the WF operator on the time line of a UPN interface of Movieshop as any Movieshop operator.

A window opens, where you can set the editing mode, the effect group and the desired effect using list gadgets.

The operator shows instead of the effect name the name of the group and the Wildfire effect within the cut list. On the right and on the left near the operator icon you find the operator name WfMS. Within the status line the current computing step is shown, if the operator window stays open.

The three included MovieShop macros WFControl, WFHide and WFShow make it possible to iconify resp. to activate the Wildfire window out of MovieShop.

WFControl iconifies resp. activates the window depending from the current status of Wildfire. WFHide iconifies the window and WFShow activates the window.

Some powerful special effect scripts will be made available, when buying the operator, for free. Oberland Computer will continue selling various effect collections for each 50 DEM - 100 DEM (approx. 25.51 - 51.02 Euros) or bundled with Wildfire 7 for approx. 500 DEM (approx. 255.10 Euros).

The user can create his own effect sessions in Wildfire and use them out of Movieshop, of course. Of course, experience is necessary, to do this.

Due to the simple concept of WF the chreation of these scripts is nearly a children's play.

Movieshop and Wildfire run on a normal Amiga as well as on a Draco.


There are four modes within the operators: 
1. Single Image 
 This mode is to edit unique scenes using the help of pre-defined effect scripts by Wildfire. 
  
2. Compose 
 With this mode two scenes can be combined and attached to another using pre-defined 
 WF effects. 
  
3. Current Single 
 Is similar to the mode Single Image, but the effect is already loaded into Wildfire 
 resp. developed in WF. 
 Due to this an effect can be tested to be exported as effect script. 
 This mode is recommended to experienced Wildfire users, who want to generate 
 their own effects. 
  
4. Current Compose 
 Like Current Single, but analogue to the Compose mode. 
  

Some effects require some time to be computed. During this time it seems, that nothing works, but your computer works quite intensive, anyway.

During the calculation of a picture by Wildfire the operation cannot be aborted. So be patient - it's worth it !!
