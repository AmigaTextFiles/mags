6.Preview: Tales of the Heaven

Since a short time you can marvel at the demo of the Super Mario 3D clone
Tales of the Heaven. This game is a kind of 3D jump'n'run. The game was
prgorammed 100% in assembler and runs playable, but not smooth, on my 68040.
But this is understandable, if you watch the computed environment. The
controls are very individual, but after some time you get use to them.
Unfortunately the demo version quits, if you fall e.g. from the stairs.
You find yourself on the Workbench, again. But if the game is slightly improved,
it could become a hit, without doubt. Tales of the Heaven is THE Amiga game
of the 3D jump'n'run genre. Presently it supports only AGA, but a RTG support
is under development. It will not support PowerPC, because it was programmed
completely in Assember. Lets hope, there are some optimizations regarding
speed and controls. The demo can be downloaded e.g. from the Amiga Flame
website. You absolutely should watch it. Nothing has been said about the
release date of the full version, there has to be found a distributor, first.
