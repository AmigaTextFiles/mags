4. Wildfire revision 7.13 released

»Wildfire revision 7.13 - April 1999

Since April 1999 the Wildfire 7 update is available. It includes these changes:

  • Direct Movieshop support!!

    Loading and saving of WFMV pictures is supported from now on. These pictures are required from the new Wildfire oerator in Movieshop by Heinrich Angler (Heinrich.Angler@muenchen.de).

    So you can use Wildfire effect scripts directly in Movieshop! Within the directory WF:Movieshop/ you find all required files to install the Wildfire-Movieshop operator.

    (The Movieshop operator has to be released by Macrosystems)

    • 2. AREXX

      The commands wf_movieshopmode and wf_blackframes have been added.


    • 3. Non-linear-effects
      • Realtime preview can be switched on and off!!!

      • Texts of the input connection are available in German, now

      • Operator input/output texts is available in German, now

    • 4. Settings

      The selected screenmode for playing in the mode `Window' is being saved, now.


    • 5. Converters

      HAM pictures weren't saved, if a picture depth larger 8bit was chosen, before. - Fixed.


    • 6. Transition Maker (Fading-Generator)
      • The RX command is now loaded from SYS:REXXC/ .

      • Instead of filename the effect descriptions are shown. So the user gets an idea, what the chosen effect is going to create.

      • The Transition example animations can reside in the WF:TransAnims/ drawer, now. So you can e.g. copy the Transition animations of the CD into this directory!

      • Some new fadings have been added. (Tile, ScaleWipe, Perspective)

    • 7. Documentation

      In the directory WF:Dokumentation you find the German Online Help as ASCII text. So you can print it and/or search within it!
