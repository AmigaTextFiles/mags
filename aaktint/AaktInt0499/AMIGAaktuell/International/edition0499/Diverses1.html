1. Free Java-2-package announced

»DaytonaTM is a complete JavaTM 2 runtime system for AmigaOS, based on the "Java
2 Platform" by Sun Microsystems, Inc. It consists of the following parts:

DaytonaTM main package, available FREE of charge. Among other things Daytona
contains support for:

* The complete Java 2 API and class set, directly derived from Sun's reference
implementation, and therefore 100% compatible with the official Java
specifications and releases (unlike some other Java runtime engines for
AmigaOS).
* Multithreading, using a native Amiga implementation. No Unix emulation code
or libraries required.
* File and socket I/O (multithreaded) including multicasting, URL handling and
Internet protocol engines.
* AWT (Abstract Windows Toolkit): The basic Daytona package contains AWT
runtime classes, but no native bindings. AWT bindings for native AmigaOS
displays are available as part of the DaytonaPLUS package.
* JFC (Java Foundation Classes): Swing, Java2D.
* CORBA

DaytonaDEVTM package, available FREE of charge. DaytonaDEV is intended for
developers who want to write their own Java programs, and compile them into
Java byte code. DaytonaDEV contains the following items:

* Java byte-code compiler: launcher and all required runtime classes.
* Various tools (byte-code disassembler, documentation generator, archvie
management tools etc.)

DaytonaPLUSTM package, released as shareware. The precise shareware fee will be
announced later. DaytonaPLUS is an independently developed add-on to Daytona
that provides additional Amiga-specific features. DaytonaPLUS contains:

* AWT bindings for native AmigaOS displays. Includes support for CyberGraphX
and Picasso96 high/true-color screens.
* BOOPSI "look-and-feel" for Swing.
* Applet API and runtime system with a stand-alone Applet viewer, and a
documented Applet API library, allowing Daytona to be used to add Java
Applet support to AmigaOS web browsers.
* PowerPC support, for PowerUP or WarpUP (not determined yet).
* JIT (Just-In-Time) compilers for 68K and PPC.
* Support for precompiled native (68K and PPC) versions of all standard Java
runtime classes. Daytona and DaytonaDEV come with complete sets of 68K/PPC-
precompiled runtime classes for use with DaytonaPLUS.

Features are subject to change, and not all features might be available in the
first version.
