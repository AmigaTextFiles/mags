5. Schatztruhe GmbH announces Aminet Set 8

»Aminet Set 8

AMINET SET 8 offers you everything that was added to the archive since AMINET SET 7 was made, plus full versions of CygnusEd 3.5, ArtEffect 1.5, Gloom 3 und Directory Opus 5.5.

AMINET® SET 8, dated July 1999, consists of approximately 4 gigabytes of software in 5.500 archives. Since the release of Aminet 31 more than 300 MB of new software have been added and more than 600 MB did never appear on regular Aminet CDs due to space restrictions.
