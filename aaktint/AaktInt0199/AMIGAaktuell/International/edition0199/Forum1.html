1. Reply to A. Wendorf's question regarding networking (by Holger Jeromin)

»If you want to build a network with two Amigas, there is the software Envoy. (Beat me, if that's wrong, I don't use it.)
But if you want to exchange data with a PC, you can use Samba.

A step-by-step manual for the installation can be found on the homepage of Miami. www.nordicglobal.com/MSamba.txt On the Amiga side it is not comfortable, but it works. Especially with Win95 it is quite comfortable, because the Amiga appears in the normal network environment.
It could also be possible to install a ftp-server on the PC, but I don't have any experience with this matter.

I hope, I helped you.
