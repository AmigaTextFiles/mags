3. New operating system for PowerPC-AMIGAs under development?!

On the WWW site http://www.mematex.se/amiga/screens/index.html are some informations about a new operating system for PowerPC AMIGAs, which seems to be under development. Screens is supposed to have these features:

  • »Amiga only, no slow code to make it work on other computers
  • Run AmigaOS and Screens© at the same time
  • 16/24 bit screens only
  • 16 bit sound (Paula is not yet emulated)
  • New GUI system, using intuition only to open the screen
  • New ARexx system
  • 16/24 bit mouse pointer at any size, not using Amiga sprites
  • Long filenames, 255!
  • much more....

Requirements

  • 68040/68060 and PowerPC-Prozessor (WarpUP)
  • CyberVisionPPC or BVisionPPC (8 MB needed)
  • 10 MB Fast memory
  • CyberGraphX v4
  • Kickstart 3.1 or higher
  • 20 MB Harddrive space
  • Soundcard>«
