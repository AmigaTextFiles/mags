5. Preview: Max Rally

The game Max Rally by Fortress Software is a top view racing game in the style of All Terrain RAcing. The full version is expected to have 26 different courses in four different worlds. The game's target is to convince with the possibility to play, instead of graphic. This is said quite appropriate. The graphic is, if you don't mention the title screen, not a hit, but like on an A500. But the demo is quite good to play. Even if the graphic is not the latest state the 4-player-mode in the full version could be fun, because there are different game modes like Multi-Player Battle. So we hardly can't wait, if the full version can hold the demo's promisses. The demo version is not in Aminet, but on various homepages. One of them is the Amiga Flame website (http://www.amigaflame.demon.co.uk/maxdemo.lha, approx. 500 KB). Take a look!
