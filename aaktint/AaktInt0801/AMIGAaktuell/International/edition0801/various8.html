Various Amiga news

8. News about Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»The first version of the CD32 Games Install Kit v4.5 for Windows is out and it already can run the most games without problems because of the custom patches which are applied to almost every game. Even CD audio emulation is possible now! (this required a custom version of WinUAE which you can download at http://phobos.spaceports.com/~sfa/en/downloads.html). However, the Windows version of the CD32 Games Install Kit is currently only available for owners of the "The Best Of Airsoft Softwair Gold Edition" CD-ROM. If you own this CD-ROM, head over to the update section (http://phobos.spaceports.com/~sfa/en/boaupdate.htmland) and grab your copy!«

»The Updater from the "The Best Of Airsoft Softwair Gold Edition" CD-ROM is now available in a new version. An important bug was fixed, so you should download and install this version. Thanks to Mario Cattaneo for reporting this one!«
