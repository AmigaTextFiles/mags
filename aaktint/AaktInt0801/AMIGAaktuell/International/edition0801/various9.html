Various Amiga news

9. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

68k/miscellaneous

For "PageStream" an update to V2.1 of the "Universal filter" has been released, that permits the loading of many graphics formats to PageStream, BME or HHV.
[http://www.uni.coloraid.de/]

For the MAS-Player version 1.4 of the MASMPegDevice has been released.
[http://www.platon42.de/platon/files/mus/MASMPegDevice.lha]

The Voodoo 3 Warp3D drivers for the PCI busboard "Prometheus" are finished. Additionally all other drivers for the board are updated.
[http://www.matay.pl/]

Supply proof and Haage&Partner offers the compiler "StormC" for students and trainees at DM 298.-. The regular price is DM 498,-.
[http://www.haage-partner.com]

For "ArtEffect" and "AmigaWriter" toolbars in GlowIcon look have been published.
[http://www.haage-partner.com]

PowerPC

The Voodoo 3 Warp3D driver for the PCI busboard "Prometheus" are finished and are available in a PPC version, too.
[http://www.matay.pl/]

---------------------------------------------------------------------------
Software (non-commercial)
---------------------------------------------------------------------------

68k/miscellaneous

The file system "Smart File System" (SFS) is available in version 1.190a.
[http://home.wtal.de/js/index.phtml?SFS.lha]

The AOL Instant Messenger clone for the Amiga "AmigAIM" was published in version 0.9428 beta.
[http://www.missingpiece.com/downloads/AmigAIM/AmigAIM_BETA.lha]

The mail and news reader "Microdot II" was extended with several features, for instance the installation of multiple POP3 mailboxes. The current version number is 1.4.15 beta.
[http://www.vapor.com]

The compiler "vbcc" for the Amiga is available as version 0.8. Several bugs are removed and a few new features added. Furthermore faster code is said to be generated.
[http://www.compilers.de/vbcc/]

For the Workbench replacement "Scalos" a new plugin with name "DevFilter" has been released, which allows blanking out device icons from the graphical user-interface.

Another new plugin with the name "Titlebar Clock" permits the extension of the titlebar of the Scalos desktop for a clock and date display.
[http://www.scalos.com/]

"Scalos" itself has been published in version 1.3 beta.
[http://www.scalos.com/]

The paint-program "DiamondBOX" has been published in version 1.12.
[http://www.DiamondDice.f2s.com/]

The continued development of the now free file manager "Directory Opus 4" shows first success: Version 4.16 has been released and now is localised for several languages.
[http://dopus.amiga.pl/files/DOpus416b2.lzx]

The file manager "ClassAction" also received an update to version 4.2.
[http://www.martin-elsner.de/]

The midi sequencer "Bars&Pipes", released to freeware, which now will be developed, has been renamed "BarsnPipes", to make clear the new status. The recent, already extended version has the version number 1.
[http://home.t-online.de/home/alfaj/alfa.html]

The SMS transmitter "TaskiSMS" is ready for download in version 2.58.
[http://taskisms.taski.com.pl/]

For the compiler gcc a bug-fix (2.95.3-3) appeared, which solves the problems with the compiler options "-fbaserel" and "-fbaserel32".
[ftp://ftp.geekgadgets.org/pub/geekgadgets/amiga/m68k/alpha/gcc/]

The HTML viewer "SimpleHTML" has been published in version 0.17.
[http://home.t-online.de/home/sebauer/simplehtml.html]

Version 0.17 of the open source operating system "NetBSD" has appeared.
[http://www.netbsd.org/Releases/formal-1.5/NetBSD-1.5.1.html]

The virus scanner "Safe" is published in version 1.5.1.
[http://home4.inet.tele.dk/vht-dk/amiga/safe/Safe.lha]

The "WarpJPEG.datatype" was released in version 44.22.
[http://www.nanunanu.org/~oliver/warpjpeg.html]

The "WBStartEmu" for AmigaOS 3.9 has been published in version 45.2.
[http://home.wtal.de/stephanr/downloads/WBStartEmu.lha]

The sample editor is ready for download as version 0.77 "Audiomaster 2000".
[http://www.nucleus.com/~fenn/]

The "DocDatatypes", permitting the reading of many text formats, for instance of MS-Word, have been released in version 40.20.
[http://go.to/docdatatypes]

The file manager "DiskMaster2" of Rudolph Riedel has been published in version 2.6b11 and met with several bug-fixes.
[http://home.t-online.de/home/rudolph-riedel/]

The "amissl.library" appeared in version 1.1, but does not run with IBrowse V2.2.
[http://tesla.rcub.bg.ac.yu/~antony/AmiSSL/]

The Amiga port of "MySQL", compiled for the 68040, is in version 3.23.40 ready for download.
[http://sixk.maniasys.com/MySqld_32340_040_base.lzx]

PowerPC

The compiler "vbcc" for WarpOS/PowerUP is available as version 0.8. Several bugs are fixed and a few new features added. Furthermore faster code is said to be generated.
[http://www.compilers.de/vbcc/]

The movie player "SoftCinema" has appeared in version 0.13 and it can now display DivX movies.
[http://softcinema.amiga.pl/]

For WarpOS the version 3.89 of the mp3 encoder "LamePPC" has been published.
[http://www.kolumbus.fi/jami.laakkonen/ppc/download/lame.lha]

The WarpJPEG.datatype has been released in version 44.22.
[http://www.nanunanu.org/~oliver/warpjpeg.html]

MorphOS

The file system "Smart File System" (SFS) is available for MorphOS in version 1.190a.
[http://home.wtal.de/js/index.phtml?SFS.lha]

The compiler "vbcc" for MorphOS is available as version 0.8. Several bugs are fixed and a few new features added. Furthermore faster code is said to be generated.
[http://www.compilers.de/vbcc/]

The webserver "Apache" was published in version 1.3.20 including version 4.0.6 of the PHP modul.
[http://sixk.maniasys.com/apache1320_php406_mos.lzx]

The program "Webalizer", permitting statistical evaluations of Apache logfiles, has been released in version 2.01-06.
[http://sixk.maniasys.com/index_en.html]

The WarpJPEG.datatype has appeared in version 44.22.
[http://www.nanunanu.org/~oliver/warpjpeg.html]

The PC emulator "Bochs" has been published in a new version.
[http://www.morphos.de/fileslinks.php3]

Miscellaneous
---------------------------------------------------------------------------

On 03-Nov-2001 in Brentwood, Essex in England the "World of Amiga" will take place. It can be taken for sure that this fair will be the first where the AmigaOne can be bought.
[http://www.worldofamiga.com/]

Martin Elsner, programmer of "ClassAction", has taken over the development of the start menu "StopMenu" from Damir Arh.
[http://www.martin-elsner.de/]

The living legend "Dave Haynie", who among other things decisively participated in the development of the Amiga 3000, now is Chief Technical Officer at Merlancia, who have announced several computer systems, also supporting AmigaOS.
[http://www.merlancia.com]

From 20th to 22nd of September in Cologne, Germany, as part of the maxpo2001 (Apple-Mac-Show) the Amiga World 2001 will take place. The show will be run by Falke Media. *
[http://www.amiga-messe.de/]

[* The date for this show has been deferred, because maxpo2001 will not happen. For more informations read the next issue]
