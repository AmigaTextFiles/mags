»Amiga Arena News
(06-July-2001)


Amiga Arena Full Version!

In cooperation with Christian Steiner Amiga Arena makes possible the free release of "Sharks Multi Version 1.5 Final"! SHARKS! is a game of skills running under water. Our brave diver is on his way to search the bottom of the sea for treasures. But the area he is in is infested by sharks ... so WATCH OUT !!! Recover as much treasure as possible and beware of the sharks! Because of not sufficient feedback by the users the development was cancelled!

Amiga Arena "File Cutter" Code!

Stefan Berge made available the code of File Cutter! File Cutter makes it possible to split files to pieces. For instance a file of 8 MB (animation, LHA archive) can be cut down to several pieces each of a size, that this file easily can be transported on several diskettes.

On the other computer these file parts can be assembled to the complete file again.

»Amiga Arena Full Version!

With permission of Mark Ford Amiga Arena makes possible the free release of "The Prophet"! "The Prophet" is a role game, that was published by "Alive Media Soft" in the year 1999! Unfortunately there arose some problems and so one year of development was lost for Mark F. "Amiga Format" rated "The Prophet" in those days with 81%. The Prophet now is available in English language!

Some features:

  • more than 200,000 locations

  • no linear sequence of the game

  • different enemies

(21-July-2001)


Amiga Arena Full Version!

In cooperation with Denis Unger Amiga Arena makes possible the release of "Foreign Language Master V2.3"! FLM is a foreign language program, which can be used as dictionary, for the translation of texts and for learning of words (vocabularies).

FLM contains several English-German dictionaries (altogether appr. 8 MB with appr. 175000 entries).

Denis Unger proceeds with the development of "FLM"! For a new version of "FLM" Denis Unger would be pleased about substantial advices for improvements and feedback! More informations and databases for "FLM" you will find under: Homepage: http://www.flmdic.de Contact: flmdic@gmx.de

Amiga Arena - Commentary!

I for myself use "FLM" nearly daily and cannot be without it any more! My appeal at you - with just a short mail to Denis U. everybody could share, that this tool further will be improved and maintained!«

Amiga Arena - Special Price Action II !

In cooperation with Frank Reibold the Amiga Arena makes possible a special price for the programs MicroPaint and Microbase.

The most important features of MicroPaint are its functions for picture processing and the compatibilty with *nix / Linux (excerpt):

  • Up to 256 colours on computers with AA chipset

  • Support of Intellifonts, font styles, drawmodes, Colour Fonts

  • Support of all graphics modi, which are supported by your computer

  • Flexible spray modi

  • Multi-spray: Spray with random colours

  • Invert graphics and change colours

  • Switch colours

  • ARexx port

  • Lists the files used as latest in the project menu

MicroBase presently offers you following possibilities (excerpt):

  • Import and export of ASCII files in the the formates Superbase Professional 4, BECKERtext III, Documentum 2, ProWrite, TransWrite, Personal Write; accommodation to other formats is possible

  • Flexible functions for search and replace of field contents, for instance multiplication of field contents with a constant

  • Field names can have up to 20 characters and may contain spaces

  • Direct access via data-set numbers

  • Filter functions

More informations you will find under http://www.reibold-online.de
