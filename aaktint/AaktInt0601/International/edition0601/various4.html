Various Amiga news

4. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

68k/misc.

The HD recording software 'Audio Evolution' (formerly PlayHD) has been updated to version 3.20.
[http://www.audio-evolution.com]

'Digital Almanac' has been released in version 3.5 and was extended among other things with buttons in GlowIcon-style by Martin Mason Merz.
[http://members.tripod.de/achimste]

'MakeCD' has appeared in version 3.2 beta 7 and received several bug-fixes, such as at the SCSI_Atapi drivers.
[http://makecd.core.de/ftp/MakeCD32d_public_beta_7.lha]

For 'CyberGraphX 4' an update came out, which is meant for the owners of a G-Rex PCI-Bus-Board.
[http://www.schatztruhe.de]

For the graphics tool 'UltraConv' an update to version 4.1 has been released.
[http://www.innovative-web.de/reg.php?rlang=d]

The program 'Pro Station Audio' is available in the new version V4.10 and offers support for several new formats.
[http://www.audiolabs.it]

The 'isdnsurfer.device' is available in version 1.944 and supports Sync-PPP, which enables the dialling into all networks via Call-By-Call.
[http://www.jschoenfeld.de/jumpnews.htm]

PowerPC

For the iMac emulator 'iFusion' the update to V1.2 has been published, which eliminates some minor errors.
[http://www.blittersoft.com/cgi-bin/ams/shopzone30.cgi/iFusion.html]

'Digital Almanac' has been released in version 3.5 and was extended among other things with buttons in GlowIcon-style by Martin Mason Merz.

Furthermore the IFF-ANIM compression now is possible in PPC mode.
[http://members.tripod.de/achimste]

---------------------------------------------------------------------------
Software (noncommercial)
---------------------------------------------------------------------------

68k/misc.

After '3ivx' as replacement for MPEG-Video its programmers want to create an open replacement for Quicktime, which is to run on as many as possible platforms.
[http://openquicktime.sourceforge.net/]

The disk-to-harddisk installer 'WHDLoad' has been published in version 14.2.
[http://www.whdload.org]

The SMS transmitter 'TaskiSMS' is ready for download in version V2.57 and contains several improvements.
[http://taskisms.taski.com.pl]

The virus-killer 'VirusExecutor' appeared in version 2.18.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/VirusExecutor.lha]

The virus-scanner 'Safe' as well was improved and now has the version number 15.6.
[http://home4.inet.tele.dk/vht-dk/amiga/]

'Amplifier' was published in V2.26 and supports now among other things Shoutcast/Icecast MP3 streaming, too.
[http://www.katodev.de/down/amplifier226.lha]

'NetBSD' has been released for the Amiga in beta-test version V1.5.1
[http://mail-index.netbsd.org/port-amiga/2001/05/01/0000.html]

The database system MySQL has appeared in the new version Dev_3.23.38 for the 68040 processor, V3.23.32 for the 020.
[http://sixk.maniasys.com/]

The file manager Directory Opus is available in the beta-version 4.15k.
[http://dopus.amiga.pl]

'SecondSpin', a program to convert Audio-CDs to MPEG sound-files has been released in beta-version 2.17.
[http://personal.inet.fi/bailu/zuulikuuli/secondspin.html]

The popular mp3 player 'AmigaAmp' appeared in version 2.9 beta and contains improved support for Shoutcast Streaming. Additionally several errors are eliminated.
[http://www.amigaamp.de/]

'VEPatch.Brain' for VirusExecutor has been published in version V.1.34.
[http://home4.inet.tele.dk/vht-dk/amiga/]

The program 'StrICQ' has been released in version V0.1732 beta.
[http://stricq.owlnet.net/]

On the homepages of 'OnyxSoft' there are more recent versions of several programs, for instance 'DRemind'.
[http://www.onyxsoft.nu]

The paint-program 'Perfect Paint' has been released in version 2.6.
[http://gothic.fr.free.fr/amiga/download/PfPaint_V26.lha]

'MooVID' and 'PIV-MooVID', the AVI/MOV players, came out in version 1.5.
[http://www.dfmk.hu/~torokl/]

The webserver 'Apache' is available in version 1.3.20 and supports the PHP module V4.0.5.
[http://sixk.maniasys.com/index_en.html]

For registered users there is available the beta-version V3.5.14 of the IRC client 'AmIRC'.
[http://www.vapor.com]

PowerPC

Version V44.17 of the datatype 'WarpPNG' has appeared. It supports from now on libpng 1.0.11.
[http://www.nanunanu.org/~oliver/warppng.html]

As well in a new version (V 44.4a) the affiliated program 'WarpDTPrefs' exists.
[http://www.nanunanu.org/~oliver/]

The popular mp3 player 'AmigaAmp' appeared in version 2.9 beta and contains improved support for Shoutcast Streaming. Additionally several errors are eliminated.
[http://www.amigaamp.de/]

The SuperNES emulator 'WarpSNES' completely was reworked and now has the version number V6 beta.
[http://raistg4.free.fr/warpsns/]

The precompiled kernel-version 2.4.4 for Apus-Linux is available.
[http://www.amiga-linux.de.vu]

The MPEG-Video-Player 'Frogger' has been published in version 1.65rc5.

'MooVID' and 'PIV-MooVID', the AVI/MOV players, came out in version 1.5.
[http://www.dfmk.hu/~torokl/]

MorphOS

The 'guigfx.library' and the 'mysticview.library' were ported.
[http://www.neoscientists.org/~bifat/binarydistillery/]

The read-only browser 'Lynx' is ready for download.
[http://www.morphos.de/meanmachine/files/Fr3dY/lynx284dev20-mos.lha]

The picture viewer 'MysticView' has been released.
[http://www.neoscientists.org/~bifat/binarydistillery/]

The latest version of 'gcc' (V2.95.3) has been successfully ported.
[http://www.morphos.de/meanmachine/files/gcc-2.95.3-bin.tar.bz2]

The very widespread webserver 'Apache' (V1.3.19) was ported.
[http://sixk.maniasys.com/apache_1.3.19_Mos.lzx]

Hardware
---------------------------------------------------------------------------

First pictures of the 'Pegasos' of bPlan Enterprises are published. Among others a working LinuxPPC system can be seen on the computer.
[http://www.bplan-gmbh.de/news/pegasos_d.html]

The Hungarian Ferenc Valenta presents his sound-board 'VSS' for the clockport of Amiga 1200.
[http://ludens.elte.hu/~vf]

The delivery of the sound-board 'Repulse' has started.
[http://www.aliendesign-gbr.de]

Elbox announces new PCI-Bus boards, among them Mediator PCI 4000 for the Amiga4000D in a tower, Mediator PCI ZIII for all Amigas with ZorroIII-Slots and the Mediator PCI ZIV for all Amigas with ZorroIV-Slot.
[http://www.elbox.com]

In the commentary forum of www.amiga-news.de Thomas Dellert posted interesting information on the planned 'Super-IO board' for the G-REX board.
[http://www.amiga-news.de/de/news/comments/thread/AN-2001-05-00299-DE.html]

Miscellaneous
---------------------------------------------------------------------------

Paul Nolan has become involved in a legal case about the name 'Photogenics'. Anybody who acquired Photogenics in the USA before 1998 is asked to get into contact with him.
[http://www.paulnolan.com]

The Mattay company, developing the PCI-Bus board 'Prometheus', has released free of charge an SDK.
[http://www.matay.pl/files/PrometheusSDK.lha]

The sourcecode of the program 'Bars&Pipes Professional 2.5c' has been released as open source.
[http://home.t-online.de/home/alfay/alfa.html]

Warp3D V4 from now on becomes a commercial product, which manufacturers will have to license for a modest fee. According to Hyperion the development of 3D drivers for graphics boards is extremely improved.
[http://www.hyperion-software.de]
