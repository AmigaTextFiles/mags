Amiga Arena Special Price Action!

In cooperation with Michael Wagner the Amiga Arena makes possible a special price for Greatkartei.

Greatkartei is a very complex but in spite of that easy to handle database program, satisfying in all respects. Almost any type of databases can be "created" and administered.

The advantages:

Because of its flexibility the program can be used for any type of data administration.
It conforms with styleguide directives of Commodore.
It supports locale.library.
Text, sound and graphics can be added to any data record.
Every data file can be password-protected.
Every data file can be updated to a definite status (based on a date).
Any change in the database can be reconstructed.
Every important function can be executed with keyboard, menu and keys.

»Amiga Arena News
(31-May-2001)


Amiga Arena Special Price Action!

In cooperation with Stefan Kost Amiga Arena makes possible a special price for SoundFX, available from 01-June-2001 through 01-August-2001 in the Amiga Arena.

SoundFX is an editor for digitized audio-data (samples).

SoundFX is created modular and has a comfortable graphical user interface.

With SoundFX samples can be furnished with digital effects (unrivalled on Amiga) and modified.

Following is a list of several features:

  • More than 50 effects with many parameters and multiple possibilities to modulate them

  • Sound-synthesis (AM, FM, ...)

  • 3D-cube-parameter modulation (Mix, Equalize)

  • Effects e.g. Hall, Echo, Delay, Chorus/Phaser, Morph, Pitchshift

  • Operations e.g. Resample, ZeroPass (FadeIn/FadeOut), Middle, Amplify, Mix, DeCrackle, ConvertChannels

  • 2D/3D-Spectrum analysis

  • Very good resonance-filters and boosters

  • Works in mono, stereo and quadro

  • Sample-data are stored as 16bit in memory and on harddisk

  • AHI-player (for owners of sound boards)

  • System-conforming programming (tested with cyberguard, wipeout and blowup)

  • And lots, lots more

Additional information under http://www.sonicpulse.de - SonicPulse«

»Amiga Arena News
(03-June-2001)


Amiga Arena Full Version!

With permission of Frederic Laboureur the Amiga Arena presents the full version of "The Boss"!

With "The Boss" as a system manager you can start programs and applications with a mouse-click upon a simple button.

No unnerving and memory-eating menus or docks are used for the start.

A simple button alone is sufficient to start a program! Nonetheless "The Boss" is very easy to handle, flexible, compact, system-friendly and resource-saving and runs on every possible screen (Workbench, public and even its own) and is meant for all those for whom tools like Toolmanager or ToolsDaemon are too bulgy.

Features:

  • Completely user-configurable. No additional tools are needed

  • Font-sensitive GUI. Selectable screen and button fonts

  • Full support of OCS, ECS, AGA. Selectable screen modes

  • All functions on a single screen. No menus, icons or docks

  • No separate interface like MUI or Classact is needed

  • Asyncronously multitasking and extremely system-friendly

  • 16 colour WB palette to define the colours of the tool

  • Efficient use of memory - only little memory is used

  • Selectable types of buttons for drawers, scripts and executables

  • Unlimited number of buttons, use as many as you need

  • Small, optimized code makes the whole program system-friendly

  • Very easy to use and to configure

  • Supports drag & drop

Special thanks go to Kilian Servais for the German "Short Manual"!
