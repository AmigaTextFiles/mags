9. Emulator-News

This month Mathias "AmiDog" Roslund published a new version of his port of the Gamboy emulator GBE. Version 0.4.8b can use triple buffering and a problem with the memory.

This month has been published some new versions of the Amiga emulators for Windows-PC's WinUAE. The latest version is now 0.8.14 revision 3.

YaQuZa is the name of a new emulator, which emulates the Playstation on an Amiga. There is no working version up to now but is said to be published soon. The emulator will be published in a WarpOS- als well as a MorphOS- and a PowerUp version.
