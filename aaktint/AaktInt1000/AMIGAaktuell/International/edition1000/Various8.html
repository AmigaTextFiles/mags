software
---------------------------------------------------------------------------

Haage&Partner announced version 4 of their developer kit 'StormC'. Guests who had been invited to Haage&Partners annual summer festival gained some experience and were very enthusiastic about it. Meanwhile the project is in the beta stadium.
[http://www.haage-partner.com/news/news_20000828_d.htm]

DreamCDGuide V0.96 has been released. The program offers a graphical user interface for navigating through Aminet CD-ROMs recognizing them automatically.
[http://www.trogsoft.uklinux.net/site/products/downloads/dreamcdguide.lha]

DataM II, a new spreas sheet program, has been released as version 0.8 beta and may be tested by anyone who's interested in that.
[http://www.amclust-net.de/software/Kirsch.html]

OpusMI, an icon collection for DOpus 5.82 in GlowIcon style has been published in a final version.
[http://home.t-online.de/home/mason.home/zhomepag.htm]

The stock trading program Bonds has been released as version 1.78 beta.
[http://www.programmed-excelence.de/]

Diskmaster by Rudolph Riedel is available as version 2.5b7.
[http://home.t-online.de/home/rudolph-riedel/]

The anti virus program "VirusExecutor" has been published as version 2.05 beta.
[http://home4.inet.tele.dk/vht-dk/amiga/ve/VirusExecutor.lha]

The XFDmaster.library has been updated to V11.34.
[http://www.vht-dk.dk]

Jacek Rzeuski continued to work on the free source code of DirOpus 4. The latest version is V4.15.
[http://viper.pl/~opus/DOpus415JRbin.lha]

The successor of the MPEG-Players Amp, logically named AMP2, is available as demo version 5.
[ftp://ftp.amidog.com/pub/amp/AMP2pre5.lha]

Frogger has been released as version 1.62 beta.
[http://frogger.rules.pl/downloads.html]

The program Rainboot which was planned for beautifying the boot procedure is now availble as version 3 and can now be used for creating whole demos.
[http://phobos.spaceports.com/~sfa/downloads.html]

The MPEG Encoder Pegase has been released as version 1.8d.
[http://perso.pacwan.fr/kakace/pegase/]

The latest version of the printer software Turboprint is now 7.13 beta.
[http://www.irseesoft.de/tp_faqs.html#beta]

Again there's an update for the drawing program PerfectPaint bearing version #2.2.
[http://gothic.fr.free.fr/amiga/index.html]

Innovative announced fxScan 3.0 which shall have OCR as a new feature
[http://www.innovative-web.de]

The burner software MakeCD has been published as V3.2d beta 3.
[http://makecd.core.de/ftp/MakeCD32d_public_beta_3.lha]

Thomas Wenzel released his famous mp3 Player AmigaAMP as version 2.8 beta.
[http://amigaamp.amiga-software.com/Binaries/AmigaAMP28b8.lha]

The latest version of DirOpus 4, whose source code has been made public as V4.15a beta now supports New- und GlowIcons.
[http://viper.pl/~opus/DOpus4beta.lzx]

If you like the Windows taskbar you should have a closer look at Workbench2000 version 1.6.
[http://memory.overdrive.free.fr/Download/WB2000.lha]

The easy to use SMS program TaskiSMS has been published as version 2.0 and is now able to manage several SMS YAM alike.
[http://taski.com.pl/pub/taski/programs/TaskiSMS.lha]

The AOL Messenger AmigAIM, has been made availble in V0.9392 beta.

hardware
---------------------------------------------------------------------------

The PCI bus board "Mediator" is available now.
[http://www.elbox.com/]

In the British magazine "Amiga Active" you could read that for Cyberstorm/Blizzard PPC boards there shall be a PCI AddOn named "Predator" in development.
[www.amigaactive.co.uk/]

DCE announced that they were working on a PCI bus board named G-Rex. Additionally work on a Voodoo3 board was in progress.
[http://www.dcecom.de/pub/g-rex.pdf]

The company Merlancia Industries announced that they'd produce a mouse converter making it possible to use PS/2 mice on Amigas. A driver that supports up to 4 buttons and scroll wheels shall come along with the converter.
[http://www.merlancia.com/amiga/new.html]

A SCSI controller for the Ateo bus has been released.
[http://www.ateo-concepts.com/amiga/eframe.html]

If you like to do-it-yourself then you can build a digital output port for your Melody1200 with just eight parts.
[http://www.katodev.de/deutsch/ctdigio.html]

As you could derive from Blittersoft's website the paraglide module for the PicassoIV should be available soon.
[http://www.blittersoft.com]
