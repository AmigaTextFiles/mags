News about the AmigaNG

5. In a nutshell (by Oliver Tacke)

The first MPEG-Player which runs on the SDK has been released. Without optimizing the code it does 17 fps at 325x288 pixels on a Pentium 2/233 with Linux as host system.
[http://www.qdev.de/]

Patrick Roberts and Wayne Martin announced that they were working on a new standard archive format for the new AmigaOS namend "AmiPak". There are also plans to write the archive software for all common platforms.
[http://www.artixsoft.com/]

AMIGA published several documents concerning legal issues (copyrights, trademark usage , licencing, etc.) on their developer website.
[http://www.amigadev.net/index.php?sid=&subpage=home&function=legal]
