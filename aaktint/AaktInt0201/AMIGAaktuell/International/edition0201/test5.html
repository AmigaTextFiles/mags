5. Software test: System monitor »SnoopDOS v3.4« (by Klaus Moddemann)

Just in case you hadn't noticed, SnoopDOS34.lha is begging to be downloaded from Aminet. It offers quite a few extra features over those in version 1.7. The most important are:

-Support for more than 20 new system functions -Easy usage for all functions through a complete GUI -Custom formats for the output file, including a format editor -History function with scrolling -Outputs of ARexx scripts can be displayed -Commodity support -Support for Cut&Paste with Clipboard -AmigaGuide-help for all functions

Once you have downloaded the archive, installation is as simple as unpacking it and copying the contents somewhere on your hard disk. The program is started with the usual double-click on the icon, which opens the SnoopDOS main window. The title bar is resplendent with vital information like program name, author, version and hotkey. Underneath is the event window that displays all system functions as they execute. Below is the status bar, showing the current status of SnoopDOS. There are also some control buttons.

A click on "Setup" opens the preferences window. Here you can change the font, the default screen, logfile, hotkey and buffer for the display. In two string gadgets you can define the format for both the output window and the log file. If a manual edit seems too much, you can use the supplied SD-format editor. Open by clicking on "Edit" beside the string gadget. SD-format lets you change all options with Drag'n'Drop.

When you're finished, confirm the changes with "USE" to get back to the preferences window. With "Copy", you can transfer the settings to the other string gadget. Confirm with "USE" and you will be back in the main window.

With the "Functions" gadget, you can open a window to define the functions to be monitored. Once again, make your settings, confirm them with "USE" and then if you are happy with them use "Save settings" from the main window to save them permanently.

The other gadgets are quite self explanatory.

Using SnoopDOS is very easy. Start it and then start the program(s) that are causing problems. SnoopDOS gives very extensive information on the processes started by recalcitrant program(s). You can save this information into a logfile.

SnoopDOS is not only helpful for trouble-shooting, but also for deinstallation, because it shows which files are opened by a particular program.

To conclude this test: SnoopDOS is an easy-to-use program that helps you solve problems without making problems itself. The only exception I noticed was MiamiInit: With SnoopDOS running in the background, the whole system hangs when you start MiamiInit, making a reboot necessary.
