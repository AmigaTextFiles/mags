4. News from Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»The CD32 Games Install Kit v4.1 NG has just been released on the Airsoft Softwair web site <http://www.airsoftsoftwair.de/>. This update can only be used and downloaded by owners of the The Best of Airsoft Softwair Gold Edition CD-ROM.

In this version the compatibility of the inbuilt emulator has been greatly improved and some important fixes have been made. Owners of the The Best Of Airsoft Softwair CD-ROM can download this update now with the program Updater which is on their CD.«

»Because of many people asking about it, here's a more exact list, what has changed in the release of the Install Kit v4.0:

  • Install2001 supports RTG mode now for users of gfx boards

  • there's now a gadtools AND a MUI interface for the Install Kit

  • many new games can be installed / emulated

  • Install2001 uses now nearly half the memory than before; this increases the compatiblity alot

  • both interfaces have been greatly improved and are much user-friendly now

  • more settings can be configured now

  • the CD-ROM support has been significantly improved

  • the inbuild emulator supports now much more CD32-specific things

Additionally there were many bug fixes and other improvements of course. The CD32 Games Install Kit v4.0 is exclusively available on the The Best Of Airsoft Softwair Gold Edition CD-ROM which is available for only 29.95 DM from me.«
