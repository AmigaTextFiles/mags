Games section

3. News

  • Quake Deathmatch

`The Spirit of AMIGA' has organized for the first time a Quake Deathmatch on January 26th. If there is enough interest a clan could be founded and/or a dedicated server will be set up for play.

  • 'Ermentrud'

This graphics adventure (see last AA for preview) is making progress. The game engine is nearly finished. There has been additional inventory and multiple choice talks are close to reality. A new demo version will possibly be published soon. The film ´Ermentrud´ is making progress, too.

  • Redesigned homepages

The homepage of Blittersoft has a new design and an dedicated Amiga part in its online shop now.

Alternative-Games.de has a new layout, too.

  • 'Tales of Tamar'

News about ToT: the chronicle for the year 60 is finished.

  • Amiga Flame Top 10

On the homepage of www.AmigaFlame.co.uk the top ten Amiga games of the year 2000 have been voted. The first three are "Heretic II', 'Nightlong' and 'Earth2140' (in that order).

  • New adventure by X-Team Software

X-Team Software plans a new graphics adventure. Up to now only some screenshots have been published.

  • Trogladite needs developer

Trogladite Software needs developers for their projects. They also want a designer for their homepage: http://www.trogsoft.org

  • 'Another Commander' becomes freeware

'Another Commander' by Ragonsoft has been published as freeware. Download it for free at http://www.ragonsoft.org

  • PlayAmiga.de under construction

PlayAmiga.de, a well-known site about Amiga games, has dropped off the net for a redesign. After the redesign it´ll be called Play!Amiga.

  • Start of 'Star Empire II'

On 31.1. the game 'Star Empire II' started. This text-based game is played via e-mail, so it is possible for users without flatrate to join the game.

  • 'Ultra Violent Worlds' published

'UVW' is Blittersoft's first Hybrid-CD, which contains an Amiga version as well as the PC or Mac version of the game. The benefit is that one can get an Amiga game in a shop for IBM-compatible computers. `Ultra Violent Worlds` is developed by Vorlan and is a 2D shoot'em'up with vertical parallax scrolling for up to 2 players.

Originally, `UVW' was published in ´98 but that version had a lot of bugs which have been removed now.

  • New Crusader-Clone

Extraterrestrial Developing has announced a new Crusader clone named 'Silencer'. There is already a demo version 0.2b. This runs on a 640*480 screen with 256 colors. It needs at least 16 MB RAM and 15 MB space on the harddrive.
