Various Amiga news

5. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software
---------------------------------------------------------------------------

For users of OS3.5 the second BoingBag has been issued. But it is of no benefit for users of OS3.9.
[http://www.amiga.com/3.5/download/BoingBag2a.lha]

For registered users of AWeb 3.x there is a new patch.
[http://www.amitrix.com/aweb3x34pch.lha]

The mp3 player AmigaAMP at last has been issued in its final version 2.8.
[http://www.amigaamp.de/Binaries/AmigaAMP28.lha]

For the Workbench replacement Scalos a plugIn has been written to offer compatibility with OS3.9. Besides that the icon system and the volume gauge have been improved. A beta version of V1.2 can be downloaded as well.
[http://scalos.stratum.org/products/scalos/]

EasyPlayer has been released in version 1.04.
[http://www.coli.uni-sb.de/~rade/soft/e/EasyPlayer.lha.bin]

The program MainPrefs, which simplifies the handling of the preferences of the Workbench (3.5 and higher), has appeared in version 2.02.
[http://private.addcom.de/martin.elsner/deutsch/anmai.html]

The ICQ client StrICQ is available now in version 0.1734.
[http://www.multimania.com/aubioane/programmes/stricq.lha]

AmigAIM, the (unofficial) AOL Instant Messenger for the Amiga has been released in version 0.9407.
[http://www.missingpiece.com/AmigAIM/index.html]

EZP@gerNG Classic is available in version 1.6.
[http://www.sfxsoft.de/de/news.html]

The final version of the mpeg player AMP2 has been released.
[http://ftp.amidog.com/pub/amp/AMP201-lha]

For the Aldi/Medion/Tevion MD 9310 graphic tablets a new version of the driver FormAldiHyd appeared, and that is version 1.3.
[http://www.platon42.de/]

The scanner calibrating software ICS is available as version 2.01.
[http://www.ics.coloraid.de]

The IRC client AmIRC has been treated with a higher version number: 3.5.6 beta.
[http://ftp.ch.vapor.com/amirc/amirc_356beta.lzx]

There is a new version of Samba and it has the version number 2.0.7.
[http://ftp.wustl.edu/~aminet/aminet/comm/tcp/samba_2.0.7.lha]

PageStream import filters for Word8 (and Word97/2000) and for AmigaWriter have been released.
[http://www.muenster.de/~tomjoad/]

To the SMS mailer SMSEngineer an update has been released.
[http://blitz-engineers.virtualave.net/download/SMSEngineerMUI.lha]

JST, the disk-to-hd installer for many older games, has appeared in version 4.4.
[http://jffabre.free.fr/amiga/patches.html]

DirOpus V4.15e beta has been released.
[http://viper.pl/~opus/DOpus4beta.lzx]

The virus detector Save has been published in version 14.8 and is ready for download.
[http://www.vht-dk.dk/]

The browser Voyager has appeared in a new beta version: V3.3.65. (note of editor: now 3.3.79 as well)
[http://v3.vapor.com/]

For EASys a patch for version 4.84 appeared.
[http://www.eternity-computer.de/easys/bin/EASys!_patch_484.lha]

Perfect Paint appeared as version 2.4.

[http://gothic.fr.free.fr/amiga/]

An alpha version of the new eMail client Simple Mail has been released.
[http://www.schlawack.net/simplemail/simplemail.lzx]

VHI Studio has appeared in version 5.4.

[http://www.innovative-web.de]

SoftCinema has been published in version 0.11 and now also supports DIVX and 3ivX.
[http://www.amiga.pl/softcinema/]

The Macintosh emulator Shapeshifter has been released in version 3.11.
[http://iphcip1.physik.uni-mainz.de/~cbauer/ShapeShifter3_11.lha]

The stock market program Bonds has been issued in a new version.
[http://www.programmed-excellence.de/]

If at least 300 new users register for Picasso96, then official Picasso96 drivers for the Mediator PCI bus board will be created.
[http://picasso96.cogito.de/]

As was published in AmigaNews Forum, Hyperion will release only Warp3D drivers for the Voodoo3. It is extremely difficult, if not even impossible, to obtain the necessary information from the manufacturers.
[http://www.amiga-news.de/cgi-bin/UltrBoard/UltraBoard.pl]

The editorial staff of Amiga Future is looking for support.
[http://www.amigafuture.de/jobs.html]
