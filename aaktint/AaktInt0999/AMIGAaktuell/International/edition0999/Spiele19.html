Regarding the emulation speed: instead of text a number can represent it.

1 = :-] = runs in full speed
2 = :-) = stutters a bit, but can be played without problems
3 = :-| = recognizeable stuttering and can be played with patience, only
4 = :-( = that slow, that the game does not really make fun
5 = :-[ = the game is too slow for playing

Intermediate values (2-3) are allowed.
