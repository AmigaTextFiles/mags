16. Mac corner

Preface

Hello, welcome back in the Mac corner !

Who already knows the Mac corner from the last edition of the Aakt int might have noticed, that the expression "Test" at some games has changed to "Introduction". The reason for that is: Because the number of games I have for the Mac is limited, the introduction of a game, can be based on these bases:


Full version -  the game, which is available as original has been played extensively and so we can give you detailed informations about the content as well as for the emulation speed. 
   
Demo version -  only the demo version of the game was available. Due to this there can be no real evaluation, but an evaluation about the emulation speed. 
   
Others -  The introduction is based on a different basis, like an old Mac-magazine, the Mac comp list or another game description. This has the disadvantage, that there cannot be an evaluation about the speed. 
   

Of course, this differentiation has its own problems. When testing a demo you cannot tell, in how far it shows the fun of the full version. Also the test of an old Mac-magazine can be quite irritating as this example of a test of "Labyrinth of Time" shows:


MacUP 9/94 :  "Idea, graphics and sound are that impressive, that the game deserves it to be named with Myst or Journeyman Project. 
  EVALUATION: 6 of 6 points.  
   
MacMagazin 11/94 :  "Mystery after mystery, confusion after confusion, tiring and boring!"
 EVALUATION: 2 of 5 points. 
  

But I can fulfill the job of the Mac corner, which is to give an overview about the world of Mac games and to show unknown titles.

Have fun reading it!
