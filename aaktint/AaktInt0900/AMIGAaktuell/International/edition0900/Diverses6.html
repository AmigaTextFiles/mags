6. News in Brief (by Oliver Tacke)

Software

The scanner colour calibration software ICS version 1.10 has been released.
[http://ftp.wustl.edu/~aminet/aminet/hard/misc/ICS.lha]

The internetsoftware StrICQ version 0.1731 is now available.
[http://www.multimania.com/aubioane/programmes/stricq1731.lha]

Version 1.4 of the downloadmanager Charon has been released.
[http://tesla.rcub.bg.ac.yu/~antony/Charon/Charon.lha]

The controversial Napster client Amster version 0.7a has been released.
[http://wuarchive.wustle.edu/~aminet/aminet/comm/tcp/Amster.lha

The initialization program of the delfina soundcard version 2.1 has been released.
[http://www.student.oulo.fi/~jperaaho/software/DelfInit.lha]

The drivers for the IDE ports of the X-Surf network card are now available for free download.
[http://www.jschoenfeld.com/xsurfide.lha]

UnLZX version 2.12 is now available for 68k and PPC.
[http://home.t-online.de/home/LucyG/UnLZX2.lha]

The stock program Bonds is now available in version V1.75d.
[http://www.programmed-excellence.de]

Users of Voyager had to read the message"This betarelease has timed out!" since August 1st. The new update on V3.2.13 fixes this bug.
[http://www.vapor.com/download/pub/voyager/v3_3213_beta.lzx]

The group TFA will continue development of the ASM-One. Version 1.44 is already available.
[http://www.eurobet.nl/users/ftp/asmonev144.lha]

The drivers for Mustek scanners, BetaScanMustek version V0.44 - 1.0.92 is now available.
[http://ftp.wustl.edu/~aminet/aminet/hard/drivr/BetaScanMustek.lha]

DeliTracker will not be under development is is now freeware.
[http://ftp.wustl.edu/~aminet/aminet/mus/play/DeliTracker232.lha]

Various updates have been released from PerSuaSIVe SoftWorX: akJFIF V44.82, akPNG V44.82, akTIFF V44.82 and SViewIV V9.10.
[http://www.ar-kleinert.de/]

The MP3 player AmigaAMP bet #7 of version 2.8, which now supports AIFF and WAV via sndfile.library has been released.
[http://amigaamp.amiga-software.com/]

Pagestream version 4.0.8 has been released and supports the PDF export correctly, now.
[http://www.haage-partner.com]

WHD-Load, which allows saving many old disk games on hard disk is now available in version 12.0.
[http://www.whdload.org/whdload/WHDLoad_12.0_usr.lzx]

The AOL-Instant messenger clone named AmigAIM beta 0.9385 has been released.
[http://www.missingpiece.com/AmigAIM]

MagicMenu version 2.30 has been released and now there is also an own website.
[http://www.magicmenu.de]

The virus killer Safe version V13.9 is now available.
[http://www.vht-dk.dk/]

Meanwhile AMarquee raised to version 52.0.
[http://hem.passagen.se/hparting/archive/AMarquee52.lha]

The ICQ client StrICQ version 0.1732 is now available.
[http://www.multimania.com/aubioane/programmes/]

Again, there is an update for Easys, which fixes some problems on the installation. In addition service pack 2 has been released.
[http://eternity.amiga-software.com/easys/]

Version 2.1 of the drawing and picture processing program PerfectPaint is now available. Most important novelties are e.g. the loss of AGA support as well as more ARexx commands.
[http://gothic.fr.free.fr/amiga/index.html]

For the program ProStationAudion new plugins have been released: DynExpander, RingMod, AutoFilter, L-R, AAM-1, APS-1.
[http://www.audiolabs/.it]

The finance program UltraAccounts version 5.1 has been released.
[http://www.cix.co.uk/~rsmedley/amiga/ultraacc.html]

As an sister product of the HTML viewer SimpleHTML a HTML.datatype (V0.11) has been released.
[http://home.t-online.de/home/sebauer/html-dt.lzx]

The assembler ASM-One version 1.44 Rev 433 is now available and got numerous bug fixes.
[http://www.euronet.nl/users/jdm/ftp/asmonev144.lha]

Digital Almanac Version 3 has been completed, now. The full version costs 80.00 DEM (approx. 40.82 Euros), an update from version 2 only 50.00 DEM (approx. 25.51 Euros).
[http://members.tripod.de/achimste/]

The webbrowser Voyager version 3.3 Alpha 35 is now available.
[http://v3.vapor.com]

For CyperGraphX version 3 there exists a driver for graphics boards with PCI Virge chip, which can be operated using the PCI board "Mediator". A driver for Voodoo3 boards is already under development.
[http://www.vgr.com]

A replacement for the original text.datatype has been released and bears version number 44.5. This one allows to mark and copy texts.
[http://home.t-online.de/home/sebauer/]

Hardware

According to Elbox the announced PCI busboard "Mediator" is in production and will be shipped within a few days. The company rejected rumours, that only graphics board with up to 8 MByte of graphics memory could be operated. In the contrary, Mediator offered 4 GByte of address space.

The MAS player, a mp3 player for the parallel port is now available in a "Pro" version. This allows to play a 192kbps coded mp3 file in highest quality on an 68000 Amiga with 7 MHz.
[http://home.snafu.de/dcr/]

Miscellanous

Until late September it is possible to participate the petition for Village Tronic. The goal is to make authorized repair services the service documents for VillageTronic products available.
[http://www.mc68k.btinternet.co.uk/vt]

GP-Software allowed to offer the official Directory Opus4 manual as Amiga Guide file for download. Version 4 was made open source including its source code some time ago.
[http://viper.pl/~opus/files/DopusManual.lha]

In the contrary to some rumours, AsimWare does not cancel the Amiga support. The AsimCDFS would not be further developped, but MasterISO.
[http://amiga.asimware.com/]

Crystal Software & Electronics Ltd. cancel their support for the Classic Amiga and also abort running project. But the company promisses to support the new Amiga and supports the Amiga users with the online magazine "Amiga Survivor" on http://welcome.to/amigasurvivor.
[http://www.csande.co.uk/news.html]

Rumours

According to rumours AMIGA should have said on the AmiWest 2000, that there will be no more support for OS 3.5.

It has been said, that the BoXeR developerteam of Joe Torre would work on an Amiga Classic laptop design.

According to rumours the next graphic chip generation of Matrox, the G800 should be used for the AmigaOne.

According to rumours AMIGA wanted to help the current Amiga users to buy PPC boards by supporting developers (to produce?).
[http://www.myamiga.com/amiga/news/amiwest/amiwest.html]

According to rumours Stefan Burstroem works on version 2.3 of IBrowse. It should also support AmiSSL again.

According to rumours another PCI busboard, named Predator, will be made available in October. This board shall be operated using the Phase5 PowerPC boards and offer in addition to PCI slots an AGP slot as well as SD-RAM sockets.
