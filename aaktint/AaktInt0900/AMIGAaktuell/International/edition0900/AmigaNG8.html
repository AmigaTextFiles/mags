News about the AmigaNG

8. News in Brief (by Oliver Tacke)

An experimental version of the programming language Rebol has been released for Elate/intent.
[http://www.rebol.com/xpers/xpers.html]

The first text editor with graphical user interface for Elate has been released. Its name: Phat.
[http://members.xoom.com/PhatRepublic/prods.html]

Now, the Amiga SDK also runs on RedHat Linux 6.2. An additional archive, which has to be unarchived to /usr as well as this command line before starting the SDK is required:

export LD_LIBRARY_PATH=/usr/amiga/lib
[http://www.amigadev.net/developers/cvs/snapshots/amiga-ncurses.tar.gz]

A first file manager for the Amiga SDK, ACommander V0.1 has been released and makes work easier even with its few functions.
[http://www.ablaze.sk/]

Since August 23, KDH ships the AmigaNG developer system d'Amiga for 2098.00 DEM (approx. 1070 Euros).
[http://www.kdh-datentechnik.com]

Now, AMIGA takes ideas because they know, that there's much creativity in the Amiga community.
[mailto:developer@amiga.com]
