Amiga PowerPC

Amiga PowerPC

1. News In Brief (by Oliver Tacke)

AHI V4 for MorphOS as beta version released.
[http://www.lysator.liu.se/~lcs/files/ahi/beta/ahipre4-morphos.lzh]

New versions of the WarpOS picture datatypes by Oliver Roberts for the JPEG and PNG formats have been released: WarpJPEGdt V44.12, WarpPNGdt V44.7
[http://www.nanunanu.org/~oliver/]

Version 0.08 Rev4 of the PlayStation emulators FPSE has been released. It requires WarpOS V4+, CGX V3+ and 32MB RAM as well as a PSX BIOS.
[ftp://ftp.amidog.com/pub/emu/fpse008r4.lha]

Version 24.8 of the SuperView-PPC module has been released.
[http://www.ar-kleinert.de]

New versions of the emulators WarpMAME, WarpNeoMAME and WarpMESS are available.
[http://www.powerup.com.au/~cward]

New versions of Frogger as well as for WarpOS and powerUP as for MorphOS are available, now.
[http://frogger.rules.pl]

The ANSI-C compiler vbcc version 0.7d for MorphOS has been released.
[http://devnull.owl.de/~frank/vbcc07d_MorphOS.lzx]

WarpOS users can now get their hands on version 3.86 of the mp3 encoder Lame.
[http://www.kolumbus.fi/jami.laakonen/ppc/index.html]

Digital Almanac version 3 is ready and now also supports the PPC (WarpOS). The full version costs 80.00 DEM (approx. 40.82 Euros), an update from version 2 costs only 50.00 DEM (approx. 25.51 Euros).
[http://members.tripod.de/achimste/]

The MorphOS team created a public developer mailing list.
[http://www.morphos.de]

An installation guide for S.u.S.E Linux 6.4 PPC on PPC Amiga was created by Michael Heider.
[http://home.germanynet.net/101-109037/index.html]

The iMac emulator iFusion (formerly FusionPPC) will be available for public tests, soon.
[http://www.microcode-solutions.com/]

Oliver Tacke <O.Tacke@surfway.de>
