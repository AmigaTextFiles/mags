Pictris

Another new game from APC&TCP is "Pictris" a puzzle for tinkerers. Pictris is delivered in a standard CD case on a CD-R. The features of Pictris are 240 different puzzles in 3 levels, nice background pictures, multi-channesl modules and many more. Pictris requires at least an Amiga with AGA or graphics board, 1.5 Megabytes of free Chip-RAM as well as 4 Megabytes Fast-RAM.

After inserting the CD-ROM you can set various preferences like e.g. the screen mode, playing frequency for the music modules and many more. Of course, you can install Pictris also to the hard disk. Those, who don't want to do that, can start it directly from the CD-ROM. Pictris supports many different languages, you can set up. If everything is set up, you can choose to start the game automatically or not. So we can begin, now....

Pictris starts on a normal Amiga screen with 256 colours. After a short introduction you find yourself in the main menu. Here you can control everything like watching e.g. highscores, start a new game or load an old one. Of course, we choose the start of a new game. After pressing the button there is another screen, which asks for choosing a lap. First, you can only choose lap 1. Now you find yourself in the game itself. While listening to the relaxing music you can start tinkering. Here you can see, who is a bright mathematician, because there are hints for finding the right box, which are coded in numbers. To make it short: The try-and-error method doesn't help in Pictris.

Result: I'm sure, Pictris is a wonderful tinkerer's game for making a long evening short. It has an easy design, there is relaxing music in the background - the right stuff for thinking. But those, who like fast action, should better not choose Pictris, because this is not the genre of the game. All in all, Pictirs is a well-done game. Depreciatory are the background pictures and -music, because, they were mostly just downloaded from the Aminet or similar sources. It would have been better, if the author made something himself. But this game is not depending on graphics and music. This is why the fun is evalued double, while all other points are evalued simple.
