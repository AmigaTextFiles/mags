News about the AmigaNG

7. »Wildfire«: First screenshot of the AmigaNG successor and additional news

Andreas Kuessner informs:

»1. New Wildfire homepage installed

2. Wildfire successor (codename: Taifun) for AmigaOne under development

3. Email addresses of the WF developers have changed

Hello Amiga community,

1. From now on the homepage of WK-Artforts can be found on this address www.wk-artworks.de. There you can find e.g. informations about the program Wildfire7PPC and the Movieshop operator.

2. Since the show in Neuss the development of the Wildfire's successor for the new Amiga (AmigaOne) is under development. A first screenshot of the running program is on the homepage mentioned above. The project name is `Taifun` - this is also the name of the link on the homepage. Please address questions, ideas and request about the program to Andreas Kuessner <ak@wk-artworks.de>

3. The eMail addresses of the Wildfire/Taifun/YAFA developers have been changed. Please use these addresses from now on:
