3. News from Airsoft Softwair

Airsoft Softwair (http://www.airsoftsoftwair.de) informs:

»We just put version 3.8 of the CD³² Games Install Kit on the Airsoft Softwair homepage. With this version you can install some new games, now. In addition the subsystem Install2001 got major improvements.

In addition some archives with various localizations for NewInstaller v1.5 have been made available. For German users, this might not be interesting, because the German catalogue in the archive was already up-to-date.«
