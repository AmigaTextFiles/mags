9. New listed AmigaNG software (byn Thorsten Schoelzel)

These are the latest AmigaNG releases resp. supplements:

  • Kill: Abort process by signal

  • Filedialog: Filedialogue tool

  • Phat Edit: Graphic text editor

  • Rebol: Programming language

  • Easier-Emacs: Builds editing emacs operations

  • Nlookup: Searches the internet for host names or IP addresses

  • AmiFlag: AmigaVP version of the "Flag and Bombs" game

  • Clip: Clipboard tools

  • Ewinease_b035: Creates GUI-apps in easy C

  • Exev 1.0: AmigaVP "exec" command

  • Pix2PNG: Converts pixmaps to PNG

  • Povelate_b: POV-Ray 3.1g (binary only)

  • Povelate_s: POV-Ray 3.1g (elate source only)

  • Python 1.6: Python 1.6

  • Regina: GNU-Rexx interpreter

  • Which 1.0: AmigaVP "which" command

The download addresses of the mentioned programs can be found in the AmigaNG software list.
