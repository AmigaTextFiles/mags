AmigaOS 3.9

It's available ... AmigaOS 3.9, the update for OS 3.5, which can also be installed directly on top of an OS 3.1 system. (However it needs Kickstart 3.1 ROMs to operate!)

Shipping has begun! Get your personal copy, before there's a delivery bottleneck. Order now, when you want to receive it right in time for yourself or for one of your beloved ones under the christmas tree.

Features of AmigaOS 3.9

  • ACTION - AVI & Quicktime Player

  • AMPlifier - MP3/WAV/AIFF Player

  • MPEGA.Library - Software MP3 Decoder

  • PlayCD - New CD Player

  • CacheCDFS - CD ROM Support (*3.5)

  • AWEB 3.4 Special Edition - Internet Browser

  • Internet / Network Access - TCP/IP Stack

  • AmigaMAIL - E-Mail Client (*3.5)

  • AmiDOCK - Program Start Bar

  • RAWBInfo & several other tools

  • Automatic file type recognition

  • Improved Libraries & System Tools

  • Find & Clock

  • IomegaTools - tools for Zip and Jazz drives

  • New Shell & many other tools

  • Unpacker Tool

  • XAD Library - Universal Decruncher System

  • Picture Datatype - New Improved Version

  • Improved Prefs

  • WarpOS - New Version

  • WarpOS prefs

Including the features already known from OS3.5:

  • Glow Icons - New Icon Style

  • HDToolBox - Harddisk Toolkit

  • NSDPatch - Support for Harddisks >4GB

  • HDwrench.library

  • Text-Editor-Gadget

All information might change without notice!«
