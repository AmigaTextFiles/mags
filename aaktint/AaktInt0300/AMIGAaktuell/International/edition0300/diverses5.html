Various Amiga news

5. News in Brief

  • The Workbench replacement Scalos has been released as Freeware. The revision 1.2b can be downloaded on http://www.satanicdreams.com/scalos/index.html.

  • Some days ago further updates of the graphics card driver software CyberGraphX v4 have been released. The current release 4.2-r2-pre5 can be downloaded on http://www.vgr.com/v4/.

  • On ftp://ftp.haage-partner.com/3DWorld/Warp3D/ the new revision of the 3D-graphics card driver software Warp3D can be downloaded.

  • Soon, a new chat software named AmChat will be released as freeware.

  • Soon, a new revision of the WWW-browser Voyager will be released, which removes especially bugs and improves the JavaScript support. Furthermore there will be used APDF as plugin. The update of the VFlash plugin will be used a multiple times faster than the previous revision.

  • Meanwhile Village Tronic has two PCB-layouters and hopes, that the announced Voodoo-3D-addon "ParaGlide" for the graphics board Picasso IV can be shipped in late April.

  • Aliendesign has problems with the mixer with the announced Zorro sound card Repulse and presents to possible solutions to the potential customers on http://www.aliendesign.net/repulse/.

  • As each year there will be the AAA Awards, again. Further informations are available on http://www.aaa-awards.org. There you can make nomination suggestions.

    By the way, the organizers are looking for a usergroup, that wants to award a German AAA Award.


  • Michaela Pruess informs: »After the last Amiga oriented search engine closed with the shutdown of AmigaCrawler there is a REAL search engine with own database online.

    On http://amiga.ac every Amiga user can report his pages in the web (and also any other Amiga page, he knows). After a few days thousands of pages were reported and scanned, so the search functions are open since today (14 days earlier as planned). The data pool is still incomplete, but with each new hour it grows, what shows, that a search engine is not only wanted, but really required.«


  • Andreas Wendorf informs: » From now on wARPgATE can be found on www.w-9.de. Daily updated news & informations around warpOS & warp3D can be downloaded any time. In addition there is a special forum online, which is especially for all questions, problems and suggestions about the WOS/W3D.«

  • Amiga Arena (http://online-club.de/~ARENA/) informs: »Due to the positive replies of the Amiga users on the Amiga Arena action I was able to prolongue the special prize action with GeoWorld up to March 31, 2000! I want to thank all useres, who show the support and allow the development of software with their registration!«
