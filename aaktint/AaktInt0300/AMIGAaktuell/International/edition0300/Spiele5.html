5. Previews

Desert Racing of BarDos

The game 'Desert Racing of BarDos' came directly from nowhere. So I was very curious, when I found a professionally designed preview CD-ROM in my letterbox. The demo version of the game is approx. 15 Megabytes of size and can be run either directly from the CD-ROM or from hard disk. As the name of the game makes you suspect this game is a car racing game. But no common one, because this race is directly in the desert. The graphics of the game are completely in 3D and looks professional. There are many animations and the grahics are colourful. Unfortunately the only thing lag behind is, as it is very often, the controls. Currently it is quite impossible to play the game fairly, because the controls are too uncomfortable (you are very quickly attached to hills etc.). But if there are some things changing here I think, 'Desert Racing of BarDos' might become a real hit. The demo version requires at least AGA, an 68030 processor with 25 MHz as well as 2 MB ChipRAM and 8 MB FastRAM. The demo version will be available in the Aminet, soon.
