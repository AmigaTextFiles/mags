AMIGA

4. News in Brief

  • According to the Tao-Website http://www.tao-group.com/2/tao/about/custom.html AMIGA will use the Java Virtual Machine "intent (tm)" for the new Amiga.

  • The "easter egg" hidden by Olaf Barthel in AmigaOS 3.5 was found! Various secret texts were found shortly after the release of the operating system, but the maybe most interesting message can be found like this:
    1. Select "About..." in the Workbench menu. 
    2. Move the window for an odd number of pixels to the right and down. 
    3. Type "Who?" (without the quotes) into the active window. 
    4. Click the "Ok" button.
