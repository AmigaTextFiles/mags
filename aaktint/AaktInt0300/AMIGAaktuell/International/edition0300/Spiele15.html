Games section

15. HD-Installer for old games (by Ingo Schmitz)

Preface

In this section I want to introduce games, which can be installed on hard disk using programs like JST or WHDLoad, even if it was not planned originally. I tested some installers and got it running the choice is very limited I want to start a summons. If you got an NDOS-game on hard disk using the help JST or WHDLoad, you can, if you like, write a test report. It is important to mention, that the quality of the games is irrelevant (sometimes the games are as old as the hills). The evaluation should be about the game running on hard disk, if there are limitations etc., so everythings technical.
