3. New listed PowerPC software

These are the latest new entries:

  • Doggy: LightWave displayer for StormMesaPPC (WarpOS)

  • Frodo: C64 emulator (WarpOS)

  • measureconpup: Computing context switches (PowerUp)

  • Nostalgia: Multi emulation system (WarpOS)

  • Space: Space shooter (WarpOS)

  • UnLZX2: LZX unarchiver with new arguments (PowerUp, WarpOS)

  • WarpPNG: PNG Datatype (WarpOS)

  • WarpQuake: Quake port (WarpOS)

In addition a number of more or less exoctic converter tools was put into the Aminet. They will be listed in the next edition.
