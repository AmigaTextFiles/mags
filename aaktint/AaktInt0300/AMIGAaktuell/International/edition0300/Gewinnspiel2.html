And now, this is everybody's new chance for a prize:

Frank Weber gave his Programm CDCat in the brandneuw revision 3 for the competition to us:

»CDCat Release 3 (MUI) is the PPC/68k CD-ROM-catalogizer for the Amiga. In addition to that you can use it also for archiving all hard drives, floppy disks and directories (e.g. with many graphic files).

It is also quite valuable for Aminet-CD's because age, size and archive descriptions are considered. CDCat offers a super fast search mode (PPC; 68k: 85% inline assembler!), in addition you archived CD's can be viewed by the browser without put into the drive.

CDCat is also able to unarchive, view pictures, etc. But you can still configure any program.

More features:

  • XPK support
  • notepad for management of found files
  • DiskIDs for quick identification of the CDs, etc.

A demo version of the program is available in Aminet. The current revision is 3.0. More informations regarding CDCat Release 3 can be found on the support page http://www.amigaworld.com/support/cdcat

New with the revision 3 is e.g. the PowerPC support, the catalog manager for the reorganisation of the catalogs, filter functions, etc.
