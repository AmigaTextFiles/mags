6. Complaint about the PowerPC Software List (by Stefan Funke)

Unfortunately I am the one complaining. Lately there have been more and more requests for new software entries in which the URL, name, type, status or brand of the program are missing. This makes my job quite difficult and is really time-consuming. For this reason I unfortunately could not enter all reported programs to the list!!!! I'd like to call upon all developers, programmers and users, to please fill in all boxes of the registration form completely!!!! In the future, if a form is not made out completely, I cannot guarantee the acceptance of the program into the software list. Furthermore I'd like to point out that the PPC software list contains only PPC programs which are meant for the AmigaOS. So please no more reports about Mac or Windows programs!!!!!

Thank you,

Stefan Funke <ppcworld@haage-partner.com>
