Various Amiga news

7. In a Nutshell (by Oliver Tacke)

---------------------------------------------------------------------------
Software (commercial)
---------------------------------------------------------------------------

The webbrowser "Voyager" has been released in a new beta version, it again needs a keyfile for running, which only registered user have.
[http://vapor.meanmachine.ch/voyager/v_33_beta79.lzx]

For "AmigaOS 3.9" the first updates are released. There now is the ReAction version of GenesisPrefs, and furthermore there are updated locale files.
[http://www.amiga.com/3.9/]

The German localisation for "ImageFX 4.1" has been released.
[http://www.haage-partner.com/d.htm]

For "Pagestream 4" as well there is now a German localization.
[http://www.haage-partner.com/pgs/news-20000712-d.htm]

For the graphics processor "ArtEffect" (Version 4) a new colourful toolbar came out.
[http://www.haage-partner.com/arteffect/ae_d.htm]

For the PCI bus board "Mediator" an update of the software to version 1.8 has been published. Among others it contains version 1.1 of the Voodoo.card driver.
[http://www.elbox.com]

Thorsten Schoelzel (amiga-freak@t-online.de) writes: »Because for a long time nothing new was heard about the program "AmiATLAS", I asked the programmer Andreas Regul for a statement: "It is correct, during the last months there was silence about AmiATLAS. But now we found someone, who will go on developing the program. Therefore I am confident, that some day a version 6 will be released. But I do not know anything definite." If someone wants to assist Andreas and the new programmer with the further development of this ingenious program, he may please write to amiatlas@gmx.de «

---------------------------------------------------------------------------
Software (non-commercial)
---------------------------------------------------------------------------

A new version of the graph tablet driver "FormAldiHyd" has been published with the version number 2.0 beta. Experimental assistance for the WacomIV tablet is built in, too.
[http://www.platon42.de/platon/files/util/FormAldiHyd20beta.lha]

The virus scanner "Safe" appeared as version 14.8.
[http://home4.inet.tele.dk/vht-dk/amiga/div/Safe.lha]

The Disk-to-HD program "JST", with help of which even older disk games can be installed on harddisk, came out in version 4.5c with several bugfixes.
[http://jffabre.free.fr/amiga/bintools/LATEST/jst.lha]

The free version of "DirOpus 4" is continuously developed on and has been published in a new beta version.
[http://dopus.amiga.pl/files/DOpus415ibeta.lzx]

The eMail client "SimpleMail", which wants to be the successor of YAM, has been released as alpha version 0.4.
[http://simplemail.sourceforge.net/download.php?what=simplemail.lzx]

The Palm desktop "Spitfire²" has been published as version 1.5.
[http://www.windsor.igs.net/~torchia/spitfire2/Spitfire2.lha]

For the Amiga version of NetBSD several new drivers have appeared. More information you will find on the website.
[http://www.de.netbsd.org/Ports/Amiga/]

The program "WB2000", that imitates with striking resemblance the Windows taskbar upon the Workbench and has several additional functions, appeared in a new version.
[http://memory.overdrive.free.fr/Download/WB2000.lha]

The ICQ client "StrICQ" has been released in version 0.1735 and comprises as bugfixes the removal of the errors concerning the chat and download windows and the missing ICQ florets.
[http://www.multimania.com/aubioane/programmes/stricq.lha]

For web designers the useful program "SiteWay 1.0" exists, containing several helpful tools as well as a syntax checker.
[ftp://de.aminet.net/pub/aminet/comm/www/SiteWay.lha]

For route planner "AmiAtlas" there is a free map for Italy available at Amiga Arena.
[http://online-club.de/~ARENA/]

A new MPEG video player with the name "RiVA" has appeared. It supports as well AGA as Picasso96 and CybergraphX.
[http://ftp.wustl.edu/~aminet/gfx/show/RiVA.lha]

Version 2.1 of the helpful Workbench tool "MainPrefs" has been published.
[http://www.martin-elsner.de/]

The program "PlayGUI" has been released as version 3.1.
[http://narr.tsx.org]

The "opendivx.library", a shared library for 68k and WarpUP has been published.
[http://www.projectmayo.com]

SoftCinema appeared as version 0.12 beta and contains a clearly improved OpenDivX decoder.
[http://www.amiga.pl/softcinema/bin/SoftCinema_012b.lha]

The G3/G4 boards announced by Elbox will presumably appear not before the 3rd quarter 2001.
[http://www.vgr.com/cybergfx/]

Miscellaneous
---------------------------------------------------------------------------

According to Blittersoft the iMac emulator "iFusion" is in the final testing phase and is soon to be released. Microcode Solutions confirm this and came up with screenshots.
[http://www.microcode-solutions.com]

At Amiga you now may order several accessories, as there are T-shirts, stickers, etc.
[http://www.amiga.com/amigamall/]

The German version of the ImageFX manual has been published and contains as well the general manual as the supplements.
[http://www.haage-partner.com/d.htm]
