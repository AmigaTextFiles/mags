9. Hardware test: printer switch Okano CTP21C (by Axel Teichmann)

»   Tested device: OKANO Compact Auto Switch  Item no.: TS 9015-2 5297400  Type: automatic printer switch for 2 computers / 1 printer  Bought in / price: Galeria Kaufhof Hannover / 34.95 DEM (approx. 17.83 Euro)  Tester: A.Teichmann@gmx.net  Test duration: 1 day  Used hardware: Amiga4000/040 OS3.0 25MHz / 16 MB Fast RAM / 2 MB   Chip RAM   Intel Pentium II 200MHz, Canon BJC 610,   HP Laserjet 4l     Installation:

The device is easily connected with two 25 pin cables to each computer. The printer is being connected to the output of the switch. That's it the device doesn't need any external power supply.

Design:

The case has a quite "cheap" look and is much too big for the used technic inside. The device itself has a small, blue button, which is for manual input selection and is unlikely to be used.

Documentation:

The documentation is just a 4.3 inch x 5.85 inch piece of paper, which lists the features of the device.

According to the documentation the device is supposed to have DIP switches, but I didn't find them. Using them should set the "Time Out" (maybe this is true for another model of the printer switch).

Practical test:

After connecting the switch to the printer (first the BJC 610, then for another test the HP LaserJet 4L) with both computers some test printouts were made, which the switch managed automatically and switched as expected. The printout is as fast as usually, but it is no longer necessary to switch the two computers to the printer manually.

There weren't any problems with printer spoolers. (e.g. Printman by Stephan Rupprecht <stephan.rupprecht@primus-online.de>)

Positive:

It ran at once and didn't have any problems, I was afraid of before the test. Due to the fact that a new printer usually needs a bi-directional switch these days (which are quite more expensive, e.g. Highscreen fourfold Auto Data Switch (test of this device follows)), I expected the switch not to work, what might be true for Windows only due to the printer drivers, because there are printers, which want a permanent connection to the computer (e.g. HP 600 series). (Can be switched off locally with the driver.) If there are problems there is the blue button at the switch to get a direct connection.

Negative:

The cheap finish of the case and the "very short" documentation are no recommendation for the switch. In addition the help of the local seller doesn't make you buy it spontanously, if you ask for informations and co-operation with bi-directioanl printers. (Quotation: "I'm sorry, I can't help you, you have to try it at home").

Conclusion:

A good investment for all of you who have more than 1 computer and don't want to manually plug or switch their printer.

Axel Teichmann <A.Teichmann@t-online.de>«
