10. Hardware test: ARXON parallel port switch (by Axel Teichmann)

»  
Tested device: ARXON Switchbox 
Type: Bi-directional parallel port switch (triple) 
 with hard & software control for Amiga 
Bought in / price: Amiga Oberland / 70.00 DM (approx. 35.71 Euro) (remainder) 
Tester: A.Teichmann@gmx.net 
Test duration: 1 day 
Used hardware: Amiga4000/040 OS3.0 25Mhz / 16 MB Fast RAM / 2 MB 
 Chip RAM Canon BJC 610, HP Laserjet 4L, BJC 7000, 
 Soundsampler Cardattack card reader 
  
Installation:

The device is easily plugged into the parallel port and the joystick port, any other devices are connected to the switchbox, that's all.

The sofware can be copied anywhere you like. The joystick port is connected through, so you can use it for other purposes.

Design:

Due to the fact that this device is some years old (1991), but is available as a remainder from Amiga Oberland no judgement of value can be made for the design. The device itself is a "gigiantic" plastic box with 3 parallel ports and lighted buttons.

In addition the parallel input is a flat band cable of approx. 23.4 inch length as well as the jystick port 2 adaptor, which has to offer a good 5 V power supply. The included software is functional, but no beauty in design.

Documentation:

The documentation is a collection of 10 pages in DIN A5 (approx. 5.8 inch x 8.3 inch) format, which explains the functions and usage of the device well.

Practical test:

After installing the hard & software the switching has worked perfectly.

Because TurboPrint supports the Arxon box since revision 3.x the included software needn't to be used.

TurboPrint switches fast and save and addresses the correct device as set in the settings.

Also the usage of Cardattack and a sound sampler didn't result in problems, the devices have worked fine.

Positive:

Makes 3 ports out of one, no more plugging.

The support using TurboPrint is done magnificent, the switching is done silent and without mumbling in the background.

The best is, that the interior of the device is that "small" it can be built into an eventually present tower case.

Negative:

Only available as a remainder and quit expensive for that fact. Low quality finish (external flat band cables).

Conclusion:

A good purchase for anybody, who wants to have some more parallel ports to address more devices after another. For those who own more than a printer and own TurboPrint this device is strongly recommended.

Axel Teichmann <A.Teichmann@t-online.de>«
