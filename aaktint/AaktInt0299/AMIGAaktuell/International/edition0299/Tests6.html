6. Software test: CAD program AmiCAD v1.5 (by Axel Teichmann)

»   Tested program: gfx/edit/AmiCAD  Revision: 1.5  Type: giftware/ freeware  Author: Roland.Florac@wanadoo.fr (FLORAC Roland)  Tester: A.Teichmann@gmx.net  Test duration: 1 day  Used hardware: Amiga4000/040 OS3.0 25MHz /Cybervision64/3D CGX   System 16 MB Fast RAM / 2 MB Chip RAM     Installation:

The installation is done by the Commodore (Amiga, Inc.) installer. You can choose between English and French documentation.

Design:

AmiCAD runs os-friendly on its own, freely selectable screen. The colours and font can be freely selected. AmiCAD is a CAD program (Computer Aided Design), which is specialized on electronic elements using special libraries. So it is possible to draw circuits quick and clean and to put them out.

Practical test:

After the first start you have to adjust the colors and the font to allow working with the program. Using AmiCAD it is possible to draw circles, rectangles and any other basic form, but AmiCAD is predistined and designed for the use in the electronic area.

The included libraries with the electronic elements are unfortunately mostly French, so there are sometimes problems to figure out the element.

But it is possible to place elements and connections easily. Unfortunately AmiCAD doesn't offer a "current simulation", which made it possible to test a built circuit, but who just wants a clear drawing of a circuit can live with that.

The output of the drawing is done per print-out, as file in AmiCADs own format or as an IFF picture to continue editing in PPaint/DPaint.

Positive:

Runs os-friendly. The print-out is of high quality. Quick creation of electronic drawings due to included libraries is possible.

Negative:

Many parts are in French. Saving is only possible in its own format or as IFF.

Conclusion:

Who already ons the MaxonCAD revision 2.0-2.5 and the libraries that belong to it has no use for AmiCAD. Else you can create drawings quickl and easy, which cannot exported into the Windows world, but printed or exported as a low quality IFF picture to find its way into a paint program. Those who don't dare of this is good served with AmiCAD.

Axel Teichmann <A.Teichmann@t-online.de>«
