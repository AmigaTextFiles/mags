6. Schatztruhe announces Aminet-CD 31 with Directory Opus 5.5

»Aminet CD 31 - 6/99

The Aminet CD 31 - June 1999 contains nearly a Gigabyte (unarchived) software in thousands of archives. Since the Aminet 30 CD more than 500 MB of new software has been added. All the programs are described in German; many contain a German documentation. As a special highlight the Aminet 30 contains a full version of the favourite file manager and Workbench replacement Directory Opus 5.5 with a special upgrade offer to Opus 5 Magellan II.
