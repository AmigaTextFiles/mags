7. Product information: 'DragonDaemon'

»Product information: "DragonDaemon"

Status : Freeware 
Whort : Dial help and more for Hagenuk's "Speed Dragon" 
  

DragonDaemon is a Commodity, which offers some useful function of the DataBox "Speed Dragon". These functions can be reached using ARexx as well as with the Exec message port.

The Commodity is based on the preference file of "DragonTools" and offeres these basic functions:

  • Dialing of telephone numbers

  • Reading the telephone charges counter

  • Clearing of the telephone charges counter

  • Easy requesters for informations, error display and "Yes/No" queries

All the basic functions can be reached using a public message port. Developer informations are included in the German language 
 

Internal functions of the Commoditiy:

  • Number input can be activated via combination of keys

  • Selection of an analogue port for calling

  • Repeated dialling of the last 10 telephone numbers

  • Number selection from ASCII number list

  • Display of an eventually alternative provider

  • For Germany: Supports alternative telephone providers with half automatic call-by-call (configureable)

Functions of the included tools and ARexx scripts:

  • Configuring of call-by-call conditions

  • Dial macro for address management program "ADM"

  • Dial macro for address management program for "DFA"

  • Activating of the dial module of Workbench and Shell. Number and port can handed over as Shell parameters and tool types.

  • Tool for reading the telephone charge counter and saving the data to a log file (for Workbench and Shell).

  • Tool for dialing telephone numbers using the Tools menu of the Workbench.

Requirements

  • Amiga with Amiga-OS 3.0+

  • Installed "DragonTools" V2+

  • DataBox "Speed Dragon" with Firmware 2.14+ (previous revisions were not tested)

  • Hard disk

  • If ARess script are to be used the ARexx interpreter "RexxMast" has to be active
