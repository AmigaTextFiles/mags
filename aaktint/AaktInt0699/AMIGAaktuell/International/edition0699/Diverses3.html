The following changes have been made with the new revision of Wildfire 7:

  • 1. Pictures

Direct Movieshop suppport!!

Loading and saving of WFMV pictures is supported from now on. These pictures are being needed from the new Wildfire operator in Movieshop by Heinrich Angler (Heinrich.Angler@muenchen.de).

So it is possible to use Wildfire effect scripts directly in Movieshop! In the directory WF:Movieshop/ you will find all the necessary files to install the Wildfire-Movieshop operator.

  • 2. AREXX

    The wf_movieshopmode and wf_blackframes commands are new.


  • 3. Non-Linear-Effects
    • Texts of the input connection are in German available, now

    • Operator input/output texts is in German available, too

    • Realtime preview can be switched on and off!!!

  • 4. Settings
    The selected screen mode for playing in the 'Window' mode is being saved now. 
      

  • 5.Converter

    HAM pictures were not saved, if the picture depth was previously set higher than 8 bit - fixed.


  • 6. Transition Maker (Fading generator)
    • The RX command is loaded from SYS:REXXC/ , now.

    • Instead of the file names the effect descriptions are shown. So the user gets an impression, what type of fading the desired effect will do.

    • The transition example animations can now reside in WF:TransAnims/ . So you can copy e.g. the example transition animations from the CD into this directory!

    • Some new fades have been added (Tile, ScaleWipe, Perspective).

  • 7. Documentation
    In the directory WF:Dokumentation the German online help can be found as 
    as ASCII text! So it can be printed and/or you can search within it! 
     

  • 8. Direct Wildfire-ImageFX3 connection

    The Wildfire - ImageFX 3 connection was improved intense. Some important ImageFX 3 effects can be controlled and animated from Wildfire, now! To make this possible, ImageFX 3 has to run and no effect window must not be open. With the desired operators you can work as with any other Wildfire operators! This is a list of the present Wildfire-ImageFX effects:

    • IFXFire

    • IFXFireRadIn

    • IFXFireRadOut

    • IFXLensflare

    • IFXBubble

    • IFXBubbleCol

    • IFXStraw

    • IFXCrystal

    • IFXRadialStar

    • IFXClouds

    • IFXLightning

  • 9. Wildfire-Movieshop effect session 4 effect collections for the Wildfire-Movieshop connection were completed and can be gotten from Andreas Kuessner. So it is possible to use many prepared Wildfire special effects directly from Movieshop. The Wildfire-Movieshop operator by Heinrich Angler is needed!
