7. CDCat 3.0 with PowerPC support and new functions

»CDCat Release 3:

CDCat, the powerful and easy to use CD-ROM catalogizer, that uses MUI, is now available in a new version, CDCat Release 3 (v3.0), after a long period of development.

During the development many time was invested to consider the users wishes for more flexibility.

So the users get an optimum of control about the scan catalogues with the catalogue manager, in the Release 3 of CDCat. CDCat 3 is now able to search directories. To make complete use of the computing power, CDCat Release 3 now offers PowerPC search routines, but the routines for the MC680x0 still can be used, of course.

Also old catalogues and config files can be used with the new release.

Some features of CDCat are:

  • able to catalogize complete CDs, hard disks and PD disks
  • superfast Aminet mode, that scanns Aminet CDs within a few seconds and considers age, size and description
  • quick and flexible search function (PowerPC; 68k: 85% inline assembler!!)
  • Direct searching (without scan catalogue) of disks or directories possible. So you can find easily lost files on your hard disk.
  • Browser function, which shows the directories' contents of a CD like like a directory utility, that also shows the mentioned Aminet extensions, so you don't have to read all the index guides of the Aminet CDs.

    => Rummage the CDs for a file without having the desired CD in the CD-ROM!!!

  • 15 freely configureable user buttons, so you can unarchive, show pictures, play music modules etc. directly from CDCat.

Some features new with Release 3:

  • PowerPC plug-in to speed up the search (The old routines still work, of course!)
  • Catalogue manager to rearrange catalogue contents. Individual disk can be erased or copied into other catalogues etc.
  • Now it is possible to search directories.
  • Filter functions for the result of the search (available using the menu)
  • Progress display, if commands are being executed using the user buttons
  • Possibility to start programs in anychrone mode. Now you no longer need to execute a program with "run", which resulted in problems with the Shell output window.
  • The results of the search are displayed using the MUI class NLIS (by Gilles Masson), so the list has e.g. a horizontal scroller, now.
  • various bugfixes

Registering:

The register fee is about 40.00 DM resp. 20.00 Euros. Then the users get a keyfile to get use to all the functions without limitations.

Upgrades:

I invested very much work in this release, but because it is most unlikely that the number of Amiga users has grown, I also need the support of the already registered users of CDCat Relase 1 and 2.

The common register fee was risen to 40.00 DM resp. 20.00 Euros, registered users of CDCat Release 1 and 2 can get their upgrade for 20.00 DM resp. 10.00 Euros.

Users, who registered after November 1, 1998 get their update for 15.00 DM resp. 7.50 Euros.

A test of the revision v1.2 is in the (German) AmigaPlus 12/97, a test of v2.2 in the AmigaPlus 9/98.

More informations are on the CDCat support page http://www.amigaworld.com/support/cdcat There you always find the latest revision for download, in addition CDCat is available in Aminet util/misc/CDCat30.lha.
