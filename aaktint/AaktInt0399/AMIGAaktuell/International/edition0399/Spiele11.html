11. Preview: Super Pino

This month there's another Jump 'n' Run, titled Super Pino. The game was developped by Virtual Arts and the developers name theeir game 'The Last Platform Game on Amiga'. Somehow this is true. I don't know, if the want to release it on a commercial base or just on a PD base. But the game is really not better than PD. The graphics are quite colourful, but not state of the art. But I don't want to anticipate anything, because this is just a very early demo, which was published and there is much, that could be changed. So, we have to wait. The demo dan be found on Aminet under game/demo/SuperPinoDemo.lha.
