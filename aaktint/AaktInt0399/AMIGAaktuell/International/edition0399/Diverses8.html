8. Product information: Configuration program 'ITA1004Config'

»Due to the growing number of ISDN users, the range of ISDN hardware also grows. The most important investment is, as I think, an ISDN box, to manage and operate multiple telephones.

One of these boxes is e.g. the "I-TA 1004 II", which is distributed by Conrad-Electronic. As with many (all?) ISDN boxes it is delivered with a configuration program for Windows PC only. The Amiga user has to enter long groups of numbers using a connected telephone to configure the box.

With my program I offer Amiga users, who own an "ITA1004Config", a possibility to configure their box in a comfortable way.

The program is freeware and offers these features:

  • Graphical User Interface
  • Doesn't require MUI
  • Comfortable selection of the serial port
  • Allows programming of MSN list, MSN assign, ISDN features, internal redirection, short dial numbers and provider prefix.
  • Can read any of the five call-fee indicators and erase them individually.
  • individual settings of fee factor and currency (so it is ready for the Euro :) )
  • Provider list with net numbers included
  • Current configuration can be printed

To use the "ITA1004Config" you require these:

  • Amiga with Amiga-OS 3.0 or better
  • a free serial port (RS232)
  • 2MB of RAM
  • the mentioned ISDN-box

The program is available on my homepage and in future in Aminet in the subdirectory comm/misc
