10. Preview: Nuxelia

For a long time there wasn't a new Jump 'n' Run game for the Amiga, now there is one: Nuxelia. Starting the demo causes some problems, at the end it was only possible with 'Boot with no Startup-Sequence'. After that the title screen appears and you are in the game at once. Some nice parallax effects as well as many independent animation of the enemies and the main actor make the game of this Jump 'n' Run nicer. Nuxelia is controlled fine and the music is nice, too. The demo can be found in the Aminet game/demo/NuxeliaP.lha. Let's see, what the future will bring for this game.
