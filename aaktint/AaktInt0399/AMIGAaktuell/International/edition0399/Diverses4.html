4. New Amiga magazine announced

Some days ago a new German Amiga magazine was announced (remark of the translator: there are already two new ones, which started in last November). The details sound quite interesting, but I have to ask myself, if the theory can be converted into reality, especially the features. But read for yourself:

»"NG-AMIGA Magazin" next year?

Before we start, the name "NG-AMIGA Magazin" is not under copyright, but I already began negotiations with Amiga International, Inc. I am allowed to keep the name of my magazine for now. Now I want to introduce you to the NG-AMIGA Magazin. The NG-AMIGA Magazin will have the NG-Amiga as a theme for up to 80%, in addition we will notice the PPC, but only with with 20%.

68k-Amigas will not be noticed, this is true for hardware tests and software tests.

With this magazine we want to quick start the Amiga into the future. Now I use the next lines to enlist the planned features and columns:

Features:
  • Edition at the start of the "NG-AMIGA Magazins" 100,000 copies
  • appears on monthly basis
  • completely in colour
  • cool layout
  • 130 pages, all filled with Amiga power!
  • A DVD-ROM every third month
  • low advertisement prices for Amiga dealers and clubs
  • We test everything from A-Z

But of course only that, the Amiga user wants.

  • The NG-AMIGA Magazin is a magazine from users for users!
  • Will be created "only" on NG- and PPC-Amigas!

Columns:
  • Tests of Amiga dealers
  • Introduction of Amiga users and Amiga clubs
  • Thee latest news
  • Hardware tests and software tests of NG-and PPC-Amigas
