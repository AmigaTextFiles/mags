1. Two new WWW browsers under development

Two new WWW browsers have been announced:

  • 'Dry Ice' is being written in C++ and will offer these features:
    • complete HTML 4.0 support
    • JavaScript 1.3
    • "common AmigaOS-Dinge", as ARexx, AmigaGuide, etc.
    • Standard for PlugIns

    Possible release date is late summer. Further informations will soon be available on http://www.scifi1.demon.co.uk/wv.html.

  • The project "AmiExplorer" is still in the first stadium, so it cannot be guaranteed that all the features will be implemented. But the programmer tries to reach his really lofty mission.

    AmiExplorer is to be released in a light and a professional version. The lightversion will be about 20.00 to 25.00 DEM (approx. 10.20 to 12.76 Euros) and will offer these features:

  • HTTP
  • FTP
  • eMail
  • JavaScript
  • Java

The professional version will be about 45.00 to 65.00 DEM (approx. 22.96 to 33.16 Euros) and additionally offer:

  • HTML-Editor
  • Javascript-Editor
  • Java-Editor
  • RealAudio
  • RealVideo
  • ICQ
  • IRC
  • Telnet
  • Gopher

Both version are to know the various HTML languages and dialects (HTML4.0/MathML/DOM/CSS1/CSS2/SMIL/DSIG/XML).

A release date is not known, of course, but AMIGA aktuell international will keep you informed.
